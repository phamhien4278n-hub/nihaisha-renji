## 眩晕鉴别速查（临床三步法）

### 第一步：问"起则头眩"还是"不动也眩"

| 特征 | 方证 | 病位 |
|------|------|------|
| 起则头眩（动才眩） | 苓桂术甘汤 | 中焦横膈水饮 |
| 不动也眩 | 真武汤/泽泻汤 | 下焦寒水/脾积饮 |
| 眼睛不能睁 | 术附汤 | 中焦寒湿 |
| 眼前发黑+左肋痛 | 泽泻汤 | 脾积水 |
| 平躺不能枕枕头 | 术附汤 | 中焦寒湿 |

### 第二步：看伴随症状

| 伴随症状 | 辨证方向 |
|----------|---------|
| 口苦咽干+胸胁苦满 | 少阳（小柴胡汤） |
| 恶寒发热+项强 | 太阳少阳并病（刺大椎） |
| 汗出不止+脚冷 | 真武汤（下焦寒水） |
| 呕吐清水 | 小半夏加茯苓汤（膈间水） |
| 全身肢节疼痛+脚肿 | 桂枝芍药知母汤 |
| 发黄+食即眩 | 谷疸（茵陈蒿汤） |
| 遗尿+吐涎沫 | 肺中冷（甘草干姜汤） |
| 遗精+发落 | 失精家（桂枝龙骨牡蛎汤） |

### 第三步：结合"急则治标缓则治本"排序

1. 恶心眩晕急症 → 先下公孙+内关
2. 再根据辨证选用主方针灸或汤药
3. 水肿明显者 → 三皇穴利水
4. 虚证 → 灸关元、肾俞
