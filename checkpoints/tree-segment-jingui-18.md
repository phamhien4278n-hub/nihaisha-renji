# 金匮要略 第18段 知识树

## 一篇章定位
本篇围绕「饮」的辨证论治展开——留饮、悬饮、溢饮、支饮，从肋间积水到四肢浮肿到胸膈心下，层层递进。

---

## 一、甘遂半夏汤 —— 留饮（肋间积水）

### 方剂组成
| 药 | 剂量（1剂） |
|----|------------|
| 甘遂 | 1钱 |
| 半夏 | 2钱 |
| 芍药 | 2钱 |
| 甘草 | 2钱 |
| 蜂蜜 | 适量（同煮） |

### 煮法关键
1. 四味药 + 2碗水 → 煮成1碗
2. 去渣 → 加蜂蜜 → 再煮成1碗
3. **煮法同乌头——必须蜂蜜煮！蜂蜜解甘遂毒**

### 病机与用药逻辑
- **留饮在肋间**：不用甘遂半夏汤不会好
- **白芍** = 酸收 → 像挤海绵，把肋骨/肺下方的水挤出来。肺的下方 = 阴位 → 用酸收的阴药
- **半夏** → 利水
- **甘遂** → 攻水导出（甘遂/芫花/大戟 = 极凶的药）
- **甘草** → 保护肠胃津液（防半夏+甘遂伤津）
- 甘草 + 甘遂 = 反药配伍？倪师说蜂蜜解甘遂毒，同时甘遂半夏汤本身是经典的相反相成配伍

### 使用注意
- **一剂一剂开**，不一次给3付。给多了吃太过
- 服后**必然吐泻同时来**

## 二、十枣汤 —— 悬饮（肺下积水）

### 方剂组成
| 药 | 剂量 |
|----|------|
| 芫花（熬） | 各等分 |
| 甘遂 | 各等分 |
| 大戟 | 各等分 |
| 肥大枣 | 10枚（不拘泥，10/12/15均可） |

### 煮法与服法
1. 三味打粉（生用）
2. 先煮红枣取汤
3. 用红枣汤吞服药粉
4. **平旦**（清晨6点）温服
5. 强人1钱匕，羸人半钱匕
6. 不下者明日加半钱 → 剂量不够
7. 得快利后**糜粥自养**

### 脉证
- **脉浮而细滑** → 伤饮
- **脉弦数** → 寒饮，冬夏难治
- **脉沉而弦** → 悬饮内痛（十枣汤）

### 为什么红枣？
- 红枣**保护胃**——芫花/甘遂/大戟是强碱性强攻药
- 预防吐泻伤津液
- 十枣汤 = 朱雀汤（红枣 = 朱雀 = 南 = 火）

### 来源（伤寒论）
- **太阳伤寒误下** → 表寒下陷 → 水停在肺下方 → 十枣汤
- 只有太阳伤寒误下会变十枣汤（中风/温病误下不会）
- 也可能因自己吃坏东西 → 上吐下泻 = 等同误下

### 临床应用要点
- 唯一处方——肺底积水没有第二个方
- 不用抽水！抽了还会回来，十枣汤排出后**不会回来**（太久可能需2-3次）
- 吃后1-2小时开始上吐下泻，持续约3小时
- **侧坐**（面对洗手台，侧过来）——吐泻同时来
- 肺癌+肺积水 → 必须瞬间排水，否则水滋养癌症扩散
- 剂量控制：因为用粉剂，可以精确控制量

## 三、溢饮 —— 水到四肢

### 辨证关键
- 水跑到了四肢 → 手脚肿、身上浮肿
- **病在表 → 当发其汗**
- ≠ 赤丸证（里寒极盛的四逆水肿，心脏/小肠功能不够）

### 大青龙汤（外寒里热）
| 药 | 剂量 |
|----|------|
| 麻黄 | 6两 |
| 桂枝 | 2两 |
| 甘草 | 2两 |
| 生姜 | 3两 |
| 杏仁 | 40个 |
| 大枣 | 12枚 |
| 石膏 | 鸡子大1枚 |

- 麻黄:杏仁 = **等量**（麻黄6两 + 杏仁6两等量 → 再重也不会出问题）
- 杏仁补强肺 → 麻黄发汗不伤津液
- 先煮麻黄去沫
- **舌苔黄、口渴、无胃口** → 大青龙
- 汗多者温粉扑之（收毛孔）

### 小青龙汤（外寒里寒）
| 药 | 剂量 |
|----|------|
| 麻黄 | 去节 |
| 芍药 | 3两 |
| 干姜 | 3两 |
| 炙甘草 | 3两 |
| 细辛 | 3两 |
| 桂枝 | 3两 |
| 五味子 | 半升 |
| 半夏 | 半升 |

- **肺里有寒** → 小青龙
- 干姜+炙甘草 → 温肺寒
- 五味子 → 酸收降逆气
- 半夏 → 去痰
- 麻黄去节 → 节止汗，身发表
- **肺寒 = 小青龙，肺热 = 大青龙**——寒热二分法
- 大青龙也治**瘟疫/时疫**（禽流感等）
  - 表证 → 大青龙
  - 渴+小便不利 → 五苓散

## 四、木防己汤 —— 膈间支饮（面色黎黑+心下痞坚）

### 方剂组成
| 药 | 剂量 |
|----|------|
| 木防己 | 3两 |
| 桂枝 | 3两 |
| 人参 | 4两 |
| 石膏 | 鸡子大12枚 |

### 核心辨证要点（极重要！）
**面色黎黑** = 核心鉴别点：
- 脸黑 + **胃口好** = 木防己汤（水的问题）
- 脸暗黄 = 肝癌/肝硬化/脾肿大（不是黑是暗黄）

### 病机（肾阳外泄到脸上）
1. 膈间有支饮 → 肺气无法下降（肺为肾之母，肺阳应下降入肾）
2. 肺气不降 → **肾阳不足**
3. 肾阳不足 → **阴（肾的营养/黑色）外泄** → 跑到脸上
4. 脸 = 诸阳之汇（只有阳经）→ 阳不够黑色就上脸
5. 但胃气仍在 → 胃口仍好

### 脉证
- 喘满（肺气不降，回逆）
- 心下痞坚（胃处硬块，摸得到）
- 面色黎黑
- 脉沉紧（沉 = 病在里，紧 = 寒）
- 得之数十日（久病）
- 医吐下之不愈

### 为什么加石膏？
- **久病必有伏热/宿食**
- 清掉陈年宿食时，伏热会跑出来 → 石膏清之
- 久病也必有淤 → 治疗肿瘤必须考虑淤血

### 虚vs实的分岔
| 类型 | 处方 | 效果 |
|------|------|------|
| 虚证 | 木防己汤 | 当场即愈 |
| 实证 | 木防己汤去石膏+茯苓芒硝汤 | 虚者吃3天复发 → 改此方 |

- **实证变方**：石膏去热不去实 → 换芒硝攻实 + 茯苓排水

### 防己的经方定位
- **木防己**（不用汉防己——张仲景时代就知道区别）
- 入**三焦系统**——所有腺体/淋巴病都可用
  - 甲状腺、扁桃腺、耳朵、腮腺、腋下淋巴、睾丸
- 防己黄芪汤也用防己

## 五、泽泻汤 —— 心下支饮（脾积水）

### 方剂组成
| 药 | 剂量 |
|----|------|
| 泽泻 | 5两 |
| 白术 | 2两 |

### 核心辨证要点
**左肋痛** = 脾在左边（区别于其他眩晕方）
- 苓桂术甘汤 → 中焦水饮眩晕
- 五苓散 → 水逆眩晕
- 真武汤 → 肾阳虚眩晕
- 术附汤 → 中风极度眩晕
- **泽泻汤 → 脾积水眩晕（左肋痛 + 眼前发黑）**

### 泽泻汤加味→脾肿大
泽泻5钱 + 白术2钱 + **白芍1两 + 茯苓5钱 + 厚朴** → 脾脏肿大专用方
- 白芍 → 酸收挤水
- 茯苓 → 利水+滋补
- 厚朴 → 入脾宽肠

### 严重脾肿大 → 腹部全脾脏肿、垂到腹股沟

## 六、厚朴大黄汤 —— 支饮胸满（下焦便秘致饮）

### 方剂组成
| 药 | 剂量 |
|----|------|
| 厚朴 | 1尺 |
| 大黄 | 6两 |
| 枳实 | 4枚 |

### 本质 = 小承气汤（厚朴重用的版本）
- **病因**：小肠下焦不通 → 便秘堵到 → 引起支饮胸满
- **不需排水药** → 大便通了胸满自消
- **重用厚朴**的理由：降逆气
  - 经常放屁 → 气盛 → 重用厚朴
  - 经常嗝气 → 气逆 → 重用厚朴

## 九、贯穿全段的核心辨析框架

### 各类「饮」的快速对照
| 饮型 | 位置 | 主方 | 关键鉴别 |
|------|------|------|----------|
| 留饮 | 肋间 | 甘遂半夏汤 | 白芍→酸收挤水 |
| 悬饮 | 肺下 | 十枣汤 | 脉沉弦、误下史 |
| 溢饮(热) | 四肢 | 大青龙汤 | 舌苔黄、口渴 |
| 溢饮(寒) | 四肢 | 小青龙汤 | 肺寒 |
| 膈间支饮 | 胸膈→脸黑 | 木防己汤 | 面色黎黑+胃口好 ✓ |
| 心下支饮(脾) | 胃下/脾 | 泽泻汤 | 左肋痛+眼前发黑 |
| 支饮胸满 | 胸+便秘 | 厚朴大黄汤 | 便秘+排气多 |
| 支饮不得息 | 肺 | 葶苈大枣泻肺汤 | （待续） |

### 倪师关键治则
1. **阴阳调和 = 病愈**：要知道什么药是阴、什么药是阳、什么配伍是阴/阳
2. **肺下 = 阴位** → 白芍酸收挤水
3. **久病（数十日）→ 必有伏热+宿食** → 加石膏清伏热
4. **久病必有淤** → 肿瘤/癌症须考虑淤血
5. **红枣/蜂蜜**在峻剂中 = 保护胃气 → 攻邪不伤正
6. **麻黄:杏仁等量** → 麻黄再重也不伤津液

### 跨课连接
- **十枣汤**：伤寒论太阳篇已学过（太阳伤寒误下→悬饮）
- **大小青龙汤**：伤寒论已学过（太阳病发汗剂）
- **葶苈大枣泻肺汤**：肺系重症利水方（伤寒论+金匮他篇）
- **赤丸**：与溢饮水肿鉴别——赤丸是极里寒+心/小肠功能不足，不能用汗法
