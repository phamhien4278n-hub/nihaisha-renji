# 金匮要略 段37 知识树

> 段37涵盖了倪师多个临床大案的实战讲解，核心是「如何用经方治疗第四期癌症、尿毒症、红斑狼疮」。
> 四个案例：乳癌转肝癌（重点）→ 尿毒症 → 红斑狼疮（男）→ 肺癌。

---

## 案例一：第四期乳癌转肝癌、骨癌、肺癌

### 病况
- 右乳癌，左乳已开刀但复发变硬
- 转移到肝癌、骨癌、肺
- 第四期，硬块很重
- 严重失眠

### 核心处方（初诊）
| 药物 | 作用 |
|------|------|
| 柴胡 | 入上焦三焦+入肝 |
| 防己 | 直入三焦 |
| 黄芩 | 去肝家热（小柴胡汤思路，不用黄连黄柏） |
| 郁金 | 疏肝气 |
| 龙胆草 | 清肝 |
| 黄连+白芍+阿胶+蛋黄 | **黄连阿胶汤**——治失眠 |

### 黄连阿胶汤的倪师解读
- **为什么治失眠从肝和心下手**：
  - 黄芩疏肝，黄连补心阴（焦苦入心）
  - 白芍酸收——心脏无力把血拉回来→重用白芍把下肢血收回心脏，减轻心脏负担
  - 阿胶补血
  - 蛋黄悬浮在正中间→引导血跑到心脏中央（心神藏处）
- **关键错误示范**：失眠不能用茯苓泽泻——那是利水药，跟心脏问题无关（除非心脏全身水肿才用）

### 攻坚药（针对乳房）
| 药 | 特点 |
|----|------|
| 牡蛎 | 全身性咸味软坚 |
| 瓦楞子 | 长得像乳房，着重乳房 |
| 紫根 | 乳癌扩散时用（紫根牡蛎汤） |
| 露蜂房 | 长得像乳房，通乳极强——**一钱就够**，开五钱会"水灾" |

### 治疗策略：「抓头抓尾」
> 乳癌是源头，肝是末端（厥阴最里）。抓头（乳房）抓尾（肝脏），中间（肺、骨）自然通通顾虑到。转移一定是相通的。

- **乳癌转移到肝癌**比原发肝癌好治——病由外入内
- 原发肝癌难治

### 随诊变化
- **二诊（3/7/2007）**：睡眠从12点→早上6点不停，身热仍在（更年期身热非燥热）
- **三诊（3/20）**：睡眠6h+，肝区耳诊有压痛+T9压痛 → **加鳖甲（灸）、茜草**——攻肝家阴实
  - 鳖甲：攻坚力量强，鳖本性潜（钻泥）——攻坚入阴
  - 败龟板：补虚用（龟长寿，龟灵膏）
- **四诊（5/2）**：肝肿瘤缩小50%（4cm→2cm），肺肿瘤完全消失！
  - 大便排不尽感 → **加大黄一钱半**（不是三钱，不能下利——里虚掉肝癌会作怪）
- **五诊（5/16）**：乳癌完全软了，右边乳房软了，左边开刀硬块也软了
- **全程一方到底**，只做微调

### 治疗哲学
> 「中医治病就像修车子，车子在高速路时速80码的状况下把它修好。西医是停在车库拆下来也没修好。」

## 案例二：尿毒症（肾衰竭）

### 初诊（2/9/2007）
- 2006年9月19日开始洗肾
- 症状：眼睛出血、头昏、恶心
- **肾衰竭的两大主症：头昏 + 恶心**

### 病机：为什么头昏和恶心？
> 肾阳没了→水不气化→冷水积在体内
> - 冷水积到胃→恶心
> - 冷水积到横膈膜→身体一动水在晃→头昏（像瓶子里冷水晃）
> - 五苓散/苓桂术甘汤/真武汤的头昏→都是冷水造成
> - 正常身体里是**热水**（小便时摸摸就知道是热的）

### 初诊问诊要点
- 胃口不好
- 肾主二便：小便没便意、大便秘（要问频率——决定大黄/芒硝剂量）
- 小便黄→热实便秘
- 身体多数时间冷、双足冰

### 治肾原则
> **第一个治心，跟强肺。兼顾疏肝，让水去生木。**
> - 守护心→病人不会死，肾衰竭时心肺不衰竭
> - 金生水、水生木→正常循环

### 初诊处方（当归四逆汤加减）
| 药 | 作用 |
|----|------|
| 桂枝4钱、白芍4钱、炙甘草、生姜、红枣 | 桂枝汤——开胃气 |
| **生附子** | **壮心阳，守护心脉**（四逆汤思路） |
| 干姜 | 色白辛辣入肺 |
| 白术、茯苓 | 健脾整胃 |
| 木通、当归、细辛 | **当归四逆汤**——四肢逆冷 |
| **炮附子** | 固肾阳 |

> **关键区别**：生附子壮心阳，炮附子固肾阳。四逆汤用生附子+干姜+炙甘草，为防心肺衰竭。

### 随诊变化
- **二诊（3月）**：大便好了（靠木通+当归油脂，没用大黄）、胃口好转、恶心消失、小便便意增加、血压下降、**开始出汗**（心功能恢复的标志！——心主五液）
- 三诊（3/23）：Creatine 4.7→3.1，西医说"赶快吃中药不要停"
- **加补骨脂+败龟板**（滋阴）→4/13 Creatine再降到2.8
- **加熟地**——之前治肾衰竭完全没用熟地！因为当时是当归四逆汤证+里寒湿
- 5/4：气色恢复（原来洗肾病人都是黑的——"黑色天罗"），半夜四点饿醒→胃气回复=阳回头
- 停洗肾一周后再诊：加乌药（频尿）+ 利尿剂分担肾脏负担
- **加黄芩+黄连**——解尿毒！手掌青紫色（尿毒标志）→加后三个礼拜不用洗肾
- 6/22：**月经来了！**（心脏管月经，心功能恢复=月经恢复=可以怀孕）

### 核心方义
> 主力是当归四逆汤加减 + 生附子。先用阳药开到尽头，再加滋阴药（补骨脂、败龟板、熟地）。阳回头后再加黄芩黄连解尿毒。

## 案例三：男性红斑性狼疮

### 病机（倪师理论）
- 男人有红斑狼疮或乳癌：**精子残渣顺任脉上行**
- 正常：残渣到嘴巴→胡子（第二性征）
- 异常：残渣入心→红斑狼疮；入乳房→男性乳癌
- 此患者20岁，无胡子（第二性征缺失）

### 治疗方向
- **主力：生附子**
- 配合桂枝芍药知母汤（关节痛）
- 久病虚→补气补血

### 关键转机
- 生附子开重→阳气恢复→伏饮被逼出→咳嗽（白痰→黄痰=寒转热=好现象）
- 7/5/2007：西医检查说"你没有红斑狼疮"——8年的病被清零

### 治疗原则
> 「想尽办法让阳气恢复，让阴不要累积。」
> - 表无汗→发汗
> - 没大便→通便
> - 里寒→去里寒
> - 表虚→固表
> - 宁可虚，不可实。营养不要太多——超级浓缩维他命不能吃。

## 案例四：肺癌（第三~四期）

### 初诊（2/12/2007）
- 咳嗽6个月，抽烟极重（每天两包）
- 右胸痛（高处），呼吸咳嗽都痛
- **可以平躺**→肺里没积水（不能平躺=有水=危险→心肺衰竭）
- 胃口好、脚温、睡好（因为西医还没碰他）
- 痰：**白色泡沫痰**——肺癌标志
  - 抽烟者焦黄色痰不是肺癌
  - 白色泡沫痰→桔梗甘草汤证

### 初诊处方
| 药 | 作用 |
|----|------|
| 桔梗1两、炙甘草3钱 | 桔梗甘草汤——白色泡沫痰 |
| 生半夏 | 涤痰+排水+排寒水气（比制半夏好很多） |
| 干姜 | 去肺家寒 |
| 皂荚3钱+红枣 | 针对抽烟焦痰 |
| 射干 | 寒痰咳嗽必用 |
| 紫苑3钱、冬花 | 射干麻黄汤——止咳化痰 |
| 麻黄3钱 | 宣肺 |
| 紫根 | 上焦癌症攻坚（本来紫参最好但没货） |

### 治疗过程
- 一周后：胸痛消失！咳嗽好转，身热→出现炙热感（阳不入阴）
- 加丹皮桃仁冬瓜仁（千金苇茎汤——胸腔有血咳出）
- 2/27开始头晕、咳嗽增加——**这是排病反应**，呼吸反而变好，阻力消失
- **层层脱皮理论**：阴实去掉一层，里面还有一层，反复剥
- 4/5：血压从160-170降到120/70，低烧消失
- 下利（20次/天）出现——肺癌治疗中下利≠错，是排病
- 7月：白痰→黑色胶痰（抽烟的焦痰，在最底层被剥出来）
- 开始喜欢喝冷水→阳气恢复（里寒癌症患者只喝热水）

### 后期加味
- 金铃子+延胡索——止癌症疼痛
- 皂荚丸——去抽烟焦痰
- 麦冬、党参、五味子——补肺气

### 重要理念
> 「我们人都会死，但是不要饿死。」——从头到尾胃口好=胃气在=阳气在
> 病人拒绝西医介入，所以整个过程胃口从未丧失
> 「抽烟病人习惯了一辈子，瞬间停掉即使没肺癌也会生其他病」

## 跨案例共通法则

### 1. 头尾策略
> 抓源头+抓末端，中间自然通。转移必相通（厥阴最里）。

### 2. 阴实层层剥
> 癌症治疗过程是反复的——好→差→好→差→最终好。每一层去掉，下一层露出来。

### 3. 阴阳用药顺序
> 先用阳药开道（生附子、炮附子、干姜），阳药开到尽头再补阴（补骨脂、熟地、败龟板）。

### 4. 胃气是底线
> 胃口好=阳气在=不会死。所有治疗必须保胃气。

### 5. 中西医差异
> 美国西医：诚实告知，鼓励患者继续中药（"你不要停"）
> 台湾西医（倪师批评）：不行了还开刀

### 6. 药物鉴别
| 对比 | 此 | 彼 |
|------|----|----|
| 生附子 vs 炮附子 | 壮心阳 | 固肾阳 |
| 鳖甲 vs 败龟板 | 攻坚入阴 | 补虚 |
| 牡蛎 vs 瓦楞子 | 全身软坚 | 乳房专用 |
| 紫参 vs 紫根 | 肺癌最优 | 代替品 |
| 黄芩 vs 黄连/黄柏 | 中焦/肝热 | 心火/下焦 |

### 7. 症状速查
| 症状 | 病机 |
|------|------|
| 白色泡沫痰 | 肺癌标志 |
| 抽烟焦黄痰 | 不是肺癌 |
| 头昏+恶心 | 肾衰竭（冷水在体内） |
| 不能平躺 | 肺积水→心肺衰竭危险 |
| 手掌青紫 | 尿毒标志 |
| 不出汗 | 心脏功能差（心主五液） |
| 半夜饿醒 | 胃气回复=阳回头 |
| 舌苔从无→有 | 病情进步 |
| 舌苔从有→无（镜面舌） | 病情恶化 |
| 白痰→黄痰 | 寒转热=好现象 |
| 喜欢冰水 | 阳气恢复（里寒者只喝热水） |
