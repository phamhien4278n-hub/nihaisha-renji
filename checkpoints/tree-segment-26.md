# 金匮要略 第26段 知识树

## 一、黄疸病续篇

### 1. 猪膏发煎（诸黄，猪膏发煎主之）
**不可见黄就用——有特定适应症**：
- 两个主症：①大便黑 ②肌肤甲错（皮肤干燥如蛇蜕皮）
- 病机：久湿→湿干掉→身体极干燥
- 方义：猪油滋润（猪与人接近），乱发（发为血余）活血攻瘀
- 乱发用量：如鸡子大三枚，松放不可压紧
- 第二个用途：金匮妇科末篇——阴吹（子宫阴道间膜破洞，屁从阴道出）

### 2. 茵陈五苓散（黄疸病，茵陈五苓散主之）
**与茵陈蒿汤的区别**：
- 茵陈蒿汤：身黄+**大便秘**
- 茵陈五苓散：身黄+**小便不利**（五苓散主症：口渴+小便不利+发黄）
- 用法：茵陈蒿末十分+五苓散五分，饭前服，粉剂

### 3. 大黄硝石汤（黄疸腹满，小便不利而赤，自汗出，此为表和里实，当下之）
**方**：大黄、黄柏、硝石各四两，栀子十五枚
**适应症**：黄疸+腹满+小便不利而赤+自汗出 → 表和里实，热证
**关键药性知识**：
| 药 | 功能 | 备注 |
|----|------|------|
| 茵陈 | 去黄（不管寒热）+去湿 | 见黄即可用 |
| 栀子 | 清肝胆湿热 | 性寒 |
| 黄柏 | 膀胱不利、小便排不出、膀胱发炎 | 小便赤痛用 |
| 黄芩 | 胃和十二指肠发炎 | 泻心汤中用 |
| 柏子仁 | 肝胆润肠剂 | 虚人便秘替代大黄 |

### 4. 胆结石治疗（倪师临床精华）
**诊断四步**：
1. 右上腹/胁肋痛
2. 放射痛→背后+肩膀
3. 阳陵泉下一寸（胆石点）压痛
4. 耳朵胆囊点压痛

**基础方**：四逆散加减——柴胡、黄芩、枳实、甘草
**化胆结石专药**：五倍子（醋炒）+ 海金砂
- 五倍子：形如胆，醋炒→酸入肝胆，膨胆如气球清出结石
- 不同于肾结石治法

**胆的生理功能**：胆汁储存缓冲，随食物消化需要喷射——有袋必有功能

### 5. 小半夏汤证（黄疸病，小便色不变，欲自利，腹满而喘，不可除热）
- 关键鉴别：小便颜色——**色不变（正常）=只有湿无热；色赤/深黄=有热**
- 误治：见黄就开栀子黄芩等寒凉药 → 热除必哕（打嗝）
- 哕的轻证：小半夏汤（生半夏+生姜）——胃中有水
- 哕的重证（危证）：胃癌/胰脏癌/肝癌末期，胃气将绝→不断打嗝无法控制→将死

### 6. 小柴胡汤（诸黄，腹痛而呕者，宜柴胡汤）
**基础方**：柴胡、半夏、黄芩、党参、炙甘草、生姜、大枣
**加减法**：
- 急性肝炎身黄：郁金+龙胆草（血中湿热+肝胆湿热）
- 黄盛：加茵陈三钱
- 便秘：加大黄三钱
- 干呕舌苔白（胃寒）：加吴茱萸+干姜
  - 吴茱萸止呕 vs 半夏止呕：吴茱萸偏肝胆（木克土、肝顶胃），半夏偏水饮痰饮

### 7. 小建中汤（男子黄，小便自利，当与虚劳小建中汤）
- 虚劳诊断：山根青筋+腹痛拒按→小建中汤
- 两补虚方：小建中汤（腹痛为主）、炙甘草汤
- 肝病兼虚劳腹痛：可用小建中，但有腹水时去甘草（甘草蓄水）

## 二、惊悸吐衄下血胸满瘀血病脉证治第十六

### 8. 脉诊区分惊与悸
- 寸口脉**动**（左右跳，非上下）→惊（受惊吓）
- 寸口脉**弱**（太细太小）→悸（水饮所致）
- 脐下悸→茯苓；心下悸→半夏麻黄丸

### 9. 衄血预后——看眼睛
- 尺脉浮+目睛晕黄（眼白仍有黄色）→衄未止，还会再犯
- 晕黄去，眼白恢复白色→衄已止
- 病机：肝不藏血→木克土→土的黄色上泛眼白

### 10. 出血类型脉证鉴别
- 面无血色+无寒热+脉**沉弦**→衄（流鼻血）
- 面无血色+无寒热+脉**浮弱**+手按之绝→下血（大小便血）
- 面无血色+无寒热+**烦欬**→必吐血
- 病机：阳不固阴→血管破裂→血离经叛道

### 11. 瘀血判断法（全套）
**望诊**：
- 舌边齿痕重 → 内有瘀血
- 舌上有黑点 → 内有瘀血
- 唇痿（不丰润）、舌干口燥、不欲喝水 → 瘀血
- 妇人左腹腔压痛 → 瘀血
- 但欲漱水不欲咽 → 瘀血
- 舌青口燥 → 瘀血

**问诊**：
- 病者如有热状、烦满、口干燥而渴，**脉反无热**（一息四至正常）→ 阴伏=瘀血

**治则**：瘀不去，新不生
- 保守：攻瘀三+补虚七
- 经方：半攻半补（各半）
- 瘀血排出标志：大便变黑→变黄（瘀血褪尽）+嘴唇丰润+舌正常

### 12. 阳不入阴——重病判断标准
- 肝癌标准：1-3点醒→慢慢扩大到半夜→再到24小时不能睡→阴阳相隔→死
- 治疗好转：晚上能慢慢回来睡+持续热退掉
- 正常人：上寒下热（额头背冷、手掌脚底热）
- 阳不入阴的脉：一息九下十下→危

### 13. 亡血不可发其表
- 失血状态（阴已虚）+发汗（阳又虚）= 阴阳两虚→寒栗而振
- 发汗能力=心脏功能正常标志（不流汗=心脏有问题）
- 正常流汗：运动/走路才流；静坐流汗=虚汗

### 14. 三种伏邪
| 类型 | 成因 | 治法 |
|------|------|------|
| 伏饮 | 喝水太快/汗未发透/吐后补水快 | 苓桂术甘汤/五苓散等 |
| 伏热 | 宿便未完全排掉，肠壁形成薄膜包住宿便 | 保持大便通畅=上工治未病 |
| 阴伏 | 瘀血 | 当下之（活血化瘀攻下） |

## 三、本篇方剂精要

### 15. 桂枝去芍药加蜀漆牡蛎龙骨救逆汤（火邪）
**适应症**：火灾逃生/被火烤/厨师等→津液大伤
**核心理念**：行阳固阴（不同于温病滋阴）
- 去芍药（阴药）→加速阳的运行
- 蜀漆（今用常山三钱替代）→祛痰（火邪必有浓痰）
- 龙骨→敛阳潜阳；牡蛎→滋阴入肾入骨
- 关键：有伏饮/瘀血/伏热时，滋阴药进不去——必须先去邪

### 16. 半夏麻黄丸（心下悸）
**适应症**：胃有停水→胃部动悸
**方义**：
- 半夏→去胃中水+涤痰（利水快，口渴者不加）
- 麻黄→宣肺升提，把大肠津液往上蒸
- 不用甘草（甘草蓄水）
**两个使用场景**：
1. 胃中停水动悸
2. 肺家寒实（非阴实=癌症）——半夏涤痰+麻黄宣肺（=小青龙汤核心思路）

### 17. 柏叶汤（吐血不止）
**适应症**：上焦出血——食道出血/胃出血/肝硬化呕血（非咳血——咳血来自肺）
**方**：柏叶三钱+干姜三钱+艾叶三钱，原方有马通汁一升（今可不用）
**方义（经方北派观点——出血是寒导致）**：
- 柏叶（侧柏叶）→止血（肚脐以上出血用柏叶）
- 干姜→温肺胃，促肠胃蠕动
- 艾叶→引血归经，不让血乱走
- 寒则血管破裂（如冬天水管冻裂）→用热药止血，与温病派相反
- **一剂就知！**
- 马通汁可选：白马粪煮水→清肠胃宿血，今可不用

### 18. 黄土汤（下血，先便后血，此远血也）
**适应症**：胃出血/胃下垂/胃溃疡/胃癌——大便是黑色（远血，血经过小肠大肠）
**方**：甘草、干地黄、白术、炮附子各三两、阿胶三两、黄芩三两、灶中黄土（伏龙肝/灶心土）半斤
**方义**：
- 灶心土→补破洞（胃/肠破洞）
- 地黄→补血；白术甘草→健脾整胃
- 炮附子→胃寒（寒才出血）
- 阿胶→止血
- 黄芩→**胃**发炎用黄芩（小肠发炎用黄连，膀胱用黄柏）——三黄泻心汤层次：胃黄芩→小肠黄连→水道黄柏

### 19. 利膈汤（喉闭/食道癌/吞不下）
**方**：半夏+栀子+炮附子
- 半夏最多→去痰去寒水（寒则缩→闭塞）
- 栀子→除烦去虚热
- 炮附子→温通
- 加减：痰多→半夏加重；热感→栀子加重；闭塞重→炮附子加重
- 可加：厚朴行气、苏叶开胸膈、络石藤（喉癌舌癌特效——包石头生长的藤）
- 未化疗开刀者效果好，化疗开刀后只能延长寿命
- 禁甜食（人工糖喂细菌病毒→代谢物是酸）+禁咖啡（酸）→酸逆腐蚀食道→破洞吐血→胃癌胰脏癌

### 20. 胃癌治疗整合方
| 症状 | 用方 |
|------|------|
| 吐血 | 柏叶汤加减 |
| 下血（黑便） | 黄土汤加减 |
| 吞不下 | 利膈汤加减 |
| 呕酸 | 旋覆花+代赭石各三钱 |
- 禁：甜食+咖啡（呕酸的根源）

## 四、文式易筋经第十二式
- 下半身站稳绷紧（固阴）
- 双手握拳胸前，缀紧如张弓→做49下（练阳气）
- 舌顶上颚，眼睛瞪目
- 原理：下半身固阴+上半身练阳→放松后阴阳流行极快

## 五、本段新学核心药对/药性
| 药对 | 功能 | 鉴别 |
|------|------|------|
| 柏叶 vs 阿胶 | 止血 | 柏叶=肚脐以上（吐血）；阿胶=肚脐以下（妇科血） |
| 吴茱萸 vs 半夏 | 止呕 | 吴茱萸=肝胆（木克土）；半夏=水饮痰饮 |
| 黄芩 vs 黄连 vs 黄柏 | 清三焦热 | 黄芩=胃；黄连=小肠/心；黄柏=膀胱/水道 |
| 茵陈 vs 栀子 | 治黄 | 茵陈=去黄+湿（不辨寒热）；栀子=清肝胆湿热（性寒） |
| 五倍子 vs 海金砂 | 化结石 | 五倍子醋炒=胆结石；海金砂=胆肾通用 |
| 蜀漆 vs 常山 | 祛痰 | 蜀漆有毒需先煮，今用常山替代 |
