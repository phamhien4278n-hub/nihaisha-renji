# 金匮要略 第15段 知识树

## 段15：乌头桂枝汤 → 脉诊阴阳 → 宿食 → 五藏风寒积聚（肺·肝·心）

---

### 一、乌头桂枝汤（腹满寒疝宿食篇收尾）

#### 核心方剂
- **乌头桂枝汤**：乌头五枚 + 蜜二斤煎减半去滓 → 以桂枝汤五合解之 → 得一升
  - 服法：初服二合→不知→三合→又不知→五合
  - 中病反应：「如醉状，得吐者，为中病」
  
#### 病机与适应证
- **适应症**：寒疝、腹中痛、逆冷、手足不仁、身疼痛，灸刺诸药不能治
- **病机**：寒在肌肉 → 桂枝汤解肌调阴阳，但只能去风不能去寒 → 需乌头
- **倪师临床用法**：用乌头桂枝汤治疗风湿关节炎（而非标准乌头汤）
  - **原因**：很多病人心脏不好（西药吃太多），麻黄会导致心悸，所以不用乌头汤而用乌头桂枝汤
  - **标准治法应是用乌头汤**（乌头+麻黄+黄耆+甘草+芍药），但考虑到心脏问题改用乌头桂枝汤

#### 乌头蜜制作
- 乌头用两碗水煮成一碗 → 去渣得乌头汤
- 乌头汤 + 一碗蜂蜜 → 两碗煮成一碗 → 服时加一匙蜂蜜
- 乌头热性极强，走四肢全身，排肌肉/胃/胸腔寒湿

### 二、脉诊阴阳理论

#### 核心经文
> 「脉数而紧，乃弦，状如弓弦，按之不移。脉弦数者，当下其寒。脉紧大而迟者，必心下坚。脉大而紧者，阳中有阴，可下之。」

#### 倪师解释
- **根本原则**：阴阳相合则吉，阴阳分割则凶
- **阴中有阳（吉）**：病人阴症（躲家里、不讲话、面色苍白、不渴）→ 但脉有力 → 可治
- **阳中有阴（吉）**：病人阳亢（烦躁、声音大、目不能闭）→ 但脉阴 → 可治
- **脉症不合（凶）**：
  - 极虚之人脉反大数 → 阳浮于外，最后一口元气 → 危
  - 肝癌末期精神亢奋但脉细小无力 → 脉症不合 → 危
- **弦数脉内有寒**：摸到阳脉但内里有寒 → 可攻下

### 三、宿食（大承气汤 + 瓜蒂散）

#### 宿食定义
大便/食物糟粕未排出，隔24小时即为宿食。

#### 大承气汤三症（都是大承气汤）
1. **寸口脉浮而大，按之反涩，尺中亦微而涩** → 大承气汤
2. **脉数而滑者，实也** → 大承气汤
3. **下利不欲食者** → 大承气汤（便秘引起的下利）

#### 大承气汤诊断要点
- **纯阳明腑热**（区别于阳明经热——白虎汤/白虎人参汤）
- **关键症状**：不放屁、小便黄、便秘、可有高热
- **便如鞭**：大便细长如指条 = 大承气汤症
- **便秘性下利**：大肠堵死，只有水能通过 → 下利 → 按天枢剧痛
- **排除条件**：有恶心 → 不是大承气汤，是大柴胡汤
- **前额痛** = 宿食特征 → 针中脘（腑会中脘）即消

#### 大承气汤用药技巧
- 大黄3钱、芒硝2钱（分两包冲服）、厚朴、枳实
- 无腹痛：厚朴枳实少，大黄芒硝多 → 速排
- 有腹痛：厚朴枳实加重 → 止痛
- 必须用芒硝（否则大便出来但引爆痔疮）

#### 宿食在上脘 → 瓜蒂散（吐法）
- 瓜蒂熬黄 + 赤小豆 → 为散 → 香豉煮汁和服
- **倪师替代**：浓盐汤（最方便），盐多温水化开即吐
- 瓜蒂极苦，倪师自己都不用
- 赤小豆作用：排脓、去湿痰（上焦哽塞有湿痰）

#### 脉诊宿食
- 脉紧如转索无常 → 有宿食
- 脉紧，头痛风寒，腹中有宿食不化 → 前额痛

### 四、五藏风寒积聚病脉证并治第十一（开篇）

### 4.1 肺

#### 肺中风
- **症状**：口燥而喘、身运而重、冒而肿胀
- **病机**：风入肺→肺增热→津液干枯→脾湿上不去（缺津液载体）→身重→上焦燥热无津→头昏如烈日直射
- **比喻**：正常=天上有云（津液适中）；肺中风=万里无云太阳直射

#### 肺中寒
- **症状**：吐浊涕（寒则津液停滞成浊涕）

#### 肺死脉
- 浮之虚、按之弱如葱叶、下无根 → 死
- 对比正常肺脉（浮如羽毛）来判断

---

### 4.2 肝

#### 肝中风
- **症状**：头目瞤、两胁痛、行常伛（弯曲）、两臂不举、舌本燥、善太息、嗜甘
- **病机**：风热入肝 → 肝开窍于目故头目动（如帕金森）→ 两胁=少阳 → 小柴胡汤思路
- **嗜甘**：《内经》「肝苦急，急食甘以缓之」
- **发病前提**：体虚则风趁虚而入

#### 肝中寒
- **症状**：胸中痛、不得转侧、食则吐而汗出
- **病机**：寒=不动 → 肝肿大 → 顶脾脏（木克土）→ 中膈满 → 胃受不了 → 食则吐
- **胸中痛**：肝脾肿大 → 横膈膜下降受阻 → 吸气不饱
- **治则**：肝脏肿起/脾脏肿起多为寒症 → 开温热药

#### 肝死脉
- 浮之弱、按之如索不来、或曲如蛇行 → 死（真脏脉，肝属木故扭曲）

#### 肝着（旋覆花汤）★新方
- **症状**：常欲蹈其胸上（喜欢人踩/按摩胸部）、未苦时但欲饮热
- **病机**：寒着在肝 → 肝血不能入肺 → 踩压产生压力让血入肺 → 暂时舒服
- **不是肝硬化/肝癌**，只是寒堵在肝上
- **旋覆花汤**：旋覆花三两（金沸草）+ 葱十四茎 + 新绛（茜草）少许
  - 旋覆花：降逆气（酸=肝味，降胃酸）
  - 葱：行阳
  - 茜草（新绛）：入肝活血化瘀，古人用来染红衣
  - 茜草也用于妇科子宫肌瘤（肝经走少腹）

### 4.3 心

#### 心中风
- **症状**：翕翕发热、不能起、心中饥、食即呕吐
- **病机**：风入心 → 风助火势 → 心火不降反升 → 胃有火（饿）但小肠无火（食则吐）
- **方剂**：防己黄芪汤（倪师说也用于少运动腹部肥胖者减肥）

#### 心中寒
- **症状**：心如噉蒜状（胸口辣感）、剧者心痛彻背背痛彻心、如虫注
- **病机**：寒痰 → 脉浮者可自吐而愈（运动/易筋经/撞墙）
- **吐不出**：乌头赤石脂丸（极热药，对治寒痰）
- **寒热鉴别**：寒症不渴，热症口渴

#### 心伤
- **症状**：劳倦、头面赤而下重、心中痛而自烦、发热、当脐跳、脉弦
- **病机**：
  - 心火固定量（如100匹马力）→ 受伤后火不降反逆 → 面赤
  - 小肠热量下降 → 食物不化 → 里急后重
  - 脐跳：小肠温度降 → 水无法蒸化回肺 → 水停在脐部跳动
  - 脉弦=水脉
- **病因**：外伤、西药、情志伤害

#### 心死脉
- 浮之实如麻豆、按之益躁疾 → 死（真脏脉，像小气球在跳）

### 五、跨课连接 & 新认识

#### 连接伤寒论
- 大承气汤症 → 阳明腑热（伤寒论阳明篇核心）
- 大柴胡汤 vs 大承气汤：有恶心=大柴胡汤
- 防己黄芪汤 → 风水（金匮水气病篇）

#### 连接内经
- 肝苦急，急食甘以缓之 → 解释肝中风嗜甘
- 腑会中脘 → 宿食前额痛取中脘
- 五藏真脏脉 → 各藏死脉特征

#### 新学方剂
- **旋覆花汤**（肝着）——仅三味：旋覆花+葱+茜草，寒堵肝脏非器质性病变
- **乌头赤石脂丸**（心中寒之心痛彻背）——极热药对治寒痰
- **瓜蒂散**（宿食上脘吐法）——倪师用浓盐汤替代

#### 倪师独特临床经验
- 用乌头桂枝汤代替乌头汤治类风湿——因现代病人心脏多不好
- 大承气汤不加芒硝→引爆痔疮
- 便秘性下利按天枢鉴别
- 浓盐汤代替瓜蒂散——方便实用
- 前额痛=宿食→针中脘
