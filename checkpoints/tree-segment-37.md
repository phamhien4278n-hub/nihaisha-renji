# 金匮要略 段37：倪海厦临床案例合集

## 案例一：乳癌第四期转移肝癌/骨癌/肺癌

### 核心病机
- 乳癌为源头（厥阴最里），转移由外入内
- 治疗策略：**抓头抓尾**——守乳癌（源头）和肝脏（末端），中间自然贯通
- 转移性癌症比原发性好治（由外入内 vs 内生）

### 初诊处方（2月11日2007）
| 药物 | 作用 |
|------|------|
| 柴胡 | 入三焦上焦 + 入肝（病人有肝癌） |
| 防己 | 直接入三焦 |
| 黄芩 | 清肝家热（小柴胡汤选黄芩不选黄连黄柏的原因） |
| 郁金 | 疏肝气 |
| 龙胆草 | 清肝 |
| 黄连 | 失眠→心阴不足，补心 |
| 白芍 | 酸收，入肝，把下肢血拉回心脏（减轻心脏负担） |
| 阿胶 | 补血 |
| 酸枣仁+知母 | 酸枣仁汤，除烦助眠 |
| 川芎 | 活血化瘀，收敛血管防破裂（四物汤中用川芎的原因） |
| 炮附子+细辛+半夏 | 开胸膈，病人吞咽困难/喉部肿瘤 |
| 瓦楞子+牡蛎 | 咸味攻坚——牡蛎全身性，瓦楞子专攻乳房 |

### 黄连阿胶汤设计原理（深度解析）
- 黄芩 → 疏肝（肝）
- 黄连 → 心阴不足，焦苦入心（心）
- 白芍 → 酸收，拉回外散的血（心脏无力把血拉回）
- 阿胶 → 补血
- 蛋黄 → 悬浮正中，指引血到心脏中央（心神所藏之处）
- **关键**：治失眠从心肝下手，不加利水药（茯苓泽泻跟失眠无关）

### 二诊（3月7日2007）
- 睡眠：12点→早6点不停
- 身热：更年期身热非燥热（下肢冰冷，热气上冲=更年期特征）
- 舌苔：淡白（初诊淡黄湿热→淡白=胃气恢复）
- **舌苔铁律**：有苔→无苔（镜面舌）=恶化；无苔→有苔=好转

### 三诊（3月20日2007）
- 睡眠每天6小时以上
- 压诊：第四椎压痛、肺俞第三椎压痛、耳诊肝点痛→肝内仍有问题
- **加鳖甲+茜草**——攻肝家阴实的核心药对

### 鳖甲 vs 龟板
- 鳖甲：攻坚（鳖潜水泥中，攻坚力强），用于肝癌/乳癌/肺癌
- 龟板：补虚（龟长寿），龟灵膏用

### 四诊～后续（4月1日→5月2日→5月16日）
- **4月1日**：睡眠越来越好，守方
- **5月2日**（2.5个月后）：肝肿瘤缩一半（4cm→2cm），**肺肿瘤完全消失**，乳癌仍在
- **5月2日**：大便排不尽→加大黄一钱半（小剂量，防里虚）
- **5月16日**：睡眠6h+，身热退，腿温，手脚温，右乳完全软掉

### 核心教训
- 一方到底，从头到尾基本没变方
- 第四期乳癌+肝癌+肺癌→2.5个月肺癌消失、肝癌减半
- 没被西医碰过的病人，不需要这么大处方，几味药就够
- 乳房硬块不是乳癌时：四物汤+炒麦芽即可（去乳房积水），或汉唐二号（桂枝汤+葛根+化乳药）
- 硬块有痛或表面光滑→初期乳癌→加牡蛎+瓦楞子即可，不需要鳖甲茜草
- 破溃流脓恶臭→加紫根+牡蛎

## 案例二：尿毒（肾衰竭）

### 病机推理链
肾阳无 → 水不气化 → 冷水积聚 →
- 水积到胃 → **恶心**（首要症状）
- 水积到横膈膜 → 一动水晃 → **头昏**（冷水晃才昏，热水晃不昏）
- 肾功能无 → 血管不蠕动 → 血管变窄 → **血压飙升**

### 初诊（2月9日2007）
症状：胃口不好、大便秘（小便黄=热实）、双足冰、月经6周未来

### 治疗策略（治肾三步）
1. **先守心**（心脏不衰就不会死）
2. **强肺**（金生水）
3. **疏肝**（水生木）

### 初诊处方（当归四逆汤加减）
| 药物 | 作用 |
|------|------|
| 桂枝4钱+白芍4钱+炙甘草+生姜+红枣 | 桂枝汤（开胃气） |
| 生附子 | **壮心阳**（非炮附子！守护心脉） |
| 干姜 | 色白辛辣入肺 |
| 白术+茯苓 | 健脾整胃 |
| 当归+木通+细辛 | 当归四逆汤（手足逆冷） |
| 炮附子 | 固肾阳 |

**关键区分**：生附子壮心阳，炮附子壮肾阳。四逆汤用生附子是为防心肺衰竭。

### 疗效追踪
- **二诊（3月）**：大便好（靠木通+当归油脂，没用大黄）、胃口好、恶心无、小便便意增加、泡沫减少、出汗→心功能恢复
- **三诊（3月23日）**：Creatinine 4.7→3.1，加补骨脂+败龟板（滋阴）
- **四诊（4月13日）**：Creatinine 3.1→2.8
- 加熟地（之前治肾衰竭一直没用熟地！因为主力是当归四逆汤+生附干姜）
- **五诊（5月4日）**：气色完全变回（尿毒病人的乌黑退去），半夜4点饿醒→胃气回复=阳回头且稳定潜伏
- 停洗肾一周→频尿→加乌药5钱+利尿药（分担肾脏负担）
- **六诊（6月1日）**：手掌仍青紫→尿毒素还在→加黄连2钱+黄芩3钱（解尿毒）
- **七诊（6月22日）**：三个礼拜没洗肾！验血报告比洗肾时还好，血中毒素无。月经来了（6月13日）→心脏功能恢复→可以怀孕了

### 核心教训
- 治肾先治心，心脏守住就不会死
- 当归四逆汤加减+生附子治尿毒/血癌/红斑狼疮，无一不好
- 先阳药攻，后加滋阴（补骨脂/败龟板/熟地）
- 黄连+黄芩解尿毒（手掌青紫消退）
- 月经恢复=心功能恢复=可怀孕

## 案例三：红斑性狼疮

### 核心病机（倪师理论）
- 男人得红斑狼疮/乳癌：精子残渣本该走任脉到嘴边生胡子→系统出问题→残渣跑心脏（红斑狼疮）或乳房（乳癌）
- 男人第二性征消失（无胡子）=此通路已断

### 病案概要
- 20岁男性，17岁起吃类固醇3年，无胡子

### 治疗思路
- 主力：**生附子**
- 关节痛：桂枝芍药知母汤加减
- 无阳明燥热（胃口大）不用石膏
- 红斑狼疮关节痛、多发性骨髓瘤关节肿瘤→均可用此思路

### 诸积大法（金匮核心原则）
- 积是阴积，阳气流走时阴不积
- **宁可虚，不可实**——吃7-8分饱，不吃超级浓缩维他命
- 治癌不需找"哪种药治哪种癌"——**只要恢复阳气，让阴不累积**
- 表无汗→发汗，没大便→通便，里寒→去里寒，表虚→固表

### 另一女性病例
- 黑人女性8年红斑狼疮
- 处方：生附子为主
- 结果：7月5日西医宣布"你没有红斑狼疮"
- 伏饮被生附子逼出→白痰变黄痰→寒转热（好现象）

## 案例四：肺癌第三~四期

### 初诊（2月12日2007）
- 咳嗽6个月，右胸高处痛（呼吸/咳嗽加重）
- **可平躺睡**=肺无积水（关键鉴别！不能平躺=有积水→危险）
- 胃口好、脚温、身热=西医还没碰他
- **白泡沫痰**=肺癌标志（抽烟者焦黄痰=不是肺癌）
- 痰中带血（吃阿司匹林血管壁薄破裂，非肺癌本身）
- 舌苔：白厚=寒湿（抽烟本应黄湿，白厚反常=肺寒湿）

### 初诊处方
| 药物 | 作用 |
|------|------|
| 桔梗1两+炙甘草3钱 | 桔梗甘草汤（白泡沫痰主力） |
| 生半夏 | 涤痰+排水+排寒水气（比制半夏好很多） |
| 干姜 | 去肺家寒 |
| 皂荚3钱+红枣 | 去抽烟焦痰（皂荚必须配红枣） |
| 射干 | 降逆，寒痰咳嗽必用 |
| 紫苑3钱+冬花 | 止咳化痰 |
| 麻黄3钱 | 宣肺 |
| 紫根 | 攻坚（上焦癌症用，真正最好的是紫参但已无） |

本质：**射干麻黄汤**（金匮治寒痰咳嗽）+紫根攻坚

### 疗效追踪
- **一周后（2月19日）**：咳嗽好很多，右胸痛消失→但出现炙热感（阳不入阴），身热退，需盖被（之前燥热不需），白痰
- **2月27日**：加丹皮桃仁冬瓜仁（千金苇茎汤，咳血湿热）→出现特殊反应：头昏+咳嗽增加+无痰出来+呼吸变好+胸炙热退→**在排脓！**
- **3月15日**：稳定
- **4月5日**：血压从160-170/95-100→120/70，体温正常，但**下利（一天10次大便）**
  - 肺与大肠相表里，肺病下利=仍在排病
  - 加麦门冬+党参+五味子（补肺）
- **4月24日**：咳嗽减少，肺痛又出现（癌症脱皮现象！脱一层里面还有一层）
- **6月12日**（离开一个月后）：睡眠好，3点必醒（肺经时间），咳血仍有
  - 加南星3钱去痰
- **6月26日**：又下利（一天20次），回到最初方（丹皮桃仁都不加了）
  - 胸痛性质变了：从刺痛+堵→只有痛不强烈
- **7月12日**：喜欢喝冷水（阳气恢复！癌症恶化时里寒盛只喝热水）
- **7月26日**：舌苔从白厚→黄干裂（寒湿退了），痰从白泡沫→黑胶痰（抽烟焦痰出来了！最底层）

### 核心发现
1. **癌症脱皮现象**：肺阴实去了又回来→一层一层剥，最底层是抽烟焦痰
2. **白泡沫痰**=肺癌标志，焦黄痰=抽烟但非肺癌
3. **可平躺**=肺无积水（最重要预后指标）
4. **下利是排病**不是坏事（肺与大肠表里）
5. **阳气恢复标志**：开始喜欢喝冷水
6. **血压**是观察指标：肺癌时高血压→治好血压正常
7. 一旦肺癌被攻到只是"肺家咳嗽"，就是寒热并结的咳嗽阶段→寒痰药+热痰药同用
8. 坚持不让西医碰（不开刀不化疗）→胃口始终好→不会被饿死

## 跨案例通用原则

### 治癌总纲
- **恢复阳气，让阴不累积**（诸积大法）
- **抓头抓尾**（转移癌守源头+末端）
- **一方到底**，不轻易变方
- **先阳后阴**：先用阳药攻，后期加滋阴
- **在80码车速下修车**：攻邪同时补虚，不能让病人里虚

### 关键药对
| 药对 | 应用 |
|------|------|
| 生附子 vs 炮附子 | 生附壮心阳，炮附壮肾阳 |
| 鳖甲+茜草 | 攻肝家阴实（肝癌） |
| 瓦楞子+牡蛎 | 乳房攻坚（瓦楞专乳，牡蛎全身） |
| 黄连+黄芩 | 解尿毒/心肝同治 |
| 紫根+牡蛎 | 乳癌破溃流脓 |
| 桔梗+甘草 | 白泡沫痰（肺癌标志） |
| 皂荚+红枣 | 去抽烟焦痰 |

### 诊断铁律
- **可平躺**=肺无积水（肺癌预后指标）
- **舌苔变化**：有→无=恶化，无→有=好转
- **白厚苔+抽烟史**=肺癌警示（抽烟本该黄苔）
- **冷水vs热水偏好**：喜冷=阳气恢复，喜热=里寒加重
- **胃气回复**（半夜饿醒）=阳回头且稳定
- **月经恢复**=心功能恢复
- **手掌青紫**=尿毒素仍在
- **出汗**=心功能恢复（尿毒病人本无汗）

### 药物使用铁律
- 川芎：收敛血管防破裂（四物汤中用意）
- 露蜂房：催乳（一钱就够，烤干炖鸡汤，多则奶水喷涌）
- 大黄：小剂量（一钱半）排不尽，大剂量→里虚→癌作怪
- 生半夏优于制半夏：涤痰+排水+排寒水气
