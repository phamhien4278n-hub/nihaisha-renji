# 金匮要略 段32：产后病 + 妇人杂病开头

## 一、产后三病（核心病机）

**师曰**：新产血虚 → 三种病：
| 病 | 直接病因 | 核心病机 |
|----|---------|---------|
| 病痉（刚痉/柔痉） | 血虚+多汗出+中风 | 气血两虚，表虚易中风 |
| 郁冒（产后忧郁/头昏） | 亡血复汗+寒多 | 血虚阳浮，孤阳上出 |
| 大便难 | 亡津液胃燥 | 津液亏虚，肠燥 |

**三者同现 → 小柴胡汤**（和解少阳，担法——太阳阳明合病以少阳解）
- 小柴胡是产后妇人第一方
- 人参（原方用参，省钱可党参）
- 小柴胡非汗剂，但能调少阳使表里自和

## 二、产后郁冒详解

**原文**：脉微弱，呕不能食，大便反坚，但头汗出。
**病机链条**：血虚而厥 → 厥而必冒 → 冒家欲解必大汗出
- 血虚下厥 → 孤阳上出 → 头汗出（阳浮不固）
- 汗血同源：产后本就血虚，再汗出伤阴
- 阴阳协调时不该流汗不流汗；阳虚掉后毛孔不束 → 阴出 → 汗流不止

**治法**：小柴胡汤 → 汗出阴阳乃复，大便坚、呕不能食自解

## 三、产后各证型

### 3.1 产后胃实 → 大承气汤
- 病解能食，七八日更发热
- 胃口恢复 = 津液未伤 → 可放心用大承气
- 鉴别：白虎汤（燥渴无便秘）vs 大承气（便秘+高热）

### 3.2 产后腹痛 → 当归生姜羊肉汤
- 并治寒疝、虚劳不足
- 羊肉：肉中之王，温中祛寒+补血

### 3.3 产后腹痛烦满不得卧 → 枳实芍药散
- 病因：子宫内有残渣（浊水/羊水/恶露）未排净
- 枳实：去浊物浊水（不限于大承气/胸痹中作用）
- 芍药等分，烧黑 → 打成散，日三服，大麦粥下
- 并主痈脓（排脓汤中也有枳实）

### 3.4 枳实芍药散不愈 → 下瘀血汤
- 药力不够 = 腹中有瘀血着脐下
- 组成：大黄一两、桃仁三十个、蟅虫（地鳖虫）二十枚去足
- 炼蜜为四丸，酒煮，取八合顿服 → 新血下如豚肝
- 亦主经水不利
- 倪师补充：蟅虫难找时可用三菱莪术+炮附子替代
  - 炮附子温下焦阳 → 瘀血动 → 活血化瘀药清出
  - 可加丹皮、川芎

### 3.5 产后七八日恶露不尽 → 大承气汤
- 无太阳证，少腹坚痛（恶露）
- 不大便 + 烦躁发热 + 脉微实
- **关键鉴别**：日哺时（黄昏）倍烦躁发热
  - 黄昏阳要入阴，大肠燥屎堵住 → 阳进不来 → 加倍烦躁
  - 不食，食则谵语，至夜即愈
- 大承气汤主之
- 倪师经验：大承气对症时，烧退→大便出→烧全退

### 3.6 产后风（阳旦证） → 阳旦汤
- 持续数十日不解：头微疼、恶寒、时时有热、心下闷、干呕、汗出
- 虽久（一两个月甚至三个月）仍可阳旦汤
- **阳旦汤 = 桂枝加桂汤（加牡桂/肉桂）**
  - 桂枝三两、芍药三两、甘草二两、生姜三两、大枣十二、牡桂四两
- 牡桂**生用**不煮：平半匙洒入汤中 → 强心行阳 → 阳回阴自回
- 病机：产后气血两虚中风，阳回不来但不至极度虚弱

### 3.7 产后中风发热面正赤 → 竹叶汤
- 面赤 + 喘而头痛 + 发热
- **关键鉴别**：虚热脸红 vs 实热脸红
  | 类型 | 面色 | 口渴 | 方 |
  |------|------|------|-----|
  | 实热（白虎汤证） | 赤红发亮 | 喜冷饮 | 白虎汤 |
  | 虚热（竹叶汤证） | 粉红无光 | 喜热/温饮 | 竹叶汤 |
- 竹叶汤组成：竹叶、葛根、防风、桔梗、桂枝、人参、甘草、附子（炮）、生姜五两、大枣十五枚
- 葛根：防刚痉；防风：通利三焦；桔梗：排痰排湿（产后多虚热夹湿）
- 附子使用指征：手脚冰冷
- 呕者加半夏半升

### 3.8 妇人乳中虚 → 竹皮大丸
- 喂母乳时虚烦、烦乱呕逆
- 病因：产奶量不足（奶水1升，baby要3升 / 双胞胎）
- 组成：生竹茹+石膏各二分、桂枝+白薇各一分、甘草七分 → 枣肉和丸
- 白薇：入阳明经虚热
- 生竹茹：通利三焦、去湿、淋巴/腮腺/甲状腺/乳房硬块均可用
- 桂枝甘草：辛甘发散为阳
- 有热倍白薇；烦喘加枳实

### 3.9 产后下利虚极 → 白头翁加甘草阿胶汤
- 白头翁汤原方（白头翁、秦皮、黄连、黄柏） + 甘草 + 阿胶
- 阿胶：补血止血，不煮（烊化）
- 甘草：解肠胃毒素
- 鉴别：非产后热痢里急后重 → 单纯白头翁汤

## 四、退奶与乳房硬块

### 4.1 退奶方
- 时机：决定停止喂母乳前两周
- 原理：心管奶水/月经 → 奶水下行化为月经 → 胸压解除
- 处方：炒麦芽五钱 + 四物汤（当归、川芎、白芍、熟地）
- 正确处理退奶 → 预防日后乳癌

### 4.2 乳房硬块方（非哺乳期）
- 基础：退奶方 + 竹茹 + 防己 + 紫根三钱 + 牡蛎五钱（咸软坚） + 瓦楞子（祛痰水/痰积）
- 三焦内奶水 = 痰湿 → 瓦楞子去
- 服药反应：胸部刺痛 → 硬块缩小
- 需保持心情开朗

## 五、妇人杂病（开头）

### 5.1 热入血室（三种情况）

| 情况 | 原文表现 | 治法 |
|------|---------|------|
| **中风七八日** | 续来寒热发作有时，经水适断 | 小柴胡汤（如疟状） |
| **伤寒发热** | 昼日明了，暮则谵语如见鬼状，经水适来 | 无犯胃气及上二焦，必自愈（可用小柴胡） |
| **中风发热恶寒** | 七八日热除脉迟身凉，胸胁满如结胸，谵语，经水适来 | 刺期门 → 小柴胡亦可 |

- 核心：月经期感冒 = 热入血室
- 问诊铁则：女病人感冒 → 先问月经来了没有
- 谵语见鬼（黄昏时）= 热入血室 or 大承气汤 → 男女都要鉴别
- 小柴胡汤使用铁则：一证即可，胸胁苦满最明确
- 往来寒热鉴别：桂枝二麻黄一（寒少热多）、麻黄二桂枝一（寒多热少）、小柴胡（伴胸胁苦满）

### 5.2 阳明病下血谵语 → 热入血室
- 刺期门，濈然汗出者愈

## 六、跨课连接

1. **痉病**：刚痉葛根汤、柔痉栝蒌桂枝汤 → 痉病篇
2. **小柴胡汤担法**：三阳合病以少阳解 → 伤寒论
3. **阳旦汤 = 桂枝加桂汤**：奔豚也用 → 伤寒论
4. **枳实**：排脓汤中也有 → 疮痈肠痈篇
5. **白头翁汤**：热痢里急后重 → 伤寒论厥阴篇
6. **大承气汤谵语**：大肠沼气上冲头部 → 伤寒论阳明篇
7. **白虎汤 vs 大承气**：燥渴 vs 便秘 → 伤寒论阳明篇
8. **汗血同源**：内经基础理论
9. **心管奶水/月经**：核心生理观 → 贯穿妇科全篇
10. **热入血室**：伤寒论已有 → 此处重复强调

## 七、新学到的核心要点

1. 产后三病小柴胡是第一方——搞不清就用和解
2. 虚热面赤（粉红）vs 实热面赤（赤红发亮）——口渴冷热偏好是鉴别关键
3. 黄昏烦躁 = 阳入阴受阻 = 大肠燥屎堵 → 大承气
4. 枳实不限于承气汤——浊水浊物通用
5. 蟅虫缺货 → 三菱莪术 + 炮附子的替代方案
6. 牡桂生用不煮——半匙洒入，强心行阳
7. 退奶 = 奶水化月经 → 防乳癌
8. 月经期感冒 = 小柴胡汤（一证即用，不必等胸胁苦满）
