# 金匮要略 第21段 知识树

## 段21：淋病→小便不利三方→糖尿病专论→猪苓汤→水气病脉证

### 一、淋家禁汗

**原文**：淋家不可发汗，发汗则便血。

**病机**：淋家小便浓稠→津液已不足→肠胃津液本亏（才会消渴/淋病）。此时发汗→肠胃津液更亏→便血（小便带血）。原因：膀胱尿已浓→汗发更浓→结石干结输尿管/膀胱→通过时刮破出血。

---

### 二、栝蒌瞿麦丸

**原文**：小便不利者，有水气，其人若渴，栝蒌瞿麦丸主之。

**辨证关键**：小便滴滴答答 + 口渴 → 判定内有炎证（热）。炎证必口渴（津液不够+发热）。

**方剂组成**：
| 药 | 量 | 作用 |
|----|-----|------|
| 薯蓣（淮山） | 三两 | 补脾 |
| 茯苓 | 三两 | 利水 |
| 栝蒌根 | 二两 | 止渴 |
| 附子（炮） | 一枚 | 温阳 |
| 瞿麦 | 一两 | **利水消炎/去热**（倪师：西医叫消炎） |

用法：炼蜜丸，梧桐子大，饮服2丸日三；不知增至7-8丸，以小便利腹中温为知。倪师说放胶囊即可。

**新药——瞿麦**：利水消炎、攻坚活血化瘀，小便刺痛非常好用。淋病比西医定义广——瞿麦+土茯苓治淋病（土茯苓清膀胱病毒）。瞿麦不够力时加土茯苓。

**倪师辨证示范——不被病名吓倒**：
- 病人写"梅毒"→问证：不好睡+热+渴+舌苔黑干+脉数+便秘+不排气+尿深黄 → **大承气汤证** → 大承气汤下去梅毒也好了
- 核心原则：**问证状，不要被病名吓倒**

**与五苓散鉴别**：
- 五苓散：小便不利+口渴，但小便清（非粘稠）——不是淋病
- 栝蒌瞿麦丸：小便不利+口渴+**小便粘稠**——淋病

### 三、小便不利三方

#### 1. 蒲灰散（外用药）
- 蒲灰（水菖蒲晒干烧灰）半分 + 滑石三分
- 用法：外涂（和麻油涂身上）利水，用于无法吞药或全身水肿
- 也可内服

#### 2. 滑石白鱼散
- 滑石 + 乱发（烧，即血余炭，活血化瘀攻瘀血）+ 白鱼各二分
- 病机：膀胱内有瘀血+石头同时堵住
- 白鱼争议：有人说是白鲤鱼（利水），有人说是书中蛀虫
- 倪师：此方很少用

#### 3. 茯苓戎盐汤（重点！）
- 茯苓半斤 + 白术三两 + 戎盐（弹丸大一枚）
- 煮法：先煮茯苓白术，入戎盐再煎，分温三服
- **青盐 vs 海盐**（中国医学统计学）：
  - 东南沿海：吃海盐 → **多肾结石，无胆结石**（因为吃醋多）
  - 青海内陆：吃青盐（内陆湖盐）→ **无肾结石，多胆结石**（因为吃辣多，不吃醋）
  - 结论：青盐打肾结石，醋洗胆
- 临床：胆结石→多吃醋；肾结石→用青盐
- 青盐很咸难喝，倪师做胶囊
- 盐汤取吐法：浓盐汤→当场吐掉（瓜蒂散的替代，用于膈上痰涎/异物）

### 五、糖尿病专论（核心重头戏）

**经方两个糖尿病方**：①白虎人参汤 ②肾气丸（金匮肾气丸）
**但光靠这两个不够**——倪师展开完整治法。

#### 5.1 初诊糖尿病（未吃西药）
- 燥渴+胃口大 → 白虎人参汤一剂即中
- 石膏2-3两（把胃火降下去→胃口变小→减肥），知母3钱除烦
- 人参补津液（没人参用党参）
- 仍渴→加生地3钱、栝蒌根3-5钱生津止渴

#### 5.2 糖尿病15年（吃西药→打胰岛素→上热下寒）
**病程**：降血糖药5年→胰岛素10年→越用越重→脚破洞、冰冷、趾不能弯、伤口1年不愈、趾发黑

**上热下寒证**：
- 上热：仍口渴、胃口大、舌苔黄、胖
- 下寒：脚冰冷、伤口不愈

**为什么寒在脚上**：血糖被药压下去但没离开身体→屯积脚上→脚冰冷→伤口不愈

#### 5.3 肾脏衰竭两大主证
1. **恶心**（东西吃下去就吐）
2. **晕眩**（头昏很厉害，躺床上不能动）

**病机——肾阳的两个去处**：
1. 肾阳→顺输尿管入膀胱（肾与膀胱表里）
2. 肾阳→入胃下方（驱动肠胃蠕动）← 这就是先天（肾）与后天（胃）的关系

肾阳不足→极阴寒→水往上走（不是热）→胃蠕动受影响（一吃就吐）→水上到膈膜→头昏

**鉴别**：脚温热+头昏=五苓散证；脚冰冷+头昏=肾衰竭上热下寒≠五苓散

#### 5.4 上热下寒核心处方

| 药 | 量 | 作用 |
|----|-----|------|
| 石膏 | 3两起 | 清上热（阳明燥热）、降胃火 |
| 炮附子 | 3-5钱 | **固肾阳**、固表 |
| 生附子 | （量未明说） | **化下焦寒湿**，心阳直下少腹 |
| 白芍 | 1两 | 活血化瘀+血流回心脏（去杖汤） |
| 炙甘草 | 5钱 | 芍药甘草汤组成 |
| 牛膝 | 3钱 | 引药下行，治脚不能走 |
| 桂枝 | （量未明说） | 导引生附子→小肠 |
| 丹皮 | 3钱 | 活血化瘀 |
| 桃仁 | 3钱 | 活血化瘀 |
| 三七 | 3钱 | 收敛伤口 |
| 黄连 | 2钱 | 解毒→护心脉（苦入心，解尿毒） |
| 黄芩 | 3钱 | 配合黄连护心脉 |
| 半夏 | 3钱 | 止呕、排水 |
| 生姜 | 2片 | 去胃中停水、止呕 |
| 干姜 | 3钱 | 温中（胃太冷蠕动慢） |
| 白术 | 3钱 | 利水（有水肿时加） |
| 茯苓 | 3钱 | 利水（有水肿时加） |

**石膏+炮附子寒热并用**：经方家看到上热+下寒→寒热药并用。上热下寒时寒热药同煮一起去掉。

**关键服药法——冷服**：药煮好放冷再喝。身体感觉冷→接受→炮附子偷偷跑到下焦→等胃发现"不该有炮附子"时它已下去了→石膏上来清上热。趁热喝会吐。

**生附子的关键作用**：光靠炮附子不能完全去下焦寒→要生附子（极热）→心阳壮起来→心火直下少腹→下焦寒湿开始融化。桂枝在此处只是导引药，把生附子导到小肠。

**去杖汤（芍药甘草汤）**：白芍+炙甘草。白芍量要大（1两），活血化瘀+让血流回心脏。

#### 5.5 加减法
- 便秘（大便不通）→ **大黄附子细辛汤思路**（已有附子+细辛，加大黄）：
  - 一周一次→大黄5钱
  - 隔天有→大黄2钱
  - 三四天一次→大黄3钱
- 伤口→加三七（收敛），骨断才用续断
- 出血易淤→加川芎
- 仍渴→加栝蒌根、石斛增润津液
- 水肿→白术+茯苓（必须配温阳利水——生附子化寒，否则利不掉）
- 小便无便意→细辛（麻黄附子细辛汤利尿）

#### 5.6 糖尿病外用方——蔗糖水泡脚
- 蔗糖半斤溶于一盆水→脚泡进去→蔗糖吸脓（脓从伤口喷出来）
- 每天泡两次（早+晚）→开始收口
- 配合内服处方

#### 5.7 更深的寒——手脚冰冷（非仅脚冷）
→ **当归四逆汤**加减：
- 当归四逆汤：桂枝+芍药+当归+细辛+木通（没有炮附子/生附子）
- 仍冰冷→加炮附子（肾阳不足）或生附子（心阳不足，水克火）
- **炮附vs生附**：
  - 心脏无力→**生附子**
  - 肾脏功能不行→**炮附子**
  - 肾脏极端不行（水克火，心脏快停，脉没了）→炮附子没用→**生附子**

#### 5.8 疗效判断
- 初诊上热下寒→治疗后一个两周
- 脚越来越冷+上面越来越热 → **方向错，没效**
- 正常方向：上面变冷，下面变热

#### 5.9 外用药思路
知母（除湿）+ 黄连（解毒、去热）→ 粉剂外敷伤口（湿热并除）。加血竭收敛伤口、肌肉重生。药性熟→缺药也能重新组合。

### 六、猪苓汤（肾结石）

**原文**：脉浮发热，渴欲饮水，小便不利者，猪苓汤主之。

**与五苓散鉴别**：
- 五苓散：脉浮发热+渴+小便不利——发热是**表证**（感冒发烧）
- 猪苓汤：脉浮发热+渴+小便不利——发热是**里面发炎**（结石引起）

**方剂**：猪苓+茯苓+阿胶+滑石+泽泻（各一两）
- 泽泻：利上焦水
- 茯苓：利中焦水
- 猪苓：利下焦水
- 滑石：打碎石头利出来
- 阿胶：止血（石头锋利刮破尿道）

煮法：先煮四味，去滓，入阿胶烊消，温服七合日三。

**肾结石来源**：
- 喝生水（自来水不煮开、美国水龙头直饮）
- 维他命C → 造成肾结石
- 钙片 → 造成肾结石
- 天然高钙食物（芝麻等）→ 不会造成结石

### 七、水气病脉证并治第十四（引言）

**水病 vs 饮病**：
- 饮病：**内发**（生活习惯不好，躺着喝茶等）
- 水病：**外来**（表证失治——该用麻黄汤/桂枝汤/葛根汤却用温病派药→表邪未去→水累积体内）

**五种水病分类**：
| 类型 | 脉 | 外证 | 恶风 | 其他 |
|------|-----|------|------|------|
| 风水（最浅） | 浮 | 骨节疼痛 | 恶风 | 似麻黄汤/桂枝汤证 |
| 皮水 | 浮 | 跗肿，按之没指 | 不恶风 | 腹如鼓，不渴 |
| 正水 | 沉迟 | 自喘 | — | — |
| 石水 | 沉 | 腹满不喘 | — | — |
| 黄汗 | 沉迟 | 身发热，胸满，四肢头面肿 | — | 久不愈→必致痈脓 |

**发展顺序**：风水→失治→皮水→失治→正水/石水/黄汗（里水）

**治疗原则**：风水+皮水 → **汗法**（发汗解）
- 发汗前必确认：病人能否承受汗药（疮家禁汗等禁忌）
- 不能发汗→改用小便排掉
- 西医只用利尿剂→不治源头→排了又回来

### 八、新学到的东西

1. **淋家禁汗**：淋家小便浓稠=津液不足→发汗会更伤津液→便血
2. **瞿麦**是新药——利水消炎/攻坚活血化瘀，治淋病小便刺痛
3. **土茯苓**治淋病→清膀胱病毒，瞿麦不够力时加
4. **辨证不被病名影响**：梅毒病人→大承气汤证→大承气汤下去梅毒也好了
5. **青盐vs海盐**的统计差异：青盐打肾结石，醋洗胆
6. **盐汤取吐法**：浓盐汤→膈上痰涎/异物吐出来
7. **糖尿病上热下寒核心思路**：石膏清上+炮附子固肾阳+生附子化下焦寒湿→寒热药同煮冷服
8. **肾阳两个去处**：①入膀胱（与小肠火共温膀胱）②入胃下方（驱动肠胃蠕动=先天与后天的连接点）
9. **肾衰竭两大主证**：恶心+晕眩
10. **去杖汤**：白芍1两+炙甘草5钱——活血化瘀+血流回心脏
11. **生附子vs炮附子**：心阳不足用生附子，肾阳不足用炮附子
12. **蔗糖水泡脚**：糖尿病伤口外用——蔗糖吸脓
13. **猪苓汤vs五苓散**：发热性质不同——表证发热vs发炎热
14. **水气病五大分类**：风水→皮水→正水→石水→黄汗，前两种用汗法
15. **表证失治→水病**：经方有温病（葛根汤），表证及时发掉就不会有水病
