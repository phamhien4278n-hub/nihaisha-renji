### 一、肺癌阴实（续上段）

#### 核心病机
- 肺家阴实：肺与大肠表里 → 肺气不固守 → 肛门收不住 → 下利严重+肺痛
- 阴实=寒+实，治疗先祛寒→寒去后热现→再清热→到最后纯热→阴实退

#### 关键治则
1. **紫参汤失传问题**：金匮原方紫参+甘草，今紫参无 → 替代方：丹皮、桃仁、冬瓜仁、紫菀、冬花、射干、桔梗、甘草
2. 未被西医碰过（未开刀/化疗）的肺癌 → 希望很大，一定可治好
3. 治疗周期：6个月（2月→8月），症状时好时坏实则寒热交替转化

### 二、肺炎/肺脓疡（湿热证）

#### 鉴别诊断
| 特征 | 肺阴实 | 肺炎/肺脓疡 |
|------|--------|-------------|
| 舌苔 | 又白又厚 | 又黄又湿又厚 |
| 性质 | 寒+实(癌) | 湿热(发炎) |
| 大便 | 下利(肺气不固) | 便秘(热壅于上) |
| 脉 | — | 数(发炎) |

#### 核心病机推理链
肺中极热 → 肺气不降 → 大便堵 → 壅热郁热 → 化脓 → 肺脓疡

#### 处方组合（一剂知、二剂已）
1. **苇茎汤**（千金方补金匮之缺）：苇茎、薏仁(可代冬瓜仁)、丹皮、桃仁
2. **桔梗甘草汤**（排脓汤，金匮原文）
3. **大承气汤**：大黄、芒硝、厚朴、枳实——通大便排热
4. **栝蒌实**：固心阳，肺热易伤心（天地定位：肺为天，壮热之下需护心）

#### 关键辨证技巧
- 问大便→便秘→立即问小便颜色→深黄=承气汤证→有无放屁→无屁=大承气汤
- 肺化脓=咳白色泡沫痰（非焦黄痰，焦黄是抽烟痰）
- 大承气汤解高烧时，大便排同时冒汗退烧
- 肺与大肠表里的临床运用：便秘使肺热无法下降，通大便则热从大便排

#### 疗效
- 高烧：第一碗药下去即退（大便出+汗出）
- 化肺脓疡者：9-10天痊愈
- 单纯肺炎：2-3天
- 第二诊加减：苇茎汤+麦冬+西洋参（补肺津液）+减量大承气→便秘复发→恢复原方

#### 百合证
- 表现：善变，问什么都"好也不好"，模棱两可
- 病机：肺藏魄，魄不守
- 用药：百合（色白入肺，黏润，固肺守魄）

### 三、子宫肌瘤 + 不孕

#### 病史
- 1997年：3颗肌瘤(7/5/3cm) → 开刀
- 2002年：7颗 → 第二次开刀
- 2006年12月26日：28颗 → 第三次开刀
- 2007年1月开始中药 → 3月中旬怀孕

#### 核心处方（一方到底，不换方）
当归四逆汤加吴茱萸生姜汤 + 炮附子 + 生附子 + 大建中汤元素：

- 桂枝3、白芍6、当归2、生姜2片 → 当归四逆汤底
- 吴茱萸3、干姜2 → 吴茱萸生姜汤（恶心胃口不好）
- 炮附子5 + 生附子3（棉布包）→ **下焦寒湿极盛时两药同用**
- 蜀椒2 → 大建中汤（少腹大寒痛）
- 柴胡3、郁金2 → 疏肝
- 熟地3、艾叶 → 温暖子宫+补血（胶艾汤思路）
- 续断3 → 手术后疤痕
- 丹皮3、桃仁3 → 活血化瘀
- 辛夷3、菖蒲3 → 鼻窦炎（99%）或心窍不开（1%）

#### 关键药性原理
| 药 | 作用 |
|----|------|
| 炮附子 | 壮肾阳 → 血到小肠 |
| 生附子 | 壮心阳 → 配合桂枝到小肠 |
| 仅炮附子不行 | 下焦寒湿极盛必须生附子推动心阳 |

#### 辨证要点
- 手脚冰冷+恶心+胃口不好 = 当归四逆汤加吴茱萸生姜汤
- 少腹痛：小建中汤（一般）、大建中汤（大寒痛）、大乌头煎（大建中汤不效+吐白津）
- 手术后：续断处理疤痕

#### 阳极生阴 / 阴极生阳
- 寒极 → 用黄芩黄连等凉药（寒中点火）反而热起来
- 热极 → 用生附子阳药 → 热药中阴的成分让患者冷下来
- 关键心法：阳药中仍有阴，阴药中仍有阳

#### 怀孕后调养（金匮原文）
- 前20周：当归散（养母）
- 后20周：白术散（强子，含牡蛎）
- 服阳药→怀孕多为男孩（精先到位，气后到位）

#### 病史
- 19年前左肾癌切除 → 2000年右肾又长5cm瘤 → 西医5次化疗差点死

#### 治则：虚则实之
**不可攻！** 化疗5次已是攻后虚极 → 先用补法

效果：第二天恢复体力，整理家务

### 六、易筋经全集

#### 文式易筋经（12式，每式49下）
- 基本姿势：脚趾抓地，膝盖与脚平行，下半身（脚→臀→少腹）绷紧如磐石，上半身放松
- 呼吸：自然呼吸，舌顶上颚
- 十二式：按跷→指拳→平拳→肚脐前→正前方→胸前→侧拉弓→转身(碌辘式×2)→后拉→抱弓式
- 顺序可乱，不影响

#### 武式易筋经（每式1-3次，不可49次——会昏迷）
1. 韦驮献杵（三式）：马步→吸气拉开→吐气→内八→外撑→吐气收工
2. 拨云见日
3. 摘星换斗：手旋转上伸，脚趾翻过去，把身体拉到最长
4. 拔刀式：侧退弓步→弯腰→旋转→碰脚（模拟长刀出鞘）
5. 青龙探爪：马步→爪在腰际→左右伸展
6. 倒踩九牛尾：旋转甩手，去+回都出劲（不像甩手功只有去没有回）
7. 出爪亮翅（抖动）：**放松全身→高频震动→排出筋脉神经肌肉里的痰**（可代藜芦甘草汤治手抖/帕金森）
8. 结尾式（最难）：**手先着地→脚慢慢走过去→踮起来→手摸脚**。正面弯下去做不到，必须反过来做。目的：调和阴阳

#### 结尾式的意义
- 阴阳调和到极限才能做到
- 身体有问题/阴阳失衡者做不到此动作
- 保持姿势1-2分钟，摆多久算多久

#### 阴跷脉拉伸
- 脚弯起→外翻（阴跷脉在内侧，往外拉即拉伸阴跷）

#### 练功目的
- 医生自己身体好→精神气色好→病人信心足
- 经方+功夫 = 完整的医者修养
