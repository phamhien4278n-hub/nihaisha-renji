# 段1：伤寒论总论与太阳病辨证（条辨1-35）

## 一、核心概念

### 1. 伤寒有五（《难经·五十八难》）
- **中风**：伤于风 → 有汗、恶风、脉浮缓
- **伤寒**：伤于寒 → 无汗、恶寒、体痛、脉浮紧
- 温病、热、湿热（后续展开）

### 2. 正气 vs 时疫
- **正气**：冬日本应寒冷，感寒得病 → 一般病，不传染
- **时疫**：冬行春令（气候反常），男女老少症状相同 → 疫病，有传染性

### 3. 六经辨证起源
- 来源：《黄帝内经·热论第31篇》
- 六经：太阳→阳明→少阳→太阴→少阴→厥阴
- 六气运转图：厥阴(亥巳)→少阴(子午)→太阴(丑未)→少阳(寅申)→阳明(卯酉)→太阳(辰戌)
- **顺走=健康常态**，**逆走=病进**
- 阳明无死症：病到阳明即止，不进入阴经。死于阳明=医之过

### 4. 经方定义
- **绝对处方**：组成不可改
- **黄金比例**：比例不可变，按体格缩放剂量
- 经方传自黄帝、岐伯、伊尹，张仲景是集大成者

### 5. 八纲辨证
- 阴阳、表里、虚实、寒热

### 6. 太阳病 = 表证
- 再分为：中风（有汗）、伤寒（无汗）、温病（葛根汤）

### 7. 病传关键——心脏
- 情志（生气、吵架）→ 病传
- 抗生素伤心 → 舌变地图舌 → 病传
- 心情好、开心 → 病不传

### 8. 健康标准
- 头面冷 + 手脚温热 = 常态
- 脚冷 = 心脏力量不够、有淤血
- 每天大便 = 肠胃好 = 不易病
- 24小时食物排出 = 清灵 = 健康

### 9. 时间概念
- "一日"= 六到七天（一候），非一天

### 10. 体液循环
- 汗和尿同源（夏汗多尿少）
- 喝的水→胃气化；食物水→小肠气化→入血循环
- 中医反对只看验血（未验水，水分多寡影响血指数）

## 二、病机推理链

1. **太阳伤寒体痛**：毛孔开→寒入→毛孔缩→汗停在肌肉与皮肤之间→新水不来代旧水→压力=痛 → 发表→水流通→痛去

2. **太阳中风**：运动流汗→毛孔开→突进冷气→毛孔来不及闭→风吹难过→汗出不止（病汗）

3. **病传途径**：太阳 → 阳明（止，无死症）或 少阳→太阴→少阴→厥阴（危）

4. **误下后气上冲**：攻下把表邪内陷→但攻下不够强→邪还在表→气反弹上冲→继续用桂枝汤。不上冲→不可再发

5. **身痒（桂麻各半汤）**：汗到皮肤表面但透发不出→停在皮下→身痒→微发汗即解

6. **热多寒少（桂二麻一汤）**：冷几分钟，热一两小时

7. **白虎加人参**：太阳证未好→转入阳明经热→大烦渴、脉洪大（非桂枝汤发汗太过导致）

8. **桂二越婢一汤**：无阳不可发汗→麻黄+石膏（青龙+白虎）→阳回收→微汗

9. **桂枝去桂加苓术汤**：素有中焦湿→服桂枝汤后心下满微痛+小便不利→湿未去

10. **误用桂枝汤救逆三部曲**：
    不该发汗而发汗→津液伤→
    ①甘草干姜汤（复阳→脚温）
    ②芍药甘草汤（补阴→脚伸）
    ③调胃承气汤（胃中宿食堵→谵语）

11. **风温（误治变证）**：津液不断流失→脱水的连锁反应
    - 发汗→小便不利（汗尿同源）
    - 攻下→直视失溲（水源断绝）
    - 火熏→发黄惊痫（最后一层津液烧干）

## 三、关键治则

1. **表里先后**：有表+有里 → 先解表再攻里（大原则）。例外：表里双解（分两包先后吃）

2. **桂枝汤 = 调和阴阳第一方**
   - 桂枝（阳·辛甘发散）+ 白芍（阴·酸苦涌泄）等量
   - 阳虚→加重桂枝；阴虚→加重白芍
   - 姜草枣 = 补肠胃津液 + 甘草解毒
   - 炙甘草→强心；生甘草→肠胃解毒
   - 啜粥助胃气发汗

3. **发汗标准**：微似汗（摸有湿感看不见汗珠）= 最佳。大汗→病不去+伤津液

4. **桂枝加葛根汤**：葛根 > 桂枝量（四两>三两）是关键。葛根升提津液、去痈脓。不啜粥（只需项背出汗）

5. **葛根汤（温病主力方）**：桂枝汤+麻黄+葛根
   黄金比例：葛4 麻3 桂2 芍2 草2 姜3 枣12
   葛根最大→力达项背；麻黄<葛根→只发项背汗
   适用：小孩跑跳流汗后感冒、冬伤于寒春发温病、刚痉（脑膜炎角弓反张）、水痘

6. **白虎汤**：阳明经热·但热不寒 → 石膏起价一两，五钱以下=刷牙漱口

7. **调胃承气汤**：大黄酒洗（升提停留胃部）+ 芒硝生用（软坚）。只清胃，不下大肠

8. **芍药甘草汤**：治脚挛急、静脉瘤。重用至一两→暝眩（药不暝眩疾弗瘳）

9. **甘草干姜汤**：救逆复阳。炙甘草:干姜=4:2 → 强心+强肺→脾自健

10. **去桂 vs 去芍**：胃不舒服（心下满痛）→去桂枝；胸满→去白芍

11. **白术+茯苓（等量）**→去中焦湿+利小便

12. **术附汤**（白术+附子等量）→排脓汤（深处脓疡）

13. **面部中风** → 桂枝加葛根汤，葛根加倍（六两）→快煮取药气

14. **鼻窦炎**：清涕→葛根汤+术附；黄稠涕→葛根汤+黄芩

15. **痛风** → 甘草附子汤（桂枝去风+白术去湿+炮附子去寒）

## 五、新学到的东西（与内经对比）

1. 六气运转图的具体时辰分配（内经热论只提六经，未提时辰）
2. 病传方向：逆走为病进，顺走为痊愈——这是临床判断预后的大框架
3. "一日"=六七天——这个时间尺度很重要，医者要在六天内判断传变
4. 抗生素伤心的观念——舌诊验证（地图舌）
5. 寒热造成淤血的物理逻辑：冷→液体变固体→脚冷→血管淤血
6. 发表与利尿的物理关系：汗出→毛孔开→水从表走→小便也通
7. 单味麻黄不会发汗，必须配合其他药才有发汗作用
8. 石膏的收敛阳气的功能（不只是去热）
