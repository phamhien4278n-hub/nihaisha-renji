# 金匮要略 第11段 知识树

## 核心病机推理链

### 虚劳类证
```
虚劳 + 大便3-4次/日(寒利) → 桂附八味丸（肉桂附子止寒利，茯苓利水）
虚劳 + 大便1次/日(正常)   → 六味地黄丸
虚劳 + 常感冒              → 薯蓣丸（倪师不用，主张经方辨证治感冒断根）
虚劳 + 虚烦不得眠          → 酸枣仁汤（老年人/大病初愈）
虚劳 + 羸瘦+腹满+肌肤甲错+眼眶黑 → 大黄蟅虫丸（内有干血，瘀不去新不生）
虚劳 + 汗出而闷+脉结悸    → 炙甘草汤（补阴虚）
```

### 肺痿→肺痈→肺癌 进阶链
```
重亡津液（汗/吐/消渴/便难/快药下利）
  → 肺痿（脉数虚，浊唾涎沫，隐隐痛）
  → 肺痈（脉数实，咳唾脓血，刺痛持续+下利）
  → 脓成如米粥 → 死（肺癌）
```

### 咳嗽辨证链
```
咳嗽+痰白+不渴+遗尿        → 肺寒   → 甘草干姜汤
咳嗽+痰白+喉中水鸡声+吐气难 → 寒饮   → 射干麻黄汤
咳嗽+痰黄浊胶+但坐不得眠   → 湿热痰 → 皂荚丸
咳+脉浮+表证               → 肺气肿有表证 → 厚朴麻黄汤
咳+脉沉+无表证             → 肺气肿里证   → 泽漆汤
喘急                        → 急救     → 甘草麻黄汤
```

## 关键治则

### 1. 桂附八味 vs 六味地黄
- **鉴别点**：问大便！一日3-4次→八味丸（寒利可止）；一日1次→六味丸
- **注意**：下利必须是寒利（不渴、不热），热利（便脓血、燥渴）禁用桂附八味丸
- **原理**：肉桂附子止寒利，茯苓把大肠津液从小便利出

### 2. 酸枣仁汤（失眠）
- **适应症**：老年人虚劳失眠；大病初愈失眠
- **药解**：
  - 酸枣仁：酸入肝，收敛血回肝，肝藏魂→魂归肝则能眠
  - 知母：除烦（入肺）
  - 茯苓：入三焦利水
  - 川芎：活血化瘀
  - 甘草：中和
- **关键理念**：经方治失眠是**消导**不是强补——失眠者再补精神更好更睡不着
- **鉴别**：黄连阿胶汤治心血不足失眠（心藏神，神不守）

### 3. 大黄蟅虫丸（干血痨）
- **诊断标志**：
  - 肌肤甲错：**只在小腿胫骨**足三里→三阴交段
  - 两目黯黑：眼眶周围黑暗
  - 羸瘦腹满不能食
- **病机**：内有干血→瘀不去新不生→吃任何补药都进不去
- **倪师态度**：不用！原因：
  - 动物药来源不可靠（虻虫/蛴螬/蟅虫/水蛭各地说法不一）
  - 干漆有毒，不知来源
  - 可用针灸+活血化瘀药替代
- **替代方案**：
  - 肚脐以下瘀血→白芍为主
  - 肚脐以上瘀血→桂枝为主
  - 川芎、丹皮、红花、牛膝（脚瘀血）、茜草（子宫瘀血）

### 4. 肺痿 vs 肺痈（鉴别要点）

| 鉴别项 | 肺痿（虚热） | 肺痈（实热） |
|--------|-------------|-------------|
| 脉 | 数虚 | 数实 |
| 痰 | 浊唾涎沫 | 咳唾脓血→米粥 |
| 痛 | 隐隐作痛 | 刺痛持续 |
| 大便 | 可正常 | 必下利（上虚不能制下） |
| 病机 | 重亡津液，肺中干 | 热伤血脉→血凝滞→蓄结痈脓 |
| 预后 | 可治 | 始萌可救，脓成则死 |

### 5. 射干麻黄汤（寒饮咳嗽）
- **来源**：小青龙汤去白芍、甘草 + 紫菀、款冬花、射干
- **关键**：去甘草因甘味增腻增痰饮
- **辨证要点**：寒饮→不渴、舌苔白、吐气困难（肺的问题）
- **吸气困难 vs 吐气困难**：吸入困难=肾不纳气；吐气困难=肺的问题
- 倪师：十几年咳嗽气喘一剂就好

### 6. 皂荚丸（胶痰咳嗽）+ 戒烟方
- **皂荚丸**：皂荚（碱性去胶痰）+ 红枣汤（隔离胃酸）
- **戒烟方**：皂荚三钱+红枣十枚+苦参三钱+黄芩二钱+黄柏三钱+黄连二钱+半夏二钱+南星二钱
- **戒烟秘诀**：当地当季水果大量吃（倪师亲自经验：葡萄柚+柳丁汁）

### 7. 肺气肿二方

| 方名 | 脉象 | 表证 | 核心药 |
|------|------|------|--------|
| 厚朴麻黄汤 | 浮 | 有 | 麻杏甘石去甘草+厚朴半夏干姜细辛小麦五味子（加小麦助消化，久病体虚） |
| 泽漆汤 | 沉 | 无 | 大戟(泽漆)+紫参+半夏生姜白前甘草黄芩人参桂枝 |

### 8. 急救喘：甘草麻黄汤
- 麻黄2钱+甘草3钱，三碗煮一碗
- 肺立刻打开，喘停

### 9. 消肿溃坚汤（甲状腺/淋巴肿瘤）
- **核心思路**：柴胡入三焦腺体 + 三黄泻心清热 + 昆布海藻咸软坚 + 活血化瘀
- **变方（无效时）**：桂枝龙骨牡蛎汤（心悸盗汗）+防己（入三焦淋巴）+瓦楞子（咸软坚，乳房/甲状腺/淋巴硬块）
- **瓦楞子**：血蚶壳，形似乳房内有血，咸味软坚，治乳房硬块/甲状腺/淋巴癌
- **牡蛎攻坚量**：1两

### 10. 三黄泻心汤止血铁律
- **冷服**才止血：用粉+冷水配服
- **热服**止不住血
- 适用：眼睛出血、鼻血、耳血、口血、牙龈出血

## 新学到的东西

1. **虚劳→中风因果链**：虚劳不治→年轻人脸部中风、老人半身瘫痪
2. **肺痈→肺癌下利机制**：上虚不能制下，肺与大肠表里，肺实→肛门失控→下利无度（一天二十几次）
3. **肺痈口渴变化**：不渴=脓多津液多（严重）→治到渴=好转（正常人会渴）
4. **肾气旺→头发**：发油光亮=肾气旺（案例：17岁女孩3小时不洗头头发不飘→三黄泻心汤寒泻撤热）
5. **紫参失传**：紫参汤（紫参+甘草）治下利肺痛（肺癌），现已无此药，倪师用地纪去找
6. **白前 vs 麻黄**：麻黄透到皮肤毛孔，白前只透到肺打开肺
7. **小麦在厚朴麻黄汤中**：久病体虚加小麦助消化，不要加太多（两汤匙即可）
8. **消渴三型**：上消（渴饮千杯）肺、中消（饥食百碗）胃→白虎人参汤加减；下消→桂附八味丸
9. **温病滋阴药可制造肺痿**：咳嗽当阴虚治，开滋阴药→津液更伤→肺痿
10. **苦参多用**：阴部发痒、皮肤发痒、腋下湿疹、寄生虫，内服外洗均可

## 跨课连接

- 桂附八味丸/六味地黄丸 ↔ 伤寒论金匮前面虚劳篇
- 酸枣仁汤 vs 黄连阿胶汤（伤寒论）→ 肝不藏魂 vs 心不藏神
- 射干麻黄汤 ← 小青龙汤变方（去芍药甘草）

- 大黄蟅虫丸 ↔ 伤寒论抵挡汤（水蛭虻虫）
- 三黄泻心汤 ↔ 伤寒论（大黄黄连黄芩）

- 消渴病 ↔ 伤寒论厥阴篇

## 薄弱点（需重点记忆）

- 大黄蟅虫丸中动物药鉴别（虻虫/蛴螬/蟅虫各是什么）
- 肺气肿二方鉴别（厚朴麻黄汤有小麦、泽漆汤有紫参+白前）
- 消肿溃坚汤全方组成
- 紫参=肺癌关键药（已失传）
