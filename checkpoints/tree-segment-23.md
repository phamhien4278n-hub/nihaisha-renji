# 金匮要略 段23 知识树：水气病脉证并治（风水·皮水·里水·黄汗·气分）

## 核心概念

### 水病进展三阶段
风水（表初起）→ 皮水（入皮肤下/三焦淋巴）→ 里水（深入里）
- 初症无渴 → 病久则渴

### 黄汗
汗出入水中浴，水从汗孔入 → 三焦营养外渗（黄色=三焦本色）
- 非肝胆黄，是汗黄

### 气分
阴阳不交 → 大气一转其气乃散 → 桂甘姜枣麻辛附子汤

---

## 病机推理链

### 风水→皮水→里水递进
```
风水（表）→ 未治好 → 皮水（皮肤下/三焦淋巴）→ 未治好 → 里水（深入里）
     │                    │                          │
  ┌──┴──────────┐    防己茯苓汤              ┌───┴──────────┐
  │             │    四肢聂聂动                │              │
防己黄芪汤   越婢汤   肌肉跳动               越婢加术汤   甘草麻黄汤
身重+汗出   无身重                           小便自利+渴  小便不利+无渴
湿在脾/肌肉  肺热伏内                         亡津液       发汗法
```

### 防己黄芪汤机理链
冷气/汗不透发 → 湿积肌肉 → 脾主肌肉，湿阻 → 新水无法代谢旧水 → 旧水停→身重 → 湿太盛→汗出 → 脉浮身重汗出恶风

### 越婢汤机理链
肺热伏内 + 表束 → 无大热（热伏） + 续自汗出 → 麻黄敛汗（反常！）+ 石膏清肺热

### 黄汗两方机理对比
|      | 表虚（收敛法）           | 表实（发散法）             |
|------|------------------------|--------------------------|
| 方   | 黄芪芍药桂枝苦酒汤       | 桂枝加黄芪汤               |
| 机理 | 黄芪清旧汗+桂芍循环+醋收 | 桂枝汤解肌发汗+黄芪退黄     |
| 黄芪 | 五两（重用）             | 二两（减量，防碍汗）        |
| 服法 | 正常服                  | 啜热稀粥+温覆取微汗        |
| 注意 | 可能心烦（醋）           | 黄芪减量关键               |

## 关键治则 & 方剂速查

### 一、风水（水病初起）

#### 防己黄芪汤
- **主证**：脉浮身重，汗出恶风
- **组成**：防己一两，黄芪一两，白术五钱，炙甘草五钱
- **机理**：白术去湿（脾主肌肉），防己入三焦通利水道，黄芪固表去旧汗
- **加减**：腹痛+芍药，短气+麻黄，咳+杏仁
- **临床应用**：
  - 冷气病（表汗不透发）
  - 虚胖/小腹微突
  - **淋巴癌/肿瘤**：防己+茯苓+瓦楞子（去痰饮）+牡蛎（咸软坚），身重+白术，汗出气虚+黄芪
  - 肿瘤鉴别：表面圆滑=良性，粗糙硬如石=癌症（治法同，剂量加重）
  - ⚠️ 不切片——切片处药力难达

#### 越婢汤
- **主证**：风水恶风，一身悉肿，脉浮不渴，续自汗出，无大热
- **组成**：麻黄六两，石膏半斤，生姜三两，甘草二两，大枣十二枚
- **机理**：肺热伏内，表束→麻黄在此**敛汗**（非发汗！），石膏清肺热
- **舌象**：舌苔黄干，燥热
- **加减**：恶风+附子一枚，风水+术四两（→越婢加术汤，治黄汗/黄疸）
- **与防己黄芪汤鉴别**：无身重用越婢，有身重用防己黄芪

### 二、皮水（水入三焦淋巴系统）

#### 防己茯苓汤
- **主证**：四肢肿，水气在皮肤中，四肢聂聂动者（肌肉跳动）
- **组成**：防己、黄芪、桂枝各三两，茯苓六两（重用），甘草二两
- **机理**：桂枝导心火入小肠+行阳，茯苓生三焦油膜利水
- **对比防己黄芪汤**：去白术，加桂枝+茯苓
- **病久必渴**
- **临床应用**：帕金森（肌肉跳动）、尿毒

### 三、里水（水深入里）

#### 越婢加术汤
- **主证**：一身面目洪肿，脉沉，**小便自利**，渴（亡津液）
- 上见越婢加术

#### 甘草麻黄汤
- **主证**：一身面目洪肿，脉沉，**小便不利**，无渴
- **组成**：甘草二两，麻黄四两（2:1）
- **机理**：纯发汗利水
- **经方两味药规律**：主药:甘草=2:1
  - 大黄甘草汤、桂枝甘草汤、茯苓甘草汤同理

### 四、水之为病（脉鉴别）

| 脉象 | 病位 | 方 | 机理 |
|------|------|-----|------|
| 脉沉小 | 少阴（里阳不足） | 麻黄附子汤 | 麻黄返魂壮里阳 |
| 脉浮 | 表有水气 | 杏子汤（已佚） | 麻黄+杏仁等量发汗 |

#### 麻黄附子汤
麻黄三两，附子一枚，甘草二两

#### 杏子汤（倪师推断）
麻黄+杏仁等量（麻黄发肺汗，杏仁配麻黄保安全——等量即不伤）
- ⚠️ 陈修园说麻杏甘石汤 → 倪师反对：麻杏甘石汤用于肺燥热+渴+喘，与此不同
- 杏子汤是单纯发肺汗，不加石膏

#### 麻黄在经方中的双重身份
- 青龙：发汗
- 返魂汤：阳虚屯水时壮阳利尿（如少阴症麻黄附子细辛汤）

### 五、蒲灰散
- **适用**：厥而皮水，汤药不能下肚
- **用法**：外敷包身排水
- **日常少用**（有防己黄芪汤等替代）

### 六、黄汗

#### 病因：汗出入水中浴，水从汗孔入
#### 主证：身体肿，发热，汗出而渴，状如风水，汗沾衣色正黄如柏汁，脉沉

#### 黄芪芍药桂枝苦酒汤（表虚·收敛）
- **组成**：黄芪五两，芍药桂枝各三两，苦酒（醋）一升
- **机理**：黄芪清表旧汗+桂枝芍药循环加速+醋收敛→黄收三焦
- **副作用**：心烦（醋阻），停药自愈
- **效速**：一剂知

#### 桂枝加黄芪汤（表实·发散）
- **组成**：桂枝芍药各三两，甘草黄芪各二两，生姜三两，大枣十二枚
- **服法**：啜热稀粥+温覆取微汗
- **关键**：黄芪减量（二两≠五两），否则止汗与目的相反
- **主证**：腰以上汗出下无汗，身疼重，烦躁，小便不利

#### 黄汗鉴别
- 两胫自冷 → **历节**（非黄汗）
- 食已汗出+暮盗汗 → **荣气**（非黄汗）
- 汗出反发热不止 → 必生恶疮（湿热）

### 七、气分

#### 病机链
寸口迟涩（寒+血不足）+ 趺阳微迟（气不足+寒）
→ 手足逆冷 → 营卫不利 → 腹满胁鸣 → 膀胱营卫俱劳
→ 阳气不通=身冷 / 阴气不通=骨疼
→ "大气一转，其气乃散"

#### 桂甘姜枣麻辛附子汤
- **主证**：心下坚大如盘，边如旋盘
- **组成**：桂枝生姜各三两，细辛甘草麻黄各二两，附子一枚炮，大枣十二枚
- **服后反应**：汗出如虫行皮中即愈
- **阴阳观**：腹为阴胸为阳，8字循环（肚脐=阴阳交汇点）

## 新学到的东西

1. **风水两方鉴别核心**：身重（有/无）——有=防己黄芪汤，无=越婢汤。简单但关键。
2. **麻黄在越婢汤中敛汗**：这个反常用法很重要——肺热+表束时，麻黄不发表而敛汗
3. **经方两味药规律**：主药:甘草=2:1，统一规律
4. **麻黄+杏仁等量=安全**：无论麻黄多重，杏仁等量即不伤人——杏子汤的核心
5. **黄芪用量决定方向**：重用清表固表（黄汗虚），轻用（配桂枝汤）不碍发汗
6. **三焦黄色=营养外渗**：黄汗的本质理解——不是肝胆问题
7. **不切片原则**：淋巴肿瘤切片后药力难达——临床重要警示
8. **返魂汤概念**：麻黄在阳虚时不是发汗而是壮阳利尿
9. **大气一转其气乃散**：气分的核心治则——让阴阳重新交泰循环
