# 金匮要略 · 段19：支饮诸方辨治——小青龙汤变证链

## 核心框架

本段围绕**支饮**展开，前半段鉴别多种支饮证型及其对应方剂，后半段详述**小青龙汤误汗后本虚之人的变证处理链条**（三条递进方）。

---

## 一、支饮的多种面貌鉴别

### 1. 葶苈大枣泻肺汤（复习）
- **证**：支饮、胸胁苦满、不能平卧
- **病机**：肺热盛→日久化脓→肺痈
- **药性**：葶苈子苦寒，泻肺中热
- **鉴别点**：肺热标志

### 2. 小半夏汤
- **条文**：呕家本渴，渴者为欲解，今反不渴，心下有支饮故也
- **证**：呕吐后不渴（反常）
- **诊断逻辑**：正常人呕吐→津液伤→应当渴。不渴=有支饮
- **方**：半夏一升 + 生姜半斤
- **用法**：生半夏（利水力强）+ 生姜（解半夏毒+去胃中停水）
- **定位**：最轻的止呕方，半夏去胸膈间水，生姜去胃中停水
- **特别说明**：用生半夏而非姜半夏（姜半夏=生姜+半夏，再用生姜煮多此一举）

### 3. 己椒苈黄丸 ★★★（倪师常用）
- **条文**：腹满，口舌干燥，此肠间有水气
- **证**：便秘 + 腹压有水声（咕噜）+ 右侧天枢压痛 + 有排气
- **病因**：原便秘（大承气/大柴胡证）→ 受惊吓 → 燥屎顶到肝下与大肠血管连接处 → 水与糟粕分离 → 水渗到肠外
- **方**：防己 + 椒目 + 葶苈 + 大黄各一两，蜜丸
- **药性分析**（四味经典配伍）：
  - 防己：排三焦/肠间之水
  - 椒目（蜀椒）：辛辣入肺大肠，开肺气下降，驱虫，消炎解毒
  - 葶苈子：苦寒，泻肺水化痰（大肠与肺表里）
  - 大黄：推实
- **加芒硝**：渴者加芒硝半两，芒硝汤吞丸——芒硝打散硬便
- **必须丸剂**：汤剂走太快，大便吸住退不下来
- **疗效**：大便如potato chips一片片排出，水也排出，肚子变平
- **鉴别**：大柴胡/大承气/大黄附子细辛汤/大黄牡丹皮汤均无效，非此方不可

### 4. 小半夏加茯苓汤
- **条文**：卒呕吐，心下痞，膈间有水，眩悸者
- **证**：平时好好的→突然呕吐+胃脘不适+头眩心悸
- **病因**：偶发——喝水太快/躺倒喝水/运动后灌水
- **方**：半夏一升 + 生姜半斤 + 茯苓四两（小半夏汤+茯苓）
- **与五苓散鉴别**：偶发一次用小半夏加茯苓；经常吐奶/每天吐用五苓散
- **儿科**：小朋友偶尔吐奶→小半夏加茯苓；经常吐+水肿+口水多→五苓散

### 5. 五苓散
- **条文**：假令瘦人脐下有悸，吐涎沫，而癫眩，此水也
- **证**：肚脐下动悸 + 口吐涎沫 + 头昏眩（严重可昏倒）
- **方**：泽泻一两六铢 + 猪苓/茯苓/白术各十八铢 + 桂枝半两
- **服用**：为末，白饮服方寸匕，多饮暖水，汗出自愈
- **核心病机**：阳不足→水积。用阳药（桂枝）行阳，不用阴药（无白芍）
- **温阳利水法**：桂枝行阳→白术收湿→茯苓泽泻猪苓利小便
- **临床应用扩展**：
  - 尿毒症（处方之一）
  - 偏头痛：头痛+口渴+小便不利=水包血管
  - 晕车晕船
  - 小儿口水过多（一天换十几张围兜）
  - 见水即发的癫痫
  - 瘟疫（毒从外来，停在身上）
- **脐周动悸定位**：脐下=茯苓；脐上=桂枝；脐中=桂枝+茯苓

### 6. 十枣汤（再提）
- **条文**：咳家其脉弦，为有水；支饮家咳烦胸中痛，不卒死，至一百日或一岁
- **证**：脉弦（如琴弦）+ 咳烦胸痛 + 支饮
- **案例**：小姐气喘二十几年，上身下身像两个人中间堵住，十枣汤一剂"变成一个人"
- **鉴别小青龙vs十枣**：小青龙=表寒里寒；十枣=无表里寒，纯支饮

## 三、小青龙汤变证处理链 ★★★（核心）

### 前置条件：本虚之人
- 平常体弱（走路一段就累、爬楼二三楼就喘）
- 得了小青龙汤证（表寒里寒）
- 发汗过度→津液大伤→出现变证

### 变证链条

#### 第一方：桂苓五味甘草汤
- **条文**：青龙汤下已，多唾口燥，寸脉沉，尺脉微，手足厥逆，气从小腹上冲胸咽，手足痹，其面翕热如醉状，因复下流阴股，小便难，时复冒者
- **证**：
  - 气从小腹上冲胸咽（冲气）
  - 面热如醉状（阳浮于面）
  - 时复冒——**如帽戴头**（特征性症状！倪师凡见此症必用此方）
  - 手足痹（阳不能达四肢末梢）
  - 小便难（阳散+阴津不足）
- **病机**：本虚之人汗后→津液（阴）流失→阳无阴守→上冲（气冲）→无法照顾四肢→手足冷痹
- **方**：桂枝四两 + 茯苓四两 + 五味子半升 + 炙甘草三两
- **药性**：
  - 桂枝：行阳，让阳回头
  - 茯苓：肚脐下动悸（水）+ 白入肺增润津液+利尿
  - 五味子：酸收，降冲逆
  - 炙甘草：补散失津液
- **案例**：车祸后耳聋+如帽戴头，加此四味一剂愈

#### 第二方：苓甘五味姜辛汤
- **条文**：冲气即低，而反更咳，胸满者，用桂苓五味甘草汤去桂，加干姜细辛
- **证**：冲气降了→但咳嗽加重+胸满
- **病机**：冲气降=津液回来→但寒仍在胸中（原支饮补位）
- **为什么去桂枝**：胸满=胸阳已足，不需桂枝行阳
- **方**：茯苓四两 + 甘草三两 + 干姜三两 + 细辛三两 + 五味子半斤
- **药性**：干姜温中入肺，细辛止咳去水

#### 第三方：苓甘五味姜辛半夏汤
- **条文**：咳满即止，而冲气复发者，以细辛干姜为热药也，服之当遂渴，而渴反止者，为支饮也；支饮者，法当冒，冒者必呕；呕者复内半夏以去其水
- **证**：咳嗽胸满好了→但冲气又复发+呕+冒（头昏）
- **病机**：细辛干姜热药→本当口渴→不渴=支饮还在→冒（支饮头昏）+呕
- **方**：茯苓四两 + 甘草二两 + 细辛二两 + 干姜二两 + 半夏半斤 + 五味子半斤
- **即前方减甘草量 + 半夏去水**

## 五、新学到的关键观念

1. **"呕后不渴"诊断支饮**：常识→呕吐伤津应渴，不渴=水饮内停
2. **惊吓致己椒苈黄丸证**：惊吓→燥屎移位→水糟粕分离——病理链条完整
3. **本虚+发汗的变证链**：三递进方（桂苓五味甘草→苓甘五味姜辛→苓甘五味姜辛半夏）——张仲景展示的治法弹性
4. **"如帽戴头"作为桂苓五味甘草汤的特征性症状**：倪师临床独到经验
5. **丸剂vs汤剂**：己椒苈黄丸必须丸剂——大便吸住推不动，汤剂走太快
6. **五苓散温阳利水法**：桂枝行阳+白术收湿+苓泽猪利小便，全方无一味阴药
7. **胸满去桂枝**：胸阳已足不需行阳，胸阳不足才用桂枝
