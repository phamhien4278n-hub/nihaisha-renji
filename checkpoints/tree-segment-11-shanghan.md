# 伤寒论 第11段 知识树（227-251条）

## 段概述
本段涵盖阳明病谵语危重症鉴别、大小承气汤多维鉴别体系、茵陈蒿汤（阳黄）、猪苓汤、栀子豉汤、四逆汤、白虎加人参汤、小柴胡汤在阳明兼证中的应用，以及肝病腹水虚实治疗（分消汤/补气建中汤）、骨三阴、蜜煎导法等。

## 一、谵语死证判断（227-228条）

### 227条：阳脱死证
| 症状组合 | 预后 | 病机 |
|----------|------|------|
| 谵语+直视+喘满 | **死** | 阳往上脱，眼睛无神定视，呼吸短促 |
| 谵语+下利 | **死** | 阳绝不能固阴，阴往下脱 |

**核心病机**：孤阴不生，独阳不长。正常→阴在上阳在下（地天泰卦）。阳绝时→阳往上、阴往下，阴阳相背→死。

**临床验证法**：上吊者——大小便未出=有救，大小便一出=阳绝已死。

### 228条：发汗亡阳判断
- **前提**：已发汗很多→再发汗→亡阳→谵语（津液丧失→肠胃水分被发掉→燥屎）
- **脉短者死**：短脉=阴脉，代表里面无法自己恢复
- **脉自和者生**

**桂枝汤vs麻黄汤在此**：桂枝汤有炙甘草/生姜/红枣→不易造成亡阳；麻黄汤/葛根汤有麻黄/葛根→更容易导致此情形。

## 二、大小承气汤多维鉴别体系

### 第一维度：屁的有无（经典鉴别）
| | 小承气汤 | 大承气汤 |
|------|------|------|
| 放屁 | **有屁** | **无屁** |
| 压痛 | 小腹（关元周围）按之痛 | 肚脐两旁**天枢穴**压痛 |
| 燥屎位置 | 小肠 | 大肠 |

### 第二维度：能否进食（232-233条）
- **不能食** → 大承气汤（肠胃堵到燥屎往回逆，不想吃）
- **能食** → 小承气汤（只是大便硬）

### 第三维度：脉象（231条）
- 脉滑而**虚**（按下去脉就没有了）→ **小承气汤**
- 应是脉实但虚→不应用大承气

### 第四维度：潮热+谵语+手足汗（238条）
- 二阳并病，太阳证罢 → 潮热+手足漀漀汗出+大便难+谵语 → **大承气汤**
- 注意：手足汗多也可能是里虚（桂枝汤+龙骨牡蛎），必须确认是阳明燥实证

### 承气汤使用铁律
1. **过经乃可下**：必须太阳→少阳→阳明纯阳明证才可攻下（235条）
2. **攻下后不出**：第二天不大便+脉微涩→里虚→**不可再给承气汤**（231条）
3. **一服利止后服**：大承气汤大便一通就停。汤剂一剂就够，粉剂不行（229条）
4. 小便颜色验证清干净：正常→淡白/淡黄；黄→没清干净
5. **换方不当的后果**：小承气开成大承气/反之都不对

## 三、三阳合病与二阳并病

### 三阳合病（237条）
- **症状**：腹满、身重、难以转侧、口不仁而面垢、谵语
- **治法**：**小柴胡汤**（和解少阳，担法）——唯一处方
- **误治后果**：
  - 发汗→谵语更严重（津液不够）
  - 攻下→额上汗出+手足厥冷+遗尿（危险！）
  - 若自汗出→白虎汤救逆

### 二阳并病（238条）
- 太阳证罢 → 潮热+手足漀漀汗出+大便难+谵语 → **大承气汤**

## 四、栀子豉汤证（240、244条）

### 适应证
- **心中懊憹**（胃里灼灼难过）→ 舌上苔者 → **栀子豉汤**
- 阳明病下之后 → 外有热+手足温+心中懊憹+**饥不能食**+但头汗出 → **栀子豉汤**

### 药性
- 栀子→去胃里/上焦的**虚热**
- 多用于病后调理

### 鉴别：饿的不同类型
| 症状 | 方剂 |
|------|------|
| 饥不能食 | 栀子豉汤（虚热） |
| 不能食 | 大承气汤（燥屎） |
| 能食 | 小承气汤 |
| 一饿马上要找东西吃（手抖冷汗） | 白虎汤 |

## 五、白虎加人参汤证（240条）
- **渴欲饮水+口干舌燥** → 白虎加人参汤
- 与猪苓汤鉴别：猪苓汤也有渴，但关键区别是**小便不利**

## 六、猪苓汤（240-241条）

### 适应证
- **脉浮+发热+渴欲饮水+小便不利** → 猪苓汤
- 临床：肾结石、膀胱结石、膀胱炎

### 方剂组成
猪苓一两、茯苓一两、阿胶一两、滑石一两、泽泻一两（等量）

### 药理解析
| 药 | 作用 |
|----|------|
| 滑石 | 攻坚，打碎石头，滑出来 |
| 阿胶 | 止血（石头刮破尿道） |
| 茯苓 | 利中焦水 |
| 猪苓 | 利下焦水 |
| 泽泻 | 全身水往下走 |

### 猪苓汤 vs 五苓散
| | 猪苓汤 | 五苓散 |
|------|------|------|
| 方向 | 直接攻下（结石/炎症） | 收表水（汗未透发，停在肌肉皮肤间） |
| 关键区别 | 去桂枝白术+阿胶滑石 | 有桂枝（强心阳解肌）+白术（收敛湿） |
| 小便痛 | **有**（结石刮尿道） | 无 |

### 241条禁忌
- 汗出多而渴者 → **不可与猪苓汤**（胃中已干燥，再利小便更伤津液）
- 但真有结石堵到 → 仍可急下存阴

## 七、四逆汤证（242条）

### 适应证
- **脉浮而迟+表热里寒+下利清谷**（吃青菜排青菜）→ **四逆汤**

### 里寒鉴别
- 胃中虚冷+不能食+饮水则哕（打嗝）
- 对比：承气汤/白虎汤证→喝水不会打嗝，反而越喝越想喝

---

## 八、衄血（243条）
- 脉浮+发热+口干+鼻燥+能食 → **衄**（流鼻血）
- 热在肺（肺开窍在鼻），胃没问题（能食）
- 发烧+流鼻血 → 烧应退（正常）
- 无故流鼻血+无发烧 → **坏病**（可能是鼻咽癌/肺癌）

## 九、小柴胡汤在阳明证中的应用（245-247条）

### 245条
- 阳明病+发潮热+大便**溏**+胸胁满 → **小柴胡汤**
- 大便硬+胸胁满 → 大柴胡汤
- **原则**：有丝毫牵扯到别的脏，就不先用承气汤

### 246条
- 胁下硬满+不大便+呕+舌上白苔 → **小柴胡汤**
- 倪师评：看起来更像大柴胡汤，但小柴胡不坏事
- 药后：上焦得通→津液得下→胃气因和→身**濈然汗出**（微汗，最佳）

### 247条：阳明中风（最复杂）
**症状**：脉弦浮大+短气+腹满+胁下及心痛+久按之气不通+鼻干不得汗+嗜卧+一身面目悉黄+小便难+潮热+时时哕+耳前后肿

**处理**：
- 耳前后肿 → 刺之稍好
- 病过十日 → 脉弦 → **小柴胡汤**
- 脉但浮+无余证 → **麻黄汤**
- **不尿+腹满+加哕** → **不治（死症）**

## 十、肝病腹水——分消汤与补气建中汤（247条延伸）

### 共同5味基础药
苍术3、白术3、茯苓3、陈皮2、厚朴2

### 分消汤（实证）
基础药 + 香附子2、猪苓2、泽泻2、**枳实1**、大腹皮1、西砂仁1、木香1、灯芯草1把、干姜1、生姜1

- 来源：《万病回春》五脏门
- 适应证：精神还好、有力气、肚子大、脚肿、颜色暗但不虚
- 枳实→枳壳 = 实脾饮

### 补气建中汤（虚证）
基础药（去厚朴） + **人参**、泽泻3、麦冬3、黄芩2

- 来源：《济生方》臌脏门
- 适应证：气色暗黄、元气无、形瘦干干、肚子大

### 加减法
- 冷 → 生附子5钱（棉布袋包），壮里阳（心脏力量）
- 水肿 → 生硫磺（补命门火，三焦水道），6钱-1两
- 绝对**不加甘草**（蓄水）

### 临床提示
- 西医动过（切片/栓塞/开刀）→预后不好
- 针灸配合：期门、章门、肝关、肝1肝2肝3
- 倪师用甘遂+牵牛（等量各三钱，九碗煮三碗），配合：
  - 胸胁苦满 → 小柴胡汤加减
  - 无胃口无胸胁苦满无恶心 → 桂枝汤加减
  - 元气无+打嗝 → 茯苓四逆汤加减

## 十一、骨三阴（247条）
人体三处阴骨：
1. **泥丸宫**（头顶，小时软骨头，2-3岁闭合）
2. **耳后高骨**（耳朵后面大骨头）
3. **膝盖骨**

耳后高骨肿起 → 病入阴实（危险信号），如血癌、多发性骨髓瘤
膝盖骨肿 → 阴实证（骨髓瘤常见于此）

## 十二、蜜煎导法（248条）
### 适应证
- 阳明病自汗出 → 发汗 → 小便自利 → **津液内竭** → 大便硬但不可攻
- 年老气衰，津液回不来，无力排便

### 制作方法
蜜七合，铜器微火煎至凝似饴状 → 手捻作挺（头锐，大如指，长二寸）→ **趁热**做，冷则硬 → 纳谷道中

倪师态度：可用甘油锭替代，只要对病人有利就可用。

## 十三、桂枝汤与麻黄汤在阳明病中的应用（249-250条）

### 249条：有汗
- 阳明病+脉迟+汗出多+微恶寒 → 表未解 → **桂枝汤**
- 只要阳明病有汗→开桂枝汤，因为有炙甘草/生姜/红枣滋阴，无需担心伤津液

### 250条：无汗
- 阳明病+脉浮+**无汗而喘** → **麻黄汤**
- 麻黄入肺，喘/咳嗽都可用

### 核心原则
**有表证先解表，解完表再攻下。** 不能麻黄承气汤一起煮。

## 十四、茵陈蒿汤——阳黄（251条）

### 发黄机制
- 正常：发热+汗出→热越→不会发黄
- 异常：**但头汗出+身无汗+剂颈而还+小便不利+渴饮水浆** → **瘀热在里** → 身必发黄

### 三证模型
| 层次 | 证 | 方中对应 |
|------|-----|------|
| 上 | **热**（头汗，胃热上冲） | 茵陈 |
| 中 | **湿**（湿来补位，梗在中间） | ？ |
| 下 | **实** | ？ |

**茵陈蒿汤三味药**：一去热、一去湿、一去实，药简力专。

⚠️ 原文中此处倪师解释茵陈蒿汤的药理部分不完整（讲三证模型后未对三味药逐一解析）。茵陈蒿汤标准组成：茵陈蒿、栀子、大黄。

### 黄的类型
| 类型 | 特点 | 危险程度 |
|------|------|------|
| 阳黄 | 光鲜亮丽 | 可治（茵陈蒿汤） |
| 阴黄 | 暗、黑 | 危险（肝硬化腹水，分消汤/补气建中汤） |
| 黄汗 | 衣服腋下黄 | 另有处方（非茵陈蒿汤） |

## 十五、关键临床思维总结

### 谵语鉴别速查
| 伴随症状 | 方剂/预后 |
|----------|----------|
| 谵语+直视+喘满 | **死** |
| 谵语+下利 | **死** |
| 谵语+潮热+不能食 | **大承气汤** |
| 谵语+潮热+能食 | **小承气汤** |
| 谵语+大便难+手足汗+潮热 | **大承气汤** |
| 谵语+发潮热+脉滑虚 | **小承气汤** |
| 谵语+热入血室+下血 | 刺**期门** |

### 渴的鉴别
| 症状 | 方剂 |
|------|------|
| 渴欲饮水+口干舌燥 | **白虎加人参汤** |
| 渴欲饮水+小便不利+脉浮发热 | **猪苓汤** |
| 汗多而渴（无小便不利） | **不可**猪苓汤 |

### 不能食的鉴别
| 症状 | 病因 | 方剂 |
|------|------|------|
| 不能食（阳明谵语潮热） | 燥屎堵到大便逆 | 大承气汤 |
| 饥不能食（心中懊憹） | 虚热 | 栀子豉汤 |
| 胃中虚冷+不能食 | 里寒 | 四逆汤 |
