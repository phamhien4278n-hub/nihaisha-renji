# 金匮要略 段4 知识树

## 一、痉病（脑膜炎）三处方总结

### 1.1 三处方对应表
| 类型 | 处方 | 病机 |
|------|------|------|
| 柔痉 | 栝蒌桂枝汤 | 有汗 |
| 刚痉 | 葛根汤 | 无汗；温病（小儿夏天感冒发烧喉咙痛即葛根汤证，津液发散过多所致） |
| 痉病（重症） | 大承气汤 | 大便沼气上冲入脑（西医称无菌性脑膜炎） |

### 1.2 大承气汤证详析
- **症状**：胸满口噤（嘴不能言），卧不着席（角弓反张），脚挛急，必齘齿（牙关紧咬）
- **病机**：大便的沼气（浊气）往回冲到头部
- **方剂**：大黄四两（酒洗）、厚朴半斤（炙去皮）、枳实五枚（炙）、芒硝三合
- **煎法**：先煮枳实厚朴→取五升→去滓→入大黄煮二升→去滓→入芒硝→微火一两沸→分温再服。得下利，余勿服。

### 1.3 大承气汤配伍精义
- **厚朴+枳实**：宽肠→刺激大肠产生津液，使大便排出不腹痛
  - 枳实入心→入小肠，厚朴入大肠（宽肠气）
  - 大黄+芒硝攻下时若无枳朴则大便干燥而腹痛
- **大黄+芒硝**：攻下去实+打碎大便
- **原方比例**：厚朴枳实剂量 > 大黄芒硝（倪师临床常反过来用，取迅捷）
- **芒硝生用**：取其速度快。煮一两沸仅为了完全融化，非煮熟。
- **禁忌**：下利即停药，非承气汤证不可服（会伤津液、阳虚、下利清谷）

### 1.4 大承气汤在伤寒论中的扩展症状
- 高热 + 敛衣摸床（沿床边摸床单走）
- 弃衣而行、登高而歌（纯热症，神昏谵语）
- 阳明症但热不寒

## 二、湿病体系——五层湿邪递进模型

### 核心治则
- **脉沉 = 病在里**（关键判断）
- **湿痹**：脉沉而细（细=被寒束到），小便不利，大便反快 → 但当利其小便（治所有风湿病都在利尿）
- **湿家不可下**：攻下用寒药 → 胃气哕
- **湿家头汗出**：湿阻中焦，肺津液无法下入肾，往上回头→头面汗出
- **湿家身色如熏黄**：脾主湿，湿束脾藏，黄色营养无法入脾→外走皮肤（非黄疸，是营养色外溢）

### 2.1 第一层——湿在皮肤表面
| 项目 | 内容 |
|------|------|
| **症状** | 身烦疼（皮肤肌肉疼），湿疹、荨麻疹、蚊虫咬红疹、手指湿疹皮翻起 |
| **处方** | **麻黄加术汤** |
| **方剂** | 麻黄三两（去节）、桂枝二两（去皮）、甘草一两（炙）、白术四两、杏仁七十个（去皮尖） |
| **煎法** | 先煮麻黄减二升去上沫→内诸药→煮取二升半→温服八合，覆取微汗 |
| **配伍关键** | 白术剂量最重——白术敛湿，把麻黄汤锁定在去湿上，使发汗变微汗（不加白术则大汗） |
| **扩展用途** | 肾脏病水肿、缺氧急救（山洞沼气昏迷）、鼻病（加辛夷菖蒲） |

### 2.2 第二层——湿在肌肉
| 项目 | 内容 |
|------|------|
| **症状** | 一身尽疼（肌肉痛），发热，日晡所剧（黄昏加重=阳明时），不痒的疣、头皮多（皮脂漏性皮肤炎）、手脚皮厚 |
| **处方** | **麻杏薏甘汤**（麻黄杏仁薏苡甘草汤） |
| **方剂** | 麻黄半两、甘草一两（炙）、薏苡仁半两、杏仁十个 |
| **病机** | 汗出当风，或久伤取冷（常喝冰的），湿阻体表→阳气不透达→黄昏阳明时最热 |
| **配伍关键** | 麻黄+杏仁可防麻黄发汗太过（杏仁润肺生津）；麻黄+薏仁像桂枝+白芍一样搭配；薏仁可加重；粉剂效果也好 |
| **与麻加术区别** | 麻加术治痒的皮肤湿疹；麻杏薏甘治不痒的肌肉层疣/头皮/皮厚 |

### 2.3 第三层——湿在血脉周围
| 项目 | 内容 |
|------|------|
| **症状** | 脉浮身重，汗出恶风（阳不入阴→无故流汗），肥胖，中年妇女运动少发福，走路喘、身重 |
| **处方** | **防己黄芪汤** |
| **方剂** | 防己一两、甘草半两（炒）、白术七钱半、黄芪一两一分（去芦）+ 生姜四片、大枣一枚 |
| **病机** | 湿包血管周围→两套循环系统负担→身重；湿阻阴阳之间→阳不入阴→汗出不止 |
| **加减** | 喘者加麻黄；胃中不和（肠子）加芍药（治腹痛）；气上冲加桂枝（降逆气）；下有陈寒加细辛 |
| **药后反应** | 如虫行皮中（皮肤下蚂蚁爬），腰以下如冰→盖被温令微汗（寒湿从肚脐以下发散） |
| **防己特性** | 入血脉神经，长像车轮，刚好在血脉周围 |
| **黄芪 vs 炮附子** | 黄芪行气带动水液（不收敛汗）；炮附子固表收汗（会让胖子更胖）——此处怕流失津液但不可收汗，故用黄芪不用附子 |
| **炙甘草** | 入心，强固心脏——防己+白术大量去湿可能动心 |

### 2.4 第四层——湿在肌肉+中焦（桂枝附子汤 / 白术附子汤）
| 项目 | 桂枝附子汤 | 白术附子汤 |
|------|-----------|-----------|
| **症状** | 伤寒八九日，风湿相搏，身体疼烦，不能自转侧，不呕不渴，脉浮虚而涩 | 同上+大便坚，小便自利 |
| **病位** | 肌肉（风+寒=到处走痛，西医称fibromyalgia肌纤维痛） | 中焦（湿在中焦压迫肠子→大便难排但非便秘） |
| **方剂** | 桂枝四两、附子三枚（炮）、生姜三两、甘草二两（炙）、大枣十二枚 | 白术二两、附子一枚半（炮）、甘草一两（炙）、生姜一两半、大枣六枚 |
| **方理** | 桂枝汤去白芍加炮附子——去白芍因脉虚涩，白芍酸收更虚；桂枝=祛风（解肌），附子=祛寒 | 去桂枝加白术——白术去中焦湿，炮附子逐水气 |
| **配伍精义** | 风到处走→桂枝；寒→炮附子。汗出受寒（运动后进冷气房）是典型病因 | **术附=排脓汤之一**（另一为当归赤豆散）——能把深层脓疡提上来。糖尿病脚骨头发炎不收口→术附生皮 |
| **药后反应** | — | 一服觉身痹，三服其人如冒状（头昏）=中病，术附并走皮中逐水气 |
| **禁忌** | — | 湿家不可下！若误将大便难排当便秘攻下→虚+寒+湿三证 |

**桂枝附子汤 vs 桂枝汤加附子**：
- 桂枝汤加附子 = 发汗太过后的弥补措施（收表止汗，桂枝汤生津）
- 桂枝附子汤 = 去白芍加附子，专门治风湿在肌肉（不是弥补措施）

### 2.5 第五层——湿在骨节（风寒湿三邪齐聚）
| 项目 | 内容 |
|------|------|
| **症状** | 骨节疼烦，掣痛不得屈伸，近之则痛剧，汗出短气，小便不利，恶风不欲去衣，或身微肿 |
| **病名** | **痛风**（关节红肿热痛——阳不入阴，非真热证！） |
| **处方** | **甘草附子汤** |
| **方剂** | 甘草二两（炙）、附子二枚（炮去皮）、白术二两、桂枝四两 |
| **配伍关键** | **桂枝重用（四两）**：行阳，把附子带到四肢末梢关节（附子本身不入关节/不走末梢）；白术去关节内湿；炙甘草顾护心脉 |
| **禁忌** | 绝不可用寒凉药（黄芩/黄连/石膏）→阳不入阴的真寒证，用凉药更痛 |

## 三、新学到的东西（核心收获）

1. **湿病五层递进模型**：皮肤（麻加术）→肌肉（麻杏薏甘）→血脉周围（防己黄芪）→肌肉中焦（桂附/术附汤）→骨节（甘草附子汤）
2. **大承气汤也可治脑膜炎**（痉病第三证）：大便沼气上冲入脑→角弓反张、齘齿——不是所有脑膜炎都是细菌性
3. **术附 = 排脓汤**：深层脓疡（骨头发炎不收口）→用术附升提排脓
4. **痛风是阳不入阴**：关节红肿热痛是风寒湿阻在中间，阴在内阳在外进不去——绝不可清热
5. **桂枝把附子带到四肢末梢**：附子不入关节末梢，需桂枝行阳带到
6. **麻黄+杏仁**防发汗太过（杏仁润肺生津）；**麻黄+薏仁**搭配调和表湿；**白术+麻黄汤**使大汗变微汗
7. **问小便判寒热**：小便利且清白=寒湿；小便暗黄=热秘
8. **防己黄芪汤用黄芪不用炮附子**：此处汗出是阳不入阴所致，用黄芪行气带动水液，不可用附子收汗（越收越胖）
