# 伤寒论 第7段 知识树（条文106-128）

## 一、小柴胡汤核心体系

### 1. 小柴胡汤证的主证（条文106）
- **四大主证**：往来寒热、胸胁苦满、默默不欲饮食、心烦喜呕
- **副证**（可有可无）：胁下痞鞭、心下悸、小便不利、不渴
- **治则**：但见一证便是，不必悉具（条文113）

### 2. 小柴胡汤的生理基础
- 邪入少阳→三焦淋巴系统→水饮积聚→压迫胃→恶心
- 病位：介于太阳和阳明之间（半表半里）
- "血弱气尽，腠理开，邪气因入，与正气相搏，结于胁下"

### 3. 小柴胡汤的方义
- **柴胡**：疏肝（木要疏，如果树枝太茂盛树会死）
- **半夏**：排水（树根才能在水里，树上水太多树会被淹死）
- **生姜**：滋生好的津液
- **炙甘草**：生津液
- **郁金**：疏郁气
- **人参/党参**：补气
- **黄芩**：清热

### 4. 小柴胡汤的临床应用范围（超广谱）
- **肝脏病**：每天1-3点准时醒→治肝（小柴胡汤做基础方加减）
- **黄疸**：身黄加去黄药（茵陈等，金匮会介绍）
- **淋巴癌**：全身淋巴结肿块→小柴胡汤+生硫磺（硫磺去实）
- **脑积水**：小柴胡汤加重生半夏（排脑部积水）
- **乳癌**：小柴胡汤加减（月经来加桂枝导奶水入子宫）
- **肾脏病**：往来寒热、胸胁苦满、恶心呕吐→一样用小柴胡汤
- **月经痛**：月经来时感冒→一律小柴胡汤（唯一处方）
- **甲状腺肿大/大脖子**：小柴胡汤加减
- **胰脏癌**：同样辨证
- **总结**：只要问题在脏腑之间，胸胁苦满→小柴胡汤

### 5. 小柴胡汤证的转归
- **服柴胡汤已，渴者→属阳明**（条文107）：好转现象（阳明无死症）
- 少阳→阳明=好；少阳→厥阴=完蛋

## 二、柴胡汤家族进阶

### 大柴胡汤（条文116）
- **使用时机**：少阳+阳明兼证（往来寒热+胸胁苦满+恶心+**便秘**）
- **与大柴胡汤区别**：
  - 太阳+少阳→小柴胡汤
  - 少阳+阳明→大柴胡汤
- **关键症状**：心下急、郁郁微烦、呕不止
- **临床应用**：大肠癌（未开刀者≈阳明少阳证，大柴胡汤效果好）

### 柴胡加芒硝汤（条文117）
- **使用时机**：柴胡证+阳明燥热（日晡潮热）+大便坚
- **芒硝的作用**：攻坚（坚硬大便包括大肠癌）
- **与大黄区别**：大黄去实，芒硝攻坚
- **方剂**：小柴胡汤+芒硝二两
- **煮法**：纳芒硝→更煮微沸（芒硝生用）

### 柴胡加龙骨牡蛎汤（条文118-120）
- **病机**：太阳病误下→表邪未解反入三焦→小便不利+惊烦+一身尽重不可转侧
- **组成**：柴胡四两（重用）、半夏、大枣、生姜、人参、龙骨、铅丹、桂枝、茯苓、牡蛎、大黄
- **龙骨牡蛎配伍法则**（重要！）：
  - 精失（遗精、梦遗、早泄）→重用龙骨
  - 津失（夜尿、小便失禁）→重用牡蛎
  - 仅惊烦焦虑→等量
- **适用人群**：素虚之人（身体虚弱者）
- **铅丹**（青铅）：镇心定神，今用常山替代（但仅蜀漆时用常山）

## 三、建中汤体系

### 小建中汤（条文111-112, 115）
- **来源**：桂枝汤（芍药加倍+饴糖）
- **病机**：里虚寒→腹痛、心悸而烦
- **脉象**：阳脉涩（阳气不够）、阴脉弦（里寒）
- **辨证要点**：
  - 小孩子不吃饭+山根青黑→小建中汤
  - 虚寒腹痛→先给小建中汤，不好再给小柴胡汤
- **禁忌**：呕家不可用建中（太甜，甜会加重呕吐）
  - 有恶心→进入少阳→用小柴胡汤
- **体质应用**：
  - 瘦人想胖→小建中汤
  - 胖人不想胖→桂枝汤
  - 预防保健→小建中汤

## 四、三阳并病原则（条文110）
- **身热恶风颈项强**（太阳）+ **胁下满**（少阳）+ **手足温而渴**（阳明）→三阳并病
- **治法**：和解→小柴胡汤
- **原理**：少阳居中间，如同担法扎中间

## 五、火劫误治与救逆

### 火劫的危害（条文123-124, 125-128）
- **核心禁忌**：伤寒/中风不能用火烤、火熏、火瓦熨背发汗
- **后果层级**：
  1. 火热入胃→胃中水竭→烦躁谵语（大便干）
  2. 两阳相熏灼→身发黄
  3. 阳盛→流鼻血（流出来烧退则已）
  4. 阴虚→小便难
  5. 阴阳俱虚→身体枯燥、但头汗出（剂颈而还）
  6. 严重→哕（呃逆不断）→危险
  7. 最重→手足躁扰、捻衣摸床→大承气汤可救（小便利者）

### 桂枝去芍药加蜀漆龙骨牡蛎救逆汤（条文125）
- **主治**：火劫后亡阳→惊狂、起卧不安
- **方义**：
  - 去芍药→去掉收敛，让阳药回头的力量更强
  - 桂枝→走肌肉达皮肤表面
  - 蜀漆（今用常山）→去痰（受惊吓痰涎壅塞）
  - 龙骨牡蛎→潜阳收敛
- **拓展应用**（倪师临床）：不仅仅火灾，安眠药/抗忧郁药导致的焦虑、失眠、坐卧不安均可使用
- **蜀漆安全**：先煮蜀漆（因为有毒），户外烧（烟有毒）

### 烫伤外敷
- 牡蛎打粉+麻油外敷（牡蛎排水）
- 硫磺外用性寒凉（外用寒凉，内服燥热）

## 六、其他条文精华

### 肝乘脾/肝乘肺（条文121-122）
- **肝乘脾（纵）**：腹满、谵语、寸口脉浮紧→木克土→刺期门
- **肝乘肺（横）**：发热、啬啬恶寒、大渴欲饮水、腹满→木侮金→刺期门
- 倪师评：张仲景针灸不强，刺章门（脏会+脾募）比期门好

### 误下后的逆证（条文108-109）
- 脉迟浮弱（阴居阳位）+医攻下→胁下满痛+面目身黄+小便难→已变逆证
- 此时开柴胡汤→下重（里急后重）、食谷则哕→为时已晚
- 救逆用茯苓四逆汤（生附子+干姜+炙甘草+人参+茯苓）

### 津液回头的判断
- **小便利**→津液回头→病欲解
- **口渴**→好现象（病人愿意喝水）
- **最怕**：不渴+哕（打嗝）→胃气已无→危险

### 大便坚+小便判断
- 大便鞭+小便数→还好
- 大便鞭+小便不数反不多→危险（津液枯竭）

## 七、关键鉴别诊断

| 证型 | 方剂 | 关键区别 |
|------|------|---------|
| 单纯少阳 | 小柴胡汤 | 往来寒热+胸胁苦满+恶心，无便秘 |
| 太阳+少阳 | 小柴胡汤 | 太阳表证+少阳证 |
| 少阳+阳明（便秘）| 大柴胡汤 | 少阳证+便秘+烦躁 |
| 少阳+阳明（坚便+潮热）| 柴胡加芒硝汤 | 少阳证+日晡潮热+大便坚 |
| 少阳+误下+惊烦 | 柴胡加龙骨牡蛎汤 | 小便不利+惊烦+身重 |
| 里虚寒腹痛 | 小建中汤 | 腹痛+山根青黑+不渴+无恶心 |
| 火劫惊狂 | 桂枝去芍药加蜀漆龙骨牡蛎救逆汤 | 起卧不安+焦虑 |
| 误下阴实逆证 | 茯苓四逆汤 | 胁下满+身黄+不渴+哕 |

## 八、倪师临床心法（本段特别强调）
1. **治肝=疏木**：郁金疏郁+半夏排水（树根在水里OK，树上不能积水）
2. **少阳证治好了，内脏不会生病**——小柴胡汤是预防肝病最好处方
3. **大肠癌未开刀**≈阳明少阳证→大柴胡汤/柴胡加芒硝汤→效果好
4. **大肠癌开刀后**→癌细胞跑肝→变厥阴证→难治
5. **补药定义**：对证即补药——硝黄对便秘者就是补药
6. **病人口渴=好现象**，不渴=危险信号
7. **手足温热=胃气足**：手掌脚掌热、手背脚背冷=正常
8. **黄昏发热=阳明热**（阳明热都在黄昏发作）
9. **捻衣摸床+小便利**=大承气汤可救（阳明燥屎）
