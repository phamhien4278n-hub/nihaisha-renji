# 金匮要略 · 第14段 · 腹满寒疝宿食病脉证治（下）

## 核心概念

### 1. 便秘三层次辨证——闭而不痛 vs 痛而闭
- **小承气汤**：闭而不痛（便秘但肚子不痛），堵在小肠→大肠空→有屁
- **厚朴三物汤**：痛而闭（大便不出+肚子痛），重用厚朴枳实，大黄量少
- **大承气汤**：堵在大肠→连屁都不放一个，腹满不减需再攻

### 2. 大柴胡汤——少阳阳明合病
- 病在少阳与阳明之间，此段无死症
- 辨证关键：脉弦+恶心+便秘+小便黄+胸胁苦满+往来寒热（必有兼症）
- 方剂结构：小柴胡汤去人参+芍药+枳实+大黄
- 倪师临床：治大肠癌（未开刀未化疗者），大柴胡加芒硝治燥屎

### 3. 腹痛寒热深浅三层
- **少腹寒痛**：小建中汤（浅）
- **中焦寒痛**：大建中汤（中）——上冲皮起，出见有头足
- **胸寒痛**：乌头赤石脂丸（深）——有痰饮

### 4. 寒实便秘辨证体系
- **热实**（承气汤证）：小便黄+痛+烦躁+脉洪大+舌苔黄干→干黑
- **寒实**（大黄附子细辛汤）：小便清白+不痛+不为所苦+脉细+舌苔白→湿黑
- **三焦寒实层次**：
  - 上焦寒实：三物小白散
  - 中焦寒实：甘草干姜汤
  - 下焦寒实：大黄附子细辛汤

### 5. 寒疝——上中下全寒
- 定义：上焦、中焦、下焦全部是寒
- 主方：大乌头煎（经方最热最凶的药）
- 标志症状：白津出（阴阳分隔，阳津无法入阴被迫外走）
- 与癌症阴实吐白津的区别：寒疝只是寒没有实，阴实是肿瘤堵到+阴阳分隔
- 大青龙/小青龙：表寒；肺热（大青龙）vs 肺寒（小青龙）

### 6. 四肢厥逆两型鉴别
- **四逆汤证**：四肢冰冷，无水肿
- **赤丸证**：四肢冰冷+全身水肿（寒气厥逆）

## 关键治则

### 厚朴三物汤
- **组成**：厚朴八两、大黄四两、枳实五枚
- **煮法**：先煮厚朴枳实，后下大黄
- **剂量规律**：厚朴4钱、大黄2钱、枳实3钱（大黄量最少）
- **功效**：厚朴宽肠，枳实去心包湿→津液回小肠（心与小肠表里），大黄去实（不分寒热）

### 大柴胡汤
- **组成**：柴胡半斤、黄芩三两、芍药三两、半夏半升、枳实四枚、大黄二两、大枣十二枚、生姜五两
- **芍药指征**：少腹痛（肠痛/月经痛/跌打损伤）
- **加芒硝指征**：屁都不放+恶心+往来寒热→燥屎硬如石

### 大建中汤
- **组成**：蜀椒二合（炒去汗）、干姜一两、人参二两、胶饴一升
- **煮法**：先煮三味去渣→加麦芽糖再煮
- **主症**：呕不能饮食+腹中满+上冲皮起（肠蠕动可见）+出见有头足+上下痛不可触近
- **蜀椒功效**：杀虫+去痰饮+去寒湿+辛辣温中
- **变方**：蜀椒换半夏→人参半夏干姜汤（治孕妇呕吐）

### 大黄附子细辛汤
- **组成**：大黄三两、附子三枚（炮）、细辛二两
- **功效**：温下寒实——细辛温小肠+利水+壮胆气
- **剂量**：倪师习惯 炮附子三钱、细辛二钱、大黄三钱

### 赤丸
- **组成**：茯苓四两、乌头二两（炮）、半夏四两（洗）、细辛一两
- **制法**：丸剂，朱砂外包（极少量，一斤药丸加一颗药丸量的朱砂摇匀）
- **服法**：酒饮下三丸，日再夜一，不知稍增
- **主治**：寒气厥逆=全身水肿+四肢冰冷+脉微细+吐气冷

### 大乌头煎
- **组成**：乌头大者五枚（熬去皮）
- **煮法**：水煮乌头→去渣→加蜜再煮
- **剂量**：壮人七合/弱人五合，不可一日再服（极毒）
- **安全措施**：倪师在诊所预熬乌头蜜，一平匙混入汤药
- **适应证**：寒疝绕脐痛+白津出+手足厥冷+脉沉紧

### 当归生姜羊肉汤
- **组成**：当归三两、生姜五两、羊肉一斤
- **加减**：寒多加生姜至一斤；痛多呕加橘皮二两、白术一两
- **用法**：大乌头煎去寒后补血用
- **羊肉**：肉类顶级，补血又去寒，高原必备

## 病机推理链

### 便秘痛否→承气汤鉴别链
```
便秘→肚子痛? 
  ├─ 不痛 → 有屁? → 有屁=小承气汤（堵小肠）→ 没屁=大承气汤（堵大肠）
  └─ 痛 → 厚朴三物汤（重用厚朴枳实宽肠去湿+少量大黄去实）
```

### 寒热辨证→便秘攻下链
```
便秘→小便颜色?
  ├─ 小便黄+烦躁+痛+脉洪大+舌苔黄干 = 热实 → 承气汤
  └─ 小便清白+不痛+不为所苦+脉细+舌苔白/湿黑 = 寒实 → 大黄附子细辛汤
```

### 腹痛寒热层次链
```
腹痛部位：
  少腹（脐下）→ 小建中汤（浅寒）
  中焦（胃脘）→ 大建中汤（蜀椒干姜人参胶饴）
  胸口 → 乌头赤石脂丸（有痰饮）
  上中下全寒 → 大乌头煎（寒疝，白津出）
```

### 四肢厥逆鉴别链
```
四肢冰冷→水肿?
  ├─ 无水肿 → 四逆汤
  └─ 全身水肿+吐气冷 → 赤丸（寒气厥逆）
```

### 白津出病因鉴别
```
吐白津（白色口水/津液）：
  ├─ 癌症阴实 → 阳不入阴（肿瘤阻隔）→ 燥热+持续恶化 → 阴阳分隔时间越来越短
  └─ 寒疝 → 里寒极盛（无实）→ 阳津无法入阴被迫外走 → 脉弦紧+无燥热
```

## 新学到的东西

1. **便秘+痛 vs 便秘+不痛是完全不同的方**：厚朴三物汤 vs 小承气汤，关键在厚朴枳实比例
2. **大柴胡汤治大肠癌**（倪师临床验证）：前提是未开刀未化疗，还在少阳阳明之间
3. **"出见有头足"就是肠蠕动可见**——大建中汤的标志性症状，一剂即止
4. **寒实便秘三焦分层**：上（三物小白散）→中（甘草干姜汤）→下（大黄附子细辛汤）
5. **白津出的两种病因**：癌症阴实（有实阻隔+燥热）vs 寒疝（纯寒无实+无燥热）
6. **大乌头煎的安全用法**：水煮去渣→加蜜再煮→平匙混入，不可一日再服
7. **枳实的特殊功效**：去心包膜湿→津液回小肠（心与小肠表里关系）
8. **小青龙汤治喘+失眠**：倪师案例——肺寒喘+失眠，舌白不渴，小青龙含麻黄仍能安眠
9. **当归生姜羊肉汤定位**：大乌头煎后的补血善后方，非急救方
10. **赤丸用朱砂外包技巧**：一斤药丸只需一颗药丸量的朱砂，摇匀即全红
