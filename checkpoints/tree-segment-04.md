## 倪师案例（本段涉及的倪案）

> nishizhu 中本段药物无独立倪案条目，但段内有倪师临床讲解：
- **鼻窦炎通用方**：辛夷+苍术+菖蒲，倪师说"光这三味药，鼻窦炎就好了，不管寒热"
- **人参滥用类比**：倪师把人参滥用类比抗生素滥用——平时天天吃，真正需要时就无效了
- **十枣汤不加甘草**：攻水急证，甘草缓性会拖延排水利水
- **炙甘草汤强心**：脉结代心动悸，重用炙甘草，原理是蜂蜜炒过焦糖味入心火

## 倪师辨证思路

### 倪辨（来自 nishizhu）
- **生甘草 vs 炙甘草**：生甘草解毒泻火缓和；炙甘草（蜜制）强心补脾胃。心动悸脉结代用炙甘草汤重用炙甘草
- **生地 vs 熟地**（干地黄条目）：生地（鲜地黄）性寒凉血滋阴，适合女子补血凉血；熟地（九蒸九晒）性温补肾填精，适合男子

### 倪诀（来自 nishizhu）
- **生地凉血（女子用），熟地补肾精（男子用）**

### 段内倪师辨证思路
- **辛夷+苍术+菖蒲配伍原理**：三味药直取鼻窦堵塞物，寒热不分；寒热从肺治（肺开窍于鼻），寒加干姜热加杏仁麦门冬
- **苦味药三功能**：能泻、能燥、能坚（菊花处讲述）
- **甘味药三功能**：能补、能和、能缓（甘草处讲述）
- **中药中和特性**：人参为例——同一味药治主证，对下痢病人和便秘病人的副证都能自动调整，多余的排掉不足的补上
- **攻水去甘草**：急证排水时甘草缓性会阻碍攻下（十枣汤不用甘草的反面教训）
- **补能恋邪**：人参禁忌——病在发散时用补药，病就发不出来了
- **平时不吃人参**：类比抗生素——日常服用后真正需要时药力"石沉大海"
- **温病vs伤寒立场**：以经方（伤寒论）为基础，温病药做附件补充——经方不足时用温病药补，不能舍经方只用温病（治不了重症急症救逆）
- **菊花不宜陈年**：与大多数中药（陈皮、艾叶等）越陈越好相反
- **苦平药性**：菊花苦平→苦味泻/燥/坚，平性不偏
- **甘草急迫缓痛**：筋抽痛时「急食甘以缓之」，甘草甘味缓急止痛

## 跨课连接（倪联，来自 nishizhu）

- **炙甘草汤**：炙甘草 + 人参 + 生地 + 阿胶 + 麦冬 + 麻仁 + 桂枝 + 生姜 + 大枣 → 脉结代，心动悸（伤寒论）
- **桔梗甘草汤**：桔梗 + 甘草 → 肺痈吐脓，肺癌咳血（金匮要略）
- **小柴胡汤**：含人参 → 少阳证，半表半里（伤寒论）
- **理中汤**：含人参 → 脾胃虚寒（伤寒论）
