# 金匮要略 段20 知识树

## 段20：小青龙汤后变证 + 消渴小便不利淋病脉证治第十三

---

### 一、核心概念

#### 1. 小青龙汤后的连锁变证（痰饮咳嗽病篇续）

**复习：前方已用苓甘五味姜辛汤（段19）**——干姜+细辛是热药。

| 变证次序 | 症状 | 病机 | 加味 | 处方名 |
|----------|------|------|------|--------|
| 第一变 | 咳满止，冲气又来 | 热药下去本应口渴→不渴反止→内有支饮 | 加半夏 | 苓甘五味加姜辛半夏汤 |
| 第二变 | 水去呕止，形肿（浮肿） | 本虚之人，肺津液未复 | 加杏仁（不可用麻黄！） | 苓甘五味加姜辛半夏杏仁汤 |
| 第三变 | 面热如醉（脸红） | 胃热上冲熏其面 | 加大黄（大黄去实） | 苓甘五味加姜辛夏杏大黄汤 |

#### 2. 支饮的病理特征
- 支饮从胸膈、肋间上冲 → 恶心、呕吐
- 胸胁苦满 = 水停在淋巴丰富区域（淋巴结大处易停水）
- 小柴胡汤用半夏正是因为胸胁苦满

#### 3. 血虚不可用麻黄——核心禁忌

**血的阴阳分法**：
- 血的阴 = 血球（红细胞、血小板等）
- 血的阳 = 血管中的水

**麻黄伤血的机制**：麻黄发阳→把血管里的水发散掉→血虚。不是直接伤血球，是伤血中之水。

**杏仁+麻黄并用原理**：杏仁入肺，润肺补津液，可补足血中的水，防止麻黄伤血。

**贫血/血虚之人→不可用发表药/麻黄**。

#### 4. 饮家诊断法
- **先渴后呕** = 水停心下 = 饮家 → 小半夏加茯苓汤
- 喝水进去又吐掉 → 水停在中间
- 水是疾病最佳温床（癌症、病毒、寄生虫都需要水）→ 必须排饮
- 练功夫（扭转身体）目的之一 = 排饮

### 二、消渴（糖尿病）——消渴小便不利淋病脉证治第十三

#### 1. 消渴总论

- 中医"消渴"范围 > 西医"糖尿病"
- **三阳结，名消渴**（《黄帝内经》）：胃太热 + 大肠太热 → 中焦（小肠/十二指肠/肝胆）也被热起来
- **糖水杯比喻**：血糖高本质 = 血中水不够（不是糖太多），水加进去甜度就降了
- 西医降血糖药 = 把糖压到脚底（物质不灭），手指尖验血糖是假的（脚趾头验就穿帮）
- **判断药对不对**：吃药后脚越热 → 对；脚越冰/越硬/越冷 → 错

#### 2. 糖尿病三大成因（倪师观点）

| 成因 | 机制 |
|------|------|
| ① 吃宵夜（吃完就睡） | 食物停在胃里闷着→胃蠕动减缓→血糖释放→胃热→越吃越多。**不是基因遗传，是生活习惯遗传** |
| ② 甜食+精制糖+运动少 | 蔗糖OK，精炼人工糖有害；吃到嘴里甜的，到身体里变酸的；开车代步→脚趾退化弯起 |
| ③ 西药/化学物释放水分 | 降压药=利尿剂→排水→血中水量↓→糖浓度↑。味精、西药酵素都释放水 |

#### 3. 第一型 vs 第二型糖尿病

| | 第一型 | 第二型 |
|------|--------|--------|
| 倪师观点 | **不存在**，是疫苗/西药制造出来的（胰脏功能被毁） | 真正的消渴 |
| 治疗 | 极难（胰脏无功能） | 初诊一剂就好；吃西药15-20年+则难治 |
| 底线 | 无法恢复 | 即使20年+，用中药至少可保脚不截肢 |

#### 4. 消渴三分法

| 类型 | 症状 | 病机 | 处方 |
|------|------|------|------|
| **上消** | 渴饮千杯不止渴 | 肺津液不够 | **白虎加人参汤**（石膏+知母+甘草+粳米+人参/党参） |
| **中消** | 饿——吃一百个汉堡还饿，手脚发抖冒冷汗（低血糖症状） | 三阳结+便秘，胃蠕动太快 | **白虎汤+承气**（石膏+知母+甘草+粳米+大黄+枳实+厚朴） |
| **下消** | 喝一斗小便一斗，不举（性功能丧失） | 肾气不足 | **肾气丸**（六味地黄丸+桂附） |

**关键洞察**：血糖太高用白虎汤，血糖太低（饿得发抖冒冷汗）也用白虎汤——胃蠕动太快是共同病机。剂量：石膏一两起，美国人五两起。

#### 5. 厥阴消渴
- 饥而不欲食，食即吐，下之利不肯止
- 看起来像宿食，攻下后利不止→不是宿食，是厥阴病
- 真正宿食：大黄甘草汤下去→舒服→能吃→不吐

#### 6. 口渴鉴别诊断（五种情况）

| 证型 | 症状特征 | 处方 |
|------|----------|------|
| 上消 | 渴饮千杯不止，小便多 | 白虎加人参汤 |
| 五苓散证 | **小便不利** + 渴 | 五苓散 |
| 水逆 | 渴欲饮水，水入则吐（成人运动灌水/婴儿吐奶） | 五苓散 |
| 文蛤散证 | 渴不止，**小便正常** | 文蛤散（蛤蜊壳煮汤） |
| 肾气丸证 | 喝一斗尿一斗，不举 | 肾气丸 |

### 三、淋病

#### 1. 淋的定义与分类
- 核心症状：小便如粟状，小腹弦急，痛引脐中
- 中医淋的范围 > 西医淋病（性病）

| 分类 | 特征 |
|------|------|
| **膏淋** | 小便浓稠白色粘粘的 |
| **砂淋** | 小便有石头/砂 |
| **血淋** | 小便带血 |
| 类似淋病 | 小便有白白一片片（发炎） |

#### 2. 结石定位诊断
- 痛引脐中（肚脐后面）→ 石头在**膀胱**
- 小腹弦急（剧烈痛）→ 石头在**尿道/输尿管**

#### 3. 针灸急救结石剧痛
- **太溪穴**（内踝）找压痛点下针 → 当场痛止 → 石头开始移动
- 先针灸止痛→再煮药排石
- 经方大师（如张仲景）不会针灸是短板，好中医必须针药并强

### 四、脉诊

#### 寸口脉与趺阳脉
| 脉象 | 病机 |
|------|------|
| 寸口浮而迟 | 浮=卫气不足（虚），迟=营气竭（劳） |
| 趺阳浮而数 | 浮=气，数=消谷→大便坚+小便数→消渴 |
| 趺阳脉数 | 胃中有热→消谷引饮→大便必坚+小便则数 |

**"胃中"=胃+小肠**（读金匮/伤寒时看到"胃中"常要想到小肠）

### 六、新学到的东西（个人笔记）

1. **血的阴阳模型**：阴=血球，阳=水。麻黄伤的是血中之水，不是血球本身
2. **杏仁+麻黄配对的深层原理**：杏仁润肺补水入血，抵消麻黄发散血中水的作用
3. **先渴后呕=饮家**——简洁而精准的诊断法
4. **血糖高低都可用白虎汤**——病机是胃蠕动太快，不是血糖数值
5. **消渴治则核心**：补血中之水（食物中的水才能入血管，光喝水只入三焦）
6. **太溪穴压痛点**下针是结石止痛第一选择
7. **生活习惯遗传 ≠ 基因遗传**——糖尿病的家族性可能是宵夜习惯的家族性
