# 段36：金匮要略 —— 心脏病全案 + 肝癌案 + 乳癌案

## 一、核心概念（本段新学/强化的诊断与治则框架）

### 1. 心脏病诊断两要件
- 耳心点压痛 → 动脉血管堵塞/冠状动脉硬化（胸腔内血脉问题）
- 第五椎（T5）压痛 → 结构问题（瓣膜闭锁不全/心率不整） OR 血管堵塞
- **耳压痛+五椎同时压痛 = 心气心血同时受阻，须气血双治**

### 2. 心脏-小肠表里关系（大便的预后价值）
- 心与小肠搏动速度相同。大便正常 = 即使血管堵塞，心脏搏动节奏未失
- 大便正常是心脏病患者的"唯一有利因素"
- 反之：大便不规则 + 睡眠差 + 心口刺痛 + 手脚麻 = **随时走人**

### 3. 瓣膜闭锁不全的生理机制（倪师原创）
- 心脏血液回流 → 肺脏短暂失压 → 坐飞机后肩膀酸痛 → 瓣膜损伤标志
- 瓣膜问题 = 结构问题 → T5压痛为主，耳穴反而不痛
- 血管堵塞 → 耳穴压痛为主

### 4. 心脏病恢复的五大指标（停药标准）
1. **睡眠品质恢复**（最重要）
2. **正常出汗**（运动出汗/不盗汗/不无汗）—— 心主五液，心主汗
3. **手脚温热**、无麻痛
4. **大便正常**
5. **胸口无闷痛**

### 5. 心脏病病因：心脏本身不受病
- 病因全在外来：① 情志（生气/惊吓）② 饮食（油炸/甜食/咖啡/可乐/维他命/酸食）
- 规律生活+易筋经+逛沃尔玛 → 心脏不会有病

## 二、病机推理链

### 心脏病治疗推理链
```
诊断 → 耳心点+T5压痛 → 确定心气+心血均堵
     ↓
方药 → 川芎+丹皮+桃仁（血管三胞胎，固血管防破裂）
     + 红花（通血管，《金匮》红蓝花酒单味治心痛）
     + 栝蒌实+薤白+枳实+桂枝（胸痹主力，通心气）
     + 地龙（活血化瘀重武器，钻入组织间隙清血）
     + 当归（补血，攻血必补血否则人昏倒）
     + 白芍+炮附子（白芍收血回心，炮附加热融化血块）
     + 细辛+炮附子（下焦火力不足/小便无力）
     + 黄连（补心）+ 黄芩
     ↓
追踪 → 睡眠/汗/手足温/大便/胸痛 五指标
     ↓
收尾 → 五指标全恢复才停药；四肢末梢最敏感，先麻先冷先恢复
```

### 呃气被误判为心脏病的纠正链
```
初诊：呃气 + 心脏病 → 误判为心气问题
六诊后心脏指标已恢复，呃气仍在 → 重新审视
发现：呃气来自胃酸 → 改旋复花+代赭石+半夏+枳实+黄连（小陷胸汤变法）
关键教训：即使病人已知有心脏病，胸痛也可能是胃酸烧灼食道（小陷胸汤证）
```

## 三、关键治则（本段新增/强化的用药法则）

### 1. 血管三胞胎（必处）
**川芎 + 丹皮 + 桃仁** —— 任何血管问题（动脉堵塞/血脉剥离/易出血/伤口不收）必用的固血管基础

### 2. 胸痹主力处方
**栝蒌实 + 薤白 + 枳实 + 桂枝** —— 胸闷/心气郁结
- 栝蒌实：色白入心包，形如心脏
- 薤白：色白+辛臭（大蒜味）→ 入肺 → 治心先治肺
- 枳实：色深形如小橘，入心脏
- 桂枝：加强心脏喷射力量

### 3. 攻血必补血
单用活血化瘀药（红花/地龙）→ 病人当场昏倒（血虚）
**必须加当归等补血药**

### 4. 红花 vs 地龙的分工
- **红花（川红花）**：清血管内的血（脑溢血/血管内瘀血）—— 头（天）
- **地龙（蚯蚓）**：钻入组织间隙清除溢出到组织间的血 —— 脚（地）
- 动物药"很凶"，活血化瘀常规药去不掉时才用

### 5. 白芍 + 炮附子 —— 脚上瘀血的标准配伍
- 胸闷原则上不用白芍（酸收加重闷）
- 但脚痛/脚麻必须用白芍收血回心
- **白芍收血时必配炮附子**——炮附加热融化被收回去的血块，防止血块进入主动脉堵塞
- 脚上化瘀不是靠牛膝/地龙猛攻，而是**热药融化**（热胜寒）

### 6. 治心必治肺 —— 薤白的选择逻辑
- 薤白：色白（入肺）+ 辛臭厚味（入心）= 心肺同调
- 为何不选石膏：石膏色白但淡味，只入肺不入心，且寒凉伤胃（肺母=脾胃）

### 7. 心痛彻背、背痛彻心 = 痰饮在纵膈腔
- 痰不会进内脏，只在纵膈腔上移动 → 像剑从胸前穿到胸后
- 主力方：栝蒌薤白桂枝汤（气滞血瘀型）/ 乌头赤石脂丸（痰饮型，倪师改为乌梅赤石脂丸）

### 8. 针灸心脏病选穴
- 公孙 + 内关（八脉交会）
- 巨阙（心募）+ 关元（小肠募）
- 心俞（背俞）+ 俞募治疗
- 耳针心点埋针
- 厉兑（心绞痛/心下满，井主心下满）
- 然谷（肾经路线痛时代替涌泉——涌泉太痛病人会跑）
- 列缺（任脉旁）

### 9. 肝癌/肝病的不二法门：保持大便通畅
- 宿便毒素经肝脏解毒 → 肝已有病再解毒增负 → 必须通便减负
- 肝病病人绝不允许隔日大便
- 大便黑色 = 排毒（阴实排出）—— 治肝病的正常现象
- 替代疗法原理：用药取代部分脏腑功能，让病脏有时间休息恢复

### 10. 说服病人的沟通原则
- 对危重病人不能模棱两可（"可能""也许""大概"）
- 直接说"死定了，准备丧礼"比耐心劝说更有效
- 西医在肝癌病人身上放任何东西（导管/阻断剂）= 清不出来 → 必死

## 五、新学到的东西（之前课程未涉及/本段深化）

1. **耳心点 vs T5的分工诊断** —— 之前学心脏病只知道T5压痛，本段明确了耳穴专查血管问题、T5兼查结构问题
2. **地龙的真正用法** —— 组织间瘀血红花清不掉→地龙钻进去清；动物药不可滥用
3. **白芍+炮附子的配伍原理** —— 不是单用白芍收血（风险：血块回心堵塞），必须炮附加热融化
4. **呃气与心脏病的鉴别** —— 胃酸烧灼食道的胸痛会被误判为心脏病胸痛（小陷胸汤vs胸痹方）
5. **肝癌的"不能放东西"原则** —— 西医导管/阻断剂=人工异物=中药清不掉=必死
6. **说服晚期病人** —— 直接吓比温和劝有效
7. **手脚末梢是最敏感的阴阳平衡指标** —— 发病前几年手脚已麻→西医说与心脏无关→实际是早期信号
8. **镜面舌 = 病进**（肝癌案例中舌苔退→镜面舌出现）
9. **治肝必须通大便不是中医理论问题，是减负原理——解毒脏已有病不能再让它解宿便毒**
