# 金匮要略 段16：五脏风寒积聚病脉证并治

## 一、邪哭·魂魄不安 → 癫狂病机

### 核心病机链
```
血气少（肝血少 + 肺气虚）→ 影响心脏（心管魂和魄）
→ 心气虚 → 其人则畏 → 合目欲眠 → 梦远行精神离散 → 魂魄妄行
→ 阴气衰者为癫，阳气衰者为狂
```

### 关键概念
- **魂属肝，魄属肺，统由心（神）管**
- 心取得木（肝）和金（肺）的平衡 = 正常
- 血气少 → 心脏受伤 → 虚症
- 做梦走远路 = 神不能守，更耗气血

### 治则
- 精神问题从**肝、肺、心**三脏下手
- 癫狂可治，从虚论治

### 新认识
- "邪哭" = 无缘无故哭，动作诡异，疑神疑鬼 → 心气血不足
- 怕到躲进衣橱 = 心气虚的表现
- 癫（阴气衰）vs 狂（阳气衰）的起源

## 二、脾中风

### 脾的管辖范围（总纲）
- **脾主四肢、主肌肉、主少腹、主湿**
- 脾统血 = 管静脉（非动脉）
- 脾脏收缩 → 胃撑大；脾脏撑大 → 胃缩小（脾推动胃蠕动）

### 脾中风病机
```
风入肌肉/脾脏 → 湿盛（下焦湿盛）
→ 心火欲入小肠，湿阻不得入 → 心火上冲 → 翕翕发热、形如醉人
→ 腹中烦重（湿多）→ 皮目瞤瞤（眼袋属脾）→ 短气（心肺之气不降）
```

### 治则
- **利小便，不发汗**（发汗伤津液，湿出得慢，肚子仍肿大）
- 主方思路：**泽泻 + 白术 + 茯苓 + 白芍**

### 核心药法
| 情况 | 药量配比 |
|------|---------|
| 中焦湿 | 白术 = 茯苓（等量）|
| 下焦湿/肚子肿大 | 茯苓重用（白术三钱，茯苓六钱）|
| 静脉回流 | **白芍重用至一两**（管静脉，不治动脉）|

### 白芍统血原理（核心！）
- **脾统血 = 管静脉**（不管动脉）
- 白芍重用 → 静脉血回流 → 心脏负担减轻（只需管动脉）
- 动脉问题 = 静脉回流不够影响心阳
- 这就是"脾统血"的实际临床意义

## 四、脾约 → 麻子仁丸（重点！）

### 趺阳脉诊断
```
趺阳脉浮而涩
浮 = 胃气强（胃火大）
涩 = 小便数
浮涩相搏 → 大便坚 → 脾约
```

### 完整病机链（第二遍讲解，更深入）
```
胃火太大 → 约束脾脏（脾约）
→ 心火到胃后不下小肠 → 小肠温度下降
→ 膀胱无蒸汽（冷水停底部）→ 压迫 → 小便频数（一点点就要上）
→ 脾被约束不能供湿给小肠大肠 → 小肠干燥
→ 大便如羊粪，一颗一颗
```

### 小便能否喷射 = 小肠温度指标
- 能喷射 → 小肠温度够
- 不能喷射 → 小肠温度不足 → 需练易筋经

### 膀胱气化二次利用
膀胱水气化 → 水生木 → 肝脏 → 胆汁（胆汁中的水来自膀胱）

### 麻子仁丸方药分析
| 药物 | 作用 |
|------|------|
| 麻子仁 | 润肠（仁剂）|
| 杏仁 | 润大肠 |
| 枳实+厚朴+大黄 | 小承气汤底（通便）|
| 白芍 | 管腹部（少腹属阴，白芍管）|
| 桂枝 | 管胸腔（胸闷用桂枝）|

### 设计思路
知道病理学 → 处方自动出来。不是死记方子，是理解为什么每味药在那里。

## 五、肾着病 → 甘姜苓术汤（肾着汤）

### 病因
身劳汗出 → 衣里冷湿 → 久久得之（反复汗湿衣服不换，吹干再出汗）

### 症状
- 身体重，腰中冷，如坐水中
- 反不渴，小便自利（与水肿区别！）
- 饮食如故
- 腰以下冷痛，腹重如带五千钱

### 肾着 vs 水肿鉴别
| | 肾着 | 水肿（真正积水）|
|---|---|---|
| 口渴 | 反不渴 | 口渴 |
| 小便 | 小便自利 | 小便不利 |
| 本质 | 湿累积（寒湿）| 水积 |

### 病机
**脾湿过剩 → 脾主少腹 → 湿排不出 → 下坠 → 寒湿在腰以下**

### 甘姜苓术汤方（肾着汤）
```
甘草 白术 各二两
干姜 茯苓 各四两
```
- **干姜**：去寒
- **茯苓**：去湿（下焦重用，超白术量）
- **白术**：去湿
- **甘草**：和中

### 临床应用扩展
- 带脉病（环腰一圈冷）
- 坐骨神经痛
- 频尿、遗精（下焦湿）
- 脚萎弱、肌肉萎缩（下半身太冷）

### 倪师实战案例
- 最重一例：站起来肚子贴地，十几年未愈
- 甘姜苓术汤一两一两下，腰冷减，但肚子未全消
- 说明：早期治则快，拖延太久难回

## 七、三焦寒热辨证

### 上焦寒 → 善噫（打嗝）
- 病机：中焦寒 → 上焦受中焦气未和 → 不能消谷 → 噫
- 治疗：**橘皮竹茹汤**（橘皮辛香发散 + 生姜排胃水）
- 很浅的病

### 下焦寒 → 遗溺失便
- 小肠温度不够 → 膀胱不气化
- 不须治（久则愈）/ 练功（易筋经、洗髓经）/ 灸

### 热在上焦 → 咳为肺痿
- 肺津液不够 → 心火大 → **麦门冬汤证**

### 热在中焦 → 大便坚

### 热在下焦 → 尿血、淋闭不通

### 大肠寒热
| | 寒 | 热 |
|---|---|---|
| 大肠 | 鹜溏（控制不住，稀便）| 便肠垢（浊物多，味重）|

### 小肠寒热
| | 寒 | 热 |
|---|---|---|
| 小肠 | 下重便血（食物消化成血但无法供身）| 必痔（湿热下注）|

### 小肠寒致便血/精血 → 关键误诊陷阱！
- 精中带血 → 容易被误判为"发炎"→ 用寒凉药 → 血更多
- 实际是**小肠温度不够**，血无法生化成精
- 寒症用寒药 = 雪上加霜

## 八、积·聚·榖气鉴别

| | 积 | 聚 | 榖气 |
|---|---|---|---|
| 部位 | 脏（固定不动）| 腑 | — |
| 移动性 | 终不移 | 发作有时，展转痛移 | 胁下痛，按之则愈，复发 |
| 预后 | 难治（肿瘤癌症）| 可治 | 可治 |
| 疼痛 | 持续固定痛 | 移动痛 | 按之缓解 |

### 核心判别法
- **痛会移动 = 没事**
- **痛固定不动 = 麻烦**
- 肺癌：持续刺痛，不动，吸吐皆痛，日夜不休

### 直接发 vs 转移
- 直接发脏病（如原发肺癌）→ 难治
- 转移（如乳癌转肺癌）→ 较好治
- 往外走 = 吉，往内走 = 凶
- 乳头凹陷 = 往内走 → 凶

## 九、诸积大法 → 附骨脉诊断（核心！）

### 附骨脉
```
脉来细而附骨者，乃积也
```
- 重按到骨边，脉还在 = 附骨脉
- 细 = 肿瘤把血管包住，血管变细小

### 各部位对应
| 脉位 | 积的位置 |
|------|----------|
| 寸口 | 胸中（肺癌）|
| 微出寸口（脉横向走手掌内侧）| 喉中（喉癌/舌癌/脑瘤）|
| 关上 | 脐旁（肝癌/胰脏癌）|
| 上关上 | 心下（胃下方/胰脏）|
| 微下关 | 少腹 |
| 尺中 | 气冲（子宫/卵巢/前列腺）|

### 左右对应
- 脉出左 → 积在左
- 脉出右 → 积在右
- 脉两出 → 积在中央

### 治疗追踪
| 向好 | 向坏 |
|------|------|
| 脉变大（从细变粗）| 脉持续细小 |
| 至数减少（8→7→6→5）| 至数增多 |
| 痛减、睡眠好 | 痛加剧 |

## 十、倪师肺癌临床案例

- 1949年生，第四期肺癌
- 持续固定胸痛，刀刺般
- 不戒烟——94岁老人戒烟一周就走，肺癌晚期戒不戒已无关
- 治疗原则：不化疗，中药调理

## 十一、本段新增核心临床技能总结

1. **精神问题** → 从肝肺心下手（虚则补之）
2. **脾湿** → 利尿不发汗；中焦白术=茯苓，下焦茯苓>白术
3. **脾统血 = 管静脉** → 白芍重用收静脉血回心
4. **脾约 → 麻子仁丸** → 胃火约束脾，小肠冷→小便数+大便坚
5. **肾着 → 甘姜苓术汤** → 寒湿腰以下，干姜+茯苓为主
6. **附骨脉** → 定位肿瘤的金标准，比西医检查更快更准
7. **痛移vs不移** → 移动可治，固定难治
8. **小肠寒致便血/精血** → 切勿当热症治！
