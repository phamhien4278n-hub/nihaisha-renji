# 伤寒论 段4 知识树：发汗后变证与补救（63-73条）

## 核心概念

### 1. 阴阳自和（63条）
汗吐下后亡津液，若出现以下表现→阴阳自和→必自愈，不给药：
- 脉：寸尺俱缓，不浮，一吸四至（医生一吸病人脉跳两下，吐跳两下，呼吸间暂停→一息约五至）
- 外症：病人自觉舒服但比较累，不燥渴，汗已停
- 原则：张仲景不补阴，津液要自己回头才是真正痊愈

### 2. 烦躁 = 危险信号（核心贯穿本段）
- 病人平稳只虚→不给处方，休息自愈
- 只要出现烦躁→必须开方（66条干姜附子汤、后续多处）
- 安宁病房 = 烦躁病房——病人死前都要烦躁

### 3. 津液自回原则（张仲景核心立场）
- 张仲景从不提"阴虚阳虚"，只说"内外俱虚"
- 不补阴，让阳气恢复后阴液自回
- 反对滋阴——补阴药让身体退化（类比：人工胰岛素→胰脏退化）
- 只有人参、生姜、甘草、芍药这些助阳药，不用生地麦冬天冬

### 4. 补位理论
人体无间隙，汗出后津液流失，身体自动补位。肠胃不好的人补位慢→水停中焦→动悸/头眩。

## 八张方剂的病机推理链

### 63-65：三种不处方的情况

```
63: 汗吐下→亡津液→阴阳自和（脉缓、舒服、不渴）→ 自愈（不处方）
64: 大下+发汗→小便不利（津液不够，非结石炎症）→ 不治，小便自利即愈
    鉴别：小便少+不痛+不血+不滴答=津液不够；痛/血/滴滴答答=结石/炎症
65: 下+发汗→怕冷+脉微细→表里俱虚（非阴虚阳虚）→ 无危险，食补即可
```

### 66：干姜附子汤——阳虚烦躁（张仲景第一个处方！）

```
下+发汗→昼日烦躁不得眠+夜安静+不呕不渴+无表证+脉沉微+身无大热
→ 阳虚（白天=阳，晚上=阴；病只在白天=阳虚）
→ 干姜附子汤

干姜：温中、温胃、温肺（上焦）
生附子（去皮）：温心阳、命门火
  生附子增加心脏摩擦力（质变）≠ 麻黄加速心跳（量变）
  生附子去皮=去纤维，不用棉布包
  心脏阳不够=炮附子；四逆汤证（心阳不够）=生附子

要点：烦躁=必须开方！平稳只虚=不给药
```

### 67：新加汤——过汗伤津身疼痛

```
发汗后身疼痛+脉沉迟（沉=里，迟=寒）
→ 表已解但津液伤到
→ 桂枝加芍药生姜人参新加汤（桂枝汤+芍药一两+生姜一两+人参三两）

关键鉴别：
  新加汤：汗已止，身疼痛脉沉迟 → 补津液
  桂枝加附子汤：汗不止 → 固表止汗
  开反了：汗不止给新加汤→越吃越流汗（新水进旧水出）
```

### 68：麻杏甘石汤——汗后喘无大热

```
发汗后汗出而喘+无大热 → 麻杏甘石汤

黄金比例：麻黄 = 石膏+杏仁（等量）
  例：麻黄6钱+杏仁2钱+石膏4钱→6=2+4，不伤津液
  麻黄不发表只治喘（吃了不流汗！）
  石膏清热+杏仁去痰润肺

关键：麻杏甘石汤不治发烧！发烧用大青龙/小青龙/桂枝汤/麻黄汤
```

### 69：桂枝甘草汤——心下悸（麻黄汤太过）

```
发汗过多（麻黄汤太过）→ 叉手自冒心+心下悸+喜按
→ 肺津液外泄→下焦水来不及补位→水停心下（心下方/胃部）
→ 水克火（心不受水）→动悸
→ 桂枝甘草汤（桂枝4:甘草2，顿服）

桂枝：降冲逆，把水气化掉
喜按=虚证（无实邪）；拒按=实证（有肿块/宿便）
```

### 70：苓桂甘枣汤——脐下悸（桂枝汤太过）

```
发汗后（桂枝汤太过）→脐下悸→欲作奔豚（还没发作）
→ 苓桂甘枣汤

69 vs 70 差异：
  69心下悸=麻黄汤太过，肺津液不够，水停心下
  70脐下悸=桂枝汤太过，肠胃津液不够，水停脐下

甘澜水：水二斗以杓扬之，水上有珠子五六千颗相逐→取用
```

### 71：厚朴生姜半夏甘草人参汤——虚胀腹胀满

```
发汗后腹胀满 → 肠胃津液干→肠气多→虚胀
→ 厚朴生姜半夏甘草人参汤

虚胀 vs 实胀鉴别：
  小便淡白=虚胀（厚朴生姜半夏甘草人参汤）
  小便黄=实胀（承气汤）
  喜按=虚；拒按=实

剂量参考：厚朴3钱+生姜3片+半夏3钱+人参1钱+炙甘草2钱
厚朴：去湿，脾脏湿移到肠子
```

### 72：苓桂术甘汤——起则头眩（横膈停水）

```
伤寒+若吐若下→心下逆满+气上冲胸+起则头眩+脉沉紧
→ 发汗则动经，身为振振摇
→ 上吐下泻伤肠胃津液→再发汗伤肺津液→水停横膈
→ 横膈像海绵吸水，人一动水跟着动→头眩
→ 苓桂术甘汤（茯苓4:桂枝3:白术2:甘草2）

关键鉴别：
  起则头眩（动才晕）= 苓桂术甘汤（横膈水饮）
  躺着头眩 = 真武汤等其他方剂
  临床常加半夏（恶心时）
  常见病因：运动后喝水太快也会
```

### 73：芍药甘草附子汤——汗后恶寒里虚

```
发汗病不解+反恶寒 → 里阳不足（虚故也，非医之过）
→ 芍药甘草附子汤

芍药+甘草：补里虚（张仲景补里虚的标准搭配）
炮附子（重用）：补里寒
不需要生附子——只是里寒，没到四逆
```

## 学习心得

本段核心是张仲景处理"过度治疗"的补救措施。大原则：
1. 能自愈的不开方（阴阳自和、小便自利）
2. 虚但平稳的不开方（食补）
3. 一出现烦躁必须开方（干姜附子汤）
4. 所有补救方都不补阴，靠助阳让津液自回
5. 动悸的位置决定方剂：心下/脐下/横膈各不同
