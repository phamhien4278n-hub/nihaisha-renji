# 金匮要略 段2 知识树

## 一、伤寒论总结（段2前半·过渡部分）

### 1. 通脉四逆汤加猪胆汁
- **猪胆汁作用**：取代人体胆汁，帮助消化肠胃宿食，应急用
- **四逆汤组成**：炙甘草、干姜、生附子（大者一枚，去皮破八片≈三钱）
- **剂量经验**：救逆时可用四钱，好转后减至三钱→两钱；倪师离开病人时只给两钱
- **药性配合**：炙甘草（焦苦入心+甘味入脾）→心和脾之间走动；干姜温胃；生附子强心→救逆
- **煮法**：水三升煮取一升，加猪胆汁，分温再服。无猪胆汁可用羊胆汁代

### 2. 病后饮食禁忌
- 发汗后脉平但略有烦躁→肠胃不适的正常恢复现象
- 伤寒病后饮食：稀饭+青菜，不加蛋肉；白水煮面加葱白（葱白通脉助肠胃吸收）
- 等完全恢复后再正常饮食

## 二、伤寒论113方总纲（三法统万方）

### 核心框架：辛散·寒泻·温补
| 三法 | 代表药 | 目的 |
|------|--------|------|
| **辛散** | 麻黄、桂枝、细辛、柴胡、葛根 | 去寒、驱寒 |
| **寒泻** | 石膏、黄芩、黄连、大黄（膏黄） | 撤热、急下存阴、救阴 |
| **温补** | 干姜、附子、吴茱萸、蜀椒 | 回阳、救逆 |

### 张仲景用药铁律
- **非寒不泻**：不用寒药无法宣泄——石膏大黄黄芩黄连
- **非温不补**：药性不温补不到身体——用温药做补药，不用四物汤思路
- **人参用法**：养阴退热——津液不足时用（胃蠕动无津液→发热），非补品

### 六经辨证与辛散药对应
- 太阳→麻桂 / 少阳→柴胡 / 阳明→葛根（葛芩连汤）/ 少阴→细辛

## 三、脏腑经络先后病脉证第一（正文开始）

### 1. 上工治未病——肝病传脾·当先实脾

#### 肝病传变规律
- 肝病→传脾（相克传变，肝木克脾土）
- 实脾目的：阻断传变，病止于肝

#### 肝病早期诊断（五点）
1. **睡眠**：丑时（1-3am）不能睡→肝已出问题
2. **大便**：便秘→肝脏压力大，积毒无法从大肠宣泄（肝有两条排泄通路：胆+通大肠的血管）
3. **小便颜色**：发黄→肝病前兆
4. **胃口**：饭前不饿→肝有问题
5. **手掌颜色**：粉红→深红→暗红→肝病进展

#### 肝虚补法——酸·焦苦·甘
- **补用酸**（酸入肝）：乌梅丸、酸枣仁
- **助用焦苦**（焦苦入心）：吴茱萸、黄连
- **益用甘味调之**（甘入脾）：小建中汤
- **适用**：肝虚证（如肝炎），不适用于肝硬化/肝癌（实证）

#### 五行传变推演
- 实脾→脾克肾→肾气微弱→水不行→心火气盛→心火克肺→肺金不行→肝气盛→肝自愈
- 此条仅适用于虚证

### 2. 肾病传心·当先实心
- 肾水不升木（肝）→直冲心脏（水克火，肾水凌心）
- **诊断**：手掌心发黑发青（劳宫穴/少府穴区域）
- **治法**：黄连解毒——苦味入心，解肾脏之毒
- **案例**：10年不洗肾的美国病人，手掌黑脸黑但活着；倪师用黄连解毒治之
- **肾衰竭前两大症状**：①呕吐不止（心火不能生脾土）②头目昏眩（毒气上升至脑）
- **好转标志**：呕吐消失+晕眩消失+小便便意恢复（脾藏意→心强则脾恢复→便意回来）

### 3. 实证死·虚证不死
- 虚证不会死人（瘦弱长寿），实证才会死人（阴实等）
- 遇到实证→赶快变成虚证
- 维他命之害：虚证人为补成实证→致命
- **治阴虚的铁律**：经方家不用补阴药，用阳药使阴自回

## 四、南派六大误治（补虚之弊）

| 误治场景 | 错当阴虚补 | 造成的病 |
|----------|-----------|---------|
| 下痢不止（疟证） | 补阴 | **痞证**（心下痞→泻心汤证） |
| 下痢/痢血 | 补阴 | **臌胀**（腹部胀大） |
| 咳嗽（支气管炎/肺炎/TB） | 滋阴 | **肺痿**（肺萎缩） |
| 痰饮（胸腔积水） | 补阴 | **浮肿**（脸肿） |
| 吐血+泄精 | 补阴 | **劳证** |
| 湿阻中焦/食滞肠胃 | 补阴 | **格**（阴阳相格） |

## 五、病因三条（千般疢难不越三条）

| 类型 | 内容 | 说明 |
|------|------|------|
| **内因** | 经络受邪，入脏腑 | 外邪由经络深入脏腑（未及时治） |
| **外因** | 四肢九窍，血脉相传，壅塞不通 | 外皮肤所中 |
| **不内外因** | 房室、金刃、虫兽所伤 | 房事过度、刀伤、虫蛇咬伤 |

- 外因包含：风寒暑湿燥热
- 内因包含：喜怒忧思悲恐（七情过度）

### 养慎预防法
- 邪风一入经络→立即医治（导引/吐纳/针灸/膏摩）
- 四肢才觉重滞→即导引吐纳——不要等九窍闭塞
- 无犯王法（心中安宁）、房室勿令竭乏、服食节其冷热苦酸辛甘
- **腠理**：腠=三焦通会元真之处（血气所注）；理=皮肤脏腑之文理

## 六、文式易筋经第一式——按蹻式

### 基本功架
- **下半身**：绷紧不动，脚趾如爪抓地，肚脐以下肌肉硬到拍打弹起
- **上半身**：松的——松紧对比是核心

### 握拳法
- 拳面调平（无三角形角度），尺骨桡骨成平行直线
- 目的：发力不伤手腕

### 按蹻式动作
- 双手下按——吸气——指头往上抬（感觉像手撑桌面跳起）
- 仅在一寸之间发紧
- 做49下→换下一个动作
- 出汗后摸背→冰的（太阳寒水，正常）

## 七、望诊——面部气色

### 鼻头（山根）色诊
| 气色 | 诊断 | 备注 |
|------|------|------|
| **青** | 腹中痛（小建中汤证） | 小儿常见：两眼间青筋=小肠冷 |
| **微黑** | 有水气（肾病） | 黑色营养应入肾，肾虚则反逆于面 |
| **黄** | 胸上有寒（甘草干姜汤） | ≠鼻头黄有横财（玩笑话） |
| **白** | 亡血（贫血） | |
| **微赤（非时）** | 死症 | 秋冬季见赤色=心火非时 |
| **目正圆（痉）** | 不治 | 脑中风/脑膜炎/抽筋→尽人事听天命 |

### 一般望色
- 青→痛 / 黑→劳 / 赤→风 / 黄→便难 / 鲜明→留饮

## 八、闻诊——听声音

| 声音特征 | 诊断 |
|----------|------|
| 语声寂然·喜惊呼 | **骨节间病**（关节剧痛——如类风湿，乌头汤/乌头桂枝汤治） |
| 语声喑喑然不彻 | **心膈间病**（中风后想说说不出） |
| 语声啾啾然·细而长·重复不停 | **头中痛**（讲话减轻头痛，大声则更痛） |

|----------|------|

## 十、新学到的重要观念

1. **伤寒三法统万方**：所有113方归为辛散·寒泻·温补三类——前所未有的简化视角
2. **非温不补**：张仲景不用四物汤思路补虚，必用温药（姜附吴茱萸蜀椒）
3. **人参=养阴退热**：不是补品，是津液不够时用
4. **治阴虚用阳药**：让阴自回，不开补阴药——六大误治就是反例
5. **实证死·虚证不死**：临床最高原则——遇实证立刻转虚
6. **肝病两大排泄管道**：胆+通大肠的血管——保持大便通畅=保护肝脏
7. **肾病先治心**：肾衰竭前兆=呕吐+眩晕，好转标志=便意恢复
8. **文式易筋经**：松紧对比原则——下半身绷紧、上半身松、仅一寸发紧
