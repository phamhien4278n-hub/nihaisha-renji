# 金匮要略 段12 知识树

## 一、肺痿肺痈咳嗽上气篇（续）

### 核心方剂对比表

| 方名 | 主证 | 病机 | 关键鉴别点 |
|------|------|------|-----------|
| 麦门冬汤 | 火逆上气，咽喉不利，干咳失音 | 肺津不足，虚火上逆 | 干咳无痰+失音 |
| 葶苈大枣泻肺汤 | 肺痈喘不得卧 | 肺实热，已成脓 | 不能躺+舌黄脉数 |
| 千金苇茎汤 | 咳嗽黄痰，喘，但可躺 | 肺实热，脓未成或初成 | 能躺+舌黄脉数 |
| 桔梗汤 | 咳而胸满，振寒，时出浊唾腥臭，吐脓如米粥 | 肺痈成脓，胸阳已虚 | 吐米粥样痰+反觉冷 |
| 越婢加半夏汤 | 咳而上气，肺胀，喘，目如脱状，脉浮大 | 肺气胀+内有热 | 有汗+痰黄 |
| 小青龙加石膏汤 | 肺胀，咳而上气，烦躁而喘，心下有水 | 表寒里寒+肺胀 | 无汗+不渴+舌白 |

### 1. 麦门冬汤（火逆上气）
- **组成**：麦门冬七升，半夏一升，人参、甘草各二两，粳米三合，大枣十二枚
- **核心病机**：肺中津液不足 → 虚火上逆 → 干咳+失音
- **辨证关键**：干咳无痰 + 声音沙哑/失音
- **倪师临床用法**：
  - 老年人气血两虚，常干咳 → 麦门冬汤
  - 孕妇咳嗽 → 必用麦门冬汤（孕妇多里热，里寒根本不会怀孕）
  - 肺结核（TB）→ 麦门冬汤是最佳处方，舌黄+气逆+吞不下
  - 食道癌吞不下东西 → 麦门冬汤
  - 突然唱歌沙哑 → 麦门冬汤
- **煎服**：水一斗二升煮六升，温服一升，日三夜一（一天四次）

### 2. 葶苈大枣泻肺汤（实热脓疡）
- **组成**：葶苈（熬令黄色捣丸），大枣十二枚
- **煎法**：先煮枣取汁去枣，再入葶苈煮
- **辨证关键**：喘不得卧 + 舌黄 + 脉数 + 口渴
- **与千金苇茎汤区别**：不能躺 → 葶苈大枣泻肺汤；还能躺 → 千金苇茎汤
- **倪师趣谈**：不能相信病人会按分步煮法——美国人吃西药多记忆不好，到Wal-Mart门口忘记来干嘛

### 3. 千金苇茎汤（孙思邈）
- **组成**：苇茎、冬瓜仁、桃仁、薏仁
- **作用**：苇茎+冬瓜仁（去痰降气），桃仁（活血化瘀），薏仁（去湿，可大剂用一两）
- **辨证**：实热证而尚可平卧

### 4. 桔梗汤（肺痈成脓）
- **组成**：桔梗一两，甘草二两
- **辨证关键**：咳而胸满 + 振寒（反而不热！）+ 脉数咽干不渴 + 吐腥臭浊唾 + 吐脓如米粥
- **病机转折**：化脓后反而不热 → 胸阳已虚 → 病人感觉冷
- **为什么反冷**？胸部是诸阳之会，化脓后阳虚 → 振寒
- **米粥样痰** = 肺癌前兆（没治好就转肺癌）
- **服后反应**：吐脓血（桔梗清脓，甘草解毒+增肠胃津液）

### 5. 越婢加半夏汤（肺胀有汗）
- **组成**：麻黄六两，石膏半斤，生姜三两，大枣十五枚，甘草二两，半夏半升
- **辨证关键**：有汗 + 咳嗽发烧 + 痰黄（里热）
- **与桂枝汤鉴别**：有汗发烧无咳嗽 → 桂枝汤；有汗发烧有咳嗽 → 越婢汤
- **倪师注**：麻黄在越婢汤中可止汗（非发汗）

### 6. 小青龙加石膏汤（肺胀无汗）
- **组成**：麻黄、芍药、桂枝、细辛、干姜、甘草各三两，五味子、半夏各半升，石膏二两
- **辨证关键**：无汗 + 不渴 + 舌白（表寒里寒）+ 心下有水
- **与越婢加半夏区别**：一个是里热有汗，一个是里寒无汗

## 二、奔豚气病脉证并治第八

### 核心病机：水的气化过程

```
心脏（火）→ 移热小肠（火）
食物水 → 胃 → 小肠（火蒸化）→ 糟粕+水 → 大肠
大肠水 → 小肠火在下蒸化 → 水气化 → 肺（金金相通）
```

- **为什么肺癌会下利**？肺有阴实堵住 → 水回不去 → 从大肠出 → 下利
- **治好的标志**：下利停止
- **小肠温度不够**（心脏力量不足）→ 水不气化 → 水+糟粕同出 → 寒利
- **寒利判断**：脚冷 → 脚越冷寒越重

### 奔豚病因：惊发

四种病皆从惊发：奔豚、吐脓、惊怖、火邪

**三种奔豚路径**：
1. **浊气入三焦（水道）**：小肠紧缩 → 大肠沼气 → 进入三焦水系统 → 往来寒热 → **奔豚汤**
2. **浊气入血分**：大肠干燥+受惊 → 浊气入肝→入心→心脏加强搏动排出去 → 动悸 → 另一种奔豚
3. **过汗致表虚**：发汗太过 → 表虚 → 肺津不足+小肠津来不及补 → 动悸

### 奔豚汤
- **组成**：甘草、芎䓖、当归、黄芩、芍药各二两，半夏、生姜各四两，生葛五两，甘李根白皮一升
- **辨证**：气上冲胸，腹痛，往来寒热
- **方义**：小柴胡汤底（甘草、黄芩、半夏、生姜）+ 当归川芎（入血）+ 芍药（腹痛）+ 甘李根白皮（少阳）
- **腹痛用芍药，腹满用厚朴** — 核心经方原则

## 三、胸痹心痛短气病脉证第九

### 脉诊基础

- **正常**：一息五至（医生的一呼一吸，病人脉跳5下）
- **热**：6→7→8→9→10（8以上为凶）
- **寒**：4→3→2→1（2以下为死症）
- **结脉**：如绳打结，心脏有问题
- **代脉**：跳跳停停。54下一止=正常；50下一止≈5年；10下一止≈10年
  - 吃了药从10下一止→15→20→54→好了

### 核心诊断：阳微阴弦

- **阳微** = 寸脉微 = 上焦阳虚
- **阴弦** = 尺脉弦 = 有水
- **病机**：阳弱 → 水往上冲 → 胸痹而痛

### 胸痹方剂

#### 1. 薏苡附子散（风湿性心脏病）
- **组成**：薏仁、炮附子等量，打散剂，每次一平汤匙，日三服
- **辨证**：胸痹缓急（随气候改变——天冷或潮湿就发作）
- **病机**：寒湿 → 同气相求，天冷/湿引发
- **倪师言**：一剂知，二剂已

#### 2. 桂枝生姜枳实汤（心包炎）
- **组成**：桂枝、生姜各三两，枳实五两
- **辨证**：心中痞（胃不舒服），诸逆，心悬痛（心脏像悬在空中）
- **方义**：桂枝行胸阳，枳实入心脏，生姜去湿+和胃

#### 3. 乌头赤石脂丸（最严重心脏病）
- **组成**：乌头一分（炮），蜀椒、干姜各一两，附子半两，赤石脂一两
- **辨证**：心痛彻背，背痛彻心（穿心痛）
- **病机**：纵膈腔浓浊痰湿跑动
- **倪师临床实际用乌梅**：乌梅（酸入肝）+ 蜀椒干姜（辛辣入肺）+ 附子赤石脂 → 苦温化湿
- **为什么乌梅而非乌头**：治心必治肺+强肝（木生火）。乌梅酸入肝，蜀椒辛辣入肺
- **预后**：此证出现 → 半年内心脏病发作走人

### 动脉血管堵塞（四大症状）

1. **失眠**（无原因）— 心藏神，神不回心
2. **大便不规则**（时秘时利）— 心小肠相表里，速度同步
3. **手脚冷** — 心脏温度不够，从末梢开始
4. **心胸痛** — 刺痛如刀刺

**治则**：苦温化湿
- **基础方思路**：黄连阿胶汤（苦） + 桂枝炙甘草（胸满胸痛） + 炮附子（手脚冷） + 川芎（活血化瘀，血管内堵塞） + 枳实（胸闷心痛）
- **白芍拿掉**（胸闷不用白芍）
- **可加丹皮、桃仁**活血化瘀

### 心脏病治疗总诀

- **保冷治心**：冷水冲脚 → 心脏受刺激把热导到末梢 → 自愈
- **天冷少穿一件** → 不容易发心脏病（《黄帝内经》）
- **胸满用枳实，腹满用厚朴，腹痛用白芍** — 经方铁律
- **耳针心点**：动脉堵塞有压痛点
- **第五椎**：心脏结构问题（瓣膜等）有压痛点
- **针灸急救**：公孙+内关；心胸正痛→厉兑（井主心下满）

### 西医批判
- 支架 = 活炸弹，松脱射向心脏当场死
- 绕道手术 = 人体自己能做
- 阿司匹林/华法林 = 破坏血管+瘀血，瘀血到脑=脑溢血
- 吃西药越吃手脚越冷 = 没效
