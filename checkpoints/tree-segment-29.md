# 金匮要略 段29 — 知识树

## 核心方剂（4个）

### 生姜半夏汤（胃中停水）
- **组成**：半夏半升，生姜汁一升（生姜是半夏的一倍）
- **对比小半夏汤**：小半夏汤半夏重于生姜；生姜半夏汤反过来
- **主治**：胃中停水过多→"胸中似喘不喘，似呕不呕，似哕不哕，彻心中愦愦无奈者"——胃里闷闷的，想吐吐不出，想喘喘不了
- **病机**：水停在胃里→横膈以下（不是横膈以上）→呼吸不受影响（所以"似喘不喘"）
- **病因**：躺到猛喝水→胃的火在水下面→水无法气化→停在胃里。站起来喝→火在下水在上→气化正常
- **非药物治**：跑步出汗（汗液从肠胃来）→水发散；长期不治→胃寒→缩小（石膏同理减肥）
- **服法**：分四服，日三夜一。恶心好了就停，不可多喝→胃里无水干涸→食物堵胃→大黄甘草汤症

### 橘皮汤（干呕打嗝·最浅）
- **组成**：橘皮四两，生姜半斤
- **主治**：干呕+打嗝+手足厥冷
- **适用**：精神好、活蹦乱跳、就是打嗝——橘皮辛香发散（行阳）
- **不适合**：脸色暗黄、元气尽、躺床上打嗝（那是肝癌/胃癌末期）
- **注意**：橘皮≠柠檬皮（柠檬酸味，不一样）
- **民方**：塑料袋套头呼吸CO₂

### 橘皮竹茹汤（哕逆·胃虚热）
- **组成**：橘皮二升、竹茹二升、大枣三十枚、生姜半斤、甘草五两、人参一两
- **主治**：打嗝+口渴严重——吐了很多胃津液
- **竹茹**：止渴；临床用于淋巴结肿、扁桃腺肿、腮腺肿、甲状腺肿、糖尿病
- **人参**：津液伤到→用人参补津液（仲景惯例）
- **辨证**：胃家虚热（橘皮汤是浅症，这个是虚热）

### 茯苓四逆汤（危症打嗝·胃气将绝）
- **适用**：肝癌/肝硬化/胃癌末期打嗝（死症，但仍要出手救）
- **病机**：胃气要绝→内脏寒实→胃里寒湿
- **组方逻辑**：生附子（极热→心脏到小肠全热）+干姜（寒）+茯苓（排三焦湿）→药简力专
- **不用白术**：急救应急时药简力专

## 六腑气绝 vs 五脏气绝（辨证纲领）

| | 六腑气绝（阳绝） | 五脏气绝（阴/阳绝） |
|---|---|---|
| **外在** | 手足寒、上气、脚缩（横膈膜缩小、腿抽筋缩起） | - |
| **下利** | - | 利不禁（大小便失禁，阳气不固） |
| **严重** | - | 下甚者手足不仁（血分→麻木） |
| **临床举例** | 晚上睡觉脚抽筋缩起来→阳不足 | 吊死者大小便失禁→阳气绝 |

## 下利辨证体系（大块核心内容）

### 一、脉证合参——判断预后

| 脉证 | 预后 | 病机 |
|------|------|------|
| 脉沉弦+下重 | 里寒重 | 病在里+寒 |
| 脉大 | 未止，阴还多 | 阴未排完 |
| 脉微弱数 | 自愈，虽发热不死 | 阳气回头（不是回光返照那种大脉发热） |
| 脉缓+微热+汗出 | 自愈 | 胃气回来（任何病脉缓=胃气回） |
| 脉紧 | 未解 | 还有寒 |
| 脉数而渴 | 自愈 | 胃气回来 |
| 脉数而渴+没好 | →必清脓血（大肠发炎） | 热故也 |
| 脉反弦+发热身汗 | 自愈 | 津液回头 |
| 寸脉反浮数+尺中自涩 | 必清脓血 | 阳不入阴+血不足 |
| 脉迟而滑 | 实也，急下之大承气汤 | 里实结到，水能过食物不能过 |
| 三部脉皆平+按之心下坚 | 急下之大承气汤 | 燥矢堵住 |

### 二、死症判断

- **手足厥冷+无脉**：灸之不温，脉不还+微喘→死（仅剩一点阳气）
- **下利后脉绝+手足厥冷**：晬时（一个时辰）脉还+手足温→生；脉不还→死→通脉四逆汤+灸关元·中脘
- **回光返照**：下利到最后脉突然变大发热→像蜡烛燃尽前亮一下
- **少阴负趺阳为顺**：尺脉<关脉（趺阳胃脉）→正常→下利不死
- **下利+不渴**：最怕！津液不断丧失但不渴→身体功能丧失
- **下利+喝东西吐**：不能接受新东西又排旧东西→完了

### 三、下利的分类

| 类型 | 表现 | 治法 |
|------|------|------|
| **热利（一般）** | 下利+热 | 葛芩连汤（葛根黄芩黄连汤） |
| **脓血利** | 大便+血混合（非先血后便/先便后血） | 葛芩连汤没用→后面有方子 |
| **寒利（虚寒）** | 下利清谷（吃黄瓜拉黄瓜），大便没味道，小便淡白 | 温里（四逆汤/真武汤） |
| **燥矢利（热结旁流）** | 下利腹胀满拒按，硬块，馊水极臭，小便深黄 | 大承气汤急下 |
| **气利** | 下利+放屁很多 | 利小便（五苓散/猪苓汤） |

### 四、先便后血 vs 先血后便 vs 血便混合

| 顺序 | 诊断 |
|------|------|
| 先血后便（血先喷出来） | 痔 |
| 先便后血 | 肠胃出血 |
| 血便混合 | 大肠发炎（此处讨论的脓血） |

### 五、治疗原则（铁律）

1. **先温里后攻表**：下利+表证→先解里（四逆汤温里），后解表（麻黄汤）
2. **下利清谷不可攻表**：一攻表→寒从里走到外→虚胀满（用真武汤）
3. **下利腹胀满必须辨虚实**：
   - 虚寒→喜按→四逆汤温里
   - 实（燥矢）→拒按+硬块+馊水极臭→大承气汤急下

### 六、大承气汤证的下利（热结旁流鉴别）

- 「下利是便秘引起的」——燥矢堵住大肠，只有水能通过
- 三部脉皆平+按之心下（少腹）坚→急下
- 脉迟而滑→实也→急下
- 小便深黄（阳明）vs 虚寒小便淡白
- 大承气汤下去大便没出来：
  - 剂量不够→加量
  - 大肠癌肿瘤堵到→仍在阳明，不会死
  - 六经辨证→阳明→大柴胡→少阳阳明之间→手术→厥阴（死症）

### 七、下利自愈的正面信号

1. 微热而渴+脉弱：身体功能恢复，想喝水
2. 脉缓+微热+汗出：胃气回来
3. 脉数而渴：胃气回来
4. 下利气（放屁多）：利小便即愈（非重症）

## 倪师临床思维精华

1. **看病时脑筋空白，听病人说**——病人说头痛欲裂+吐→吴茱萸汤已讲完，摸脉只是"装装"
2. **下利+声音大**→顺（胃气还在）；下利+无元气+面无血色+手指末梢无血→危
3. **饿起来慌+流冷汗+血糖低**→白虎汤
4. **减肥=胃寒**：石膏（寒凉→胃缩小）；增胖=胃热
5. **肠病毒来源**：感冒病毒+肠胃不好+抗生素→表邪陷下→病毒跑到肠子
6. **凡事桂枝汤**：身体不好、每次感冒必中→平时吃桂枝汤调和阴阳
7. **治肝要治大肠**：肝与大肠有血管直连（唯一无瓣膜的血管）→切大肠→癌细胞跑肝→肝癌死。不动手术→阳明/少阳→可治

## 新学到的东西

- 生姜半夏汤 vs 小半夏汤的剂量区别（生姜量决定命名）
- 胃中停水的物理机制（躺喝vs站喝）→commonsense
- 橘皮竹茹汤中竹茹的广谱应用（止渴+淋巴+腺体）
- 茯苓四逆汤用于危症打嗝的配伍逻辑（生附子+干姜+茯苓，不用白术）
- 下利全套脉证合参体系（预后判断表）
- 燥矢下利=便秘引起的下利（看似矛盾实则常见）→大承气汤
- 先温里后攻表的铁律
- 大肠癌→切→肝癌的死亡路径（无瓣膜血管）
- 「脉症合」原则：下利+脉虚=合（好）；下利+脉大=不合（危险）
