# 金匮要略 段22：水气病篇（上）—— 五水病机 + 血分水分 + 治则

## 一、核心概念：五水分类

### 1. 风水 (Wind Water)
- **病因**：发汗时受风→汗水不透→水肿
- **特点**：脉浮、恶风、面目肿大、颈脉动、时时咳、按手足陷而不起
- **治则**：汗法可解（越婢汤：麻黄+石膏+生姜+甘草+大枣）
- **望诊**：眼袋肥厚如卧蚕（脾土不克水）

### 2. 皮水 (Skin Water)
- **病因**：风水未治好→水入三焦淋巴系统
- **特点**：脉浮、腹如鼓、脚肿、**不渴**、不恶寒（水不在表在里）
- **治则**：仍可发汗（比风水深一层）
- **关键鉴别**：渴而不恶寒者为皮水（vs 越婢加术汤证：渴+汗+小便不利+肌肉酸痛）

### 3. 正水 (Proper Water)
- **特点**：脉沉迟（里+寒）、外证**自喘**
- **病机**：水入内脏→里寒很盛→吸气时横膈膜无法下降→喘
- **治法**：麻黄附子细辛汤类（去里寒）

### 4. 石水 (Stone Water)
- **特点**：脉自沉、腹满**不喘**
- **鉴别**：与正水的区别就在"喘 vs 不喘"

### 5. 黄汗 (Yellow Sweat)
- **病因**：汗出入水中（不是被风吹，是整个人跳进水里）
- **病机**：汗湿停在三焦系统→日积月累湿热蓄积→三焦营养渗出→腋下黄汗
- **特点**：脉沉迟、身发热、胸满、四肢头面肿；上焦有寒可伴口多涎（肺寒→甘草干姜汤）
- **预后**：久不愈→必致痈脓（任何水肿滞留都如此——新水不代旧水→皮肤干燥→化脓）
- **治法**：黄芪苦酒汤（不可发汗！）

## 二、发汗三法总则

| 部位 | 治法 |
|------|------|
| 肚脐以上 | 吐法 |
| 肚脐以下 | 攻下 |
| 全身性 | 汗法 |

**不可发汗的情况**：渴而下利 + 小便数 → 津液已伤 → 先补津液（人参/甘草/姜枣/水果），再发汗。误汗→抽筋痉挛、小便浓少。

## 三、汗水的两个源头（核心生理）

### 源头一：饮水→胃→气化→三焦系统
喝下去的水→胃气化→进入三焦（淋巴/内分泌系统）→在脏腑之间运行

### 源头二：食物→小肠→大肠→肺→皮毛→肾
食物水→小肠火气化→大肠→水上至肺→①滋养皮毛（肺主皮毛）②多余的水金生水→入肾

**二者会合**：三焦系统的水 + 皮毛的水 → 在四肢混合

**发汗过度**（麻黄汤太强）：胃津液+小肠津液干→肺干→三焦水干→出现五苓散证（口渴+四肢皮下有水+无小便）

## 四、关键方剂速查

### 越婢汤
- **指征**：有汗 + 发烧 + 咳嗽/喘（有汗而咳→越婢汤；有汗不咳→桂枝汤）
- **组成**：麻黄、石膏、生姜、炙甘草、大枣

### 越婢加术汤
- **指征**：**里水**——水肿+有汗+口渴+小便不利+肌肉酸痛（素有湿）
- **原理**：越婢汤证 + 湿家体质 → 加白术去湿

### 麻黄加术汤
- **指征**：太阳伤寒 + 身重而酸（非骨节痛）→ 麻黄汤证 + 湿
- **应用**：瘾疹/荨麻疹（运动出汗+吹冷气→毛孔闭+皮下有水→一热就痒）
- **演变**：风久不愈→痂癞（皮肤粗糙）

### 桂枝茯苓丸（推导过程）
- **病机**：表实（不爱流汗）+ 胃气不足（冷水化不掉）+ 少腹寒 → 水寒结合 + 瘀血 → 子宫肌瘤
- **拆方思路**：
  - 茯苓 → 祛水
  - 桂枝 → 心阳下导子宫（去寒）
  - 丹皮+桃仁 → 化瘀血
  - 白芍 → 止痛（"不通则痛"）
- **加味**：寒更盛→附子（炮/生）；水不去→硫磺

## 五、脉诊核心

| 脉象 | 含义 |
|------|------|
| 寸口沉滑 | 阳（上焦）有水气 |
| 寸口沉紧 | 沉=水，紧=寒 → 结在关元 |
| 趺阳脉微而弦 | 胃气不足 + 有水 |
| 少阴（尺）沉滑 | 里实 → 血结胞门（子宫肌瘤） |
| 脉沉迟 | 里+寒（正水） |

## 六、血分 vs 水分（妇科水肿鉴别）

| 类型 | 顺序 | 预后 | 病机 |
|------|------|------|------|
| **血分** | 先停经 → 后水肿 | **难治** | 子宫内有瘀血/寒（实证） |
| **水分** | 先水肿 → 后停经 | **易治** | 水肿堵住月经通道，去水即经自下 |

---

## 七、妇科肿瘤病机（完整推理链）

1. 表实（不流汗/冷气房）→ 胸腔热散不掉
2. 胃气不足 → 喝的水化不掉 → 水停
3. 少腹寒冷 → 经血排不干净（寒则不动，如冰川）
4. 水 + 寒 + 瘀血 → 结在胞门 → **子宫肌瘤/巧克力囊肿**
5. **诊断**：尺脉沉滑 + 三阴交压痛 + 耳穴子宫区压痛
6. **疗效标准**：脚热起来 = 寒去了 = 好了

---

## 八、阳虚水肿 + 误治链

**时间线**：年轻时阳盛不觉 → 阳衰后阴上冲（肾气沿冲脉上冲咽喉）→ 医生误诊为留饮→大下→不愈→误吐→伤胃津→虚烦咽燥→小便不利→面目手足浮肿→再用葶苈丸下水→稍好→饮食过度→复肿→胸胁苦痛如奔豚→咳喘逆

**治序原则**：先攻击冲气令止 → 乃治咳 → 咳止喘自差 → **先治新病，病当在后**

## 九、新学到的重要概念

1. **麻黄止汗而非催汗**：有汗+咳嗽用越婢汤（含麻黄），桂枝汤无咳嗽
2. **喉病与肾有关**：肾津液沿冲脉上至喉咙，阳衰阴冲→喉咙梗塞（看似发炎，实为阴上冲）
3. **关元的重要性**：小肠募穴→相火→点火肾脏命门火；阳气衰→阴失控上冲
4. **寒湿→肿瘤的因果链**：寒湿不除→癥瘕（子宫肌瘤/卵巢囊肿/子宫颈癌皆寒湿）
5. **渴vs不渴的寒热辨证**：寒→不渴；热→渴。治疗后"由不渴变渴"=寒化热=好转
6. **"忘记喝水"的诊断价值**：里寒很盛→内有水积→不会渴→忘记喝水
7. **新水不代旧水→皮肤干燥→痈脓**：任何水肿滞留都会导致皮肤异常干燥
8. **吐法/下法的误用后果**：误下→下焦虚→元气下陷→胸满短气（横膈膜无法下降）
9. **茯苓+桂枝的药对逻辑**：茯苓去水+桂枝导心阳下行=温阳利水→治疗寒湿之根
10. **桂枝汤可作补药**：协调阴阳、固表（虚人常服）
