# 伤寒论 段14：少阴病症治（三）—— 条313~326

## 核心概念：少阴死证判断 + 六大主方

---

### 一、少阴死证（313-314）

#### 313：息高者死
- **症状**：少阴病六七日，呼吸极短——吸一点气就吐出来
- **病机**：心肺衰竭，水积满到横膈膜无法下降，肺无法完全张开
- **关键**：水肿病人呼吸急促 = 息高 = 危证
- **倪师**：少阴病开始出现死证。比例50%会死，后面有救逆方

#### 314：脉微细沉 + 烦躁不得卧 = 死
- **症状链**：
  - 脉微细而沉（细=寒，沉=里）
  - 但欲寐，汗出不烦（阳往外走，阴盛逼阳）
  - 自欲吐 + 自利（里寒湿盛）
  - 五六日后烦躁不得卧（阳要绝，阴阳相隔）
- **病机**：睡觉流汗 = 阳外走；烦躁不得卧 = 阳要离开身体
- **倪师**：不可不治，大剂四逆下去救

---

### 二、麻黄附子细辛汤（315）—— 少阴第一方

#### 方证
- 少阴病始得之，**反发热，脉沉者**
- 比例：麻黄2：细辛2：炮附子3

#### 病机
- 心肾中间寒湿，里阳不足，阳往外走
- 寒 + 湿阻在中焦

#### 药性拆解
| 药 | 作用 | 关键理解 |
|----|------|----------|
| 麻黄 | 发阳、利水 | 在这里不走表不发汗，配附细走中央 |
| 细辛 | 去寒水（下焦/小肠积水） | 性热，能化寒水；与茯苓泽泻不同 |
| 炮附子 | 固阳、去寒 | 阳要失掉时固阳 |

- 三药合用 → 不走表不发汗，跑身体正中 → **小便解，非汗解**
- 服药后小便不断排出，次数多、量大
- 吃到脉浮起来、小便正常为止

#### 大黄附子细辛汤（对比）
| | 麻黄附子细辛汤 | 大黄附子细辛汤 |
|----|----|----|
| 病位 | 中膈寒湿 | 大肠寒实 |
| 替换 | 麻黄 | 大黄（去实） |
| 适应症 | 但欲寐，脉沉细 | 便秘无感觉+不痛+小便淡白 |
| 核心 | 寒湿利尿 | 寒实攻下 |
| 停药指征 | 小便正常 | 小便淡黄+手脚温热 |

- 寒实便秘 **不痛**，阳明燥屎便秘 **绞痛**——关键鉴别点

### 三、麻黄附子甘草汤（316）

#### 方证
- 少阴病得之二三日，**无里证**，微发汗
- 比例：麻黄2：甘草2：炮附子（1枚）

#### 关键
- 刚得少阴病，里寒还没结很实 → 稍发汗没关系
- 细辛换成甘草（润肠）
- 麻黄仍要先煮去沫

---

### 四、黄连阿胶汤（317）—— 少阴虚烦失眠主方

#### 方证
- 少阴病二三日以上，**心中烦，不得卧**

#### 药方
- 黄连4：黄芩1：芍药2：阿胶3：鸡子黄2枚
- 煮法关键：黄连黄芩芍药先煮 → 去渣 → 趁滚下阿胶 → **温后下生蛋黄**（不能滚，滚了变蛋花汤）

#### 核心病机：心肾不交
- 寒湿梗在少阴（心肾中间）→ 心火无法下达 → 水火不交
- 心中那一滴血跑掉了 → 神无处藏 → 失眠

#### 药物原理（离中虚）
- 心中永远有一滴血在中间，神明藏于其中
- 黄连：苦味入心，补心（对心脏是滋补）
- 阿胶：补血
- **鸡子黄**：药性永远浮在中间 → 把阿胶导到心脏正中央，补那一滴血
- 不用蛋黄 → 无效（关键！）

#### 临床
- 倪案：脉沉细+手脚冰冷+口渴（少阴口渴）+溏泻+失眠 → 黄连阿胶汤一剂愈
- 25美金治好多年失眠

### 五、附子汤（319）—— 寒湿在里的身痛

#### 方证
- 身体痛 + **手足寒** + 骨节痛 + **脉沉**

#### 鉴别：麻黄汤 vs 附子汤
| | 麻黄汤 | 附子汤 |
|----|----|----|
| 脉 | 浮紧 | 沉 |
| 病位 | 寒在表 | 寒湿在里 |
| 手足 | 不一定 | 寒 |
| 治法 | 发表 | 利尿去寒湿 |

#### 药方（附子汤）
- 炮附子2枚（6钱，重用！）| 茯苓3 | 人参2 | 白术4 | 芍药3
- 附子汤 vs 真武汤：人参换生姜
  - 真武汤：茯苓多白术少（排水为主）
  - 附子汤：白术多茯苓少（去寒湿为主，慢慢利尿）

#### 方义
- 附子 → 管心肾
- 白术茯苓 → 去中间湿
- 人参 → 补肺（太阴传少阴，脾肺两虚）
- 白芍 → 滋阴补脾（贫血）
- 药简力专，五味同时解决心肾脾肺
- 附子必须重用（3钱不够），否则寒水化不掉

### 六、桃花汤（320-322）—— 下利脓血

#### 方证
- 少阴病下利，大便脓血（血与大便混在一起）
- 一天二三十次跑厕所
- 腹痛、小便不利（水都拉光了）

#### 药方
- 赤石脂一斤：一半煮、一半打碎生用
- 干姜一两
- 粳米一升（一汤匙）

#### 煮法
- 粳米+干姜+半量赤石脂煮到米熟 → 去渣 → 加赤石脂末生吞

#### 原理
- 赤石脂 = 涩剂，让食物停在肠道不排出去
- 与大黄附子细辛汤相反（那个要排出去）
- 西医：大肠炎 → 用抗生素，杀掉好菌留坏菌 → 治不好
- 倪案：下利一年多，桃花汤一剂愈；一年后复发，两剂再愈

#### 针灸配合
- 可刺关元、天枢、中极 + 桃花汤

### 七、吴茱萸汤（323）

#### 方证
- 少阴病，**吐得严重** + 手足厥冷 + 烦躁欲死

#### 止呕药鉴别
| 药 | 适用 |
|----|------|
| 吴茱萸 | 阴证呕吐（少阴病） |
| 生半夏 | 阳证，胃周围有水 |
| 干姜 | 温中止呕 |
| 生姜 | 散胃中水 |

#### 药方
- 吴茱萸（止呕主力）+ 人参（补津液）+ 生姜（散胃水）+ 红枣（补津液）

---

### 八、猪肤汤（324）

#### 方证
- 少阴病，下利 + 咽痛 + 心烦

#### 病机
- 寒湿挡在中间 → 心火往上走 → 咽痛
- 肾寒水往下走 → 下利

#### 药方
- 猪皮一斤，煮出胶质 → 去猪皮 → 加白蜜一升 + 白粉（米粉）五合 → 熬香

#### 原理
- 猪皮汤色白 → 入肺，补肺津液
- 用食补不用补药（补能恋邪——病邪在身时补药会让邪不走）
- 米粉 + 蜂蜜增加润泽
- 与桃花汤鉴别：桃花汤治脓血下利，猪肤汤治普通下利+咽痛

---

### 九、咽痛三方（325-326）

#### 甘草汤（325）
- 单纯咽痛 → 甘草单味滋润津液
- 轻证

#### 桔梗汤（325）
- 甘草汤不好 → 加桔梗
- 桔梗去寒湿转成的痰（白色泡沫痰）

#### 苦酒汤 / 半夏苦酒汤（326）
- **重症咽痛**：喉咙生疮，不能讲话，声音没了，吞东西都吞不下去
- 生半夏（泡在乌醋中）+ 鸡蛋清（半壳）+ 醋
- 放在蛋壳里，火上烧到起一两个泡 → 给病人含咽
- 吞不下去的 → 一吞就进去
- 最重的 → 巴豆+贝母（三物小白散）喷喉咙打洞

## 新学到的东西

1. **寒实 vs 热实便秘鉴别**：寒实（大黄附子细辛汤）不痛+小便白；热实（承气汤）绞痛+小便黄
2. **细辛去寒水 vs 茯苓泽泻利水**：细辛性热化寒水，苓泽只利不化
3. **心中一滴血**：「离中虚」——神明藏在心脏中间那滴血里，黄连阿胶汤就是补这一滴血
4. **鸡子黄的药性**：永远浮在中间，把阿胶导到心脏正中——不能用化学分析理解
5. **补能恋邪**：有邪在身时用食补不用补药
6. **身痛鉴别**：麻黄汤（脉浮紧，寒在表）vs 附子汤（脉沉+手足寒，寒湿在里）
7. **桃花汤的涩法**：赤石脂一半煮一半生用——涩剂让肠道不再排空
8. **止呕药分级**：吴茱萸（阴证）> 生半夏（阳证水）> 干姜（温中）> 生姜（散水）
