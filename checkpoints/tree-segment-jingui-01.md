# 金匮要略 · 段1：续伤寒霍乱篇

> 倪师在正式进入金匮之前，先把伤寒论霍乱篇补进来——因为宋朝拆分《伤寒杂病论》时，不属六经辨症的篇章被放入金匮，造成断裂。合并后伤寒论才是完整的六经辨症体系。

---

## 一、核心概念

### 1.1 霍乱定义
- **经典定义**：呕吐而利，是名霍乱。又吐又泻 = 霍乱。
- **不与西医霍乱对号入座**——西医拿中医名词去用，不是中医拿西医名词。
- 最早出处：《汉书·严助传》；《素问·六元正纪论》「太阴所至，中满霍乱」；《灵枢》「太阴，厥气上逆」也称为霍乱。

### 1.2 病因病机
- **核心病因**：湿 + 热 夹杂
- **地理**：北方少，南方多
- **季节**：春分→秋分（夏至到冬至，少阳相火 + 少阴君火 + 太阴湿）
- **天地模型**：地中有火（上蒸） + 太阳之火（下降） + 中间土湿 → 人处湿热交杂之中 → 霍乱
- **现代诱因**：夏夜汗多 + 凉风吹 + 冷饮冰食 → 肠胃不消化 → 食物坏死 → 上吐下泻

## 二、病机推理链

### 2.1 霍乱 vs 太阳症鉴别
| 症状 | 太阳症（纯表） | 霍乱 |
|------|-----------|------|
| 发热/头痛/身疼/恶寒 | ✅ | ✅ |
| 吐利 | ❌ | ✅（关键鉴别点） |

**结论**：太阳表症 + 吐利 = 霍乱

### 2.2 脉象鉴别
- **伤寒**：脉浮紧
- **霍乱（本当霍乱后见伤寒）**：脉微涩（已伤津液，脉变小变细）

### 2.3 病程转归
1. 霍乱吐利 → 邪在肠胃
2. 四五日后「阳转入阴」→ 病由表入里 → 必利
3. 「欲似大便而反矢气，仍不利」→ 胃气开始恢复，病转阳明 → 便鞭 → 十三日经尽而愈
4. 若不能食 → 不属阳明 → 需警惕

### 2.4 津液丧失病理链（最关键的危重链）
```
大量出汗/吐利
  → 津液干涸 → 肠中无津 → 大便堵（阳明实）
  → 继续丧失 → 利止（非病愈，是无物可利）→ 恶寒脉微（心阳不足）
  → 复利 → 亡血（血中水份外泄） → 危症 → 四逆加人参汤
```

### 2.5 阴阳模型
- **正常人**：阳在里，阴在外（阴封阳）。外表凉，内里热。
- **病理**：阴入里 → 阳被迫外走 → 内寒外热（非真热，是阳被挤出）
- 此「外热」≠ 发高烧，是心脏搏动动能（阳）被挤出体表。

## 三、关键治则与方剂

### 3.1 五方速查表

| 方名 | 主症 | 病机 | 关键药物 |
|------|------|------|----------|
| **五苓散** | 热多欲饮水 + 小便不利 + 得水则吐 | 表水停（肌肉/皮肤），中焦无水 | 泽泻(集表水)+茯苓(中焦)+猪苓(下焦)+桂枝(强心解肌)+白术(健脾去湿) |
| **理中丸** | 寒多不用水 + 胃口差 | 胃寒蠕动慢 | 人参(补津液)+干姜(温胃)+白术(去湿)+炙甘草(温中) |
| **四逆汤** | 四肢厥冷 + 恶寒脉微 | 心阳不足，热度不够 | 生附子(强心阳)+干姜+炙甘草 |
| **四逆加人参汤** | 利止复利 + 亡血 | 津液枯竭，血中水份外泄 | 四逆汤+人参大剂(救阴存胃液) |
| **通脉四逆加猪胆汁汤** | 脉微欲绝 + 汗出而厥 + 四肢拘急 | 最后阶段，津液全无 | 通脉四逆+猪胆汁(无猪胆用羊胆) |

### 3.2 桂枝汤小和之
- 吐利止而身痛不休 → 痛在肌肉 + 微有汗 → 桂枝汤
- 倪师加味：肌肉痛可加**玉竹**（单味药治肌肉痛，对应肌纤维痛症）

### 3.3 理中丸加减法（精要）
| 症状 | 加减 | 原理 |
|------|------|------|
| 脐上筑（悸） | 去术加桂四两 | 肾气动，水逆心 → 桂枝强心平冲 |
| 吐多 | 去术加生姜三两 | 生姜散胃中水止呕 |
| 下多 | 还用术 | 白术去湿→水走膀胱→利自止 |
| 悸（脐下） | 加茯苓二两 | 脐下悸 = 水在下焦 |
| 渴欲得水 | 加术至四两半 | 加重去湿力度 |
| 腹中痛 | 加人参至四两半 | 津液不够的痛（非便秘痛） |
| 寒者 | 加干姜至四两半 | 加重温胃 |
| 腹满 | 去术加附子一枚 | 下焦寒湿重 → 炮附子去下焦寒 |

### 3.4 止利终极法
- **白术止利 = 利尿止利**：白术去肠胃湿 → 水从小便走 → 大便自然收干
- **警告**：便秘引起的下利（大承气汤症）用白术会加重→需先辨虚实（拒按=实，喜按=虚）

## 四、倪师核心讲法（新学到的）

### 4.1 心脏 = 火 = 摩擦力
- 心跳速度人人一样，但**摩擦力（热度）不同** → 心脏强弱
- 桂枝 = 加强动脉血液循环速度
- 肉桂 = 加强喷射力量（涡轮引擎）
- 炮附子/生附子 = 加强摩擦力（热度）
- **脚冷 = 心脏摩擦力不够**（脚趾头是最远末梢）
- 冷到膝盖 = 极限 → 四逆汤症

### 4.2 生附子 vs 炮附子配伍
- **生附子 + 干姜**（如四逆汤）：强心阳，救逆
- **炮附子 + 生姜**：一般温阳

### 4.3 人参用法
- 人参 = 救阴、存胃液
- 救逆时大剂量用（一两二两），不省——人命千金
- 轻症可用党参替代；真正救命用高丽参

### 4.4 胃寒 vs 生姜/干姜
- **干姜**：温胃，治胃寒（蠕动慢、不渴、吃不下）
- **生姜**：散胃中水，治停水恶心

### 4.5 肚脐上下悸
- **脐上悸**：肾气动 → 水逆心 → 去白术加桂枝
- **脐下悸**：水在下焦 → 加茯苓

### 4.6 寒则拘急，热则纵
- 中风两种：手脚绷紧（寒）、手脚瘫软（热）
- 四肢拘急 + 手足厥冷 → 寒 → 四逆汤

### 4.7 三个阴骨
- 头顶聪骨（囟门）：封口后有记忆
- 耳后智慧骨：越大越聪明；扁=笨；不均匀=蒙古痴呆
- 膝盖骨：1岁后长出来才能走路
- 这三个是人体最末的骨，冷到膝盖=极限

### 4.8 正常人的「虚」
- 天空是空的 → 常人是虚的 → 空灵（灵感、思虑、意志）
- 食物进→谷道出→通畅 = 空灵之人 → 保持大便通畅

### 4.9 西医院陷阱
- 心电图机只看跳动、不看温度
- 病人手脚冰冷+心痛 → 心电图正常被赶回家 → 晚上发作再来 → 死在心电图机上
- **相信自己的感觉，手脚冰冷就动手**
