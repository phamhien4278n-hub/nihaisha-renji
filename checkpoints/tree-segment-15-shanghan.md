# 伤寒论 段15：少阴后半→厥阴开篇

## 核心概念

### 少阴咽痛三层次
- **轻**：扁桃腺发炎 → 巴豆贝母（三物小白散）——小题大做
- **中**：喉咙化脓发炎、还能吞东西 → **半夏散及汤**（生半夏+桂枝+炙甘草），生半夏去水=去病毒游泳池
- **重**：完全吞不下东西 → **半夏苦酒汤**（生半夏泡米醋+鸡蛋清）

### 中医核心理念：水 vs 血
- 中医从头到尾谈**水**，西医谈**血**
- 喉咙发炎生疮 = 喉咙有水 → 病毒在水里游泳很开心
- 生半夏下去 → 水没了 → 病毒离开 → 炎症好了
- **不是消炎药，是去水药**

### 生半夏用法
- 必须生半夏，制半夏无效
- 甘草可解半夏毒，生姜也可解
- "婆婆药"——给唠叨婆婆吃生半夏→无声；再给生姜→恢复

## 关键治则与病机

### 白通汤 vs 桃花汤（倪辨·015）
- 白通汤：**寒利不止**（下利清谷、脉微、手足厥逆），葱白通阳
- 桃花汤：下利便**脓血**（无热、滑脱），赤石脂涩肠
- 问肛门痛不痛 → 痛=热利；不痛=寒利

### 四逆汤 vs 通脉四逆汤（倪辨·007）
- 唯一区别：**干姜剂量**
- 四逆汤：干姜一两半
- 通脉四逆汤：干姜三两（加倍）——里寒更盛时用
- 通脉四逆可加：葱九茎（面色赤）、芍药（腹痛）、桔梗（咽痛）、人参（利止脉不出）

### 白通汤加猪胆汁汤——终极一搏
- 白通汤止不住 → 里面**阳已绝**，消化能力没了
- 猪胆汁：暂时取代人的胆，帮助消化
- 人尿（回龙汤/童子尿）：原汤化原食，帮助消化 + **强命门火**
- 服后脉**暴出者死**（身体不受药，阳绝）→ 微续者生
- **倪案·014**：徐济民老师闪腰喝童子尿，20分钟命门跳动→腰痛痊愈
- **倪诀·013**：原汤化原食

### 真武汤 vs 附子汤
- 真武汤 = 附子汤 − 人參 + 生姜
- 真武汤：茯苓三 > 白术二（利水为主）
- 附子汤：白术 > 茯苓（去湿为主）
- 真武汤证：头重脚轻、振振欲擗地——躺下也晕

### 通脉四逆汤——动脉血管剥离预测
- 里寒外热 → 寒在中焦（心肾之间）→ 像冬天水管结冰爆裂
- **寒会爆裂血管**！这就是为什么中焦寒湿会导致动脉血管剥离
- 出现少阴证 → 马上下手温中（干姜）→ 可**预防**破裂
- 面色赤（戴阳）= 里阳往外走，危险信号——两颧红赤，非均匀红润

### 四逆散——胆结石专方
- 心包脉直接络胆 → 胆汁喷射速度=心跳速度
- 中焦寒湿 → 不消化 → 里干 → 胆汁浓缩 → 胆结石
- 四逆散无热药！柴胡疏肝 + 枳实扩管 + 芍药止痛 + 甘草
- 加滑石、五倍子、海金沙 → 胆结石加强版
- **地域规律**：沿海吃醋多=肾结石；内陆吃辣多=胆结石

### 少阴转阳明——从阴回阳是好现象
- 阴证中出现阳明证（口燥咽干、腹胀不大便）= 病人从阴回到阳
- 大承气汤证：自利清水色纯青=燥屎堵死只过水；心下痛+口干燥+小便深黄
- 寒实便秘 vs 热实便秘：
  - 大承气汤：腹痛拒按、小便深黄、潮热
  - 大黄附子细辛汤：不大便无所苦、小便淡白、脉沉细

### 胸中饮实 vs 膈上寒（338条鉴别）
- 饮食入口则吐 + 手足寒 + 脉弦迟 + 脸猪肝色 = **胸中饮实**（东西梗到）→ 吐之
- 干呕 + 脸色苍白 + 虚弱 = **膈上有寒** → 不可吐，四逆汤温之

### 厥阴病开篇（340条）
- 厥阴 = 肝 + 心包，最后一道防线
- **消渴，气上撞心，心中疼热，饥而不欲食，食则吐蚘**
- 厥阴 = 寒热并见：热多寒少有救，寒多热少完蛋

## 倪师临床思维精华

### 生半夏——去水即消炎
> "喉咙里面有水在那边，病毒在里面游泳很开心。生半夏下去，游泳池的水当场没了，病毒说我才不干呢，我离开就好了。"

### 葱白的作用
> 上下通利时用葱白。白通汤不用炙甘草——对下利没帮助。

### 脉证相合
> 虚证+脉微=相合，白通汤即可。最怕脉证不合：人很虚脉很大，或人很壮脉很小→危险。

### 男怕脚肿，女怕头肿（倪诀·010）
> 男人脚肿=逆证=危险（活不过多久）；女人头肿=逆证。反过来女人脚肿三四十年也没死。

### 烦躁=危重信号（倪问·011）
> 出现烦躁必须开处方。手足躁扰+四逆=死证。越烦躁越危险。

### 手脚温者不死（倪诀·008）
> 手脚温度=胃气存亡指标。冷到手肘膝盖=四逆=危险。

## 薄弱点（自评）
- 真武汤加减法中"呕者去附子加生姜"——为何去附子？因为呕=胃停水，生姜散水即可，附子温下焦与此无关
- 四逆散条文位置可疑——倪师自己也觉得"很奇怪""可能放错条辨"
- 猪胆汁+人尿不煮，后下——区别于前面三味药
