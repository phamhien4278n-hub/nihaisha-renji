# 金匮要略 第33段 知识树

## 段33：妇人篇核心方证（半夏厚朴汤·甘麦大枣汤·温经汤·带下治则·土瓜根散·胶姜汤·大黄甘遂汤开头）

### 一、半夏厚朴汤（梅核气）

#### 核心症状
- 咽中如有炙脔——喉咙有梅核气哽住，咳不出吞不下
- **特殊鉴别症状**：「好像有人抓住我的心脏」→ 听到这句直接开半夏厚朴汤

#### 病机推理链
```
胃中有停水（妇人不运动+喝水→水停胃）
→ 水寒→水火不相容→水反扑心脏（火被水浇）
→ 感觉心被握住
同时：胃阻+郁闷→气逆痰阻→喉咙异物感
```

#### 方药与方义
| 药 | 作用 | 为什么 |
|----|------|--------|
| 半夏（一升/五钱） | 祛痰祛水 | 核心药，痰和水同时祛 |
| 厚朴（三两/三钱） | 行气降逆 | 喉部是气脉所过 |
| 茯苓（四两/四钱） | 祛胃中停水 | 配生姜去胃水 |
| 生姜（五两/三片） | 去胃中停水 | 胃水之源 |
| 苏叶（二两/二钱） | 宣肺行气 | 轻药，只宣喉咙不象麻黄动全肺 |

#### 新学到的东西
1. 「有人抓心脏」= 半夏厚朴汤的**特异性鉴别症状**，听到即用
2. 苏叶 vs 麻黄：苏叶只宣喉咙，麻黄动全肺——精准用药的范例
3. 为什么放妇人篇？古代妇人足不出户，情绪郁闷→痰气交阻→此证
4. 药量注意：苏叶二钱已经很大量（非常轻），比半夏五钱体积还大

### 二、甘麦大枣汤（脏燥）

#### 核心症状
- 妇人脏燥，悲伤欲哭（无法控制，眼泪不停流）
- 象如神灵所作，数欠喜伸

#### 病机推理链
```
子宫（脏=子宫，不是五脏）干燥
→ 津液从冲任脉无法下行
→ 回逆上去→从眼睛出→无故流泪
```

#### 关键扩展：小儿癫痫！
- 原理：「肝苦急，急食甘以缓之」
- 腹诊特征：腹筋一条条直的（腹筋急）→ 甘麦大枣汤证
- 病因：高烧后脑损→癫痫

#### 甘麦大枣汤下利现象（重要！）
- 吃甘草+小麦+大枣→怎么会拉肚子？
- 原理：甘味→甘生辛→生金（肺）→新津液生成→取代旧浊痰→**下利排出**
- 这是「病在下，采下法」的自然体现，不是药力攻下
- 吃到腹筋不再急、癫痫不再发为止

#### 方药与安全性
- 甘草三两 + 小麦一升 + 大枣十枚
- 全是食物级甘味药，零毒性
- 亦补脾气

### 三、妇人吐涎沫误下治则

#### 证治链条
```
上焦寒湿→吐涎沫→应去寒去湿（小青龙汤）
→ 医误下→心下痞
→ 治则：先治吐涎沫（小青龙），再治心下痞（泻心汤）
→ 先解表再攻里 = 大原则
```

#### 关联知识
- 五个泻心汤来源：表证误下→轻则心下痞，重则悬饮（十枣汤）
- 选方：呕多用半夏泻心汤；下利多使用生姜泻心汤

### 四、妇人之病总论（三大病因）

#### 三大病因
1. **虚**
2. **积冷**
3. **结气**

#### 病机演进链
```
虚+积冷→血寒积结胞门→经水断绝（历年）
→ 在上：寒伤经络→凝坚→呕吐涎唾→久成肺痿→形体损分
→ 在中：盘结→绕脐寒疝→两胁疼痛→与藏（子宫卵巢）相连
→ 结热中→痛在关元（阳要入阴进不去）
→ 在下来多→经候不匀→阴掣痛→少腹恶寒→引腰脊→膝胫疼烦
```

#### 阳不入阴的核心概念
| 表现 | 机制 |
|------|------|
| 关元痛（经期） | 阳要入阴，里面太冷进不去→痛 |
| 脉数无疮 | 阳被挡回逆→脉数，但无脓疡（摸皮肤无局部烫） |
| 皮肤粗糙如鱼鳞 | 阳往外走 |
| 女穿男装（阳亢） | 阳不入阴，阳往外亢 |
| 悲伤多嗔/看到鬼 | 非鬼神，是带下病（子宫有瘀） |

#### 四逆汤治闭经案例
- 女儿两年没月经+手脚冰冷→当归四逆汤加减→第二天来
- 经方逻辑：**寒积→血瘀=冰块→需要热药化掉**
- 生附子(强心)+干姜(温肺胃)+炙甘草→心脏热度直入小肠→子宫瘀血化掉
- **不要看到没月经就攻血——先辨寒热！**

#### 治则
- 积冷 → 热药化之
- 结气 → 散郁药开之
- 日久不治 → 人越来越瘦（冷结在里）

### 五、带下补充（临床实用处方）

#### 大前提：无湿不成带

#### 湿热型
| 特征 | 色黄、深，味重，体胖 |
|------|----------------------|
| 处方 | 黄连2 + 黄芩3 + 黄柏3 + 白术3 + 赤豆3 |
| 方义 | 黄连去心火小肠热；黄芩去肝经湿热；黄柏去膀胱下焦热；白术+赤豆去湿 |
| 加减 | 痔疮加槐花+当归（变当归赤豆散）；虚热加生地+栀子 |

#### 寒湿型
| 特征 | 白带、无味、滴漏（阳不足不能固守） |
|------|--------------------------------------|
| 处方 | 炮附子3 + 白术3 + 茯苓4 + 泽泻5 + 猪苓3 |
| 方义 | 炮附子去寒固阳；白术去湿；茯苓＞白术量（利小便从下焦出）；泽泻+猪苓利下焦 |
| 关键 | **附子要放胆开**（3-8钱），量小没感觉→病人以为药错 |

#### 寒湿vs湿热去湿路径不同
- 湿热：清热+去湿，不走小便
- 寒湿：利小便（茯苓大于白术），从小便利出

#### 针灸
- 三阴交、血海、中极、关元
- **隐白**：白带特效穴——「隐白，就是把白隐起来」
- 针下去白带就没了，药还没煎好

#### 情绪因素
- 女子有带大部分是情绪不好（先生不好/离婚等）
- 可加郁金、甘麦大枣汤等

### 六、温经汤（经方最著名调经方）

#### 主症
- 妇人年五十，下血数十日不止（更年期仍流血不止）
- 暮即发热（黄昏发热→肺无津液，心火上走）
- 少腹里急，手掌烦热，唇口干燥

#### 病机
- 曾经半产→瘀血在少腹不去
- 辨证关键：**唇口干燥 = 内有瘀血**

#### 方药（12味）
| 药 | 量 | 作用 |
|----|-----|------|
| 吴茱萸 | 三两 | 补肝虚，入肝经→络子宫（头痛呕吐亦用） |
| 当归 | 二两 | 补血+通便（油多），虚秘加重 |
| 川芎 | 二两 | 活血 |
| 芍药 | 二两 | 止腹痛，有瘀可加重 |
| 人参 | 二两 | 补气 |
| 桂枝 | 二两 | 行阳，导心热入小肠 |
| 阿胶 | 二两 | 补血止血，虚证能让月经来（实证不宜） |
| 丹皮 | 二两 | 活血化瘀（子宫） |
| 生姜 | 二两 | 行阳 |
| 甘草 | 二两 | 调和 |
| 半夏 | 半升 | 祛痰（妇人忧郁多痰） |
| 麦门冬 | 一升 | 润肺补津液→解决暮发热 |

#### 方义整合
```
桂枝+生姜+甘草→强心行阳
麦门冬→补肺津液（肺为水之上源）
桂枝→导心热入小肠
吴茱萸→补肝虚→肝经络子宫
当归+川芎+芍药+阿胶→补血活血
丹皮→子宫化瘀
半夏→祛痰
→ 全方：强心+补肺津液+导心热下行+活血化瘀+补血
```

#### 主治范围
- 少腹寒，久不受胎（助孕！100%生男孩）
- 崩中去血
- 月水来过多
- 月水不来（至期不来）
- **不限于50岁，月经不调即可用**

#### 关键加减
- 虚+便秘 → 加重当归（油多通便）+芍药（腹痛）
- 当归保存需冷藏，无油=无效

#### 阿胶虚实辨
- 虚证（人羸瘦）：阿胶补血→月经来
- 实证（泼妇骂街型）：阿胶下去→月经不来+胸闷→发狂

### 七、土瓜根散（实证，卵巢问题）

#### 主症
- 经一月再见（隔月来月经）→ 一侧卵巢功能没了

#### 病机
- 有东西堵到卵巢→实证（与温经汤虚证相对）

#### 方药（粉剂）
- 土瓜根 + 芍药 + 桂枝 + 蟅虫（各三分）
- 粉剂原因：攻坚要专一，慢慢渗透，不象汤剂跑太快

#### 现代替代
- 土瓜根难找→三菱、莪术、桃仁、丹皮
- 蟅虫=攻瘀动物药（大黄蟅虫丸同类）

#### 治到何时停
- 每个月月经正常来即停

### 八、胶姜汤（血崩）

#### 主症
- 妇人陷经漏下，黑不解（大量出血止不住，血色黑）

#### 病机
- 产后胎盘未完全清出/外伤→血崩
- 里寒→崩漏

#### 处方
- 胶艾汤去艾叶 + 加干姜
- 去艾叶的原因：艾叶入血脉神经→可能增加出血量→止血时要去掉
- 加干姜：里寒导致崩漏

### 十、本段核心逻辑串联

```
妇人病三大因：虚→积冷→结气
                ↓
        血寒积结胞门（阳不入阴）
                ↓
    ┌───────────┼───────────┐
    ↓           ↓           ↓
  在上          在中        在下
  肺痿          寒疝        带下/经乱
  (干姜)      (热药)    (辨证寒湿/湿热)
                ↓
        虚证→温经汤（唇口干燥）
        实证→土瓜根散（经一月再见）
        急证→胶姜汤（血崩黑不解）
```
