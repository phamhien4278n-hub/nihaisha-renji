## 第一部分：肺癌案例（倪师临床实录）

### 病人基本情况
- 肺癌第3~4期，持续咳嗽6个月
- 可平躺（肺无积水），胃口好，不怕（对治疗最有利）
- **关键症状**：
  - 夜间身热（阳不入阴→阴实阻隔）
  - 凌晨3-5点必醒
  - 下痢一天20+次（阴阳相隔：上热下寒，阳反逆→肛门守不住→阴脱）
  - 咳白泡沫痰、黄痰、带血
  - 服阿司匹林后开始咳血（阿司匹林破坏血管壁）

### 脉诊
- 左手无脉，右手洪大
- **未见附骨脉**（倪师纳闷，可能因做过切片或病情未到极恶阶段）

### 经方治癌三原则
1. **阻绝水**（利水）：十枣汤/甘遂半夏汤/五苓散等——癌细胞无水不能生存
2. **阻绝营养**：鼓励吃原始食物（糙米、荞麦、五谷杂粮）——癌细胞吃不到精制营养
3. **改变环境**：不杀癌细胞，改变体内环境使之不适合癌细胞生存

### 首诊处方（2007.2.20）
| 药物 | 剂量 | 作用 |
|------|------|------|
| 桔梗 | 一两 | 排脓汤核心，须≥一两才有效 |
| 炙甘草 | 三钱 | 配桔梗 |
| 皂荚 | — | 碱性去烟痰烟垢 |
| 红枣 | — | 护胃壁，防皂荚碱性伤胃 |
| 生半夏 | 四钱 | 涤水去痰力强，去肺寒湿 |
| 干姜 | 三钱 | 解生半夏毒+温中去寒 |
| 紫菀 | — | 代紫参，去寒痰 |
| 款冬花 | — | 排肺寒 |
| 麻黄 | — | 射干麻黄汤化裁 |
| 射干 | — | 射干麻黄汤化裁 |

- **合方思路**：桔梗甘草汤 + 射干麻黄汤 + 寒热并用
- 九碗煮三碗（久煮力强），五付吃一周
- 干姜+炙甘草：温中→肺寒上不制下→止下痢

### 二诊（一周后）
- 咳嗽大好转，胸痛大减
- 仍有炽热感、白痰多、咳血块
- 新增：**丹皮+桃仁**（活血化瘀，因阿司匹林致肺内瘀血）+ **冬瓜仁**（苇茎汤化裁，排脓化瘀）
- 嘱停阿司匹林

### 预后
- 持续好转中，倪师判"要死还很难"
- 紫参若可得，处方不必如此复杂

## 第二部分：痰饮咳嗽病脉证并治第十二

### 一、四饮定义

| 饮名 | 病机 | 主症 | 治法/方剂 |
|------|------|------|-----------|
| **痰饮** | 水走肠间 | 素盛今瘦，沥沥有声 | 温药和之 |
| **悬饮** | 水流胁下 | 饮后水流胁下，咳唾引痛 | **十枣汤** |
| **溢饮** | 水归四肢 | 当汗不汗，身疼重 | **麻黄加术汤**（发汗） |
| **支饮** | 水在肺 | 咳逆倚息，不得卧，形如肿 | 温阳利水 |

### 二、饮的形成
- 喝水太快/运动后冷气间不出汗 → 水饮停积内脏下方（心/肝/肺下方、横膈处）
- 判断法：每次感冒都很严重 → 有饮病
- 预防：运动出汗→喝水→新水代旧水

### 三、水在五脏鉴别

| 病位 | 症状 | 方剂思路 |
|------|------|----------|
| **水在心** | 心下坚筑，短气，恶水不欲饮 | 茯苓杏仁甘草汤 |
| **水在肺** | 吐涎沫（白泡沫），欲饮水 | — |
| **水在脾** | 少气身重（少腹水→吸气不纳） | 桂枝汤+白术茯苓 |
| **水在肝** | 胁下支满，嚏而痛 | 小柴胡汤+半夏重用 |
| **水在肾** | 心下悸 | 苓桂甘枣汤/五苓散 |

### 四、三级饮病深度

| 层级 | 名称 | 特征 |
|------|------|------|
| 浅 | **饮** | 四饮（痰/悬/溢/支） |
| 中 | **留饮** | 水停留不去 |
| 深 | **伏饮** | 潜伏无症状，一感冒就极重 |

### 五、留饮（两种）

#### 心下留饮（胃下方有水）
- **背寒冷如掌大**（胃的大小区域）
- 胁下痛引缺盆，咳嗽则辄已（咳时水移开，暂缓解）
- 与小青龙汤证区别：小青龙→背寒在两肩胛骨间如拳大（更高），由趴着工作受寒引起

#### 胸中留饮（胸腔有水）
- **短气而渴**（冷水阻隔，津液不能上达→口渴）
- 四肢历节痛，脉沉
- 平时肌肉酸/关节紧 → 已有留饮 → 应运动出汗排掉

### 六、伏饮
- 膈上病：痰满喘咳吐 → 发则寒热、背痛腰疼、目泣自出、振振身瞤
- 判断：平时无症状，一感冒极严重 → 有伏饮
- 策略：趁未发病时去伏饮

### 七、饮水生理（关键概念）
- **饮料水** → 胃 → 气化 → **三焦/淋巴系统**（血管外）
- **食物水** → 小肠 → 大肠 → 小肠火蒸化 → **入血**（血管内）
- 食少饮多 → 胃火不足 → 水停心下 → 动悸（重）/短气（轻）
- 大下后里虚 + 饮冷水 → 饮病源头

### 八、核心治则
> **病痰饮者，当以温药和之。**（温阳利水）

### 九、苓桂术甘汤

**组成**：茯苓三两、桂枝三两、白术三两、甘草二两
**煮法**：水六升煮三升，分温三服，小便则利

**主证**：心下有痰饮 → 胸胁支满，目眩

**倪师天平比喻**：横膈膜=天平基座，水饮在基座→晃动→上面不平衡→眩晕

**临床应用**：
- 神经衰弱
- 心脏肥大/动悸
- 精神病头晕（走路摇晃站不稳）
- 耳聋伴眩晕
- 晕车晕船

**药性分析**：
- 桂枝+甘草 = 辛甘发散为阳
- 白术去湿，茯苓利水
- 行胃阳+行心阳

**与五苓散区别**：
- 苓桂术甘汤：眩晕但不渴，小便正常（有时不利但非主证）
- 五苓散：渴+小便不利（水停皮肤表面，中焦缺水）

**短气有微饮**：苓桂术甘汤 OR 肾气丸（桂附八味丸——肉桂+附子温阳利水）

### 十、甘遂半夏汤

**组成**：甘遂大者三枚、半夏十二枚、芍药五枚、甘草如指大一枚
**煮法**：水二升煮半升→去滓→加蜜半升→煎取八合→**顿服**

**主证**：脉伏，自利反快（下利后暂时舒服），利后心下续坚满（水又回来）
→ **此为留饮欲去**（留饮在深处，清掉后又积回来）

**与十枣汤区别**：
| | 十枣汤 | 甘遂半夏汤 |
|------|--------|-------------|
| 病位 | 肺底积水 | 肋间/肋膜积水 |
| 能否平躺 | **不能**，必须坐 | **可以**平躺 |
| 典型感觉 | — | 上身下身"分开"，中间有东西梗到 |

### 十一、阴阳药配对原则
- 辛+甘 = **阳药**（桂枝+甘草）
- 酸+苦 = **阴药**（白芍+黄芩/黄连）
- 治病=调和阴阳

## 本段新学到的东西
1. 经方治癌三原则（利水→断营养→改环境），不直接"杀"癌
2. 夜间燥热=阳不入阴（阴实阻隔），热退=好转，热增=恶化
3. 肺癌下痢=阴阳相隔（上热下寒），非便秘
4. 皂荚必配红枣（碱性护胃），生半夏必配干姜/生姜（解毒）
5. 桔梗排脓须≥一两才有效
6. 饮料水走三焦/淋巴，食物水走血——两个不同系统
7. 四饮→留饮→伏饮三级深度，伏饮=平时无症状一感冒就极重
8. 苓桂术甘汤天平比喻——横膈水饮→眩晕
9. 十枣汤 vs 甘遂半夏汤：能否平躺是鉴别关键
10. 甘遂半夏汤顿服法——蜜煎减少刺激性
