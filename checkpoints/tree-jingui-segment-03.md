# 金匮要略 段3 知识树

## 一、脏腑经络先后病脉证（续）

### 1. 呼吸辨证法
- **息摇肩** → 心中坚（心脏问题，呼吸不深须摇肩才能吸气）
- **息引胸中上气** → 咳
- **张口短气** → 肺痿唾沫（肺萎缩，津液不能下输肾→上逆为唾沫）
- 原理：肺为津液之源（金生水），肺病→津液不下反上

### 2. 吸而微数——中焦辨虚实
- **吸而微数，病在中焦，实也** → 当下之即愈
- **虚者不治**：实不在肠胃（可能是胰脏癌/肝癌等旁边器官的实）
- **大黄甘草汤证**：朝食暮吐（胃下方食物堵）——大黄提出来+甘草补胃津
- **调胃承气汤证**：堵在十二指肠
- 胃下方常存约两分食物（灵枢：光喝水可活七天，靠此段食物）

### 3. 脉色四时相应
- 寸口脉（阳脉）随四时旺时而动
- 肝旺色青，四时各随其色
- **非其时色脉 → 皆当病**
- 局部色变轻，全脸色变重

### 4. 未至而至（节气异常→灾病预测）
- 冬至后甲子夜半少阳起
- **未至而至**（冬行春令，应冷反暖）→ 水未凝聚→夏天大水
- **至而不去**（春仍冬令）→ 水气早散→夏天必旱
- 过和不及皆不好
- 应对：节气不对时身体沉重→练易筋经发汗，病在表去之

### 5. 脉浮辨表里
- 浮在前（寸/阳脉）→ 病在表
- 浮在后（尺/阴脉）→ 病在里（腰痛背强不能行）
- 尺脉浮到尺泽 → 下半身瘫痪/脚不良于行

### 6. 厥阳独行（有阳无阴）
- **定义**：有阳无阴=厥阳，孤阴不生独阳不长→死证
- **肝癌阴实模型**：
  - 肝为阴脏，实堵在内→阴实
  - 丑时（1-3am）肝经，阳要入阴被实挡→阳回逆→发热
  - 病进：热时扩大（1-3→3-5→5-7→...）
  - 病退：热时缩小，身渐凉
  - 末期只余下午3-5点凉快，其他全热
  - 无法躺下、眼瞪大（肝开窍于目）→ 阴阳相隔
- **治则**：去实 + 阳药打开阴

### 7. 卒厥（中风）
- 寸脉沉大而滑 → 沉为实、滑为气 → 气血两旺往上冲脑
- **入脏即死**：唇口青（脾寒）、身冷 → 血不下停在脑
- **入腑即愈**：身和、汗自出
- **急救**：百会放血 + 十宣放血（头挂下，桶接血，泄实气）

### 8. 脉脱·病进退原则
- 百病皆然
- **浸淫疮**：从口→四肢可治；从四肢→口不可治
- 病在外可治，入里难治
- 针灸验证：痛从背→腿→小腿→脚 往外走=吉
- **艾灸治破伤风**：红线前一寸隔姜灸，灸一寸退一寸，逐寸逼退至指尖
- 艾草可解毒

### 9. 阳病十八·阴病十八（统计性概述）
- 阳病十八：头痛、项腰脊臂脚掣痛（外症）
- 阴病十八：咳上气喘哕咽肠鸣胀满心痛拘急（内脏病）
- 五劳七伤六极、妇人三十六病不在其中
- 不必死记，大致了解

### 10. 五邪中人规律
- 清邪居上，浊邪居下；大邪中表，小邪中里
- 风中于前，寒中于暮；湿伤于下，雾伤于上
- 风令脉浮，寒令脉急，雾令皮腠，湿流关节
- 食伤脾胃，极寒伤经，极热伤络

### 11. 急当救里救表
- 误下后**下利清谷不止+身体疼痛** → 先救里（阳虚）
- 清便自调后身体疼痛 → 再救表
- 肛门收缩=阳足；肛门开张大便直出=阳虚（吊死者大小便出=真死）
- 表里可同治时不冲突（如葛根汤+承气汤）

### 12. 猪苓汤治渴
- 肾有实堵 → 阳不入阴→发热→口渴
- 药物对应：猪苓（小腹突）、茯苓（肚子大）、泽泻（脸肿）、滑石（排石）、阿胶（止血）
- 诊断技巧：针灸腰痛→委中阴谷后仍痛→可能有肾结石→压脚踝络痛点+猪苓汤

## 二、痉湿暍病脉证治法第二·痉病

### 核心概念
- **痉** = 脑膜炎一类（角弓反张、抽搐、口噤）
- **刚痉**：太阳病发热无汗反恶寒 → 葛根汤
- **柔痉**：太阳病发热汗出不恶寒 → 栝蒌桂枝汤
- **痉脉**：按之紧如弦，直上下行（速度极快、刚强不柔）

### 痉病来源（三大禁忌）
1. 太阳病发汗太多 → 津液丧失 → 痉
2. 风病误下后又发汗 → 抽筋拘急
3. 疮家（脓疮灸疮）发汗 → 血虚致痉

### 痉病诊断
- 身热足寒，颈项强急，恶寒
- 时头热，面赤目赤
- 独头动摇，卒口噤，背反张（角弓反张）
- **暴腹胀大 = 欲解**（津液回肠胃）
- 肚子凹下 = 津液冲脑抗邪
- 痉脉沉细 = 难治（病往里走）
- 痉病有灸疮 = 难治（外用：白术去湿+黄连去热+知母安神）

### 栝蒌桂枝汤（柔痉·有汗）
**组成**：栝蒌根二两 + 桂枝汤（桂枝三两、芍药三两、甘草二两、生姜三两、大枣十二枚）
- 栝蒌根（天花粉）→ **生津**止渴（不是升津，葛根是升津）
- 病机：血25%+水75%，水不够→血高→发热抽筋
- 服法：九升煮三升，分三服，取微汗；汗不出喝热粥
- 生姜去胃停水、甘草解百毒、红枣补胃液（换掉生病期间的旧津液）

### 葛根汤（刚痉·无汗）
**组成**：葛根四两、麻黄三两（去节）、桂枝二两、芍药二两、甘草（炙）二两、生姜三两、大枣十二枚
- 比例：4:3:2（葛根:麻黄:桂枝/芍药）
- 病机：无汗+小便少=热闷在内→壮热上冲→口噤不得语
- 三药协作：葛根升提津液→桂枝推到肌肉表→麻黄开毛孔
- 原方先煮麻黄葛根去沫再纳诸药（倪师减麻黄量省此步骤）
- 服后不须啜粥（与桂枝汤不同）
- **汗尿同源**：天生毛孔密实者服后小便增多=等于是汗解
- 麻黄去节（节止汗、身发汗）
- 炙甘草：解毒+**强心**
- 恢复期禁食肉类（难消化），吃稀饭+好消化青菜

### 重要临床经验
- 冬天/寒带出生者毛孔天生密实→不易流汗→运动后昏倒=葛根汤证
- 痉病已发作（无法问有汗无汗）→手摸：有汗=栝蒌桂枝汤，无汗=葛根汤
- 控涎丹：痰多致痉（如新生儿痰水未清）→去痰即愈
