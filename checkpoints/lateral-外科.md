# 横向打通：外科

> 生成时间：2026-07-01
> 症状：外科（涵盖疮痈、肠痈、肺痈、金疮、痔疮、乳痈、瘰疬、浸淫疮等）
> 来源：五门课贯通 + Hindsight + symptom-classification-index.md

## 概述

「外科」在中医体系中涵盖一切体表可见或外治可及的疾病。倪师人纪五门课中，外科内容主要集中在：
- **金匮要略**：第十八篇《疮痈肠痈浸淫病脉证并治》及第七篇《肺痿肺痈》
- **伤寒论**：「疮家不可发汗」原则；抵当汤/桃核承气汤（瘀血）；桃花汤（脓血便）
- **神农本草经**：乳香、没药、黄芪、白芷、续断、王不留行、漏芦、营实等大量外科用药
- **针灸大成**：烧针（燔针）治痞/脓疡；隔蒜灸/隔姜灸/隔附子灸；痔疮（长强/承山/孔最）；乳痈（肩井+足三里）

## 关键法则汇总

### 诊断铁律
1. **有脓无脓**：手按肿上——热者为有脓，不热者为无脓（金匮18篇）
2. **病进退方向**：从外往内走=病进（凶）；从内往外走=病退（吉）（浸淫疮/破伤风/中风恢复）
3. **疮家不可发汗**（伤寒论第93条）
4. **凹陷为恶、凸起为良**（肿块形态判断）
5. **手掌褐斑=慢性腹膜炎信号**（倪师临床经验）
6. **阑尾穴压痛点**：足三里下一寸

### 排脓三方鉴别
| 排脓汤 | 组成 | 适用部位 |
|--------|------|---------|
| 术附汤 | 白术+炮附子 | 深部/骨边脓疡（推脓外出） |
| 桔梗甘草汤 | 桔梗+甘草 | 肺痈吐脓/上部脓疡 |
| 赤豆当归散 | 赤小豆(发芽)+当归 | 痔疮/腹膜炎/肠痈穿孔 |

### 灸法选择（外科）
| 证型 | 灸法 |
|------|------|
| 尚未化脓的硬块 | 隔姜灸 |
| 无味脓疡 | 隔附子灸（生附子或炮附子） |
| 腥臭脓疡（腐肉） | 硫磺+面粉+大蒜饼灸21壮以上 |
| 久咳/肺病 | 隔蒜灸 |
| 带状疱疹 | 两头隔姜灸截法 + 棉花火烧法 |
| 下肢瘫痪/久病 | 灸膏肓百壮 |

### 烧针三针法
脂肪瘤/良性痞块：第一针从上往下 → 第二针从前向后 → 第三针从后向前（烧麻油三次下针）
