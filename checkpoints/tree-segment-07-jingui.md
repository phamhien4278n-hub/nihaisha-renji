# 金匮要略 段7 知识树

## 一、疟病（续）——三类疟疾 + 三个补充方

### 1. 瘅疟（Dan Nue）—— 失津致热疟

**原文**：阴气孤绝，阳气独发，则热而少气烦冤，手足热而欲呕，名曰瘅疟。若但热不寒者，邪气内藏于心，外舍分肉之间，令人消铄脱肉。

**倪师解释**：
- 病机：病人失津（津液丧失）→ 血分水（阳）和血（阴），水分丧失后阴中阳不够 → 阴气孤绝，阳气独发外走
- 瘅疟本质 = 失津 → 进而亡血（血不足）
- 治疗原则：血不足补血，津液不足补津液
- **本条无方**——倪师说经方有残缺，但抓到"经方的神"就能治，不必执着于缺方
- 倪师认为此条位置摆得不很正常，但经方本身没错

**核心概念**：
- 津 = 血中之水 = 阳；血 = 阴
- 失津 → 阴中阳不足 → 阳气独发外走 → 但热不寒

---

### 2. 温疟 —— 白虎加桂枝汤

**原文**：温疟者，其脉如平，身无寒但热，骨节疼烦，时呕，白虎加桂枝汤主之。

**方剂**：知母六两，甘草二两（炙），石膏一斤，粳米二合，桂枝（去皮）三两。锉，每五钱，水一盏半，煎至八分，去滓，温服，汗出愈。

**倪师病机分析**：
- 但热无寒 → 必属阳明症 → 白虎汤症
- 骨节疼烦 + 恶心 → 加桂枝
- **桂枝 = 主四肢**，用树枝类药物入四肢末梢
  - 树枝 → 四肢（桂枝/桑枝）
  - 花蕾 → 鼻脑（最高处）
  - 根茎 → 肠胃（土，地下）
- 经方只用桂枝不用其他树枝

**关键治则**：但热不寒 = 阳明 = 白虎汤打底；骨节疼 = 桂枝引药到四肢

---

### 3. 牝疟（Pin Nue）—— 蜀漆散

**原文**：疟多寒者，名曰牝疟，蜀漆散主之。

**方剂**：蜀漆（烧去腥）、云母（烧二日夜）、龙骨（倪师认为应是牡蛎）。等分，为散，未发前以浆水服半钱。温疟加蜀漆半分，临发时服一钱匕。

**倪师分析**：
- 牝 = 源自易经"坤为地卦牝马之贞"
- 蜀漆 = 四川漆树树皮，极毒，粘手 → 目的是**去痰**
  - 蜀漆拿不到 → 用**常山**替代（常山提炼出奎宁）
  - 常山 ≠ 蜀漆（不同药），但可互代
- 云母烧两天两夜 → 金石类药，止呕 + 去中焦寒
  - 中焦过寒 → 极度晕眩 → 云母去中焦寒 → 头昏消失
- 龙骨不合理 → 倪师认为应该是**牡蛎**（龙骨与牡蛎性不同）
- 服法：未发病前服粉剂

**核心推理链**：寒疟 → 蜀漆（常山）去痰 + 云母止呕去中寒 + 牡蛎（非龙骨）软坚 → 未发前服

### 4. 附《外台秘要》三方

#### 4.1 牡蛎汤 —— 治牡疟
- 牡蛎四两（熬）、麻黄四两（去节）、甘草二两、蜀漆三两
- 先煮蜀漆麻黄去上沫 → 入余药 → 煮取三升，温服一升
- **若吐则勿更服**

#### 4.2 柴胡去半夏加栝蒌汤 —— 治疟病发渴 + 劳疟
- 柴胡八两、人参三两、黄芩三两、甘草三两、栝蒌根四两、生姜二两、大枣十二枚
- 即小柴胡汤去半夏 + 栝蒌根
- 适用：往来寒热 + 渴（躁渴）+ 不恶心 → 去半夏加栝蒌根
- 亦治劳疟（户外劳力工作者）
- 小柴胡汤本身就是很好的治疟处方
- 配合针灸：曲池、外关、委中

#### 4.3 柴胡桂姜汤 —— 治寒疟（但寒不热或微有热）
- 柴胡半斤、桂枝三两（去皮）、干姜二两、黄芩三两、栝蒌根四两、牡蛎三两（熬）、甘草二两（炙）
- 服一剂如神（原文标注）
- 初服微烦 → 复服汗出便愈
- **替代蜀漆散的简便方案**：找不到蜀漆/云母时用
- 病机：胃中寒湿 + 有痰
  - 干姜 → 去中膈寒湿
  - 牡蛎 → 咸能软坚，化痰
  - 痰还在 → 加常山、天南星

**三方对比**：
| 方名 | 主治 | 核心药组 |
|------|------|----------|
| 牡蛎汤 | 牡疟 | 牡蛎+麻黄+蜀漆 |
| 柴胡去半夏加栝蒌汤 | 疟渴+劳疟 | 小柴胡-半夏+栝蒌根 |
| 柴胡桂姜汤 | 寒疟 | 柴胡+桂枝+干姜+牡蛎 |

## 二、文式易筋经第二式 —— 练肺阳

**动作**：
1. 手握拳，大拇指上翘（其余四指下压）
2. 拳顶裤缝，轻轻提起来一寸
3. 下半身（肚脐以下）绷紧——全程不松
4. 手臂伸直，吐气时指头上提+其余指下压
5. 吸气放松 → 吐气重复
6. 做49下

**原理**：
- 大拇指 = 手太阴肺经
- 肺 = 呼吸力量 = 气动工具（Air Compressor）
- 五指中大拇指最有力 → 练肺阳
- 属按硚式

**先后顺序**：先做第一式（心包，手指往上翘）49下 → 再做第二式（肺）49下

**功效**：感冒风寒时马上去做

## 三、四物汤补充 —— 药性深度解析

**组成**：当归、川芎、白芍、熟地

**倪师逐味分析**：
| 药 | 作用 | 关键点 |
|----|------|--------|
| 当归 | 滋润润肠（有油脂） | 肠胃好→血有源头。便秘血虚者加重当归即可通便，不用大黄 |
| 川芎 | 活血，收血管 | 阿司匹林/老鼠药致血液稀薄→易淤血→川芎收血管+排肌肉淤血。头痛加重川芎 |
| 白芍 | 行血化瘀、破血 | 静脉曲张、脚冰冷→白芍从三钱→一两。白芍去旧血 |
| 熟地 | 真正补血（黑色） | 血亏+胃口不好→熟地加重 |

**核心逻辑**：四物汤 = 新陈代谢循环
- 白芍去旧血（破血化瘀）
- 当归润肠活血（保障血源头）
- 川芎活血活络
- 熟地补新血
- **不是简单的"四味补血药"——各有分工，是一个循环系统**

**加减活用**：
- 脚冷/静脉曲张 → 白芍加量
- 血亏+胃口差 → 熟地加量
- 头痛 → 川芎加量
- 血虚便秘 → 当归加量（含油脂润肠，不伤正气）

## 四、中风历节病脉证并治第五 —— 开篇

### 4.1 中风核心病机：气血两虚

**倪师核心论断**：
- 中风 = 气血两虚状态下发生
- **不要想"高血压→中风"**——如果降压药真能预防中风，世界上就没中风病人了
- 知道病因是气血两虚 → 预防方就出来了

**预防中风基本方思路**：
- 气 → 黄耆（不是人参，人参用于胃气将绝时的独参汤）
- 补气 → 党参（日常补气足够）
- 去风（次级的，非桂枝类强发汗）→ 独活、防风
- 补血 → 当归、川芎（活血+补血）

### 4.2 中风先兆三信号

| 信号 | 时间窗 | 病机 |
|------|--------|------|
| 无故失眠（持续半年） | 约6个月 | 血不归肝，肝不藏魂；心血不足，神不藏心 |
| 前三指麻（肺经+心包经+大肠经） | 约3个月 | 气血不能通达末梢 |
| 胫骨麻（足三里→解溪，胃经） | 约2周 | 胃经气血衰败，时间窗最短 |

**睡眠三条件**：肝藏魂 + 心藏神 + 脾藏意（脾主意，思虑不停则脾不藏意）

**卒厥**：情绪激动（吵架/太兴奋）→ 气血旺则不会发生 → 平时有小恙要早治

### 4.3 面部中风（喎僻不遂）

**原文**：寸口脉浮而紧，紧则为寒，浮则为虚，寒虚相搏，邪在皮肤。浮者血虚，络脉空虚，贼邪不泻，或左或右，邪气反缓，正气即急，正气引邪，喎僻不遂。

**核心机制**（倪师画图解释的关键段落）：
- 邪在右脸 → 右脸为虚 → 左脸（健侧）正气正常循环
- 正气拉邪气 → 嘴巴歪向**健侧**（左脸）
- **面瘫歪向哪边，邪在对侧**

**针灸原则**：
- 初病（新发）：针健侧（对侧）→ 地仓透颊车等 + 对侧合谷
- 久病（5年）：**两边都扎**，因为正气也虚了

**成因**：大汗后脸对冷气吹 → 风邪入面部肌肉

**处方**：桂枝汤 + 葛根（解肌+提升肌肉中寒水）
- 有头痛 → + 川芎
- 思路：面部肌肉 = 肌 → 桂枝汤解肌 → 葛根引药上行 → 川芎活血止痛

## 五、段7 薄弱点自评

1. 瘅疟无方可考——倪师也承认缺方，靠临床灵活应对
2. 牝疟三方（蜀漆散/牡蛎汤/柴胡桂姜汤）需要区分使用场景
3. 中风先兆的三个时间窗要记牢（6月/3月/2周）
4. 面部中风"正气引邪"的左右机制——易混淆
