### 一、伤寒论总结（衔接部分）

#### 1.1 通脉四逆汤加猪胆汁
- 猪胆汁 → 取代人体胆汁，帮助消化宿食
- 组成：炙甘草 + 干姜 + 附子（大一枚，生用去皮破八片，约三钱）
- 附子剂量经验：救逆时四钱，好转→三钱→两钱；离开时只开两钱
- 机理：炙甘草（焦苦入心+甘入脾，在心脾之间走动） + 干姜温胃 + 附子强心 → 救逆
- 猪胆汁应急用（肠胃有宿食），平时不用（难吃恶心）。无猪胆汁可用羊胆汁代。
- 煎法：水三升煮成一升，加猪胆汁，分温再服

#### 1.2 病后饮食注意
- 发汗后脉平但烦躁 → 肠胃尚不适，正常恢复现象
- 病后饮食：稀饭、青菜 → 连蛋都不吃，肉类更不吃
- 白水煮面（不加盐）+ 葱白（通脉助肠胃吸收）→ 正常后再恢复正常饮食

#### 1.3 伤寒论113方总结——三方概括
**核心论断：113方可归纳为三类**

| 类型 | 代表药 | 目的 | 用途 |
|------|--------|------|------|
| **辛散** | 麻、桂、柴胡、葛根、细辛 | 去寒、驱寒 | 太阳（麻桂）、少阳（柴胡）、阳明（葛根）、少阴（细辛） |
| **寒泻** | 石膏、黄芩、黄连、大黄（膏黄） | 撤热、救阴、急下存阴 | 宣泄实热 |
| **温补** | 干姜、附子、吴茱萸、蜀椒 | 回阳、救逆 | 阳不足时用温热药救回 |

**张仲景三句话**：
- 非寒不泻 → 不用寒药无法宣泄
- 非温不补 → 药性不温补不到身体里（非四物汤思路）
- 人参 → 养阴退热（津液不足→胃蠕动生热→人参补津液退热）

### 二、脏腑经络先后病脉证第一

#### 2.1 上工治未病——五行传变规律

**核心原理**：
- 见肝之病 → 知肝传脾 → 当先实脾（四季脾王不受邪则勿补）
- 中工不晓相传 → 见肝之病不解实脾，惟治肝也

**传变链条**（以肝为例）：
```
酸入肝 → 焦苦入心 → 甘入脾
脾能伤肾 → 肾气微弱 → 水不行
水不行 → 心火气盛 → 伤肺
肺被伤 → 金气不行 → 肝气盛
∴ 实脾 → 肝自愈
```

**治肝先治大肠**：
- 肝脏下方血管通大肠 → 两个排泄管道：①胆（排胆汁）②肝→大肠
- 治肝须保持大便通畅（宣泄肝脏"积毒"）
- 肝病前兆信号：
  - 晚上1-3点睡不着（丑时肝经）
  - 小便颜色变黄
  - 饭前不饿/没胃口
  - 手掌从粉红变深红→暗红

**以此类推——肾病传心**：
- 见肾之病 → 知肾传心 → 当先实心
- 肾水不克心火（正常）→ 肾有问题水直接入心（水克火，异常）
- 手掌发黑发青色（手掌中心劳宫+少府）→ 肾水凌心
- 黄连（苦味入心，解毒）→ 解肾脏之毒（尿毒入血入心）

**肾衰竭两大前兆症状**：
1. 呕吐不止（心火冷不能生脾土）
2. 头目昏眩（毒气上升入脑）

**疗效判断**：
- 治疗肾脏病 = 治疗心脏（强心→脾土恢复→便意回来）
- 吃了药小便开始自排 → 好转，不用洗肾
- 吃了药仍无便意 → 没救过来

**倪师临床案例**：
- 美国人坚持10年不洗肾，手掌脸黑色，但活着（质疑洗肾必要性）
- 37年乳癌患者，不开刀不化疗，活着；按西医治疗死的更快

#### 2.2 肝虚证治则

**补用酸 + 助用焦苦 + 益用甘味**
- 酸枣仁（酸）、乌梅丸（极酸）
- 吴茱萸（焦苦）、黄连（苦）
- 小建中汤（甘）

**关键区分**：
- **虚证**：肝炎、肝肿、没胃口 → 小建中汤 → 甘味药即可
- **实证**：肝硬化、肝癌 → 不用此法！→ 实证用甘味药：白术、陈皮、薏仁、泽泻、茯苓（平淡甘味药≠建中汤）
- **重要原则**：虚证不会死人，实证会死人。一遇实证赶快变虚证

#### 2.3 倪师维他命观
- 肝如活树 → 不能涂油漆（维他命=油漆，给死木板用的）
- 虚证不会死 → 补维他命=把虚证变实证 → 维他命致命
- "维他命是给敌人吃的，自己不要吃"

#### 2.4 南派误治——补阴六大弊

**核心批判**：南派见什么都是阴虚 → 开补虚药 → 制造新病

| 原病 | 误当阴虚治 → 造成 |
|------|-------------------|
| 下痢 | → 痞证（心下痞，泻心汤证来源） |
| 下痢/痢血（无往来寒热） | → 臌胀（腹部胀大） |
| 咳嗽 | → 肺痿（肺萎缩） |
| 痰饮（胸腔积水） | → 浮肿（脸肿） |
| 吐血/遗精 | → 劳证 |
| 湿阻中焦/食滞肠胃 | → 格（阴阳相格） |

**经方家处理阴虚**：不开补阴药，用阳药让阴自回

#### 2.5 虚虚实实
- 经曰：虚虚实实，补不足，损有余，是其义也。余藏准此。
- 非寒不泄 → 寒凉药才能泄实
- 厥阴篇常见配伍：附子+黄芩+黄连+干姜（寒热并用）

### 三、千般疢难不越三条（三类病因）

#### 3.1 总纲
- 人禀五常，因风气而生长 → 风气能生万物亦能害万物
- 五脏元真通畅 → 人即安和
- 客气邪风 → 中人多死

#### 3.2 三条

| 类型 | 路径 | 范畴 |
|------|------|------|
| **内因** | 经络受邪 → 入脏腑 | 风寒湿入经络未阻→入脏腑 |
| **外因** | 四肢九窍，血脉相传，壅塞不通 | 外皮肤所中（外感） |
| **不内不外因** | 房室、金刃、虫兽所伤 | 房劳、刀伤、虫蛇咬 |

外 = 风寒暑湿燥热；内 = 喜怒忧思悲恐（情志）

### 四、养慎防病——腠理与导引

#### 4.1 防病原则
- 适中经络 → 未传脏腑即医治之
- 四肢才觉重滞 → 导引、吐纳、针灸、膏摩
- 勿令九窍闭塞
- 无犯王法、禽兽灾伤
- 房室勿令竭乏
- 服食节冷热苦酸辛甘

#### 4.2 腠理定义
- **腠**：三焦通会元真之处，血气所注
- **理**：皮肤脏腑之文理

### 五、文式易筋经第一式——按蹻式

#### 5.1 基本要领
- **站姿**：下半身不动，肚脐以下绷紧（手打肌肉会弹起），上半身松
- **脚**：脚趾如爪抓地（赤脚草地最佳，小心火蚁）
- **握拳**：平面拳（四指平直，与尺骨桡骨平行成直线）→ 不易受伤
- **按蹻式动作**：肩松→手往下按→吸气→指头往上抬（自然，不过度用力），只在一寸之间发力。感觉像手撑桌子跳起。
- **次数**：49下
- **效果**：30下开始出汗，背后汗是冰的（太阳寒水，正常）→ 玄府打开→汗水出→肠胃津液回补

#### 5.2 倪师经验
- 大学打麻将坐冷气旁→一觉得不对→站外面练一圈→排掉（第一圈葛根汤证→不练第二圈就麻黄汤证了）

### 九、新学到的东西（个人成长）

1. **猪胆汁的用法**：不是常规药，是应急代偿消化液。无猪胆→羊胆代
2. **张仲景补药观**：非温不补——颠覆"补=四物汤"的常识
3. **人参定位**：养阴退热，不是"补气强身"
4. **肝虚vs肝实治法完全不同**：虚证建中汤甘味即可，实证需白术陈皮等平淡药
5. **治肾病=治心脏**：手掌发黑是肾水凌心→黄连解毒→强心
6. **阴自回**原则：不开补阴药，用阳药让阴自己回来
7. **维他命致命逻辑**：虚证不会死→维他命把虚变实→会死（像给活树涂油漆）
8. **文式易筋经与武式区分**：文式站姿不动下半身，重在呼吸吐纳
9. **山根望诊**：青=腹痛（小建中汤），黑=水气（肾），黄=胸寒，白=亡血
10. **闻诊三型**：惊叫=骨痛，哑然=心膈，细长自语=头痛
