### 一、泻心汤系列（本段核心）

#### 1. 大黄黄连泻心汤（一六九）
- **主证**：心下痞，按之濡（软绵绵不痛），关上脉浮
- **病机**：胃热上冲——关上脉浮说明上焦有热
- **用法关键**：**泡不煮！** 麻沸汤（滚水）冲泡，取药气不取药味
  - 泡=取气=往上走，清上焦热（眼鼻出血、牙龈出血）
  - 煮=取味=往下走，变泻药
- **大黄二、黄连一**（比例）
- **延伸应用**：
  - 五官无故出血（眼白出血、流鼻血、牙龈出血）= 胃热上冲
  - 泡服大黄黄连，苦但病人有证反觉好喝
  - 粉剂外敷收敛伤口：+白术（祛湿）+没药乳香（收敛）+三七（活血化瘀）

#### 2. 附子泻心汤（一七〇）
- **主证**：心下痞 + 恶寒 + 汗出
- **病机**：胃热（三黄证）+ 表阳虚（汗出恶寒）
- **用法关键**：大黄、黄连、黄芩三味**泡**，附子**单独煮**，然后将附子汁兑入泡好的三黄汤中
- **炮附子的核心定位**：固表阳
  - 表虚→汗出恶寒→炮附子
  - 失血多=汗血同源→也可用炮附子
  - 遗精龙骨牡蛎收不住→加炮附子
  - 老人夜尿频繁→炮附子（汗尿同源）
- **炮附子 vs 生附子**：
  - 炮附子：固表阳（表阳虚）
  - 生附子：温里阳（里阳虚）

#### 3. 生姜泻心汤（一七二）
- **主证**：胃中不和 + 干噫食臭（打嗝呕酸）+ 胁下有水气 + 腹中雷鸣 + 下利
- **生姜 vs 半夏**：
  - 生姜：排**胃里面**的水→打嗝（干噫）
  - 半夏：排**胃外面**的水→恶心呕吐
- **炙甘草在此处的关键**：胃中有水→必须用炙甘草（不能用生甘草，生甘草蓄水）
  - 利水时不用生甘草，必须用甘草时选炙甘草
- **黄芩黄连**：清肠胃热（消炎）
- **与葛芩连汤区别**：
  - 葛芩连汤：胁热利（纯热利，拉到臭）
  - 生姜泻心汤：肠胃不舒服+肠鸣下利+呕酸

#### 4. 甘草泻心汤（一七三）
- **主证**：伤寒中风误下→下利日数十行，谷不化（大便可见食物），腹中雷鸣，心下痞鞭而满，干呕，心烦不得安
- **病机**：胃中虚，客气上结（肠中下焦气往上跑→摸起来硬以为是实，实际是气）
- **虚实鉴别关键**：
  - 实证：拒按、脉重按有力、舌苔黄厚、脸色光鲜声音洪亮
  - 虚证：喜按、脉浮轻按即无、舌苔薄黄
- **重用甘草**：胃被攻下伤到，生甘草→解毒、护胃、蓄水
- **三个泻心汤总结**：
  - 生姜泻心汤 = 胃有水 + 打嗝 → 重用生姜
  - 甘草泻心汤 = 胃虚 + 下利重 → 重用甘草
  - 半夏泻心汤 = 恶心呕吐 → 重用半夏
  - 附子泻心汤 = 胃热 + 表虚 → 加附子
  - 大黄黄连泻心汤 = 纯胃热 → 大黄+黄连泡服

#### 5. 赤石脂禹余粮汤（一七四）
- **主证**：服泻心汤+理中汤后下利仍不止→病不在中焦，在**下焦**
- **治则**：利在下焦→涩肠止利
- **赤石脂+禹余粮**：收涩剂
- **后续**：若涩肠后仍利→当利其小便（水走肠间故利）
- **临床延伸**：大肠炎、大肠癌便血均可使用
- **倪案·010**：大肠炎病人一天二三十次脓血便→桃花汤（赤石脂一半煮一半生用+干姜+粳米），一剂第二天好转
- **倪辨·015**：白通汤（寒利救逆）vs桃花汤（大肠炎脓血便）

#### 6. 治痞原则（一七九）
- **先表后里**：有表证先解表，表解后再攻痞
- 解表→**桂枝汤**（凡经发汗/攻下后还有表证，一律桂枝汤，不用麻黄葛根）
- 攻痞→大黄黄连泻心汤（泡服）
- 桂枝汤可当补药日常吃（腰酸背痛=桂枝汤证）

### 二、十枣汤（段首回顾）

- **核心证**：但坐不得卧（必须到那个阶段才能用）
- **汗出鉴别（关键！）**：
  - 水汗（一般汗）→十枣汤可下
  - **油汗**（油油的、黏的）→元阳元气将绝，**不可下**，下了病人一口气梗到会死
- **甘遂、大戟、芫花分工**：
  - 甘遂：去肺+子宫周围水，**涤痰力最强**
  - 大戟：去三焦油网系统积水
  - 芫花：去胸腹部积水
- **三个合用**=肺积水在胸胁横膈
- **痰+水+热三结**→结胸→甘遂+大黄+芒硝（不用芫花大戟）
- **服用后护理**：吐泻完3小时胃缓→喝稀饭/烂面（不放油），不要吃肉（会伏热）
- **秤不离砣**：甘遂芫花大戟必配红枣，皂荚必配红枣（红枣阻隔胃液保护胃）
- **倪联·009**：十枣汤连易经坎离卦——中医是健康专家，离中虚不着相

### 三、五苓散（一七一）

- **主证**：泻心汤后痞不解 + 口渴 + 烦躁 + 小便不利
- **病机**：表水未透发——发汗剂量不够，汗从肠胃跑到皮肤表面但没透发出去，停在肌肉皮肤表面
- **核心功能**：去水而生水
- **倪辨·006**：五苓散 vs 猪苓汤
  - 五苓散：全身表水不利，水入即吐（水逆），瘟疫旅游必备
  - 猪苓汤：下焦水热互结石，小便带血痛，肾/膀胱结石专方

### 四、桂枝人参汤（一七五~一七八）

- **主证**：外证未除（有表证）+ 寒利下利（下利清谷、不渴、舌苔白）
- **组成**：桂枝、人参、白术、干姜、炙甘草
- **用法关键**：先煮四味，**桂枝最后放入**稍煮即可（不用久煮）
- **服用**：日再服夜一服=每6小时一次，24小时吃4次
- **每味药的角色**：
  - 人参：津液伤到（下利伤津）→补津液
  - 白术：健脾祛湿（肠胃+关节）
  - 干姜：温中，寒利用干姜（热利用生姜）
  - 桂枝：行阳，心脏力量加强→心火移到小肠
  - 炙甘草：入小肠
- **不用白芍**：此处纯寒利，不用白芍。白芍用于腹痛+补阴
- **寒利 vs 热利**：
  - 寒利：不渴、舌苔白、下利清谷、水白
  - 热利：口渴、舌苔黄、下利臭

### 五、瓜蒂散（一八一）

- **主证**：似桂枝证但头不痛项不强，胸中痞鞭，气上冲咽喉不得息
- **病机**：胸中有寒（实）= 异物/痰/食物梗到
- **唯一吐剂**：瓜蒂（越甜的瓜蒂越苦）+赤小豆，等量打粉，香豉煮汤送服
- **不吐少少加，吐了马上停**
- **禁忌**：亡血虚家不可与
- **吐解的广义理解**：
  - 不仅瓜蒂散能吐
  - 肾结石吃猪苓汤吐了→也是吐解（痰梗到导致结石形成，吐掉痰结石就下）
  - 汗吐下都是病情解除的现象，不可死板

### 六、藏结死证（一八二）

- 胁下素有痞，连在脐旁，痛引少腹，入阴筋者→死
- 汉朝当时医学技术不发达，此证不该死

### 七、白虎加人参汤（一八三~一八五）

- **主证**：口燥渴 + 心烦 + 背微恶寒（或完全无表证）
- **关键鉴别**：有表证（脉浮发热无汗、恶寒）→不可用白虎，先解表
- **确定无表证**（但热不寒）+ 渴→白虎加人参汤
- **津液伤到**：即使背有一点怕冷，只要口大渴→津液已伤→白虎加人参
- **无大热≠不高烧**：阳明高热可达103-104℉（壮热），依然用白虎/承气
- **倪辨·008**：阳明经热（白虎）vs 腑热（承气）
  - 白虎：大便正常、壮热、大烦渴、脉洪大
  - 承气：便秘、潮热、谵语、腹痛拒按
  - **问大便是最简单区分法**
- **倪诀·003**：「阳明无死症，死在医之过」——阳明病只要辨证正确不会死人
- **倪案·005**：艾滋病患者大烦渴壮热→白虎汤→「从来不知道当一个人可以感觉那么好」；不吃西药前都是阳明证，吃了西药变厥阴证才死
- **表里双解**：桂枝汤证+白虎汤证可合方，一剂表里双解（病人弄混也没关系）

### 八、黄芩汤（一八七）

- **主证**：太阳少阳合病 + 自下利
- **若呕**：黄芩加半夏生姜汤
- **组成**：黄芩、芍药、炙甘草、大枣
- **药性关键**：
  - 黄芩：清**肠**热（vs 黄连清**胃**热，大黄清**胃+攻下**）
    - 胃热→黄连（或大黄黄连）
    - 肠热→黄芩
    - 肠胃都热→三黄（大黄+黄芩+黄连）
  - 芍药：止腹痛 + 补阴 + 滋阴（腹部静脉多，芍药收静脉）
    - 腹痛重用芍药，胸满禁用芍药（会更闷）
  - 炙甘草：入小肠（肠胃病用炙甘草）
- **太阳少阳合病下利 vs 纯太阳下利**：
  - 纯太阳下利：葛芩连汤
  - 太阳+少阳合病下利：黄芩汤
- **服用**：日再服，夜一服（每6小时一次）

### 九、诊断铁律（多处散见+倪问整合）

#### 渴与不渴（倪问·002）
- 不渴=寒，渴=热/津液伤
- 消渴（喝了不解渴）= 五苓散或白虎加人参
- 不渴治到渴=好转（寒去欲解），渴治到不渴=好转（津液回头）

#### 小便（倪问·003）
- 色白=寒，淡黄=正常，深黄=热实（承气证）
- 承气汤攻下时：深黄攻到淡黄就停
- 小便不利+渴=五苓散

#### 喝水温度（倪问·013）
- 想喝冰水=里热（白虎/承气）
- 想喝热水=里寒（四逆/理中）
- 渴不欲饮=瘀血
- 水入即吐=五苓散水逆
- **关键：问欲望！不是问实际喝什么**

#### 虚实鉴别
- 实：拒按、脉重按有力、舌苔黄厚、声音洪亮脸色光鲜
- 虚：喜按、脉浮轻按即无、舌苔薄黄

#### 汗出鉴别
- 水汗（一般汗）= 可治
- 油汗（黏腻）= 阳气将绝，危重

### 十、跨课连接

- 十枣汤配红枣 ≈ 皂荚配红枣（秤不离砣，红枣保护胃液）
- 赤石脂禹余粮汤 → 桃花汤（少阴篇，赤石脂+干姜+粳米）
- 白虎加人参汤 → 阳明篇核心方
- 五苓散 ↔ 猪苓汤（后续阳明/少阴篇详细对比）
- 黄芩汤 → 少阳篇延伸
- 生姜/干姜 → 生姜排水（胃内），干姜温中（胃寒）
- 炮附子 vs 生附子 → 后续少阴篇深入

### 十一、饮食护理铁律

- 汗吐下后：喝稀饭/烂面，不加油，不吃肉（肉会伏热→过两天看似好了又发烧）
- 「宁可饿点，不可过饱」→ 空灵为常
- 倪师胰脏癌案例：病人遵医嘱找不到东西吃，饿→脾胃功能反而恢复得快
- 经方少用补药，多用攻坚疏导和解去热
