## 一、栀子厚朴枳实汤（心烦腹满卧起不安）

### 处方
栀子十四枚（劈） + 厚朴四两（炙） + 枳实四枚 → 水三升半煮取一升半，分二服

### 三药定位
| 药 | 主治 | 机理 |
|----|------|------|
| **厚朴** | 腹满（虚实皆用） | 降气+把脾脏湿移到小肠。脾主少腹 |
| **枳实** | 胸满/心烦/卧起不安 | 宽肠（撑开胆管/小肠/大肠），管心脏，形似心脏 |
| **栀子** | 心烦 | 去虚热 |

### 核心病机
- 厚朴+枳实+大黄 = 小承气（攻下）
- 厚朴+枳实+栀子 = 栀子厚朴枳实汤（事后补救，病已好但余症未消）
- 腹满必用厚朴，不论虚实——实证承气汤也有厚朴，虚证厚朴生姜半夏人参甘草汤也有厚朴

### 厚朴深度解析
- 白术去湿；厚朴不是去湿，是把**脾脏的湿移到小肠**
- 脾主少腹 → 厚朴治腹满的逻辑
- 承气汤不加厚朴 → 大便出来但剧痛（硬扯）
- 厚朴能增润津液

### 枳实深度解析
- 宽肠=把缩小的撑大：胆管太小→枳实撑开；小肠大肠撑大
- 枳实管心脏，形似心脏→跟着心脏搏动→小肠大肠蠕动增加
- 胆结石处方示范：炙甘草（强心）+ 柴胡（胆）+ 黄芩（湿热）+ 枳实（宽胆）+ 五倍子/海金沙（消结石）
- 心脏搏动push胆→胆汁喷出（像喷香水）。胆汁淤积原因：①两餐间隔太久 ②吃太油腻
- 胸满时一律不用白芍，用枳实

## 二、栀子干姜豉汤（伤寒大下后身热不去微烦）

### 处方
栀子十四枚 + 干姜二两 + 香豉四合 → 水三升煮取一升半，分二服

### 核心鉴别
- **干姜 vs 生姜**：
  - 恶心用生姜
  - 微烦（无恶心）用干姜
- **干姜管中焦+上焦**，不管下焦（下焦用炮附子）
- **寒的定义**：蠕动慢——正常一息四至，寒则一息二至

### 病机
- 伤寒大下（丸药攻下，多寒凉药如大黄芒硝）
- → 身热不去（表热仍在，但是虚的）
- → 里面真寒（故用干姜温中）
- 栀子去虚热 + 干姜温中 = 寒热并用

## 三、栀子豉汤禁忌（第89条）

> 凡用栀子豉汤，病患旧微溏者，不可与服之。

- 栀子豉汤偏寒凉 → 肠胃虚寒者（常下利、大便溏）忌用
- 补救：加干姜、白术、茯苓

## 四、真武汤（头眩身瞤动振振欲擗地）

### 处方
茯苓三两 + 芍药三两 + 生姜三两 + 白术二两（炙） + 炮附子一枚
→ 水八升煮取三升，温服七合，日三服

### 真武汤 vs 苓桂术甘汤（倪辨·005）

| | 苓桂术甘汤 | 真武汤 |
|---|---|---|
| **病位** | 中焦（横膈）水饮 | 下焦寒湿 |
| **头晕特点** | 起则头眩（站起来才晕） | 头重脚轻、躺下也晕、振振欲擗地 |
| **核心药对** | 桂枝降逆 + 白术茯苓排湿（4:2） | 炮附子温阳化水 + 白术去湿 + 茯苓利水（3:2） |
| **适用人** | 喝水太快横膈停水 | 久居湿地、素体中湿之人（一动就流汗、胖、脚冷、不渴、没胃口） |

### 真武汤方解
- 炮附子：热药，温阳化水（下焦寒，肾阳不够）
- 白术去湿 + 茯苓利水（从小便排出）：白术无茯苓=药不知道你要干什么
- 芍药+生姜：当补药
- **利水方一律不用甘草**（甘草蓄水）：五苓散、真武汤都没有甘草

### 骨三阴论（倪联·008 黄帝外经）
- 三处阴骨：头顶泥丸宫（聪骨）、耳后高骨、膝盖骨
- 湿伤于下 → 先伤膝盖骨（最下面的阴骨）
- 临床辨证：膝盖关节炎、耳后骨、头顶骨异常 → 阴实证预警

### 中湿之人特征
- 胖胖的、不管天气冷热一直流汗
- 一天换好几次衣服
- 湿伤于下：脚冷、没胃口、下肢无力
- → 苓桂术甘汤清不掉（湿已久变寒湿）→ 真武汤

## 五、发汗禁忌七条

| 序号 | 禁忌人群 | 发汗后果 | 机理 |
|------|----------|----------|------|
| 1 | **咽喉干燥者** | 不可发汗 | 咽喉=三阴之会（太阴/少阴/厥阴），干燥=阴津不够，汗血同源 |
| 2 | **淋家**（小便淋沥/发炎/白浊） | 发汗必便血 | 膀胱有炎症，发汗→膀胱水分没了→小便浓稠带血 |
| 3 | **疮家**（化脓性皮肤病） | 发汗则痉 | 已有化脓疮不可发汗（一般干癣可发汗） |
| 4 | **衄家**（常流鼻血） | 额上陷（天仓凹）、脉急紧、直视不能眴、不得眠 | 阴虚→发汗→阳亢（阳往上跑）→阳虚。阴阳要平衡 |
| 5 | **亡血家**（贫血、面色苍白、嘴唇无色、芤脉） | 发汗则寒栗而振 | 汗血同源 |
| 6 | **汗家**（平常没事就流汗很多）重发汗 | 恍惚心乱、小便阴痛 | 津液太干→禹余粮丸（原方已失）；替代：人参+甘草+生姜+红枣 |
| 7 | **病人有寒**（里寒）复发汗 | 胃中冷→吐蛔 | 胃下方有食物存7天，里寒+发汗→胃中冷水泡食物→坏死生虫 |

### 相关诊断技巧
- 问诊时必问："喉咙痛不痛？"
  - 喉咙痛+发烧 = 葛根汤（太阳温病：运动流汗后受风邪）
  - 喉咙干燥（不痛）= 不可发汗，津液不够
- 天仓（眉尾两侧）：凹陷=气血两虚
- 桂枝汤最能协调阴阳（发汗禁忌的兜底方）

## 六、治则与救逆

### 表里治则（第98条）
- 表证（太阳）→ 先发汗，不可攻下
- 里证（承气汤证）→ 先攻下，不可发汗
- 违反 = 逆证

### 误下救逆（第100条）
伤寒（应发汗）→ 误攻下 → 清谷不止（里寒）→ 急当救里：**四逆汤**
救里后 → 身疼痛 → 急当救表：**桂枝汤**（不可再用麻黄汤）

### 汗后脉沉救里（第101条）
发热头痛身痛（表证）→ 发汗 → 没好 → 脉反沉 → **四逆汤**
（原因：病人素虚 / 汗剂太重）

### 汗后冒证（第102条）
发汗后流汗不止（冒证）→ 桂枝汤或小建中汤

### 脉微解证（第103条）
- 阴阳脉俱微（寸尺都小）→ 自汗而解
- 阳脉微（寸）→ 汗出而解
- 阴脉微（尺）→ 下之而解，用**调胃承气汤**（最轻微的攻下剂）

### 桂枝汤证重申（第104条）
发热汗出 = 荣弱卫强 = 桂枝汤证（不管西医诊断是什么）

## 七、小柴胡汤（少阳病主方）⭐核心

### 主症（必有）
往来寒热 + 胸胁苦满 + 默默不欲食 + 心烦喜呕

### 或症（可有可无）
胸中烦不呕 / 渴 / 腹中痛 / 胁下痞硬 / 心下悸小便不利 / 不渴身微热 / 咳

### 病传路径
太阳（表）→ 两种走向：
1. → **阳明**（到此为止，白虎/承气，无死证）
2. → **少阳**（三焦/淋巴系统，需阻止→否则→太阴→少阴→厥阴）

**少阳必须截断！**

### 少阳 = 三焦系统 = 淋巴系统
包括：脏与腑之间的隔膜、脑部、乳房、胸部、女子胞（子宫）
→ 热入血室也用柴胡汤

### 处方
柴胡半斤 + 黄芩三两 + 人参二两 + 甘草三两 + 半夏半升（生） + 生姜三两 + 大枣十二枚
→ 水一斗二升煮取六升→去滓再煎取三升→温服一升，日三服

### 倪师口诀（倪诀·011）
> **「但见一证便是，不必悉具」**
> 往来寒热 / 胸胁苦满 / 恶心 / 默默不欲食——四证中只要出现一个，就可以开小柴胡汤

### 肝炎/肝癌治疗（段6重磅临床内容）
**肝炎基础方**：小柴胡汤 + 白术 + 茯苓（白术去湿，茯苓利水，茯苓略多于白术）
- 肝气郁结 → +郁金（行肝气）
- 身黄/黄疸 + 小便深黄 → +茵陈（去黄）
- 虚烦不得眠 → +栀子

### 肝癌预警（倪问·009）
> **凌晨1-3点准时醒来** → 肝已经出问题（提前2年可知！）
- 肝癌不是"无形的杀手"——是你没注意信号
- 肝炎阶段（仅有湿热无实证）→ 好治
- 变成结实（长东西）→ 难治
- 腑中实证（胃/小肠/大肠/胆）→ 不难清
- 脏中实证（肝）→ 极难清

### 肝病痊愈六指标（从阴证治回阳证的标准）
1. **睡好**：一觉到天亮，不会1-3点醒
2. **大便正常**
3. **胃口好**：包括感到肚子饿 + 吃东西有味道
4. **小便颜色正常**：淡黄
5. **体力恢复**
6. **足温热**

### 病进/病退标志
- 阴证 → 阳证（如厥阴变少阳）→ 病退，好现象
- 阳证 → 阴证 → 病进
- 西医开刀/放化疗 → 少阳变厥阴（本来14年存活→5-10年）
- 不动它 → 少阳到厥阴14年
