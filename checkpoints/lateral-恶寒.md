## 二、倪师关键辨证要点

### 2.1 恶寒 ≠ 一定有寒

- **阳明洒洒恶寒** → 纯热症！因体内纯热，故感觉外界空气是冷的。不可用热药！
- **发汗后恶寒** → 虚证（芍药甘草附子汤），不是表寒未解
- **疮痈恶寒** → 热毒蕴结、将化脓，不可当表证发汗

### 2.2 炮附子 vs 桂枝（倪师核心区别）

- **炮附子**：壮心阳（心脏摩擦生热，热=体温来源）→ 恶寒时用
- **桂枝**：导心火下入小肠（味厚入血分）→ 心火不下行时用

### 2.3 脉法鉴別

| 恶寒类型 | 脉象 | 
|---------|------|
| 太阳伤寒 | 浮紧 |
| 中暍（中暑） | 弦细芤迟（关键区别！不是浮数） |
| 少阳 | 弦 |
| 寒疝 | 弦紧/沉紧 |
| 疮痈 | 浮数（应发热却恶寒=矛盾→发痈） |

### 2.4 下利+表证铁律

下利清谷+表证（恶寒无汗）= **绝对不可攻表**！先解里，后解表。
先攻表→腹胀满（气机壅塞）。
