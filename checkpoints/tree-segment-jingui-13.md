# 金匮要略 段13 知识树

## 段13：心脏病补充治症 + 腹满寒疝宿食病脉证并治第十（开头）

---

### 一、心脏病治症总纲

#### 1.1 心脏问题二分法
- **外（结构问题）**：胸痹，金匮胸痹篇
  - 栝蒌实 — 结构问题
  - 枳实 — 心包周围湿
  - 薤白 — 涤痰（辛味，如大蒜）
  - 茯苓 — 去水（茯苓杏仁甘草汤，心下积水）
- **内（血的问题）**：动脉血管堵塞、硬化
  - 痛如刀刺，持续刺痛，固定一点
  - 舌有齿痕/地图舌

#### 1.2 痛感鉴别（关键诊断）
| 痛型 | 病因 | 方剂 |
|------|------|------|
| 闷痛、虫蛀感（跑动） | 痰在纵隔腔跑 | 乌梅赤石脂丸 |
| 痛到背后再痛回来 | 痰饮流动 | 乌梅赤石脂丸 |
| 持续固定刺痛（刀刺） | 动脉血管堵塞 | 活血化瘀方 |

#### 1.3 动脉血管堵到处方
- **活血化瘀药**：川芎（大剂）、丹皮、红花（更刺痛时加）
- **导药入心**：黄芩、黄连（苦味入心，君臣佐使之"使"）
- **攻补兼施**：阿胶（攻血同时补血）
- 结构+血管问题可合并用方

#### 1.4 炮附 vs 生附
| | 炮附子 | 生附子 |
|------|------|------|
| 配伍 | +生姜 | +干姜 |
| 用途 | 收表、止盗汗、心主五液 | 心脏无力产热、四逆证 |
| 症状 | 不正常流汗 | 手脚冰冷至肘膝 |
| 对应方 | 桂枝加炮附（发汗太过） | 四逆汤（救逆） |

#### 1.5 炙甘草的用法
- **炙甘草**：强心、调整心脏搏动
- **炙甘草汤**：脉结代、心动悸 → 重用炙甘草
- **动悸**：炙甘草汤 + 桂枝
- **脐下悸**：茯苓甘草

#### 1.6 心脏正常四大症状（治疗终点）
1. 一觉到天亮
2. 大便正常
3. 手脚温热（尤其脚趾尖—最早冷最早热）
4. 手脚不麻痛（指尖麻=堵塞信号）

#### 1.7 望诊
- **两颧红**：粉红（病浅）→ 暗红（加重）→ 黑（危险）
- **手掌温 + 手背凉** = 胸部阴阳平衡（正常）
- **手掌热 + 额头凉** = 正常（要摸病人的，不是自己的）

#### 1.8 心脏病与癌症关系
- 心脏病是七种癌症根源：乳癌、红斑性狼疮、肝癌、肺癌、脑瘤、淋巴癌
- 心脏治好 → 根源去 → 其他病变自愈
- 心管月经 → 心脏治不好无法调经
- 心脏功能恢复 → 小肠热恢复 → 子宫卵巢热恢复 → 可怀孕

#### 1.9 临床案例
- 三根动脉堵塞 + 单肾功能 → 西医拒绝手术 → 针下（公孙内关）当场痛去 → 行走不需停 → 药后两周可逛街数mile

### 二、寒极生热（心态突破）

- **常规**：热药治寒症
- **例外**：热药一直下去寒不退 → 重用寒药（黄芩黄连黄柏）
- **原理**：人体是太极旋转，寒极生热、热极生寒
- **案例**：大胖子女阳明燥热 → 石膏开到十四两仍热 → 生附下去反而冷 → 换寒药后热退瘦身
- **适用**：红斑性狼疮尿毒 → 毒入心手脚冷 → 黄连解毒即此理

### 三、腹满寒疝宿食病脉证并治第十

#### 3.1 趺阳脉微弦
- **脉位**：冲阳脉（足阳明胃经）
- **微弦** = 胃寒
- **腹满（不痛）** = 虚寒 → 温药（理中汤类）
- **不腹满 + 便难** = 实寒（便秘）
- **两胠痛**（腰后输尿管区）= 虚寒从下上 → 温药

#### 3.2 虚实触诊
- **按之不痛** = 虚（无物）→ 温药
- **按之痛（拒按）** = 实（有物）→ 可下
- **触诊位置**：天枢穴（关键）
- **舌黄** = 里有热实 → 下后黄自去；黄未退 → 继续攻
- **小便颜色**替代舌诊：深黄→中黄→淡黄（热退停药信号）

#### 3.3 腹水分虚实
| 类型 | 口渴 | 方剂 |
|------|------|------|
| 实 | 渴 | 分消汤 |
| 虚 | 不渴 | 补气建中汤 |

两方皆甘淡渗利（白术苍术大腹皮茯苓泽泻等）

#### 3.4 持续性 vs 间歇性（通用虚实鉴别）
- **持续** = 实（堵到）→ 动脉堵塞持续痛、大便堵持续胀
- **间歇/时好时坏** = 虚 → 腹满时减复如故 = 寒（虚）→ 温药

#### 3.5 死症判断
- **痿黄**（暗黄，非鲜黄）+ **躁**（手足躁扰，心神不守）+ **不渴**（胸中寒实）+ **下利不止**
- = 阴阳两脱 → 死
- 三物小白散（贝母桔梗巴豆 3:3:1）治胸中寒实 → 但前提是便秘；已下利 = 不可用
- 若躁+渴 → 不死（还没那么快）

#### 3.6 弦脉与水饮
- 弦脉 = 寒水
- **寸口脉弦** = 寒水在胁下 → 胁下拘急而痛 → 小柴胡汤
- 啬啬恶寒（水在胁下会怕冷）

#### 3.7 喷嚏与寒
- **肺寒**：喜欠、清涕出
- **发热色和者善嚏** — 体能恢复排寒时打喷嚏 = 好事
- **中寒下利 + 欲嚏不能** = 中焦寒（横膈膜无力）→ 肚中寒

#### 3.8 绕脐痛
- = 风冷（风=会移动）
- 谷气不行（消化不良）
- **忌攻下** → 攻下则：
  - 气上冲 = 奔豚 → 奔豚汤/桂枝加桂汤
  - 气不上冲 = 心下痞 → 五个泻心汤补救

#### 3.9 厚朴七物汤（太阳阳明合病）
**主证**：腹满 + 发热十日 + 脉浮而数 + 饮食如故
**组成**：厚朴半斤、枳实五枚、大黄三两、桂枝二两、生姜五两、甘草三两、大枣十枚
**拆解**：桂枝汤去芍药 + 厚朴枳实大黄（小承气变方）
**去芍药原因**：腹满无腹痛；芍药酸收与攻下矛盾
**加减**：呕+半夏五合、下利-大黄、寒多+生姜至半斤

#### 3.10 附子粳米汤（腹中寒痛）
**主证**：雷鸣切痛 + 胸胁逆满 + 呕吐
**病机**：腹中寒气→小肠套叠（寒则收）
**组成**：炮附子一枚、半夏半升、粳米半升、甘草一两、大枣十枚
**煎法**：水八升煮米熟汤成去滓
**粳米** = 糯米糙米（最易消化的甜米）
**临床扩展**：胆结石痛、子宫癌寒痛、胰脏炎、腹膜炎（寒证雷鸣皆可用）
**简捷**：一剂知，第一碗即好

### 四、新学到/加深理解的核心要点

1. **心脏病二分框架**：结构（胸痹方） vs 血管（活血化瘀方），可合并用
2. **痛感鉴别表**是临床利器：闷/虫蛀/跑动（痰）vs 固定刺痛（堵）
3. **导药入心**的使药思维：黄芩黄连苦味为"使"把活血药引入心
4. **脚趾尖温度**是最早的心脏预警——比任何仪器灵敏
5. **寒极生热**是临床突破性思维——常规热药不退寒时反用寒药
6. **小便颜色**做远程替代诊断（舌苔被刷/染时）
7. **绕脐痛忌下**——寒证误攻的奔豚/心下痞连锁反应
8. **附子粳米汤**简捷精准：五味药治小肠套叠剧痛，一剂知
9. **厚朴七物汤去芍药**逻辑严密：攻下不收敛
