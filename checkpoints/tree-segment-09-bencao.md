## 倪师案例（本段涉及）

- **[桂枝·奔豚案]**：病人陈述：「胸口一直不断跳动，心脏跳得很厉害停不下来，好像心脏要从胸口嘴巴里跳出来，同时好像有人灌热水到身体里，热一直冲上来」→这就是奔豚。三焦系统水热之气上冲，桂枝降冲逆。桂枝辛甘发散→把冲逆散掉。
  - **教训**：奔豚=心脏狂跳+一股热往上冲。听到"气往上冲"→想到桂枝。
  - **来源**：nishizhu_bencao.json 倪案#19（桂枝奔豚）
  - **补充**：便秘引发奔豚→大便堵在大肠，沼气上冲心脏→先通便再降逆。

- **[肉桂·桂附八味丸辨证]**：虚寒体质信号——常常下痢、恶寒、怕冷、喜欢喝温水、晚上睡觉冷、脚冷→桂附八味丸（肉桂+附子+六味地黄丸）。热性体质（脸红润舌黄怕热喜冷）→肉桂一吃大便更秘。
  - **核心原则**：命门火=生命蜡烛。肉桂点火，附子固火。危急救难要先把命门火点起来→火在人不死。癌症病人拖长生存期=固命门火。
  - **来源**：nishizhu_bencao.json 倪案#16（肉桂桂附八味丸）

## 倪师辨证思路（本段涉及）

### 倪辨（药物鉴别）
- **桂枝 vs 肉桂 vs 桂皮**：桂枝=牡桂/大桂枝=走血脉四肢（手麻脚麻心脏闷痛用桂枝）；肉桂=菌桂/小桂皮=入命门肾脏（命门火衰用肉桂）；桂皮=主上升。
- **桂枝 vs 白芍**：桂枝走动脉（心脏→四肢），阳药；白芍走静脉（四肢→心脏），阴药。等量用→阴阳平衡；阳虚加重桂枝；阴虚加重白芍；有时单用。
- **榆白皮 vs 茯苓**：皮水（皮肤水肿）→榆白皮更好；脾虚腹胀积水→茯苓更好。

### 倪诀（核心口诀）
- **"辛甘发散为阳"**：桂枝=辛+甘→阳药。适用：四肢关节风湿。
- **"白术去湿，茯苓利水"**：白术把湿收集→茯苓从小便排出。(nishizhu #倪诀2)
- **"桂枝降冲逆"**：气往上冲、奔豚、冲气→桂枝散掉。
- **"肉桂点火，附子固火"**：肉桂=命门火之源，附子=把火固定住不散。两者常同用。
- **"辛夷+菖蒲+苍术"鼻病三剑客**：不辨寒热，先通鼻窍。辛夷辛散开窍，菖蒲导湿下行利尿，苍术收湿。
- **"以通为补"**：王不留行活血化瘀→淤去=补；大黄芒硝攻下→通便=补。
- **"不通则痛"**：王不留行活血化瘀→淤去痛止。任何痛=有淤堵。
- **"百病都有瘀"**（唐容川）：活血化瘀是最终手段。干漆破陈年瘀血。

### 倪联（本草→经方）
- **辛夷+菖蒲+苍术 → 倪氏鼻窦炎三方**：不管寒热，三味药通治鼻塞。寒加干姜炙甘草；热加杏仁麦冬半夏。(nishizhu #倪联7)
- **桂枝 → 桂枝汤**（经方第一大方，解肌）+ 60%经方有桂枝
- **桂枝+白术+附子 → 风湿关节炎方**（桂枝祛风+白术祛湿+附子祛寒=风寒湿痹）
- **茯苓+桂枝+白术+甘草 → 苓桂术甘汤**（心动悸）
- **茯苓+桂枝+甘草+大枣 → 苓桂甘枣汤**（太阳汗后奔豚）
- **酸枣仁+知母+茯苓+川芎+甘草 → 酸枣仁汤**（虚烦不得眠）
- **黄柏+黄连+黄芩 → 三黄汤**（小便刺痛、膀胱热、结石）
- **王不留行+其他药 → 王不留行散**（经方，金疮）
- **肉桂+附子+六味地黄 → 桂附八味丸**（命门火衰、虚寒体质）
- **槐实熬膏 → 塞肛门**（痔疮专方）

### 倪问（诊断思路）
- **辨桂枝证**：听到"气往上冲""一阵热往上跑""心脏要从嘴巴跳出来"→桂枝。
- **辨肉桂证**：下痢+恶寒+怕冷+喜温水+脚冷→桂附八味丸。便秘+口渴+燥渴+喜冷水→六味地黄丸（不可开反）。
- **辨苓桂术甘汤证**：心动悸+肌肉跳动+眼皮跳→水气凌心，皮下有水。
- **辨辛夷证**：鼻塞→辛夷+菖蒲+苍术。白涕→加干姜；黄涕→加清热药。
- **辨酸枣仁证**：虚烦不得眠+表虚发汗→酸枣仁汤。实邪热者（肝炎发黄）禁用。
