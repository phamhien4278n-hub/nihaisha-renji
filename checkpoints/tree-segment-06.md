# 金匮要略 段6 知识树

## 核心概念

### 一、百合病变证（接前段百合病）

#### 1. 百合病变发热 → 百合滑石散
- **病机**：百合病本就津液伤，若再发热（低热非高热），可能膀胱有炎症或结石
- **方药**：百合 + 滑石（滑石利水通淋，合用滋阴清热利水）
- **辨证要点**：这是低热，绝非高热

#### 2. 百合病治则大法（核心辨证原则）
> 「百合病，见于阴者，以阳法救之；见于阳者，以阴法救之。见阳攻阴，复发其汗，此为逆；见阴攻阳，乃复下之，此亦为逆。」

- **核心思想**：病人虚的状态下不要强攻
- **阳证（热证）**：用阴法——开滋阴的药（麦门冬、党参等清味药补津液），不要用桂枝汤
- **阴证**：用阳法——温和扶阳，不峻攻
- **禁忌**：大黄、麻黄、桂枝、黄芩、黄连都是强攻药，虚人不用
- **经方原则**：经方药很凶（一剂知），百合病这种虚证不要挑经方里的峻剂

### 二、狐惑病

#### 1. 核心症状
> 「狐惑之为病，状如伤寒，默默欲眠，目不得闭，卧起不安。蚀于喉为惑，蚀于阴为狐，不欲饮食，恶闻食臭，其面目乍赤、乍黑、乍白。」

- 蚀于喉 = 惑（甘草泻心汤）
- 蚀于阴 = 狐（苦参汤洗之）
- 蚀于肛 = 雄黄熏之
- 面目颜色变化快（乍赤乍黑乍白）→ 有寄生虫在肚子里

#### 2. 甘草泻心汤

**组成**：炙甘草四两、黄芩三两、人参三两、干姜三两、半夏半升、黄连一两、大枣十二枚

**煎法**：水一斗煮取六升，去滓再煎取三升，温服一升，日三服

**核心主治**：
- 心下痞硬 + 腹中雷鸣 + 下利 + 恶心 = 标准甘草泻心汤证
- 特殊主治：**梦游（丢搞）、凭依症、精神异常**
  - 梦游找食物吃 → 甘草泻心汤
  - 梦游跳舞 → 甘草泻心汤
  - 被关三天的猫跳出来吓到→动作像猫（凭依症）→ 甘草泻心汤
  - 西药安眠药副作用导致梦游（眼睡了神没睡）→ 甘草泻心汤

**病机**：肠胃功能不好 → 精神方面问题。寒热药并用（干姜+黄连黄芩），半夏去水止呕，芩连去胃热，甘草人参大枣补肠胃津液

**重要临床思维**：
- 西医诊断「动脉血管堵塞」，但病人陈述是甘草泻心汤证（心下痞硬+肠鸣+下利+恶心）→ 开甘草泻心汤，不开心脏病处方
- 中医辨证论治 vs 西医看病名论治

**五个泻心汤**：甘草泻心汤特殊功能在精神/梦游/凭依

**药性规律（可推演处方）**：
- 黄连、黄芩 → 肠胃热
- 黄柏 → 肾脏虚热（看似去热，实际补肾脏、滋阴）
- 三黄泻心汤用黄柏 → 偏肾脏

#### 3. 苦参汤（蚀于下部则咽干）
- 苦参可内服可外用洗剂
- 用于阴部蚀烂

#### 4. 雄黄熏法（蚀于肛）
- 雄黄 = 三硫化二砷（As₂S₃）
- **关键危险**：加热 → 硫变三氧化二砷（As₂O₃ = 砒霜！）
- **正确操作**：瓦片加热→热时倒雄黄→青烟熏肛门，**操作者绝不能闻到**
- 适应：肛门寄生虫、同性恋肛交后发炎

### 三、赤豆当归散（脓已成）

#### 1. 核心条辨
> 「病者脉数，无热微烦，默默但欲卧，汗出。初得之三四日，目赤如鸠眼，七八日，目四眥黑，若能食者，脓已成也，赤小豆当归散主之。」

**辨证链条**：
- 脉数（内有热） + 无热（表无热） + 微烦 + 想睡觉 + 汗出
- 3-4天 → 眼睛红如鸠鸟
- 7-8天 → 眼眶下两条黑（像橄榄球面罩印）
- 能吃东西 → **脓已成！**

#### 2. 两个关键部位

**（1）胆囊破裂 → 腹膜炎**
- 胆结石/胆汁太浓/造影剂堵塞 → 胆越来越大 → 破掉 → 腹膜炎
- 破后：局部烫，旁边冷（三焦油网包住固定）
- 破前持续剧痛（压力导致，非发炎），破后压力释放痛消失——假象！实际脓已成

**（2）盲肠（阑尾）破裂**
- 急性盲肠炎来不及处理（开刀/大黄牡丹皮汤都来不及）→ 阑尾破 → 局部包块

#### 3. 赤豆当归散

**组成**：赤小豆三升（发芽晒干）、当归十两（实际等量即可，如各五钱）

**制法关键**：
- 赤小豆必须**发芽**（湿纸+浅水，2-5天，像豆芽长度即可）
- 发好芽的赤小豆冷冻保存备用
- 当归必须**冷藏**（当归油脂是药效关键，干枯破裂=无效）
- 赤小豆+当归打散，**生用，不煮**（浆水服，像生菜沙拉）
- 生用 → 药力散；煮了 → 药力集中但无法透到腹膜

**药理**：
- 赤小豆发过芽 → 除湿且能透到腹膜深处
- 当归 → 活血补血
- 两药生用 → 散的力量强，专排腹膜脓

**与术附汤的分工**：
- 术附汤 → 四肢末梢、肌肉组织下、关节里的脓
- 赤豆当归散 → 腹膜里的脓（胆区/盲肠区）

#### 4. 倪师临床经验——手掌诊断法
- 左手感情线上有褐色斑块（硬币大小）→ **慢性腹膜炎**
- 同时按阑尾点（足三里下一寸）剧痛 → 确认
- 案例：英国太太腹痛15年，西医诊为精神病，吃赤豆当归散+大黄牡丹皮汤一个月，斑块消退八成

### 四、阳毒与阴毒（升麻鳖甲汤）

#### 1. 病因——"战争伤寒"
- 群聚斗室（监狱、船舱、防空洞、偷渡货柜）
- 闷热+津液耗损+大小便不畅+食物毒素排不出 → 发病
- 传染极快，一人得二人得
- 阳毒 = 病在气分；阴毒 = 病在血分

#### 2. 阳毒
> 「阳毒之为病，面赤斑斑如锦纹，咽喉痛，唾脓血，五日可治，七日不可治」

**升麻鳖甲汤去雄黄蜀椒主之**（原文有"去雄黄蜀椒"）

**组成**：鳖甲（炙，手指大一片）、雄黄半两（研）、升麻二两、当归二两、甘草二两、蜀椒一两（炒去汗）

**煎法**：水四升煮取一升，顿服，老小再服，取汗

#### 3. 阴毒
> 「面目青，身痛如被杖，咽喉痛，五日可治，七日不可治」

**升麻鳖甲汤去雄黄蜀椒**（真正的去雄黄蜀椒在阴毒）

**组成**：鳖甲、升麻、当归、甘草（无雄黄、无蜀椒）

**病机**：毒在血分，血凝固、血速变慢

#### 4. 方解
| 药 | 作用 |
|----|------|
| 鳖甲 | 沉潜、入肝，保肝，炙用 |
| 当归 | 入血分，带鳖甲入肝（肝藏血） |
| 升麻 | 解热毒 |
| 雄黄 | 解虫蛇咬伤毒、疮毒，**必须生用**（煮=砒霜） |
| 蜀椒 | 杀虫（辛辣驱虫），巴蜀四川产的花椒 |
| 甘草 | 调和 |

**临床扩展**：带状疱疹 → 若面色青胀+咽喉痛（阴毒表现）→ 升麻鳖甲汤去雄黄蜀椒（vs 常规龙胆泻肝汤）

### 五、疟病

#### 1. 基本辨证
> 「疟脉自弦，弦数者多热，弦迟者多寒。弦小紧者下之差，弦迟者可温之，弦紧者可发汗、针灸也，浮大者可吐之，弦数者风发也，以饮食消息止之。」

- **核心脉象**：疟脉自弦
- 弦数 → 多热
- 弦迟 → 多寒
- 弦小紧 → 可下
- 弦紧 → 可发汗/针灸
- 浮大 → 可吐（病在上焦）
- 弦数 → 风发，饮食调理

#### 2. 核心治则——「无痰不疟」
- 所有疟病必有痰
- **去痰是不二法门**
- 温病派误当阴虚治 → 心下痞（六大弊端之一）
- 经方治阴虚 ≠ 滋阴药，而是**开阳药恢复功能，阴自回**（从食物中取得营养）

#### 3. 疟母 → 鳖甲煎丸
> 「病疟，以月一日发，当以十五日愈，设不差，当月尽解。如其不差……此结为症瘕，名曰疟母，急治之，宜鳖甲煎丸。」

- 一月一发 → 两个礼拜自愈；不好再等两个礼拜
- 不愈 → 疟母（痰+血混结成痞块，定时发作）
- 鳖甲煎丸：23味药，加锻灶下灰+清酒制丸，如梧子大，空心服七丸，日三服
- 倪师未用过——现在疟疾很少遇到
- 若真遇疟母 → 按症状辨证去痰即可，不一定要鳖甲煎丸

## 病机推理链

1. **百合病虚证** → 不可强攻 → 阳证滋阴/阴证扶阳 → 违之则逆
2. **肠胃功能失常** → 精神异常（梦游/凭依）→ 甘草泻心汤（寒热并调+补津液）
3. **内脏化脓** → 脉数无表热+目四眥黑+能食 → 脓已成 → 赤豆当归散（发芽赤小豆+生当归）
4. **群聚闷热津伤** → 食物毒素内积 → 阳毒（气分）/阴毒（血分）→ 升麻鳖甲汤加减
5. **疟必有痰** → 去痰为治疟核心 → 痰血互结 → 疟母（鳖甲煎丸）

## 关键治则

| 治则 | 出处 |
|------|------|
| 虚证不攻，以调代攻 | 百合病"见阳攻阴…此为逆" |
| 寒热并用调肠胃 | 甘草泻心汤（干姜+芩连） |
| 化脓后压力释放痛减≠病愈 | 赤豆当归散条辨 |
| 发芽赤小豆透腹膜+生用散力强 | 赤豆当归散制法 |
| 肝有病鳖甲+当归（引经入肝） | 升麻鳖甲汤方解 |
| 雄黄生用解毒/煮用成砒霜 | 升麻鳖甲汤禁忌 |
| 无痰不疟 | 疟病核心治则 |

## 新学到的东西

1. **甘草泻心汤治梦游/凭依症**——以前只知道五个泻心汤治心下痞，不知道甘草泻心汤有精神科的独特应用
2. **雄黄加热=砒霜的化学原理**——As₂S₃ → 加热 → As₂O₃（砒霜），操作必须瓦片+自己不能闻
3. **赤小豆必须发芽才能透腹膜**——生药散力+发芽除湿透深，不发芽无效
4. **当归必须冷藏**——油脂是药效关键，干枯破裂=废药
5. **手掌感情线褐斑=慢性腹膜炎**——倪师独门诊断法
6. **脓成后痛减是假象**——压力释放≠病愈，反而是脓已成的标志
7. **升麻鳖甲汤中当归带鳖甲入肝**——引经思维，鳖甲沉潜不知去处，当归入血分引其入肝
8. **经方治阴虚不是滋阴是复阳**——开阳药恢复脏腑功能，阴自回（从食物取营养），与温病派根本区别
