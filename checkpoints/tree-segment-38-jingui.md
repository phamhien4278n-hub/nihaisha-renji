# 金匮要略 段38 —— 收官：肺癌总结+肺炎/子宫肌瘤/肾癌/狼疮案例 + 易筋经全集 + 结语

## 一、肺癌阴实案例总结（接上段）
### 治疗时间线
- 2月→8月，半年
- 初：寒湿痰咳 → 清热（苇茎汤少量）→ 寒热交替 → 纯热 → 阴实去除

### 病理转化链（核心发现）
> 先去寒→寒去变热→热多寒少→再清热→寒热交替→最终纯热→阴实自消

### 肺癌下利机制
- 肺与大肠相表里 → 肺有阴实 → 肺气不能固守肛门 → 下利（一天二三十次）+ 肺痛严重

### 紫参汤（金匮治肺痛下利唯一方）
- 紫参 + 甘草，一剂即效
- **紫参缺货替代方**：丹皮 + 桃仁 + 冬瓜仁 + 紫菀 + 冬花 + 射干 + 桔梗 + 甘草

### 预后关键
- **未经西医（开刀/化疗）的第一诊中医**：希望极大，"一定可以治好"
- 做过开刀化疗 → 病情变样 → 无法正确追踪 → 预后差

## 二、肺炎/肺脓疡案例（1998年12月）

### 病例
72岁男性，抽烟斗。其父95岁摔伤肩膀，被家人送医院照X光→感染院内肺炎→一周死亡。患者拒绝去医院。

### 初诊症状
| 症状 | 辨证意义 |
|------|---------|
| 舌苔黄湿厚 | 湿盛+肺中极热 |
| 脉数 | 发炎发烧 |
| 五天未大便 + 无屁 | **大承气汤证** |
| 小便深黄 | 病在承气汤 |
| 咳白色泡沫痰 | 肺化脓/肺脓疡（非焦黄痰） |
| 口渴喜冷饮 | 内热 |
| 胃口差 | 腑气不通 |
| 睡眠每两小时醒 | 肺热扰神 |

### 处方（一剂知、二剂已）

| 方剂 | 组成 | 作用 |
|------|------|------|
| **苇茎汤**（千金要方） | 苇茎、薏仁（代冬瓜仁）、丹皮、桃仁 | 清湿热、排肺脓 |
| **桔梗甘草汤**（排脓汤） | 桔梗5钱、甘草（生用） | 排肺脓疡 |
| **大承气汤** | 大黄3钱、芒硝（冲服）、厚朴、枳实 | 通腑泻热 |
| **栝蒌实** | 5钱 | 固心阳，防壮热伤心 |
| **炙甘草→生甘草** | — | 太热，用生甘草生津液 |

### 治疗经过
1. **12/9 初诊**：第一碗药→不到一小时大便出→**烧退冒汗**（大承气解发烧标志）
2. **12/11 二诊**：睡眠好转、胃口恢复、还有冷汗（表阳虚）→ 减大承气量 + 麦冬/西洋参
3. **12/15 三诊**：减大承气后便秘复发 → 恢复原方大承气 + **百合**（患者善变→百合证）→ 3副痊愈
4. **总疗程：9-10天**

### 关键教训
- **肺热+便秘 = 大承气汤证**：不用大承气，肺热不会好，会死
- 肺炎进入内脏不用大承气 → 死
- **百合证**：善变 = 肺魄不守（肺藏魄） → 百合色白入肺、润肺固魄

## 三、子宫肌瘤→不孕→怀孕案例

### 病史
- 1997/7 发现肌瘤 → 11月第一次手术（7+5+3cm）
- 2002/9 第二次手术（7颗大肌瘤，月经停不掉）
- 2006/12 第三次手术（**28颗肌瘤**，最小荔枝大，血红素降至2.6）
- 2007/1 开始服中药 → **3月中旬怀孕**（一个半月）

### 处方（当归四逆汤加减，一方到底未换）

| 药物 | 剂量 | 作用 |
|------|------|------|
| 炮附子 | 5钱 | 壮肾阳 |
| **生附子** | 3钱 | 强心阳（炮附子不能替代） |
| 桂枝 | 3钱 | 阳药，推热入小肠 |
| 白芍 | 6钱 | 止痛 |
| 生姜 | 两片 | — |
| 干姜 | 2钱 | — |
| 吴茱萸 | 3钱 | 止呕（胃口不好） |
| 当归 | 2钱 | 补血 |
| 熟地 | 3钱 | 补血 |
| 艾叶 | — | 暖子宫（胶艾汤思路） |
| 续断 | 3钱 | 修复手术刀疤 |
| 柴胡 | 3钱 | 疏肝 |
| 郁金 | 2钱 | 解郁 |
| 蜀椒 | 2钱 | 大建中汤——少腹大寒痛 |
| 丹皮 | 3钱 | 活血化瘀 |
| 桃仁 | 3钱 | 活血化瘀 |
| 辛夷 | 3钱 | 鼻窦炎/开心窍 |
| 菖蒲 | 3钱 | 开窍 |

### 核心理论
1. **生附子+炮附子同用**：下焦寒湿极盛（脚冰）→ 炮附子壮肾阳不够，须生附子强心
2. **寒极生热**：极寒时不用热药，用凉药（黄芩、黄连）反能生热——如小火点燃
3. **阳极生阴、阴极生阳**：至阳药含阴，至阴药含阳
4. **大建中汤→大乌头煎**：少腹大寒痛→蜀椒干姜→无效→大乌头煎（乌头蜜慎用）

### 怀孕后用药
- **前20周**：当归散
- **后20周**：白术散（含牡蛎，强胎儿）

### 排毒反应
- 阳药推动阴积（沉积物）从头部排出 → 头冒汗臭
- 排毒后百病去："生完小孩=得贵子+百病去"

### 生男理论
- 阳中带阴的药 → 精气先到子宫（精先到位，气后到位） → **生男孩**

## 四、肾癌案例（片段）

### 病史
左肾癌切除后19年 → 右肾上方5cm肿瘤 → 5次化疗差点死 → 尿血/咳血/无胃口

### 初诊策略：虚则实之——先守后攻
- **不可直接攻癌**（化疗已攻5次，再攻即死）
- **小建中汤** + 白术/茯苓（去湿）+ 杏仁（肺上元补水）+ 厚朴（行气）+ 麦芽糖
- 一剂后第二天：精神恢复，厨房整理、衣服洗净

## 六、易筋经全集

### 文式易筋经（十二式）
- 每式49下
- 核心原则：下半身绷紧如磐石，上半身放松；脚如爪抓地；舌顶上颚，自然呼吸
- 十二式：按跷→手指→平拳→肚脐前→正前方→胸前→侧拉弓→碌辘式(九/十，身转腰不动)→后拉→抱弓式

### 武式易筋经
- 韦驮献杵（三式）、拨云见日、摘星换斗、拔马刀式、青龙探爪、倒踩九牛尾（旋转甩手有来有回）、**出爪亮翅**（震颤排痰）、最后式（手先着地→脚走进→踮起来）
- 武式每式1-2次，49下会昏迷

### 出爪亮翅式——震颤排痰
- 全身放松、马步→震动从手传至全身→筋脉血脉神经肌肉中的**痰**被排出
- 替代**藜芦甘草汤**（冶fujue/手抖，药已失传）

### 最后式（阴阳调和）
- 手先着地→脚慢慢走进→踮起来→阴阳相合至极
- 能做到此式 = 身体无大问题，阴阳平衡

## 七、倪师结语嘱托

1. **先精通经方** → 再广泛阅读各家（徐灵胎、唐容川、曹颖甫等）→ 才不会人云亦云
2. 经方不熟只看旁书 = 脱离中医正轨，无法达到高境界
3. 多练功+打坐 → 医生气血好、精神好 → 病人信心足
4. 人纪不是医学极限，鼓励后人继续研发
5. 打坐不一定坐着，躺着也可以练（掌握心法即可）

## 八、本段核心临床思路提炼

1. **肺癌阴实转化链**：先祛寒→寒热交替→纯热→阴实消——不是一直攻阴实
2. **肺脓疡三件套**：苇茎汤 + 桔梗甘草排脓汤 + 大承气汤（关键在通腑）
3. **百合证**：善变=肺魄不守，百合润肺固魄
4. **寒极生热**：极寒用凉药（芩连反生热）
5. **热极生寒**：极热用热药（生附反退热）
6. **虚则实之**：大虚先守后攻（肾癌用小建中汤先复元气）
7. **蝉蜕+蛇蜕**：两蜕并用覆盖全身皮肤+关节
8. **甘淡渗利防腹水**：淮山+薏仁+芡实——疏肝同时防腹水于未然
9. **生附+炮附同用**：下焦寒湿极盛，炮附壮肾阳+生附强心阳
10. **震颤排痰**：易筋经出爪亮翅替代藜芦甘草汤
