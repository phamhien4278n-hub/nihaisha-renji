## 一、核心病机（黄帝内经）

**总病机**：「脾色必黄，瘀热以行」——脾湿累积→脾肿大→压迫胆→胆汁逆流→入肝→入全身→发黄。

**内外二因**：
- **外因**：表风束缚 + 汗不透发 + 体内湿盛 → 湿排不出去 → 脾湿过盛 → 压迫胆 → 胆汁逆流
- **内因**：脾湿太盛 → 土克水（肾）→ 肾阳上逆 → 胃加速 → 脾慢胃快 → 不平衡 → 黄疸

**倪师核心公式**：胃快 + 脾慢 = 黄疸（胃蠕动正常/加速，脾蠕动慢 → 食物堵胃 → 浊水渗入膀胱 → 小便浓稠/不利 → 发黄）

**内经关联**：
- 「诸湿肿满，皆属于脾」——黄疸本质是脾湿
- 「木克土」——肝病先实脾，黄疸常伴肝脾同病
- 大包穴为脾之大络，总管全身大络，负责五色营养分配——黄疸为木克土，脾胃不收黄色营养[C1]
- 「肾病者，腹大胫肿，喘咳身重，寝汗出，憎风」——肾阳上逆致胃加速的链条[C1]

## 三、方剂鉴别核心口诀

```
茵陈蒿汤 = 黄 + 便秘 → 茵陈(君) > 栀子 > 大黄
茵陈五苓散 = 黄 + 小便不利 + 口渴 + 大便不秘 → 茵陈 + 五苓散
栀子大黄汤 = 酒疸 + 便秘 + **不黄**(无茵陈=不发黄)
大黄硝石汤 = 黄 + 腹满 + 小便赤 + 自汗出 → 表和里实
猪膏发煎 = 大便黑 + 肌肤甲错 → 滋润+活血
硝石矾石散 = 黑疸/肝癌末期 → 攻瘀+收湿
桂枝加黄芪汤 = 黄 + 脉浮 → 汗解
小半夏汤 = 小便色不变(无热) + 哕 → 不可除热
小建中汤 = 虚劳黄 + 小便自利
小柴胡汤 = 黄 + 腹痛而呕 + 少阳证
```

## 六、倪师临床心法（核心口诀）

1. **治黄大原则**：「诸病黄家，但利其小便」——去湿是第一要务。疸而渴者难治(利湿加重口渴)，疸而不渴者可治
2. **茵陈不辨寒热**：见黄即可用茵陈，就像大黄不辨寒热只去实
3. **阳黄→阴黄=失败**；**阴黄→阳黄=病退**=厉害(从里走表)
4. **十八日为期**：阳黄十日可瘥，十日不瘥→难治
5. **大便颜色诊断**：黄=正常；黑=瘀血；柏油样块状黑=内脏腐烂(胃癌末期将死)
6. **舌诊**：舌萎黄=湿盛；舌黄=热
7. **额黑=胃气绝**——危证
8. **胆结石取胆**→最快15年最慢20年→肝结石（更难清，因胆汁仍在累积）
9. **吃内脏要加醋**：醋清胆道；不喜欢酸的人易胆道堵塞
10. **酒客便秘不用大黄**：用柏子仁/麻子仁/当归/酸枣仁等仁剂通便，攻下→黑疸
11. **硝石矾石散禁忌**：必须确定西医没碰过病人(没开过刀/没做过介入)
12. **剂量到位一剂就中**：茵陈重用一两起；剂量不够则方子对了也慢慢好

## 八、薄弱点（需后续加强）

1. **伤寒论六经黄疸体系**：目前数据主要来自金匮+伤寒少阳/阳明。太阴寒湿发黄、厥阴寒热错杂发黄的原文引用不够充分——C2推理偏多
2. **针灸黄疸体系**：腹四针辨证(小便淡白=寒灸天枢 vs 深黄=实热四针全下)来自倪师临床，但五门十变、子午流注等治则体系在黄疸上的应用尚未充分覆盖
3. **黄帝内经原文溯源**：「脾色必黄，瘀热以行」实际出自金匮，内经关于黄疸的专门篇章(如《素问·平人气象论》「溺黄赤安卧者黄疸」「目黄者曰黄疸」等)尚未逐条对照
4. **女劳疸与黑疸的边界**：女劳疸转黑疸的转化关系需进一步理清
