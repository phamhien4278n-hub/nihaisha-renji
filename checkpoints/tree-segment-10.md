# 伤寒论 第10段 知识树

## 段10：热利三方 + 风湿三方 + 阳明篇开篇 + 承气汤全系列

### 一、热利三方鉴别（条文187区域）

#### 核心鉴别点：腹痛

| 方剂 | 症状 | 关键鉴别 |
|------|------|----------|
| **葛芩连汤** | 热利，大便臭 | **不腹痛** |
| **黄芩汤** | 热利，大便臭，**腹痛** | 肚子痛 |
| **黄芩加半夏生姜汤** | 热利，腹痛 + **恶心** | 黄芩汤+恶心 |

- 黄芩汤组成：黄芩、芍药、炙甘草、红枣（没有半夏生姜，没有人参）
- 加半夏生姜 = 黄芩加半夏生姜汤（半夏+生姜 = 小半夏汤，专门止呕）
- 生半夏 vs 制半夏：制半夏=半夏+生姜炮制过，止呕力弱；生半夏+生姜入汤剂，止呕力强很多
- 一般小孩子肠胃炎吃坏肚子，处方都在这里

### 二、黄连汤（条文188）

**条文**：伤寒，胸中有热，胃下有邪气，腹中痛，欲呕吐者，黄连汤主之。

**黄连汤方**：黄连三两、甘草三两炙、干姜三两、桂枝三两、人参三两、半夏半升洗、大枣十二枚劈
— 全部等量，三三三三

#### 核心要点

- 此"腹痛"是**胃痛**，不是小肠痛
- 黄连用在**胃/胸腔**的热（发炎）
- 黄芩用在**小肠**的热
- 大黄用在**胃之上**（肺/上焦）的热

#### 黄连汤的配伍逻辑

- 黄连 → 去胃热（消炎）
- 干姜 → 胃寒（用干姜止呕，不是半夏——因为胃里寒）
- 桂枝 → 强心阳，活血，把血带到全身
- 人参 → 补津液
- 半夏 → 胃周围有水时止呕
- 红枣 → 补胃的津液
- 炙甘草 → 和中

**关键认知**：胃痛时寒热并结——胃寒+发炎，所以寒热药并用。只要没有吐血，胃不舒服就可以用桂枝。

#### 黄芩汤 vs 黄连汤 区分
- 黄芩汤：**肠子痛 + 下利**（小肠热利）
- 黄连汤：**胃痛 + 呕吐**（胃炎，寒热并结）

### 三、风湿三方（条文189-190）

#### 3.1 桂枝附子汤（条文189前半）

**条文**：伤寒八九日，风湿相搏，身体烦疼，不能自转侧，不呕，不渴，脉浮虚而涩者，桂枝附子汤主之。

**组成**：桂枝四两、附子二枚（炮，去皮，破八片）、生姜三两、甘草二两炙、大枣十二枚
— 桂枝汤去白芍 + 炮附子

- 不呕 = 不在少阳
- 不渴 = 不在阳明
- 脉浮虚而涩 = 表证已去，涩为湿脉

#### 桂枝 vs 炮附子的角色
- 桂枝 → **去风**（痛到处游走 = 风）
- 炮附子 → **去寒**（痛来自寒堵，非发炎）

**张仲景对痛的理解**：痛来自**压力**而非发炎。寒堵住→水过不去→累积→压力→痛。炮附子去寒→寒水蒸化→水循环恢复→痛消失。证据：牙龈脓肿打洞后脓流出→压力释放→痛消失，但炎症还在。

**为什么加生姜甘草大枣**：补津液。如果只用桂枝+炮附子（桂附汤），痛会去但会便秘——因为没有补津液。

**剂量**：炮附子:桂枝 ≈ 5:2（炮附子五钱，桂枝两钱）

#### 3.2 桂枝附子去桂加白术汤（条文189后半）

**条文**：若其人大便溏，小便自利者，桂枝附子去桂枝加白术汤主之。

**组成**：白术四两、甘草二两炙、附子三枚炮、大枣十二枚、生姜三两

- 大便溏 + 小便自利 = 病人湿太盛，已经在自行排湿 → **不需要桂枝（没有风证）**
- 白术 → **去湿**
- 炮附子 → **去寒**
- 结果：寒+湿，无风 → 术附汤

**服药反应**：
- 初服：身如痹（湿在动，非附子中毒）
- 半日再服 → 三服尽 → 其人如冒状（术附逐水汽未得除）
- 如果还有游走现象 → 法当加桂枝四两（此本一方二法也）

**剂量**：炮附子 > 白术（炮附子五钱~八钱，白术三钱）

#### 3.3 甘草附子汤（条文190）— 痛风！

**条文**：风湿相搏，骨节烦疼，掣痛，不得屈伸，近之则痛剧，汗出，短气，小便不利，恶风不欲去衣被，或身微肿者，甘草附子汤主之。

**关键症状**：
- 骨节痛（不是肌肉痛！）
- 掣痛（拉扯痛）
- 不得屈伸
- 近之则痛剧（碰都不能碰）
- 汗出短气、小便不利、恶风

→ **这就是痛风！倪师说学会了这条，痛风就是一片蛋糕。**

### 四、炙甘草汤 / 复脉汤

#### 适应症：脉结代，心动悸

#### 方剂组成
- 炙甘草四两、生姜三两、桂枝三两、人参二两、生地黄一斤、阿胶二两、麦门冬半升、麻子仁半升、大枣三十枚
- 九味，用**酒三碗+水四碗**煮（酒助发散）
- 先煮八味，阿胶不入锅煮，最后烊化

#### 配伍逻辑
- **生地黄一斤** → 补**液**（黏黏的才是生地，生地炭无效）
- **阿胶** → 补心血，不能煮（生用）；另有止血功能
- **麻子仁** → 通利肠子，润肠（心脏治小肠）
- **桂枝** → 把血打到四肢 + 恢复胸阳
- **人参** → 补津液
- **麦门冬** → 润肺
- 炙甘草 → 主药

#### 津 vs 液（灵枢·决气篇）
- **津**：肌肉腠理的水分，汗——阳。津不足→人参。
- **液**：谷气入肠胃→注入骨髓、脑髓、骨节——阴。液不足→生地黄。

→ 炙甘草汤是**补阴**的，把心脏的血补足、胸阳恢复、胸腔津液恢复。

又名**复脉汤**，不限于脉结代，脉不振/脉失掉都可用。

### 五、阳明篇开篇

#### 5.1 条文195：阳明之为病，胃家实也
- 阳明 = 胃 + 十二指肠 + 小肠 + 大肠
- "实"不是"热"——实是有东西堵到
- 黄芩汤、黄连汤是去热的（发炎），承气汤是去实的（堵到）

#### 5.2 条文196：转属阳明的原因
- 发汗太多 / 利小便太多 / 亡津液 → 胃中干燥 → 转属阳明
- "不更衣" = 便秘（古人上厕所换衣服）

#### 5.3 条文197：阳明外证
- 身热、汗自出、**不恶寒、反恶热**
- 纯热证，看不到寒证

#### 5.4 条文198-199：传经
- 病得之一日（一候，6-7天），恶寒自罢→自汗出而恶热→传阳明
- 阳明居中土，万物所归，**无所复传**——病到阳明停住
- 便秘早治，脸色皮肤好（阳明经走面部：承泣四白巨髎等）

### 六、倪大黄消化系统全系列（从口到肛门）

> 注意：以下部分方剂来自金匮，但倪师在第10段总结时一并讲解。

#### 6.1 大黄甘草汤 —— 胃（坏死的食物）
- 症状：**朝食暮吐**（早上吃的黄昏吐出）
- 舌苔黄
- 实证：持续痛
- 大黄 + 甘草

#### 6.2 调胃承气汤 —— 十二指肠
- 部位：中脘到神阙之间（下脘/建里/水分）压痛
- 症状：不吐不下，心烦（梗在中间，吐不出又下不去）
- 组成：大黄（酒洗）、芒硝、炙甘草
- 炙甘草让大黄芒硝速度变慢，停在十二指肠

#### 6.3 小承气汤 —— 小肠
- 症状：**不断放屁**（大肠空，气多）+ 腹大满
- 组成：大黄、厚朴、枳实（**无芒硝**）
- 关元穴（小肠募穴）压痛

#### 6.4 大承气汤 —— 大肠
- 症状：**屁也不放一个** / 大便如长鞭 / 热结旁流（下利清水，肚子硬）
- 组成：大黄、芒硝、厚朴、枳实
- 天枢穴（大肠募穴）压痛
- 小便深黄
- 最强者

#### 6.5 大黄牡丹皮汤 —— 阑尾炎（急性盲肠炎）
- 症状：右脚弓起来舒服 + 阑尾穴压痛
- 组成：大黄、牡丹皮……（金匮方）

#### 6.6 当归赤豆散 —— 阑尾/胆囊破裂后腹膜炎
- 症状：剧痛→突然不痛，局部发烫周围冰冷
- 赤小豆**必须发芽** + 当归
- 不煮，打在一起吃

#### 6.7 己椒苈黄丸 —— 受惊吓+便秘+水气
- 症状：肚子压下去咕咚咕咚水声 + 便秘
- 组成：防己、椒目、葶苈子、大黄（丸剂）+ 芒硝汤冲服

#### 6.8 大柴胡汤 —— 大肠堵+恶心
- 阳明证不应有恶心，有恶心=兼少阳→大柴胡汤
- 屁都不放 → 大柴胡汤加芒硝

#### 6.9 麻子仁丸 —— 脾约（小肠堵，津液不足）
- 症状：大便一粒一粒像羊屎（六味地黄丸状）
- 必须用丸剂，不能用汤

#### 6.10 薏苡附子败酱散 —— 盲肠段
- 症状：右下腹隐隐作痛，拒按，**不需脚弓**
- 左手生命线出现紫斑/灰斑（开心果大小）

### 八、阳明篇其他重要条文

#### 条文211：谷疸
- 脉迟、食难用饱（吃一点就撑）→ 脾脏消化功能不好
- 硬吃→谷疸（全身发黄）
- 治脾脏，不是治胃

#### 条文212：身如虫行皮中
- 阳明病应多汗，反无汗→久虚→汗透不出来→皮肤痒
- 如运动后进冷气间皮肤痒

#### 条文213：阳明头痛 = 阙上痛（印堂痛）
- 印堂痛→针中脘，当场去痛
- 胃的清气上升→气街（头维到眼睛这段）
- 浊气下降→足三里（合穴）

#### 条文214：阳明咽痛
- 阳明病咳嗽→肺有表症 + 胃热上冲→咽痛
- 扁桃腺没有发炎化脓，就是喉咙痛——燥热造成

#### 条文218：口燥漱水不欲咽 → 必衄
- 口渴想喝水但不想吞下去 → **有淤血**（唐容川《血证论》）
- 血冲到鼻粘膜→还没破→淤血感
- 鼻血一出→热退+口渴消失

#### 条文219：小便减少 → 津液回头
- 问小便日几行 → 小便减少 → 津液当还入胃中 → 不久必大便

#### 条文220：呕多 → 虽有阳明证不可攻
- 有呕 = 病在少阳 → 先解表，后攻里
- 不然表邪下陷→结胸

#### 条文221：心下硬满 → 不可攻
- 心下（胃）硬满 = 大陷胸汤/丸证，非承气汤证
- 攻之→利遂不止者死

#### 条文222：面含赤色 → 不可攻
- 津不足 → 再攻 → 津液更伤 → 发黄

#### 条文223：不吐不下心烦 → 调胃承气汤

#### 条文224：大承气汤 vs 小承气汤
- 潮热 + 大便硬 = 大承气
- 外未解（汗多微发热恶寒）+ 热不潮 → 不可大承气
- 腹大满不通 → 小承气，微和胃气

#### 条文225：小承气测试法
- 不大便六七日，先给小承气 → 转矢气 = 有燥屎 → 可攻
- 不转矢气 = 初头硬后必溏 → 不可攻（里寒湿）

#### 条文226：实则谵语，虚则郑声
- 谵语：乱讲、胡说八道（实）
- 郑声：讲话反复、重复（虚）
- 闻诊——听病人陈述区分虚实

### 九、核心认知总结

1. **热利三方**：腹痛 = 分水岭。不痛→葛芩连；痛→黄芩汤；痛+恶心→加半夏生姜
2. **黄连 vs 黄芩 vs 大黄**：位置决定用药——黄连胃/胸，黄芩小肠，大黄上焦
3. **痛来自压力而非发炎**：炮附子去寒→恢复水循环→痛消失（张仲景汉朝就懂）
4. **风湿三方**：桂枝去风，附子去寒，白术去湿——风+寒=桂附汤，寒+湿=术附汤，风寒湿全=桂枝附子白术同用
5. **痛风 = 甘草附子汤**：骨节烦疼掣痛不得屈伸近之则痛剧
6. **津 vs 液**：津=阳=汗=肌肉水分（人参补），液=阴=骨髓脑髓（生地黄补）
7. **承气汤系列定位**：胃→十二指肠→小肠→大肠，一层一层往下，每层对应不同方剂
8. **阳明无所复传**：病到阳明停住，但必须治——便秘早治，脸色好
9. **赤小豆必须发芽**：当归赤豆散治腹膜炎，不发芽无效
10. **口燥漱水不欲咽 = 淤血**：这是唐容川《血证论》的核心观点
