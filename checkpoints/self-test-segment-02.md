# 段2 闭卷自测

## 题目1：冬不藏精，春必病温

**辨证病机**：
此患者犯了两条大忌——①冬三月当闭藏，他洗冷水澡（伤阳）+ 高强度夜跑至大汗（冬天不适合出汗，逆之则伤肾）；②果不其然冬天即现腰膝酸软、畏寒——肾气已伤。

立春后高热39.2°C = 典型"冬伤于寒，春必温病"。表面热症，实际根子在冬天伤到的寒。脉浮数（表热）但重按无力（里虚寒）印证了这一点。

为什么银翘散/白虎汤反复发作？因为寒凉药只清了表面热，里面冬天积的寒根本没动——"里面寒没有去掉"，所以退烧两天又复发。

**治则**：不能再用寒凉药。应温里散寒 + 扶阳解表。温中去寒为主，表热是标，里寒是本。

✅ 通过

## 题目2：胃气已绝的脉象危候

**辨脉**：寸口脉超过腕横纹向手掌方向冲约一寸 = **溢脉**。

**病机**：阴太盛，顶阳往外 → 寸超过正常九分往上走 → 溢脉。宽大搏指但按之空虚无根 = 真虚假实。关尺反弱 = 下元空虚。

溢脉 = "胃气已无的脉" = 反顺为逆 = 内格 = 阴阳相隔的危候。

十年胃病 + 纳食极少 + 面色萎黄无华 = 脾胃之气将绝。

**治则**：急当大补脾胃阳气，恢复胃气为先。此为危脉，预后不良。

✅ 通过

## 题目3：从疣到肿块的阳气之辨

**辨证病机**：

第一阶段（疣）：高强度训练大汗淋漓 → 立即冲冷水澡 = 汗出见湿 + 汗出遇寒。按内经——汗出见湿，乃生痤疿（疣）；汗出遇寒 → 皮疱。他汗后冲冷，湿寒双中，颈后发疣完全符合。

第二阶段（肿块）：长期反复 → 阳气阻隔。内经明言——阳不行则浊物不排、营养不藏 → 累积 → 阴积。皮下硬块（核桃大、推之不移）= 阴积的实体化，"阳气阻隔致肿瘤"的早期表现。

体力下降 + 恢复慢 + 大便黏腻 = 阳气不能宣通 + 湿困。

**治则**：核心是**行阳**——让阳气动起来。内经原话："阳气当隔，隔则当泄"。行阳则阴不积，阴不积则肿块可消。不可只治表面疣和肿块——要恢复阳气运行，祛湿散结。

✅ 通过

## 题目4：心痹案——咸伤肾，寒极生热

**辨证病机**：

五味过食所伤线：素嗜咸鱼腌菜 → 味过于咸 → "大骨伤，气劳短肌" → 肾水受伤 → 水克火 → 心气抑 → 胸闷心悸（冠心病）。面色黧黑 = 肾伤 → 黑色营养外渗（肾色黑，肾伤则黑不藏而外现）。足胫浮肿 + 腰膝冷痛 = 肾虚水泛。

寒极生热线：热水泡脚反更冷、冰水冲脚后发热舒适——这恰好验证了"寒极生热"原理。内经原文：寒极生热，"冲冲冲不冷了，然后脚开始热起来，心脏阳就会回头"。更深层是"母能令子虚"——寒水克火，但木（肝/母）来补火（心/子），心脏阳反而加强。患者是自行发现了这个机制。

脉沉细而结代 = 心肾阳虚。

**治则**：①断咸味；②补肾阳（腰膝冷痛+脉沉细是肾阳不足）；③可善用"寒极生热"原理——适当用寒引导心阳回头（如冷水冲脚法），不可一味热药猛攻。利水消肿兼顾。

✅ 通过

## 题目5：八风中的冬伤于北风

**辨证病机**：

八风致病线：北风生于冬，病在肾，俞穴在腰股、四肢关节——完全吻合！隆冬北风直吹腰背两月 → 伤肾。

四时受风病位线：冬伤于风 → 病在四肢关节。

连锁反应线：冬伤于寒/风 → 当时不发 → 春发。逆冬气则少阴不藏，肾气独沉 → 春为痿厥（肌肉萎缩+精神不振）。患者腰部酸重冷痛 + 双腿肌肉萎缩 + 精神萎靡终日欲寐 = 教科书级"痿厥"表现。

**治则**：补肾阳、强筋骨、温经散寒。肾主骨，腰为肾之府。治肾为主，兼顾脾胃以助肌肉恢复。

✅ 通过
