# 金匮要略 第25段：黄疸病——证治并方

## 一、黄疸分类（四种疸）

### 1. 女劳疸（死症/危证）
**症状**：额上黑（胃气绝）+ 微汗出（阴液流失）+ 手足中热（心肾虚热外走）+ 薄暮即发（阳不入阴）+ 膀胱急小便自利（小便失控）+ 腹如水状
**病机**：阴阳分离——阴液将尽，里阳外走。额黑=胃气无，微汗=阴不断流失，手脚心热=心包/心经+肾经涌泉虚热，黄昏发病=阳要入阴而不能入
**预后**：原文说"不治"，但要给病人希望，做最后一搏

倪师纠正：过去叫"女劳疸"认为女色过多导致——这是中医的缺点，女人也会得此病

### 2. 酒疸
**症状**：心中懊憹而热（胃中难过燥热）+ 不能食 + 时欲吐
**病机**：酒性本寒，到胃里是冷的→胃加速蠕动消化→产生热感。酒精刺激胃，胃的消化不好，常呕吐
**预后**：好治（病好治，习惯难改）

#### 酒疸的细分
- **酒黄疸（一）**：小便不利→心中热（胃热）+ 足下热。湿寒累积胃→影响肠→影响脾，脾积湿
- **酒黄疸（二）**：无热，靖言了了（安静不说话），腹满欲吐，鼻燥。中膈积湿→大肠津液不能上达肺→肺燥→鼻燥
  - 脉浮（病在上）→先吐之
  - 脉沉弦（病在下）→先下之
- **酒疸心中热欲吐**：吐之即愈（不用小题大做）

### 3. 谷疸
**症状**：阳明病脉迟（非洪大）+ 食难用饱（一饱就难过）+ 饱则发烦头眩 + 小便必难
**病机**：胃蠕动正常但脾脏蠕动慢→食物残渣堵胃→浊水（馊水）渗下入膀胱→小便浓稠难排
**预后**：预测性诊断——赶早治，否则发谷疸。虽下之腹满如故（脉迟=脾虚，攻下无效）

### 4. 黑疸（危证）
**病因**：酒疸+反复攻下→寒凉药入肝→肝脏阴寒→黑疸
**症状**：目青（眼周青）+ 面黑 + 心中如噉蒜荠 + 大便正黑（瘀血）+ 皮肤爪之不仁（粗糙脱皮）+ 脉浮弱 + 虽黑微黄
**病机**：进入肝胆内脏——肝硬化/肝癌之类。大便黑=瘀血（区别于柏油样黑便=内脏腐烂/胃癌末期）
**预后**：难治，硝石矾石散为最后一击

## 二、黄疸总论

### 病因
- **核心病因**：从湿得之（黄家所得，从湿得之）
- 两热相得（本身燥热+火劫其汗）→津液伤→肠胃堵→湿热堵→发黄
- 脉沉+渴欲饮水+小便不利→里寒+热+湿→日久发黄
- 腹满+舌痿黄+躁不得睡→湿很盛，新水不能代旧水

### 病程规律
- **十八日为期**：治之十日以上瘥（阳黄一个礼拜十天就好）
- 十日不愈→难治
- 阳黄（鲜黄）→好治；阴黄（暗黄）→难治，需很长时间
- 从阴黄治成阳黄=厉害（病从里走表）

### 治疗大原则
- **诸病黄家，但利其小便**（去湿）
- 假令脉浮→当以汗解之（桂枝加黄芪汤）
- 疸而渴者难治（利湿会加重口渴）；疸而不渴者可治

### 诊断要点
- 大便颜色：黑=瘀血；柏油样成块黑=内脏腐烂（胃癌末期）
- 舌诊：舌萎黄=湿盛
- 额黑=胃气绝
- 黄昏发热→阳不入阴（非白虎汤证时是危证）

## 三、四个处方

### 1. 茵陈蒿汤（阳黄便秘第一方）
**组成**：茵陈蒿六两、栀子十四枚、大黄二两
**煎法**：水一斗，先煮茵陈减六升，纳二味煮取三升，分温三服
**药后反应**：小便当利，尿如皂角汁状（深咖啡色），一宿腹减，黄从小便去
**方解**：
- 茵陈蒿：去黄+去湿（不管阴黄阳黄）
- 栀子：去肝胆湿热
- 大黄：去实
**剂量比例**：茵陈最大 >> 栀子 > 大黄最小
**适用**：阳黄+便秘+舌黄。胆堵塞/胆囊炎/急性肝炎

倪师剂量参考：
- 体格高大：茵陈一两、栀子六钱、大黄两钱
- 中等身材（150-140磅）：茵陈六钱、栀子四钱/三钱、大黄两钱/一钱半
- 剂量够→一剂就中；剂量不够→方子对了也慢慢好

### 2. 硝石矾石散（黑疸/肝癌最后一击）
**组成**：硝石（熬黄）、矾石（烧）等分
**服法**：为散，大麦粥汁和服方寸匕，日三服
**药后反应**：病随大小便去，小便正黄，大便正黑
**方解**：
- 硝石（火硝石=炸药原料，熬黄减烈性）：咸苦→攻坚→攻瘀血/肿瘤/硬块
- 矾石（烧后脱水变细粉）：收敛去湿
- 大麦粥：固胃（防硝石伤胃）
**适用条件**：确定西医没碰过病人（没开过刀/没做过介入）
**禁忌**：熬硝石时要隔火（铁锅隔火烧），不能直接火烤→会爆炸

### 3. 栀子大黄汤（酒疸·不黄）
**组成**：栀子十四枚、大黄三两、枳实五枚、豉一升
**煎法**：水六升煮取二升，分温三服
**方解**：
- 栀子：去肝胆热
- 豆豉：去胃热
- 大黄：去实
- 枳实：去胃中湿/浊水
**关键鉴别**：栀子大黄汤 vs 茵陈蒿汤——都有便秘，差异在**一个黄一个不黄**。无茵陈=不发黄
**适用**：酒疸，心中懊憹或热痛，舌黄脉数热

### 4. 桂枝加黄芪汤（脉浮·汗解）
**适用**：黄家脉浮→当以汗解之（不用利小便法）
（方见水气篇）

## 四、倪师临床心法

1. 胆结石取胆→最快15年最慢20年→肝结石（更难清），因胆汁仍在累积
2. 吃内脏（肝类）要加醋→醋清胆道，不喜欢酸的人易胆道堵塞
3. 茵陈蒿汤剂量到位一剂就中——好医生下重手是为病人着想
4. 阳黄治成阴黄=失败；阴黄治成阳黄=病从里走表=厉害
5. 喝酒的人便秘→不攻下（不用大黄），用柏子仁/麻子仁/当归/酸枣仁等仁剂通便
6. 酒疸下之→久久为黑疸（寒凉药入肝→内脏阴寒）
7. "小洞不补，大洞一尺五"——身体小微恙时就要动手
8. 大便颜色诊断：黄=正常；黑=瘀血；柏油样块状=内脏腐烂将死
9. 硝石矾石散是最后一线药，诊所有但不轻易让人知道
