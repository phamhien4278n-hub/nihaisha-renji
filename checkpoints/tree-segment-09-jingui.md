### 一、历节病三大病因（续上段）

#### 1. 趺阳脉型——胃气实
- **病机**：胃气实（冲阳脉旺）→ 谷气实 → 浮汗自出 → 汗出当风/入水 → 汗不透发 → 历节
- **人群**：胃气很旺的人，一吃东西就流汗
- **预防**：不要吃太饱（七八分饱即可），不要满头大汗还在吃

#### 2. 太阴脉型——妇人血不足
- **病机**：太阴脉浮弱（关上脉）→ 风血相博 → 疼痛如掣
- **人群**：妇人，在家做针线活，活动量少 → 血不足
- **特点**：非常掣痛

#### 3. 盛人型——养尊处优
- **病机**：运动少 + 常喝酒 + 汗出当风 → 历节疼痛、不可屈伸
- **人群**：有钱人，仆人多，运动少

### 二、桂枝芍药知母汤 ⭐核心方剂

#### 原文主治
> 诸肢节疼痛，身体尪羸，脚肿如脱，头眩短气，温温欲吐者，桂枝芍药知母汤主之。

#### 方药组成
| 药物 | 原方量 | 倪师常用量 |
|------|--------|-----------|
| 桂枝 | 四两 | 五钱（最重） |
| 芍药 | 三两 | 四钱 |
| 甘草 | 二两 | 三钱 |
| 麻黄 | 二两 | 三钱 |
| 生姜 | 五两 | 两片/三钱 |
| 白术 | 五两 | 五钱 |
| 知母 | 四两 | 五钱 |
| 防风 | 四两 | 五钱 |
| 炮附子 | 二枚 | 三钱 |

**剂量规律**：桂枝最重 > 白芍；甘草=麻黄=炮附子等量；白术=知母=防风等量；生姜收尾

#### 倪师核心经验
1. **膝盖风湿痛特效药**——一剂知！
2. 治疗手指关节变形（类风湿）效果不好，但膝盖风湿百分百有效
3. 此方包含甘草附子汤（桂枝去风+白术去湿+附子去寒）
4. **风湿+痛风同时有**：单用此方即可，不用另开甘草附子汤

#### 临床案例
送水工人：两膝盖+脚踝+环跳痛，右膝盖红肿，解溪肿。针（膝五针）+ 桂枝芍药知母汤 → 当晚服药第二天就能抬腿走路 → 两个礼拜后直接来送水了！

#### 桂枝 vs 炮附子（关键理论区分）
| | 炮附子 | 桂枝 |
|---|---|---|
| 作用 | 壮心阳 | 壮心阳 |
| 机制 | 增加心臓摩擦产热 → 强火 | 味厚辛香发散 → 导心火下行 |
| 方向 | 增强热量 | 往下导（入血分） |
| 适用 | 恶寒/怕冷 | 导心火到小肠 |

**桂枝的特殊作用**：将心火导到小肠 → 奶水变成月经！
- 四物汤+炒麦芽（来自《奇方异录》）→ 力量不够时加桂枝
- 桂枝入血分，将血往下导

#### 便秘注意事项
- 风湿+便秘：不用攻下，直接开桂枝芍药知母汤 → 大便恢复正常 = 风湿已去
- 若大便仍不正常 → 第二次再开通利药
- 现代人常因止痛药便秘 → 可在方中加大黄/枳实/厚朴

#### 膝盖软骨问题加减
- 扎膝五针+桂枝芍药知母汤
- 软骨磨损 → 加牡蛎、补骨脂
- 没开过刀的 → 针灸可治好
- 开过刀的 → 更坏（清理碎骨=更少软骨）

#### 去痛关键药味
- 防风 → 去痛
- 炮附子 → 去寒+去痛（寒痛用热药）
- 延伸思考：癌症骨转移的剧痛 → 白术+附子（排脓汤加重）→ 把病毒往皮肤推 → 去湿去寒

### 三、味酸伤筋，咸伤骨 → 历节源头

#### 原文
> 味酸则伤筋，筋伤则缓，名曰泄；咸则伤骨，骨伤则痿，名曰枯。枯泄相搏，名曰断泄。营气不通，卫不独行，营卫俱微，三焦无所御，四属断绝，身体羸瘦，独足肿大，黄汗出，胫冷。假令发热，便为历节也。

#### 酸伤筋
- 吃太多酸 → 筋伤 → 抽筋（名曰泄）
- **"假甜真酸"**：人工白糖 → 嘴巴甜 → 胃里变酸（细菌代谢产酸）→ 晚上抽筋
- 肝苦急 → 急食甘以缓之 → **真蔗糖（黄糖/Fructose Sugar）**才有效
- 晚上喝浓蔗糖水 → 治抽筋

#### 咸伤骨
- 吃太多咸 → 骨伤 → 痿（肌肉萎缩，名曰枯）
- 治疗骨病时要告诉病人少吃咸
- 反过来：看到痿（肌肉萎缩）→ 知骨伤 → 治骨

#### 鹤膝风
- 枯泄相搏 → 膝盖肿大 + 腿变细（像鹤腿）
- 此即历节的严重阶段

### 四、乌头汤 ⭐慈悲之药

#### 原文主治
> 病历节不可屈伸，疼痛，乌头汤主之。
> 乌头汤方：治脚气疼痛，不可屈伸。

#### 方药组成
麻黄、芍药、黄芪各三两，甘草（炙），川乌五枚（口父咀，以蜜二升，煎取一升，即出乌豆）

#### 张仲景煮法的问题
原文：乌头与蜜同煮 → 煮好后乌头拿不出来！粘在一起，一片一片挑？

#### 倪师改良版——厨师煮法
1. 乌头一两 + 两碗水 → 煮成一碗（加热减毒）
2. 过滤掉乌头渣 → 只剩乌头水
3. 乌头水 + 蜂蜜一碗 → 再煮成一碗 → **乌头蜜成品**

#### 用法
- 一茶匙乌头蜜混入汤剂，早晚各一碗
- **不要"不知尽服之"**——风险太大，容易伤心脏致死
- 慢慢来，病会好，变形关节都能恢复正常

#### 倪师为何不在美国用原方
- 美国人心臓被西药搞坏（抗生素/疫苗/止痛药/安眠药/抗忧郁药）→ 十个有九个地图舌
- 麻黄剂量下去心跳加速 → 风险太大
- **替代方案**：桂枝汤 + 乌头蜜（乌头桂枝汤）← 效果也很好

#### 原方本应最好用
脚没有力的时候最好用。如果是心脏好的人 → 麻黄芍药黄芪乌头汤比乌头桂枝汤更好

#### 红斑性狼疮
标准症状 = 全身关节痛+恶寒 → 桂枝芍药知母汤加减即可，不被病名局限

#### 治疗类风湿效果
巫婆手 → 恢复正常手！连形状都变回来，不仅是痛消失。

### 五、矾石汤（外用）

#### 原文
> 矾石汤，治脚气冲心。
> 矾石二两，以浆水一斗五升，煎三五沸，浸脚良。

#### 要点
- 矾石 → 除湿力量很强
- 外用药，泡脚
- **脚气冲心** = 脚上红肿发炎从下往上蔓延（不限于脚气病因）→ 任何脚部红肿往上走都用此方

### 六、血痹（第六篇开篇）

#### 原文
> 问曰：血痹病从何得之？师曰：夫尊荣人，骨弱肌肤盛，重因疲劳汗出，不时动摇，加被微风，遂得之。

#### 病因
- 尊荣人（有钱人）：骨弱（不劳动）+ 肌肤盛（吃得好、胖）
- 疲劳汗出（房事多）+ 不时动摇 + 微风 → 血痹
- **运动对身体好**——不动养尊处优反易得病

#### 轻症：针灸
脉微涩，寸口关上小紧 → 针引阳气，令脉和，紧去则愈

#### 重症：黄芪桂枝五物汤
> 血痹，阴阳俱微，寸口关上微，尺中小紧，外证身体不仁，如风痹状，黄芪桂枝五物汤主之。

**方药**：黄芪三两、芍药三两、桂枝三两、生姜六两、大枣十二枚

**组方逻辑**：桂枝汤去甘草 + 黄芪，重用生姜
- 去甘草、加黄芪 → 黄芪走表，皮肤麻木不仁时用
- 重用生姜 → 关脉微 = 胃寒湿 = 水多 → 生姜散寒

### 七、虚劳（劳证）

#### 劳证的定义与病机

##### ① 脉大为劳
**两种解释**：
- 寸关尺脉大
- 望诊血管粗大（更直观）

**病机**：湿在血脉中 → 撑大血管 → 血量不变但血管变大 → 等于"一个人带两个血液循环系统"
- 劳证的人怎么吃都吃不胖（唯一好处）
- **治疗**：桂枝芍药知母汤 → 把血脉中的湿排出来 → 血管缩小
- 临床：老人手脚蜘蛛网状血管 → 桂枝芍药知母汤 → 收起来变漂亮

##### ② 面色薄型
面色苍白 + 渴 + 亡血（突然短气、动悸）+ 脉浮 → 里虚劳证

##### ③ 脉虚沉弦型
无寒热 + 短气里急 + 小便不利 + 面色白 + 时目瞑 + 鼻衄 + 少腹满 → 劳证

**小便不利的病机**：湿盛 → 脾主湿主少腹 → 湿积少腹 → 膀胱负担 → 小便不利 → 治湿=利小便

##### ④ 温病派制造出的劳证
温病派见吐血/泄精就用滋阴寒凉药 → 把人治成劳证！（六大弊端之一）
精血同源、汗血同源 → 吐血=泄精=盗汗不止

#### 劳证的其他特征
- **手足烦**：晚上睡觉手不自觉动来动去（不是调皮，是病）
- **春夏剧，秋冬差**：虚劳人在热天更不舒服，冷天舒服一点
- **痠削不能行**：腿没肌肉，虚弱
- **无子/精气清冷**：脉浮弱而涩 → 男人无子，女人不孕易流产
