# 伤寒论 段2 知识树：太阳病上篇——葛根汤至小青龙汤

## 二、核心概念

### 2.1 三方鼎立：太阳病三证
```
中风 ──→ 桂枝汤（有汗）
伤寒 ──→ 麻黄汤（无汗，身疼，喘）
温病 ──→ 葛根汤（项背强几几，无汗，恶风）
```

### 2.2 表里寒热辨证框架
```
表寒里热 → 大青龙汤（烦躁，口渴，黄痰，舌苔黄）
表寒里寒 → 小青龙汤（不渴，白痰，舌苔白）
```
- 大青龙 = 麻黄汤 + 石膏 + 生姜 + 大枣
- 小青龙 = 麻芍姜辛桂五草半（八味等量除五味/半夏加倍）

### 2.3 大小青龙的季节辨证规律
- **夏天发作的喘** → 大青龙汤证（里热，同气相求）
- **冬天发作的喘** → 小青龙汤证（里寒，同气相求）

## 三、关键方剂解析

### 3.1 葛根汤——太阳温病第一方

**病机**：风寒束表，但邪在膀胱经（项背到脑），需要葛根专门提升津液到后项。
**重用葛根**的原因：葛根把肠胃津液往上提，到项背膀胱经解表。

**临床扩展**：
1. **鼻窦炎**：
   - 清涕（里寒）→ 葛根汤 + 白术（去湿）+ 附子（去寒）
   - 黄涕粘稠（湿热）→ 葛根汤 + 黄芩等去热药
2. **水痘**：葛根汤一剂，全身透发 → 烧退胃口开 → 水痘自收。水痘是胎毒未清。
3. **面部中风**：葛根加倍（4→6），桂枝汤加葛根。
4. **太阳阳明合病下利**：葛根汤提肠胃津液上升，下利自止。

**使用禁忌**：
- 平素湿热之人（舌苔黄、胖）→ 葛根汤会把湿热提到脸上（面红如苹果）
- 解法：加茯苓（湿热从小便利出）+ 黄芩/黄连（去热）

### 3.2 术附汤——排脓汤

- 白术 + 炮附子等量 → 去寒去湿，逼脓外出
- 适用：深处化脓（骨边、糖尿病脚破洞不愈），皮肤表面仅局部发热
- 关键认知：**化脓表面是热的，里面是寒的**——用寒药会恶化
- 另一个排脓汤：赤豆当归散（排肛门痔疮脓），不可混用

### 3.3 甘草附子汤——痛风方

- 桂枝（去风）+ 白术（去湿）+ 炮附子（去寒）+ 炙甘草
- 病机：标热本寒——骨髓里是寒的，表面是热的
- 风性跑来跑去 → 解释痛风痛处游走

### 3.4 麻黄汤——返魂汤

**麻黄的核心作用**：**发阳，不是发汗**。单味麻黄不会出汗（只有精神好不睡觉）。

**四味配合逻辑**：
- **麻黄**：开肺、开毛孔
- **桂枝**：麻黄的马，带麻黄到皮肤表面（无桂枝麻黄不出汗）
- **杏仁**：补肺津液（像汽车水箱的水）——无杏仁则肺津液干→燥渴→过热
- **炙甘草**：肠胃生津液

**杏仁量必须≥麻黄量**，否则肺的津液不够。

**返魂汤**：麻黄汤能发阳气——病人气息都没了，麻黄一下去肺打开→当场呼吸恢复。比用200-300克炮附子（四逆汤）更有效。

**服用**：覆取微似汗（感觉有汗，摸无汗）→ 最高境界。不需啜粥（与桂枝汤不同——桂枝要肠胃汗，麻黄要肺汗）。

### 3.5 大青龙汤——表寒里热 / 时疫方

**四大主症**：表寒（恶寒、肌肉痛、脉浮紧、无汗）+ 烦躁

**烦躁 = 危险信号**：病人越烦躁越危险。"静"的病好治，"烦躁"的不好治。

**石膏**：除烦除躁，寒凉药让烦躁平静。烧高时石膏可重用至一两。

**温粉扑**（防过汗）：龙骨 + 牡蛎粉 + 糯米粉

**时疫判断**：节气不对（春行冬令等）+ 老幼同病 = 疫。大青龙 = 张仲景为疫病而设。

**方义**：汗出→津液换掉（被感染的肺津液随汗排出，石膏+杏仁补新津液）= 瞬间焕新。

**第44条**：脉浮缓，身不疼但重（无少阴证）→ 大青龙也可治。肌肉湿盛用大青龙发表排湿。

### 3.6 小青龙汤——表寒里寒

**方歌**：麻芍姜辛桂五草半（麻黄、白芍、干姜、细辛、桂枝、五味子、甘草、半夏）

**主症**：心下有水气 → 干呕 + 咳 + 发热 + 不渴 + 白痰

**快速辨证**：喉咙痒 = 水气往上冒 = 小青龙汤证！（哪怕没有表寒发热）

**药物分工**：
- **细辛**：去水气（不是大水，是水气——喉咙痒的根源）
- **干姜**：温肺寒
- **半夏**：化痰
- **五味子**：止咳
- **麻黄 + 桂枝**：发表

**加减（副症）**：
- 渴 → 去半夏（半夏排水更渴），加栝蒌根
- 微利 → 去麻黄，加芫花（肺积水，量极少，如鸡子大≈1-2克）
- 噎/嗝逆 → 去麻黄，加炮附子（胃冷）
- 小便不利/少腹满 → 去麻黄，加茯苓
- 喘 → **不应当去麻黄！这是后人妄改——喘必须加麻黄+杏仁**

**愈兆**：服后渴者 → 寒去欲解（里寒去掉了，正常口渴回来了）

## 四、核心治则

### 4.1 表里先后原则
> **有表证时先解表，不可攻下** —— 伤寒误下会成结胸。

- 太阳阳明合病喘而胸满 → 有麻黄汤证就不可下，宜麻黄汤
- 误食坏食物被动"攻下"+ 本身太阳证 → 葛芩连汤（热利）

### 4.2 经方核心哲学
- **补阳不补阴**：阳回则阴自回（让身体功能恢复，不是填补物质）
- **药简力专**：麻黄汤只四味，经方精神
- **一症一方**：不要弄错，证对了一剂即中
- **同证同治**：不管病毒叫什么（SARS/B型/甲流），证同则方同

### 4.3 汗的重要性
- 运动出汗 = 心脏功能正常
- 完全无汗 / 不动也出汗 / 盗汗 = 心脏有问题
- 有汗 → 不会有水肿/积水
- 麻黄汤汗解后覆取微似汗 = "衣服破了皮肤没破"的境界

| 痰色 | 黄痰 | 白痰 |
| 舌苔 | 黄 | 白 |

## 六、关键药物要点

| 药物 | 核心功能 | 使用注意 |
|------|----------|----------|
| **葛根** | 提肠胃津液到项背 | 面部中风加倍；湿热人慎用 |
| **麻黄** | 发阳（非发汗）| 单味不汗；无桂枝不出汗到表 |
| **桂枝** | 麻黄的"马"，带到皮肤 | 麻黄汤中麻>桂 |
| **杏仁** | 补肺津液，涤痰 | 量≥麻黄；治肺结核（肺泡结杏仁状） |
| **生半夏** | 涤水/止呕/排脑积水 | 毒=失声，生姜解；口渴去半夏 |
| **细辛** | 去水气 | 蓄水不小，水气用细辛 |
| **石膏** | 除烦除躁，去热 | 大青龙烧高可重用至一两 |
| **五味子** | 止咳（酸苦甘辛咸五味）| 配半夏=化痰止咳 |
| **干姜** | 温肺寒 | vs生姜=散胃寒 |
| **黄芩** | 去肝胆/小肠湿热 | 葛芩连汤中=黄连 |
| **黄连** | 清胃中热 | 味厚色黄属土 |

## 七、本段倪师原话精要

1. 「麻黄不是发汗，是发阳」
2. 「最高境界：衣服砍破了，贴到皮肤上皮肤没有破」（微似汗）
3. 「经方不是开补药——阳回阴自回」
4. 「世界上最便宜的药就是经方」
5. 「有汗→不会有水肿积水」
6. 「烦躁=危险信号，越烦躁越不好治」
7. 「夏天喘=大青龙，冬天喘=小青龙」
8. 「喉咙痒=水气往上冒=小青龙」
9. 「化脓是表面热里面寒——用寒药更坏」
10. 「杏仁是肺的水箱，麻黄是引擎」
