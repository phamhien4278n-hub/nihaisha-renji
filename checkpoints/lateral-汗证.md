## 一、总论

汗为心之液（内经），营卫之气所化。倪师口决：「辛甘发散为阳，酸苦涌泄为阴。」发汗太过伤胃与小肠津液，可致五苓散证。

## 三、倪师核心口决

| 口决 | 含义 |
|------|------|
| 「有汗不用麻黄」 | 汗出说明腠理已开，忌大发汗；有汗不咳用桂枝汤，有汗而咳用越婢汤 |
| 「麻黄在越婢汤中为止汗非发汗」 | 越婢汤证（汗出+咳+发热），麻黄配伍石膏后发散力大减，反起止汗作用——石青敛阳使阳回收而不外散 |
| 「桂枝汤发汗不伤正」 | 桂枝汤为阴阳调和之第一方，桂芍等量，生姜甘草大枣补肠胃津液，汗自出而不伤津 |
| 「汗出当风则病风湿」 | 金匮原文——汗出时毛孔开，入冷水中或当风吹，水湿从毛孔渗入形成黄汗/历节 |
| 「尺脉禁汗」 | 尺脉微→里虚→靠稀饭自愈；尺中迟→营气不足血少→先补津液（水梨）再汗（C1：伤寒第49/50条） |
| 「黄汗不可发汗」 | 黄汗本质是湿蓄三焦，发汗会更虚，治用黄芪收敛法 |

## 四、跨课连接

| 课程 | 贡献 |
|------|------|
| **伤寒论** | 六经汗证完整框架：太阳（有汗/无汗）→ 阳明（大汗）→ 少阳（无汗）→ 少阴（亡阳汗）→ 厥阴（厥逆汗） |
| **金匮要略** | 黄汗专篇（黄芪芍药桂枝苦酒汤 vs 桂枝加黄芪汤）；风湿汗出；产后汗出；湿病汗出不彻 |
| **黄帝内经** | 汗为心液（五脏化五液：心→汗、肝→泪、脾→涎、肺→涕、肾→唾）；营卫之气化汗；阳加于阴谓之汗 |
| **针灸大成** | 自汗取阴郄、后溪；盗汗取阴郄；无汗取合谷、复溜；大汗灸关元神阙；面部汗取合谷 |
| **神农本草经** | 麻黄（发汗峻药）、桂枝（温经发汗）、附子（回阳止汗）、黄芪（固表止汗）、白芍（收敛止汗）、石膏（敛阳清热）、牡蛎（敛汗安神）、浮小麦（养心止汗） |

## 五、汗证——寒热鉴别速查

| 汗型 | 寒/热 | 核心方 | 关键鉴别点 |
|------|--------|--------|-----------|
| 汗出恶风 | 寒（表虚） | 桂枝汤 | 脉浮缓、发热不高 |
| 无汗恶寒 | 寒（表实） | 麻黄汤 | 脉浮紧、身痛 |
| 大汗热渴 | 热（阳明） | 白虎汤 | 四大症（大热大渴大汗脉洪大） |
| 汗出肢冷 | 寒（亡阳） | 四逆汤 | 脉微细、四肢厥逆 |
| 盗汗 | 阴虚热 | 当归六黄汤 | 五心烦热、舌红少苔 |
| 黄汗 | 湿郁 | 黄芪芍桂苦酒汤 | 汗沾衣色正黄如柏汁 |
| 头汗 | 热郁/湿热 | 栀子豉汤/茵陈蒿汤 | 心中懊憹/身黄小便不利 |

## 六、学习薄弱点（小凛自审）

- 半边汗（汗出偏沮）的中风先兆预警时间窗：倪师未给出具体时间，需查后世医家
- 当归六黄汤出自《兰室秘藏》（李东垣）非金匮原文，C2标注需更精确——倪师引用此方但不属人纪原文
- 越婢汤中石膏止汗的机制「石膏=敛阳而非寒凉」是倪师独到理解，与传统认知有差异
