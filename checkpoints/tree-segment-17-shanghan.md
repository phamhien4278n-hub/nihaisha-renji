# 伤寒论 段17 知识树——厥阴篇结尾（三七二 ~ 三九七）

## 核心概念

### 厥热胜负——判断厥阴病进退的核心指标
- **热多于寒 → 愈**（三五七）：发热四日厥三日复热四日，厥少热多，当愈
- **寒多于热 → 进**（三五八）：厥四日热三日复厥五日，阳气退，病进
- **寒热同等 → 最佳**：处方不变持续吃，吃到晚上肚子饿就好了
- **热不除过久 → 便脓血**（三五七）：四至七日热不除者
- 推论：用热药（生附、硫磺）治病完全正确——厥阴最深时，冷>热是死，热>冷是生

### 厥阴死证汇总
- 脉微手足厥冷烦躁，灸厥阴厥不还 → 死（三五九）
- 下利手足厥冷无脉，灸之不温脉不还反微喘 → 死（三七七）
- 下利脉反实者 → 死（三八五）——脉证不合，阳绝
- 下利后脉绝，睟时脉不还 → 死（三八四）
- 呕脉弱小便利身微热见厥 → 难治（三九三）

## 关键方剂（本段新方）

### 1. 麻黄升麻汤（三七二）——肝病吐血
- **组成**（14味）：麻黄二两半、升麻一两一分、当归一两一分、知母/黄芩/葳蕤各十八铢、石膏碎绵裹、白术/干姜/芍药/桂枝/茯苓/炙甘草/天冬各六铢
- **煎法**：水一斗，先煮麻黄去沫，入诸药煮三升，分三服
- **主治**：寸脉沉迟、手足厥逆、下部脉不至、咽喉不利、吐脓血、泄利不止
- **特点**：寒热并用——麻黄(热)+石膏知母(寒)+干姜(热)+黄芩(寒)
- **争议**：经方家怀疑非仲景原方（单位混用一两一分+铢、药味多达14味）
- **升麻**：脱肛、疝气、肠子下堕、中气不足下陷 → 提升阳气
- **葳蕤=玉竹**：治咽喉不利、喉咙发炎，有止血功能
- **胃出血不能用此方 → 用黄土汤**

### 2. 干姜黄连黄芩人参汤（三七四）——胰脏癌主方 ⭐最重要
- **组成**：干姜、黄连、黄芩、人参各三两（实际用三钱）
- **煎法**：水六升煮取二升，分温再服
- **主治**：食入口即吐，寒格在胃（胰脏）
- **病机**：本自寒下 → 医复吐下 → 寒格 → 更逆吐下
- **方义**：干姜温中，芩连去热，人参补虚（吐下伤津液）
- **临床应用（倪师）**：
  - **胰脏癌主力方**！99%胰脏癌在此范围
  - 另一处方：旋覆花代赭石汤+干姜+吴茱萸（呕酸重者）
  - 吃后约一周大下黑便 → 痛去 → 胃口恢复
  - 基础方不动，可加：桂枝汤(怕风)、葛根(项强)、麻黄连翘赤小豆汤(皮肤痒)
  - 往来寒热恶心 → 小柴胡汤
- **倪师案例**：牙医朋友的先生胰脏癌，喝咖啡→停咖啡→好转→感冒吃抗生素→肝脓疡→死（厥阴误治入里化脓）
- **核心认知**：所有癌症都是寒，寒实。寒格喝热水舒服、喝冷水不行 → 都可用此方

### 3. 白头翁汤（三八七、三八九）——热利下重
- **组成**：白头翁二两、黄连/黄柏/秦皮各三两
- **煎法**：水七升煮取二升，温服一升
- **主治**：热利下重，里急后重，肛门痛，一天十几次，欲饮水
- **方义**：黄连入胃，黄柏入肠，秦皮止利润肠
- **与其他热利方鉴别**：
  - 葛芩连汤 → 热利+太阳表证
  - 黄芩汤 → 热利+腹痛
  - 黄连汤 → 寒热并结
  - 白头翁汤 → 纯热利，下重
- **血便混合 → 桃花汤+白头翁汤复方**

### 4. 通脉四逆汤（三八六）——里寒外热下利
- **主治**：下利清谷，里寒外热，汗出而厥
- 表里不通，通脉四逆汤通之

### 5. 补中治湿汤（三九七）——肝癌末期腹水
- **组成**：人参/白术各四钱，苍术/茯苓/陈皮/麦冬/当归/木通各三钱，黄芩/厚朴各二钱，升麻一钱
- **主治**：哕而腹满，肝癌末期腹水，大小便不通
- **倪师加强版**：原方 + **生附子五钱 + 生硫磺七钱**（棉布包）
  - 生附子 → 强心，心脏热入小肠
  - 生硫磺 → 点命门火，通三焦，利水退肿
- **厥阴禁下铁律**：不用大黄芒硝、不用甘草！
- **木通**：通大便+通小便，心脏热移小肠
- **分消汤** → 脾脏肿大腹水

## 关键治则

### 1. 表里先后原则（三八八）
- 下利腹胀满身体疼痛 → **先温里（四逆汤），再攻表（桂枝汤）**
- 里寒盛时攻表 → 寒湿被带动，阳绝
- 为什么用桂枝汤不用麻黄汤？桂枝汤稳，即使没先温里也不会出大事

### 2. 厥阴禁下！
- 里寒湿时攻下 → 死
- 木通替代大黄通便
- 排腹水不用甘草、大黄、芒硝

### 3. 下利不可攻表（三八十）
- 下利清谷攻表汗出 → 胀满
- 胀满治法：厚朴半夏人参甘草干姜汤 + 炮附子 + 白术茯苓（去寒湿利小便）

### 4. 呕家有痈脓不可治呕（三九二）
- 脓尽自愈——不要止呕，止呕则脓排不出

### 5. 寒热同等 → 不要改方
- 处方将寒热调到平衡后维持，吃到胃气恢复（晚上饿）

## 新学到的东西

### 脉证合参与死生判断
- 下利脉微弱数 → 胃气在，不死（三八一）
- 下利脉反实 → 脉证不合，阳绝，死（三八五）
- 下利脉绝，睟时脉还手足温 → 生（三八四）
- 少阴负趺阳为顺 → 太溪脉<冲阳脉正常（三七八）
- 寸脉浮数尺中自涩 → 必圊脓血（三七九）——阳浮于上，下焦阳不足

### 戴阳（三八二）
- 下利脉沉迟面少赤 → 戴阳，下焦太虚，阳气被里寒逼出
- 常用：干姜附子汤，里虚重可用生附子

### 吴茱萸汤头痛定位（三九四）
- 干呕吐涎沫头痛 → 印堂处痛（胃的所在）
- 肝病（肝炎/肝癌/肝硬化）+胃下垂 → 肝顶到胃，胃寒，胃气上逆 → 印堂头痛

### 治病方向判断（三九五）
- 治厥阴病人 → 原来冷 → 治后发热 → 病退（厥阴→少阴→太阴→少阳→阳回暖）
- 呕而发热 → 小柴胡汤主之（回到少阳）

### 胃中寒冷成因（三九六）
- 大吐大下→极虚→复发汗→胃中寒冷→哕
- 咖啡、可乐、蛋糕、蛋挞、黑森林→长期胃寒

### 虚实判断
- 实：便秘、汗不出、小便不出、大便不出
- 虚：下利、流汗不止、小便多
- 下利后更烦心下濡 → 虚烦 → 栀子豉汤（病后失眠也可用）

### 大承气汤不一定便秘（三九O）
- 下利+谵语 → 燥矢在内 → 大承气汤（热结旁流）

## 倪师临床精华

### 胰脏癌治疗
- 干姜黄连黄芩人参汤（主力）+ 旋覆代赭石汤加减（呕酸）
- 吃后一周大下黑便 → 痛去 → 胃口恢复 → 可救
- 停咖啡、停可乐等寒性饮食 → 病势先缓
- 治好后不要做检查——吓自己导致复发
- 中医健康标准：能吃能喝能拉能睡 → 即使肿瘤还在也无害

### 肝癌末期最后一搏
- 补中治湿汤 + 生附子 + 生硫磺
- 小便出来 → 有救
- 病人不要做任何检查、切片、栓塞

### 癌症治疗三大要求（倪师）
1. 不做任何检查、切片、栓塞
2. 放平常心，不要怕
3. 停掉所有寒性饮食（咖啡、可乐等）
