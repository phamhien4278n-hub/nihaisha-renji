# 金匮要略 段35：人纪补充病例

## 案例一：淋巴癌+血癌（2006-10-25）

### 病源理论——精子的残渣逆流
- 男人的精（白色，在下部）←→ 女人的奶水（白色，在胸部）
- 精的纯营养提炼后必有残渣：女人→月经，男人→胡子（第二性征）
- 心脏受伤后，精子残渣无法顺任脉到嘴角→逆流：
  - 进入心脏 → **红斑性狼疮**（男人少得，因为精宫距心脏远，女人奶水距心脏近）
  - 进入督脉 → **血癌**（leukemia）
  - 进入淋巴系统 → **淋巴癌**（lymphoma）
- 诊断标志：胡子变少（三天刮一次→以前每天刮）

### 患者资料
- 1945年生男，2003年做过5个by-pass心脏绕道手术
- 2004年第一次查出淋巴癌+血癌，化疗控制
- 2006年10月19日第二次查出，10月25日来找倪师
- 吃类固醇→胃口好

### 六诊
1. 睡非常差
2. 胃口过好（类固醇所致）
3. 小便黄（里热）
4. 口渴，喜冷饮
5. 双足冰冷（关键！里寒）
6. 身体温热，大便每日有
- 脉洪大

### 诊断
- 里寒+湿（因脚冷+大便好）
- 上热下寒
- 第七椎压痛（血癌）→ 吃斑茸丸
- 第六椎压痛（淋巴癌）

### 处方思路（第一周）
- **小柴胡汤加减**：柴胡、黄芩、党参 → 入三焦、淋巴
- **瓦楞子**：去痰饮，尤其三焦淋巴系统痰饮
- **牡蛎**：咸能软坚，攻坚
- **防己、郁金**（各五钱）：入肝、入三焦淋巴系统
- **强心**：因为舌苔黄
- **白芍**：脚冷→静脉回流力量不够
- **黄连阿胶汤加减**：黄连、黄芩、白芍、阿胶 → 治心血不足的失眠（舌苔黄、焦、无津液）
- **生附子**：里寒（棉布包）
- **生硫磺**（四钱）：入三焦、淋巴系统，配牡蛎消瘤极快
- **巴戟天、补骨脂、熟地**：壮精补肾
- 活血化瘀药

### 疗效
- 一周后：睡眠好转，体力回复，小便棕黄→淡黄，大便一日数次，脚温很多
- 左边头痛消失，肩膀痛消失
- 第二周：睡眠非常深，大便一日两次，脚温，脉缓
- 11月9日加麦冬（口渴→肺津液不够）

### 五行治则——上工治未病
- 心脏有问题→第一个影响到肺（火克金）→ **治心必须同时治肺**
- 肝脏有问题→先固脾（木克土）
- 肾脏有问题→先固心（水克火，心固则肾水不会移转）

## 案例二：ALS渐冻人（2007-3-29）

### 患者资料
- 女性，2006年8月开始有症状，2007年2月2日确诊ALS
- 右边走路有问题，下肢无力，肌肉抽筋，脖子痛、背痛，极度倦怠

### 六诊
1. 睡不好
2. 胃口还好（折腾人——胃口好不会死那么快）
3. 口渴，喜温饮（以前喜冰饮→里寒化）
4. 双脚冰冷
5. 睡觉全身冷，一点热都没有
6. 月经非常准时（特殊！）— 当天正值经期

### 诊断：里寒重症

### 处方
- **茯苓四逆汤**：生附子三钱、干姜两钱、炙甘草三钱、茯苓五钱、白术三钱
  - 白术量<茯苓量 → 利到下焦
- **牛膝三钱**：抽筋、脚无力
- **桂枝+白芍**：阴阳气互相增强
  - 只桂枝→增强阳；只白芍→增强阴（痘疹内陷用）
  - 同用→阴阳同时起来
- **补骨脂**：壮骨髓、补肾
- **当归**：活血
- **细辛**：升阳、生发
- **泽泻**：利水

### 关键概念
- 动作慢 = 寒；动作快 = 热
- 生附子≠炮附子！四逆汤用生附（配干姜），桂附八味丸用炮附——完全不同
- 寒解时：小便多（冰块化水）

### 疗效——误打误撞的剂量发现
- 患者误将九碗煮成两碗（而非三碗），早晚各一碗
- **第一碗下去当晚一觉到天亮**（从去年八月以来第一次）
- 无口唇麻、恶心、呕吐——说明**病人需要这药时，毒药即是补药**
- 一周后：抽筋好多了，手脚温热，小便多，口渴变喜冷饮
- 生附子加至四钱

### 停药时机标准
**头面身体冷 + 手脚温热** → 可停药

## 案例三：胆切除后肝硬化腹水（极复杂，两次救回）

### 背景
- 患者40年前外伤撞树→胆破裂→胆切除
- 之后发展为肝硬化、腹水

### 核心病机：一阳不生
- 胆切除后→胆汁无法生发→一阳无法生
- 身体深处冒寒意→阳该生时生不出来
- 药物之阳维持生命之阳

### 关键药物组合
- **生附子（黑，入心）+ 生硫磺（鲜黄，入肾）**
  - 两者均为至阳至刚
  - 生附子：阴中纯阳，色黑入心
  - 生硫磺：阳中带阴，色黄入肾
  - 至阳到极→阴自升
  - 道家炼丹用硫磺：取铅补汞，坎中汞、离中铅
  - 三黄（八石之三）：雄黄、雌黄、硫磺

### 经典方对应——桂枝汤去芍药加麻黄附子细辛汤
- 患者自述：「这一圈像大盘子顶在这边不能动」
- 倪师一听→「他在讲金匮里一模一样的症状」→ 「坚大如盘」
- 开原方，刀疤变红（一阳生，阴阳回转）

### 第二次发作（2006-8-24）
- 吃了一个不新鲜三明治→肝脏负担→无法睡、恶心、小便深黄、手脚冷、无腹水
- 粉剂大黄太慢→9月1日腹水回来，双腿肿
- 吐血一次，大便黑
- 脸上气色：反光、阴阳隔离、眼神没了，只说"say help"

### 第二次处方（9月1日）
- 大柴胡汤底：柴胡、黄芩、大黄、枳实、半夏、芒硝
- 郁金、龙胆草→清肝
- 当归、熟地→补血（吐血后）
- **桂枝汤去芍药加麻黄附子细辛汤**（因坚大如盘仍在）
- 白术、茯苓→甘淡渗利
- 治肝必治大肠（胆没了，唯一出路是大肠）

### 后续调方
- 9月5日：回头用2月15日处方 + 灯芯草、苏叶 + 麝香商陆肚脐贴
- 9月7日：一晚上少4磅
- 9月9日：改**胃苓汤粉剂**（病人怕喝汤药积水）
  - 苍术、厚朴、陈皮、猪苓、泽泻、白术、茯苓、白芍、肉桂、炙甘草
  - 胃苓汤=助脾胃临时取代胆汁功能
  - 又少3.5磅
- 同时汉唐69号（强肾排水→下半身水肿要靠肾）
- **鲤鱼汤**（食疗）：赤小豆、黄耆、薏仁、芡实、莲子 + 鲤鱼
  - 鲤鱼在脏水中能活→排水力强
  - 现代理解：白蛋白含量高
  - 《神农本草经》就记载鲤鱼治水肿

### 疗效
- 9月20日：体重197→170磅（↓27磅）
- 一个月水肿退光
- 但阳仍未生→脸上黄色天罗（额头发亮=阴阳仍在隔离）
- 10大天罗：脸上气色是疾病的颜色

### 灸法
- 三皇穴、肝关、三关穴

## 案例四：心脏病心绞痛（仅开头）

### 背景
- 九种心痛之心绞痛，刀刺样痛
- 西医手术限制：肾功能不好、严重糖尿病、高血压→不能手术
- 不能手术不如不要手术→中医可治

### 关键药物——红花的两种
- **藏红花**：质轻、无味、高原5000m→入脑（脑溢血用）
- **川红花**：味厚重、臭袜子味→入心、入血分（心脏用）
- 原则：味厚入血分，质轻味淡入气分

### 患者（2001-6-8）
- 1941年生男，屋顶修缮工，体格壮如蛮牛
- 血压高、血糖高、胸口强烈刺痛
- 脚痛、手风湿关节炎、背痛、胃酸多、过胖
- **手冷**，脚痛，小便慢流，睡眠非常差，大便每日一行，手足趾麻
- **持续痛=实症**（里面有东西堵到）
- 偶尔闷痛=心气堵塞（栝蒌实、薤白、枳实即可，不需活血化瘀）

## 跨课连接
- 小柴胡汤←伤寒论
- 黄连阿胶汤←伤寒论（少阴病）
- 四逆汤/茯苓四逆汤←伤寒论
- 大柴胡汤←伤寒论
- 桂枝汤去芍药加麻黄附子细辛汤←金匮要略（水气病篇）
- 胃苓汤←平胃散+五苓散
- 斑茸丸、汉唐68号、69号、84号
- 牡蛎+瓦楞子+生硫磺→消淋巴肿瘤
- 白芍治脚冷（静脉回流）
- 牛膝治抽筋+脚无力

## 薄弱点标记
- 胆切除后的「一阳生」概念需加深理解
- 生附子+生硫磺并用的指征和比例
- 胃苓汤在腹水中的定位（何时用汤剂vs粉剂）
- 十种天罗的具体内容不清楚
