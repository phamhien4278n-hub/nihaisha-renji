# 伤寒论 · 段17：厥阴篇末（三七二→三九七）

## 核心概念

### 厥阴病治疗总纲
- **寒热并用**：厥阴证的核心治则——寒药和热药同用。「苦温并济，寒热同调」（倪诀·015）
- **热多寒少为顺**：三五七条「厥少热多，其病当愈」。用热药治病是对的——经方也在用生附、硫磺
- **病退路径**：厥阴→少阴→太阴→少阳→代表回阳。三五九条呕而发热=回少阳=小柴胡汤

---

## 病机推理链

### 1. 肝病吐血链（三七二）
伤寒大下后 → 寸脉沉迟+手足厥逆+尺脉不至 → 咽喉不利+吐脓血+泄利不止 → 麻黄升麻汤
- 里寒极盛+上部热结 → 寒热错杂
- 肝病吐血（不是胃出血！）→ 麻黄升麻汤；胃出血 → 黄土汤
- 方剂特点：14味药、寒热药并用、剂量单位不统一（疑非仲景原方但临床有效）

### 2. 胰脏癌·寒格链（三七四）★最重要
伤寒本自寒下 → 医复吐下 → 寒格（寒隔在胃=胰脏） → 食入即吐 → 干姜黄芩黄连人参汤
- **胰脏癌主力方**：干姜（温中）+ 芩连（去热）+ 人参（补虚）
- 第二方：旋覆代赭石汤 + 干姜/吴茱萸（呕酸甚时用）
- 两方覆盖99%胰脏癌
- **鉴别**：吃入即吐（无呕酸）= 干姜芩连参汤；吃得下但呕酸 = 旋覆代赭石加干姜吴萸
- 服药后一周左右大下黑便 → 痛去 → 胃口恢复

### 3. 热利下重链（三八七·三八九）
- 热利下重（里急后重+肛门痛）→ 白头翁汤
- **四方热利鉴别**：
  - 葛芩连汤 = 热利+表证（喘而汗出）
  - 黄芩汤 = 热利+腹痛
  - 黄连汤 = 寒热并结
  - 白头翁汤 = 纯热利+下重+肛门灼痛+渴欲饮冷
- 血便+热利交替 → 桃花汤+白头翁汤复方
- 倪辨·013：问肛门痛不痛——痛=热利；不痛下利清谷=寒利（四逆汤）

### 4. 寒利救逆链（三八六）
下利清谷+里寒外热+汗出而厥 → 通脉四逆汤
- 四逆 vs 通脉四逆：干姜剂量！四逆汤干姜一两半 → 通脉四逆干姜三两（加倍）
- 加葱九茎（面赤戴阳）、加芍药（腹痛）、加桔梗（咽痛）、加人参（利止脉不出）

### 5. 厥阴生死判断链（三五七→三五九·三七七·三八四等）
- **热多于寒→生**：厥少热多，其病当愈
- **寒多于热→死**：寒多热少，阳气退，故为进也
- **厥不还者死**：灸厥阴（期门/章门）后冷不回来 → 死证
- **脉不还者死**：下利后脉绝，24h脉不回+手足不温 → 死
- **脉反实者死**：下利+脉实 → 脉症不合 → 死
- **微喘者死**：下利+手足厥冷+无脉+灸不温+微喘 → 阳绝

## 关键治则

### 厥阴禁下原则
- 厥阴篇禁攻下！里寒湿时攻下=死
- 腹水/肝病末期：利小便不利大便
- 禁大黄芒硝、禁甘草（蓄水）
- 用木通代替大黄通便（当归四逆汤也用木通移心热入小肠）

### 先温里后攻表（三八八）
- 里寒+表证 → 先四逆汤温里 → 再桂枝汤攻表
- 不可反！攻表会带动寒湿 → 阳绝
- 用桂枝汤不用麻黄汤——桂枝汤稳，即使误攻表也无大碍

### 下利清谷不可攻表（三八〇）
- 攻表 → 汗出 → 里寒更盛 → 胀满
- 胀满治疗：厚朴半夏人参甘草干姜汤 + 炮附子 + 白术茯苓

### 呕家有痈脓不可治呕（三九二）
- 吐脓是排病反应 → 止呕则脓不出
- 脓尽自愈，不要加吴茱萸等止呕药

## 倪师临床案例与思路

### 倪案·011：胰脏癌（对应三七四）
- 洛杉矶病人→倪师手写「咖啡」→病人承认最爱喝咖啡→停咖啡病势缓
- 呕酸严重→用干姜芩连参汤/旋覆代赭石汤→一周后大下黑便→痛去胃口恢复
- **死因**：回台湾感冒吃抗生素→肝脓疡→荣总抗生素打死
- 教训：①癌症源头（咖啡=寒）②停病因病势缓 ③西药入厥阴化脓

### 倪诀·015：苦温并济，寒热同调
- 乌梅丸（酸苦辣治虫）、麻黄升麻汤、干姜黄芩黄连人参汤——都是寒热并用
- 厥阴治疗大法：不在同一方内只用寒或只用热

### 倪诀·012：治肝实脾 + 倪案·009：肝癌腹水
- 补气建中汤 + 生附/生硫磺 → 利小便排水
- 三七九条的补中治湿汤同理：甘淡利湿（苍白术苓陈）+ 麦冬润肺 + 当归补血 + 木通（代大黄通便）+ 黄芩厚朴宽肠降气 + 生附强心 + 生硫磺点命门火
- 排腹水三大禁：禁甘草（蓄水）、禁大黄（攻下）、禁芒硝（攻下）

## 新学到的东西

1. **胰脏癌主力方**：干姜黄芩黄连人参汤——简简单单四味药，覆盖99%胰脏癌
2. **寒格=胰脏**：倪师把「寒格在胃中间」直接解读为胰脏癌
3. **热利四方鉴别体系**：葛芩连（表）→黄芩汤（腹痛）→黄连汤（寒热）→白头翁（纯热下重）
4. **厥阴生死判**：热多寒少=生；寒多热少=死。非抽象理论——直接问病人冷热天数
5. **厥阴禁下**：肝病末期用木通代大黄，绝对禁大黄芒硝攻下——攻则死
6. **癌症源头排查**：倪师手写咖啡-病人承认-停咖啡病缓——先找病因再治病
7. **回阳标志**：厥阴四肢冷治到呕而发热→回少阳→小柴胡汤——病退路径
8. **补中治湿汤+生附硫磺是最后一搏**：不是「一定治好」，是「还有一搏的机会」
9. **脉症合判断生死**：下利+脉实=死（脉症不合）；下利+脉微弱数=生（胃气在）
10. **检查的心理杀伤**：倪师反复强调不要回去体检——照出来还有肿瘤病人吓死。能吃能喝能拉能睡=好了

## 跨课连接

- **黄土汤**（金匮）：胃出血吐血用黄土汤，非麻黄升麻汤
- **分消汤 vs 补中治湿汤**：分消汤=脾脏肿大实证腹水；补中治湿汤=肝脏虚证腹水
- **白芍甘草汤**（去杖汤）：已学过的静脉曲张用方
- **旋覆代赭石汤**：胰脏癌呕酸时与干姜芩连参汤交替
- **当归四逆汤**：也用木通移心热入小肠——与本段三七九条通便思路一致

## 薄弱点

- 麻黄升麻汤14味药的具体配伍逻辑需要再咀嚼
- 厥阴下利各类预后的条辨细节（三七五→三八五）容易混淆
- 补中治湿汤中各药剂量配比不够熟悉
