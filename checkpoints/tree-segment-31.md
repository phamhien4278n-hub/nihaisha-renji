# 金匮要略 段31：妇人妊娠病篇（下）+ 产后病篇（绪论）

## 一、妊娠诊断法
- 前三月经水利时 → 怀孕（胎也）
- 前三月月经正常 → 停经 → 怀孕
- 停经后第三个月开始流血（鲜血）→ 可能流产
- 前三月月经不正常（肌瘤/肿瘤）+ 三个月后下黑血块 → 未孕（坏血）

## 二、桂枝茯苓丸 —— 子宫肌瘤
### 组成
桂枝、茯苓、牡丹、桃仁、芍药（各等分，丸剂）

### 病机
子宫肌瘤 = **寒 + 水 + 淤血**
- 寒：月经未排完/流产后残留，寒则凝
- 水：水湿停滞

### 方解
| 药 | 作用 |
|----|------|
| 桂枝 | 导心火至下焦，**祛寒** |
| 茯苓 | **祛水** |
| 牡丹+桃仁 | **攻淤血** |
| 芍药 | **止腹痛** |

### 加减
- 癌症（重证）→ 加 **生附子** + **硫磺**（等量），祛寒实
- 硫磺：祛水又祛寒；生附子：祛寒

### 药效判断（倪师关键问诊）
- 吃药后手脚**温热** → 有效，继续吃
- 吃药后手脚**越吃越冷** → 无效，另有问题

### 适用范围
不仅限于确认肿瘤——月经痛、月经不准时，只要症状合到即可用

## 三、附子汤 —— 怀孕子宫冷痛
### 条文
妇人怀娠六七月，脉弦，发热，其胎愈胀，腹痛恶寒，少腹如扇。所以然者，子藏开故也，当以附子汤温其藏。（方见伤寒）

### 症状链
脉弦 → 寒 → 子宫冷（子藏开）→ 腹痛恶寒 → 少腹如扇（自觉有扇子扇）→ 肚子胀大很快（非胎儿大，是**水**）

### 组成（见伤寒论）
炮附子、白术、茯苓、人参、白芍

### 方解
| 药 | 作用 |
|----|------|
| 炮附子 | **祛寒**（胎儿在里全身痛，附子汤主全身痛） |
| 白术 | **祛湿** |
| 茯苓 | **排水** |
| 人参 | **补津液** |
| 白芍 | **止痛** |

### 停药信号
寒痛没了，"如扇"感消失，肚子缩小，小便排出 → 停药

### 关键鉴别
怀孕应体热（两心脏），她却寒 + 脉弦 → 必是里寒

## 四、胶艾汤 —— 四大漏下症
### 条文
妇人有漏下者，有半产后因续下血都不绝者，有妊娠下血者，假令妊娠腹中痛，为胞阻，胶艾汤主之。

### 四大适应证（核心记忆）
1. 妇人漏下不止（非孕，平日滴漏）
2. 半产后下血不断（流产后出血不止）
3. 妊娠下血（孕中出血，胎未下）
4. 妊娠腹痛（胞阻）

### 组成（原方）
干地黄六两，川芎/阿胶/甘草各二两，艾叶/当归各三两，芍药四两
煮法：水5升+清酒3升，煮取3升，去滓，**纳胶令消尽**（阿胶不煮）

### 倪师常用量
生地黄1两，川芎5钱，阿胶/甘草/艾叶/当归各3钱，白芍4钱（腹痛重→5-6钱）
9碗煮3碗：水6+酒3（酒鬼可用8酒1水；一杯倒用8水1酒——灵活调配）

### 病机：肝脾两虚
- 肝经→络过子宫；脾→主少腹、统血
- 肝脾两虚 → 血不能固守子宫 → 漏下
- 气色晄白，精神不好（血虚状态）

### 方解
| 药 | 作用 |
|----|------|
| 生地黄 | **真正补血**（含铁质，黑软） |
| 当归 | 间接补血——油脂滑利排宿便→开胃→吃新食 |
| 川芎 | **活血**，开血管 |
| 艾叶 | 阳药，**行气活血**通络（灸用艾草同理） |
| 阿胶 | **止血**（不煮，冲入融化） |
| 白芍 | 止痛 |
| 甘草 | 调和 |

### 酒的作用
升提升发 → 加速药力吸收

### 特殊脉象（脉经）
寸脉左右跷跷板（一下跳左一下跳右）→ 阳虚不能守 → 阴漏下

## 五、当归芍药散 —— 妊娠绞痛（有水无漏）
### 条文
妇人怀孕，腹中绞痛，当归芍药散主之。

### 组成
当归/川芎各三两，芍药一斤，茯苓/白术各四两，泽泻半斤 → 散剂，酒和，日三服

### 倪师常用量
当归/川芎各3钱，芍药**1两**（重用止痛），茯苓/白术各5钱，泽泻6钱

### 病机：水停在子宫
- 当归+川芎+芍药 → **入血分**
- 白术+茯苓+泽泻 → **入气分，排水**
- 吃完药→水排掉→会口渴

### 与胶艾汤的关键区别
| 方 | 痛 | 漏下 |
|----|-----|------|
| 胶艾汤 | 腹痛（胞阻） | **有漏下** |
| 当归芍药散 | 绞痛 | **无漏下** |

### 非孕应用
月经痛可用，但需确定子宫有水（非淤血型）
- 淤血型月经痛 → 加茜草/丹皮/桃仁
- **茜草**：入肝经，活血化瘀（肿瘤也用）
- 三菱+莪术：破血力强 → **堕胎用**

### 堕胎方注意
三菱+莪术 + 当归/川芎/熟地/丹皮/桃仁（不加阿胶——阿胶止血与目的相反）
桂枝茯苓丸也可堕胎

## 六、干姜人参半夏丸 —— 妊娠剧吐
### 条文
妊娠呕吐不止，干姜人参半夏丸主之。

### 组成
干姜/人参各一两，半夏二两 → 生姜汁糊为丸，梧子大，饮服十丸，日三服

### 病机：胃中焦冷 → 停水
### 方解
| 药 | 作用 |
|----|------|
| 干姜 | **温中**（中焦冷） |
| 半夏 | **祛水**（胃中停水） |
| 人参 | **补津液**（吐伤胃津） |

### 倪师经验
- **一剂就知**——吃完第一包好八成
- 可用胶囊替代丸剂
- 平素胃虚寒的人 → 一怀孕就吐得凶 → 日常应保养

## 七、当归贝母苦参丸 —— 妊娠小便难（不肿）
### 条文
妊娠小便难，当归贝母苦参丸主之。

### 组成
当归/贝母/苦参各四两（等量）→ 蜜丸，如小豆大，饮服三丸，加至十丸

### 病机：热 → 胎动 → 压膀胱
### 方解
| 药 | 作用 |
|----|------|
| 苦参 | 利膀胱湿 |
| 贝母 | 利尿 + 润肠 |
| 当归 | 补血 + 润肠（油脂） |

### 与葵子茯苓散的关键区别
| 方 | 症状 | 有无水肿 |
|----|------|----------|
| 当归贝母苦参丸 | 小便难 | **无水肿** |
| 葵子茯苓散 | 小便不利 | **全身/下半身水肿** |

## 八、葵子茯苓散 —— 妊娠水肿
### 条文
妊娠有水气，身重，小便不利，洒淅恶寒，起即头眩，葵子茯苓散主之。

### 组成
葵子一升，茯苓三两 → **比例 3:1**，散剂，饮服方寸匕，日二服（可日三）

### 主治
孕妇全身水肿/下半身水肿（非单纯小便难）

### 特点
- 中药利水后不再回来（vs 西药利尿剂可能利掉羊水，身上仍肿）
- 茯苓通利三焦水，葵子加强茯苓排水力

### 适用时间
整个孕期十个月

## 九、当归散 —— 养孕妇（3-5月）
### 条文
妇人妊娠，宜常服当归散。妊娠常服即易产，胎无所苦，产后百病悉主之。

### 组成
当归/黄芩/芍药/川芎各一斤，白术半斤 → 散剂，酒服方寸匕，日再服

### 时机：60天后，第3月到第5月
### 方解
| 药 | 作用 |
|----|------|
| 黄芩 | **去胃热**（孕后胃口大开） |
| 芍药 | 活血化瘀（不堕胎，放心） |
| 川芎+当归 | 补血活血 |
| 白术 | 排水 |

### 定位
养**妈妈**的身体 → 体力好 → 易产 + 胎无所苦

### 禁忌
漏下/水肿 → 不是此方适应证

## 十、白术散 —— 养胎（6-10月）
### 条文
妊娠养胎，白术散主之。

### 组成
白术/川芎/蜀椒（去汗）/牡蛎各三分 → 散剂，酒服一钱匕，日三服+夜一服（24h四次，六小时一剂）

### 时机：第6月到第10月
### 方解
| 药 | 作用 |
|----|------|
| 牡蛎 | **补钙**（几千年前已知） |
| 蜀椒 | **温下焦**（辛辣，大建中汤也用） |
| 川芎 | 活血化瘀 |
| 白术 | 祛湿 |

### 定位
养**小孩** → 骨骼好，不养出超级巨婴

### 加减法（灵活调配——各药分开打粉）
- 苦痛 → **加芍药**
- 心下毒痛（胃痛）→ **倍川芎**
- 心烦吐痛，不能食饮 → **加细辛一两 + 半夏（大者二十枚）**
- 若呕 → 醋浆水服之
- 复不解 → 小麦汁服之
- 渴 → **大麦粥**服之

### 小麦 vs 大麦（本草经区别）
| 谷物 | 归属 | 用途 |
|------|------|------|
| 小麦 | **心之谷** | 情志问题（忧郁、烦躁、甘麦大枣汤） |
| 大麦 | **肠之谷** | 肠胃津液不足 |

## 十一、伤胎 —— 胎位不正
### 条文（有争议）
妇人伤胎，怀身腹满，不得小便，从腰以下重，如有水状……当刺泻劳宫及关元。

### 倪师修正
张仲景针灸不可信 → **不应针关元**（孕妇禁针！）
正确选穴：
- **至阴**（膀胱经井穴）：太阳经伤到 → 井主心下满
- **隐白**（脾经井穴）：脾主少腹

效果：针下去病人说"肚子在动"，很舒服
优势：都在脚上，不碰肚子，出了问题没人怪你

---

## 十二、难产方（补充方）
### 组成
党参二两，当归三两，牛膝四两 → 浓煎顿服，食顷即产

### 方解
| 药 | 作用 |
|----|------|
| 党参 | **补气** |
| 当归 | **补血** |
| 牛膝 | **活血化瘀 + 下胎** |

- 气分充满者 → 去党参，加牛膝一两

### 适用 vs 不适用
- **难产**（胎位不正、手/脚先出）→ 可用
- **死胎** → 建议西医处理取出
