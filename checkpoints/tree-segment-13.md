# 伤寒论 第13段 知识树

## 段13：太阴病（287-294）+ 少阴病开头（295-312）

---

### 一、太阴篇（287-294）

#### 1.1 太阴病主证（287）

| 症状 | 机理 |
|------|------|
| 腹满 | 脾主腹，脾受病→湿无法代谢→湿累积→肚子胀大 |
| 吐 | 脾肿大顶到胃→胃变小→食不下、吐 |
| 食不下 | 同上 |
| 自利益甚 | 若误下（用寒凉药攻下）→湿往下沉→下利更严重 |
| 时腹自痛 | 同上 |

**核心病机**：脾主腹、脾主湿。病入脾→脾不能代谢湿→湿盛→腹满。脾喜燥恶湿。

**鉴别**：饿不饿 → 脾管饿；吃东西有没有味道 → 胰脏管味。

**禁忌**：不可攻下（里无燥矢宿便，攻下用寒凉药只会让湿更盛）。

#### 1.2 太阴中风（288）

- 四肢烦疼：脾主四肢、主肌肉。湿盛→肌肉腠理间新旧津液无法替换→酸痛。
- 脉阳微阴涩而长者为欲愈：阳脉回来、阴脉变长 = 阳气回头。
- 治疗：桂枝汤 + 白术、茯苓（等量，各三钱）。
- 原理：运动出汗→新津液取代旧津液→酸痛去。脾喜运动。

#### 1.3 太阴病欲解时（289）

- 亥至丑时（21:00-03:00）。太阴正时是子午。
- 恢复信号：半夜肚子饿醒来吃东西、小便不断排出、不吐了。

#### 1.4 太阴病脉浮可发汗（290）

- 即使腹满，脉浮者仍可发汗→桂枝汤（可加白术茯苓加速）。

#### 1.5 自利不渴属太阴（291）★核心鉴别

- **太阴：自利不渴**（湿盛=水多，不需补水）
- **少阴：自利而渴**（津液伤→虚热上炎）
- **阳明：自利而渴**（燥=阳证）
- 病机：里有寒湿→当温之→宜服四逆辈。
- **炮附子+生姜**：去水积（寒湿在腑）
- **生附子+干姜**：救逆（心脏不行、里寒、恶寒、胃口无）
- **炮附子+生附子同用**：水侵犯心脏（水太大，火要熄），排水极快。

#### 1.6 脾家实·腐秽当去（292）★仙法

- 脉浮缓+手足温 = 系在太阴（肠胃功能还在）。
- 太阴本应发黄（脾湿不受营养→营养反溢→黄色外露）。
- **小便利→不发黄**。治太阴/阴证核心：**利小便，不利大便**。
- 阳证通大便，阴证利小便。
- 七八日暴烦+下利日十数行→自止 = **脾家实，腐秽当去**（脾功能恢复，自行排垃圾）。
- 判断关键：下利同时有没有小便？小便多+下利 = 好转；只下利不小便 = 坏病。
- 实脾饮系列（分消汤/补气建中汤/实脾饮）→必问小便如何。

#### 1.7 桂枝加芍药汤 & 桂枝加大黄汤（293）

**桂枝加芍药汤**：
- 适应：腹满时痛（一阵阵痛，隐痛），属太阴。
- 方：桂枝三两、芍药六两（加倍）、甘草二两炙、生姜三两、大枣十二枚
- 不加饴糖（甜味生湿，本已湿盛）。
- 白芍专治痛：寒水积→静脉循环不好→腹痛/经痛/肠痛。
- **倪师临床应用**：血癌（太阴证）→桂枝汤重用芍药+生附+白术+茯苓。

**桂枝加大黄汤**：
- 适应：大实痛（拒按，强烈痛），腹满不解。
- 方：桂枝三两、芍药三两、甘草二两炙、生姜三两、大枣十二枚、大黄一两
- **大黄只一钱**（阴证轻下，不像阳明证可用四两）。

#### 1.8 芍药大黄宜减之情（294）

- 脉弱 + 续自便利 → 胃气弱 → 大黄/芍药减量。
- 芍药可从6减到5或4，接近桂枝汤原方比例。

#### 1.9 太阴篇总结

- 太阴只两主方（桂枝加芍药汤、桂枝加大黄汤）。
- 太阴以脾为主，不论肺。但脾=肺之母→治脾即治肺。
- 脾统血，脾受湿→造血功能受影响→贫血。

### 二、少阴篇（295-312）

#### 2.1 少阴病主证（295）

- **脉微细**（脉越来越细小=病往里走）
- **但欲寐**（总想睡觉，睡14小时还觉得没睡饱，叫不醒）

#### 2.2 少阴病核心病机（296）

- **自利而渴**（vs 太阴自利不渴）。机理：
  - 心热下不去（被寒湿挡在中间）→回逆上冲→蒸发肺津→口渴
  - 下面下利→津液流失
  - 上渴下利，中间寒湿隔断
- **小便色白** = 少阴形悉具（里寒极盛，无气化，小便像白水无颜色）
- 正常小便：淡黄清澈
- 病机延伸：心火→小肠火（温度相同=正常）。寒湿隔断→小肠冷→水不气化→肾脏水直接入膀胱→冷水压迫膀胱→频尿/滴滴嗒嗒/无便意

#### 2.3 亡阳汗出（297）

- 脉阴阳俱紧 + 反汗出（本应无汗的伤寒脉） = **亡阳**，属少阴。
- 机理：心热下不去→阳回逆→往外走→出汗。
- 法当咽痛（热上冲喉咙）+ 吐利（湿往上则吐、往下则利）。
- 喉癌未动刀可治→动了刀+化疗后很难治。

#### 2.4 强发汗禁忌（298-300）

**298条**：少阴病咳而下利+谵语 = 被火气劫（强发汗所致）。
- 肺津不足→咳；下面下利；中间胡说八道（阳虚见鬼）。
- **少阴绝对不可用汗法**。

**299条**：脉细沉数（细=寒、沉=里、数=？）→病在里，不可发汗。

**300条**：脉微不可发汗（亡阳），尺脉弱涩不可下。

**恢复信号**：治后运动能自然流汗 = 心肾相交、寒湿去。

#### 2.5 自愈征兆（301）

- 脉紧→七八日→自下利→胸暴烦→手足反温→脉紧去 = 欲愈。
- **胸暴烦**：临床体验——心热突然接通小肠的瞬间，胸口一阵强烈痛/烦，之后手脚回温，精神恢复。倪师案例：黄连阿胶汤治心脏病。

#### 2.6 可治/不可治判断

| 可治 | 不可治/死 |
|------|-----------|
| 下利自止 + 手足温（302） | 恶寒身倦+下利+手足逆冷（309） |
| 恶寒倦卧 + 时自烦欲去衣被（303） | 吐利+躁烦+四逆（310） |
| 少阴中风阳微阴浮（304） | 下利止+头眩时时自冒（311） |
| 吐利+手足不逆冷+反发热（306） | 四逆+恶寒身倦+脉不至+不烦而躁（312） |

**头眩机理**：小肠火弱→肾水不气化→督脉无水气上滋脑→脑缺营养→头昏。正常：小肠热→肾水气化→督脉上脑→脖子如冷却管降温后入脑。

#### 2.7 少阴欲解时（305）

- 子至寅上（23:00-05:00）

#### 2.8 灸少阴（306）

- 脉不至 → 灸少阴七壮（太溪、复溜等），行经阳气。

#### 2.9 热在膀胱便脓血（307）

- 一身手足尽热 + 热在膀胱 → 便脓血（肾结石/膀胱结石）。
- 小便利 → 抵挡汤；小便不利 → 桃核承气汤。

#### 2.10 下厥上竭（308）

- 少阴但厥无汗 + 强发汗 → 动血（口鼻出/目出）= 下厥上竭，难治。
- **少阴/太阴利尿是最短路径，发汗绕远路。**

### 三、跨段连接

- 太阴→少阴：太阴没治好→病往里→进入少阴（心和肾之间）。
- 五道防线：太阳→阳明→少阳→太阴→少阴→厥阴。少阴还可救，进入厥阴就难了。
- 脾统血（太阴）、心主血（少阴）、肝藏血（厥阴）→三脏都影响造血→贫血。
- 四逆汤用药规律：炮附子+生姜（去水积）；生附子+干姜（救逆/心脏）；两者同用（水犯心脏）。
