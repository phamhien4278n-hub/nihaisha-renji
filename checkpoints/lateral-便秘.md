## 一、核心框架：倪师"便秘七方"

倪师原话：「光是一个便秘，我们有七种处方来治疗，西医只有一个通便剂。」

便秘辨证第一步：**看小便颜色**——这是最关键的鉴别点。
- 小便黄 = 热实 → 承气汤方向
- 小便白/清白 = 寒实 → 大黄附子细辛汤方向

再问**放屁与否**定芒硝取舍：
- 无屁 + 小便黄 → 大承气汤（堵大肠）
- 有屁 + 小便黄 + 不痛 → 小承气汤（堵小肠）
- 有屁 + 小便黄 + 痛 → 厚朴三物汤（金匮）

再问**有无恶心**区分大柴胡/承气：
- 有恶心 + 便秘 + 小便黄 + 往来寒热 + 胸胁苦满 → 大柴胡汤
- 无恶心 + 纯便秘 → 承气汤系列

## 三、内经病机总纲

| 经文/病机 | 与便秘的关联 | 出处 |
|-----------|------------|------|
| 「魄门亦为五脏使」 | 肛门（魄门）受五脏调控——便秘不仅是肠的问题，五脏皆可致 | C2：内经理论 ⚠️ 小凛推断关联 |
| 「阳不胜其阴，则五脏气争，九窍不通」 | 阳气不足→浊阴上逆→九窍（含魄门）不通 | C1：生气通天论 |
| 「肾主二便」 | 肾气不足→大小便异常；便秘可见于肾阳虚/肾精亏 | C1：金匮真言论 |
| 「肺与大肠相表里」 | 肺气不降/肺津不足→大肠传导失常 | C1：内经多处 |
| 「热气在中」则肠胃干 | 热邪在肠胃→津液耗→大便干结 | C1：内经多处 |
| 「脾约」——脾不能为胃行津液 | 胃热约束脾→津液偏渗膀胱（小便数）→大便坚 | C1：伤寒论247条 |
| 「诸腹胀大，皆属于热」 | 腹胀便秘多与热相关 | C1：至真要大论 |
| 「中焦如沤」失利 | 中焦气化失常→水谷不化→便秘或泄泻 | C1：内经多处 |

## 六、倪师临床心法（速记）

1. **看小便定寒热**：黄=热实（承气），白=寒实（大黄附子细辛）
2. **看屁定芒硝**：无屁=大承气（加芒硝），有屁=小承气（去芒硝）
3. **看恶心定大柴胡**：有恶心=大柴胡汤方向，纯便秘=承气汤方向
4. **看腹痛定厚朴量**：不痛=小承气，痛=重用厚朴（厚朴三物）
5. **看羊屎定脾约**：大便一粒一粒=麻子仁丸
6. **看黄疸+便秘**：茵陈蒿汤/大黄硝石汤/栀子大黄汤——鉴别在「有无黄疸/酒疸」
7. **以通为补**：大便不通时，大黄芒硝就是补药；攻下后小便变淡黄=净，可停药

## 七、补充：水饮与便秘的鉴别（容易误诊的关键点）

⚠️ **湿家中焦+大便困难 ≠ 承气汤证**

| 情况 | 真象 | 误治后果 |
|------|------|---------|
| 湿家中焦，大便坚（非不大便，是难排） | 寒湿阻中，白术附子汤方向 | 误攻下→虚+寒+湿三证并发 |
| 便秘性下利（燥屎堵大肠，水从旁流） | 大承气汤证（天枢剧痛拒按确诊） | 误作虚寒利→补涩加重 |
| 肠间有水气+便秘 | 己椒苈黄丸（水走肠间） | 误作纯承气→水不去 |

出处：C1——金匮痉湿暍篇 + 金匮痰饮篇 + 倪师原话多处
