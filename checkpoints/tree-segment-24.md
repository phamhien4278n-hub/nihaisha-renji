### 段24：金匮 — 气分·五脏水·阴实治法·黄疸成因

#### 核心概念

**1. 气分 vs 血分**
- 气分 = 肠胃系统，病在阳不足 → 寒实
- 血分 = 五脏系统，病在阴实（肝硬化/肝癌/肺癌等）
- 气分先讲，血分后讲——阴实证在血分

**2. 正常人体标准（中医版）**
- 上面是阳 → 摸到是冷的（阴）
- 下面是阴 → 实际是热的
- 头面冷、脚底热 = 正常。上半身热、下半身冷 = 有病

**3. 阴阳循环的8字形**
- 阴阳气以8字形旋转（肚脐为中心）
- 中间隔断 → 阴在下面转自己的，阳在上面转自己的 → 病

---

#### 一、气分——桂甘姜枣麻辛附子汤

**病机链条**：
胃气冷 + 胸腔寒 → 饮水入胃无法气化 → 水积在肋骨下缘一圈（大圆盘）→ 腹大如盘、边如旋盘

**症状**：
- 手脚逆冷（胃冷 → 手脚冷）
- 腹满胁鸣（肋骨边有水声）
- 阴阳气不相顺接：阳前通 → 恶寒；阴前通 → 痹不仁
- 肌肉麻痹（脾主少腹，腹寒 → 肌肉麻痹）
- 实证：放屁多（食物堵久产沼气）
- 虚证：遗溺（小便无法气化，漏出）

**处方：桂枝去芍药加麻黄附子细辛汤**
- 去白芍（阴药，阴已太盛不可再加阴）
- 靠桂枝行阳 → 心火导到小肠
- 麻黄+细辛+附子 → 少阴证去下焦寒实
- 寒实 = 东西堵到 + 寒像冰块 → 麻黄附子细辛汤化解
- "一剂就知，肚子水退掉"

**服药反应**：汗出、虫行皮中 → 阴阳气大气在转，皮肤/肌肉/脾脏的阴阳都跟着协调

---

#### 二、心下坚大如盘——枳术汤

**对比鉴别**：
| | 桂甘姜枣麻辛附子汤 | 枳术汤 |
|---|---|---|
| 病因 | 阳气不循环 | 水饮所作（胃下方有水饮） |
| 关键区别 | 没有胃不舒服 | **胃很难过** |
| 病位 | 整个腹部气分 | 胃下方 |
| 治法 | 行阳让阴阳气循环 | 去脾湿+排胃中浊水 |

**枳术汤**：枳实七枚 + 白术二两
- 白术 → 祛脾湿
- 枳实 → 排胃中浊水（食物坏死后的水）。枳实功能：针对胃冷消化不好产生的浊水
- 脾和胃相表里：脾湿盛 → 胃浊排不掉 → 枳术汤

#### 三、五脏水（血分·阴实）

**总则（229页）**：五脏水应放在气分"大气一转"后面——竹简排错。

**心水（心脏积水）**：
- 身重而少气、不得卧、烦而躁
- 其人阴肿（心火不下小肠 → 水积阴囊）
- 关键辨识点：**烦躁焦虑**——心藏神，心神不守 = 心脏问题
- 病理：水停在心脏下方 → 渗入脾脏（身重）→ 中隔积水（气不能下到肾 → 少气）
- 倪师思路→处方推导：茯苓（利心包水）+ 桂枝（强心导火入小肠）+ 防己（入阴囊）+ 杏仁（补肺津液，心火上逆烧肺）+ 白术（去身重）+ 黄耆

**肝水（肝脏阴实积水）**：
- 腹大不能自转侧（不能翻身）
- 胁下腹痛
- 时时津液微生（口水还在——肾未被克）
- 小便续通（还有小便，但量不大——木克土未克水）
- 病理：肝实堵到 → 顶到脾脏（木克土）→ 脾不受营养 → 水回头积腹腔
- 治法：甘淡渗利（去水）+ 实脾（甘味入脾）。**禁甘草**——甘草蓄水
- 切片/栓塞 = 加重实证 → 断人生路

**肺水**：
- 身肿（水渗到皮肤，肺主皮毛）
- 小便难（水停在肺上面下不来）
- 时时鸭溏（肺与大肠表里，上面堵→小肠火不能完全气化大肠水→大便稀溏）
- 关键鉴别：**不能躺+身肿+鸭溏** vs 心水（不能躺+烦躁+阴肿）
- 肺水来源：大肠 → 肺。堵到后水不停上来，无法停

**脾水**：
- 腹大、四肢苦重（脾主四肢肌肉）
- 津液不生、少气（气不能下降）
- 小便难（少腹湿盛）
- 关键鉴别：**四肢沉重** → 脾的问题
- 治法思路：白术+茯苓（去湿）+ 白芍（入少腹）

**肾水**：
- 腹大、**脐肿**（肚脐突出如鸡蛋——关键鉴别点）
- 腰痛、不得溺（小便排不出）
- 阴下湿如牛鼻上汗
- 足逆冷（肾经到脚）
- 面反瘦（肾水不能上达头部：命门火气化↑+肾经冲脉↑两路都断 → 脸瘦）
- 治法：**生硫磺**（色黄入三焦、补肾阳、利水）。真武汤+硫磺（棉布包煮）
- 颈为冷却系统（风池风府风门）——热天伸脖子降温，冷天缩脖子保暖

#### 四、阴实治疗总则

**核心公式**：**去阴的药（阳药）+ 去实的药 = 去阴实**

去实的药（各不相同）：
- 茜草（鹿衔草）→ 活血化瘀，深入内脏
- 炙鳖甲 → 入肝极深处（鳖无孔不入）
- 牡蛎 → 
- 瓦楞子 → 去痰积

去阴的药（升阳药）：
- 桂枝 → 生附子 → 石硫磺（由浅入深）

**肝家阴实处方示范**：
生附子 + 石硫磺 + 桂枝 + 炙鳖甲 + 茜草 + 川芎（活血）+ 泻南补北（黄连、黄芩）+ 去黄（栀子/茵陈） + 甘淡渗利（先利小便导腹水）

栀子 vs 茵陈：
- 茵陈 → 去实热之黄
- 栀子 → 去虚热之黄

#### 五、肝病初症（阴实早期）

**早期信号**：丑时（1-3am）准时醒 → 肝经气血流注时间，阳不入阴 → 热醒 → 3点后走肺经又睡

**基础方**（肝炎/B肝/C肝均可）：
柴胡（入肝胆）+ 黄芩（去热）+ 郁金（去血中热）+ 龙胆草

加减：
- 逆经（月经从鼻出）→ 单味郁金即可
- 黄 → 茵陈 + 泽泻（从小便走）
- 便秘 → 枳实 + 大黄。治肝必治大肠
- 攻养同用：攻坚药 + 滋养药（炙甘草/大枣/生姜固胃气）

**上热下寒加减**：上 + 石膏/知母，下 + 炮附子

#### 七、黄疸成因（黄疸病并治第十五）

**黄疸分类**：
- 阳黄：鲜黄如橘子，好治（10天左右）
- 阴黄：黑暗暗的黄，难治

**基本病机**：**胃过快 + 脾过慢** → 脾湿累积 → 脾肿大 → 顶到十二指肠/胆 → 胆汁逆流 → 入肝 → 入全身 → 发黄

**外因**：
- 表风束缚 + 汗不透发 + 体内湿盛 → 湿排不出去 → 脾湿过盛 → 压迫胆 → 胆汁逆流 → 黄疸
- 导火线是表风，根子是体内有湿

**内因**：
- 脾湿太盛 → 土克水（肾）→ 肾阳上逆 → 胃加速（先天支援后天的肾阳反冲胃）→ 脾慢胃快 → 不平衡 → 黄疸
- 肾阳两个出路：①小肠（膀胱气化）②胃（蠕动动力）。肾功能衰弱 → 胃不能纳谷（吐）+ 极度眩晕

**谷疸**：
- 胃快+脾慢 → 食物堵在胃里 → 只有粘稠浊水下去 → 到小肠 → 渗入膀胱 → 小便浓稠如痰（不通）
- "热流膀胱，身体尽黄"

**脾湿的成因**：
1. 思虑过度（脾主思）
2. 缺乏运动（久坐冷气房）
3. 汗不透发（饮→湿）

**肝水 vs 脾水 小便鉴别**：
- 肝水 → 小便续通（木克土未克水）
- 脾水 → 小便没有（土克水，肾被克）

#### 新学到的东西

1. 气分=寒实（阳不足）、血分=阴实（脏有实）——两套治法完全不同
2. 桂枝去芍药加麻附细辛汤的逻辑：去阴药白芍，全靠阳药打
3. 枳术汤 vs 桂甘姜枣麻辛附子汤的鉴别点：胃难不难受
4. 五脏水逐脏鉴别体系（烦躁/胁痛/鸭溏/四肢重/脐肿）
5. 肝水小便续通 vs 脾水小便没有 —— 木克土 vs 土克水的区别
6. 阴实治疗公式：去阴药+去实药
7. 生硫磺用于肾水（脐肿鉴别点）
8. 肝病丑时醒 = 阳不入阴
9. 黄疸基本病机 = 胃快脾慢 → 胆汁逆流
10. 半夏 vs 吴茱萸止呕鉴别：水呕 vs 寒呕
11. 谷疸 = 食物堵胃 → 浊水入膀胱 → 小便浓稠不通
