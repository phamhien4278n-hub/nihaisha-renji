# 金匮要略 段5 知识树

## 一、太阳中暍（中暑）

### 核心病机
- **本质**：津液不足，脱水
- **主症**：发热恶寒，身重而疼痛，脉弦细芤迟（非浮数——区别于伤寒温病表证），小便已洒洒然毛耸，手足逆冷，小有劳身即热，口开前板齿燥
- **板齿燥原理**：舌头讲话时离开上颚→任督二脉断开→津液不足时板齿处干燥（任督交汇处）

### 误治三条
| 误治 | 后果 | 机理 |
|------|------|------|
| 发汗 | 恶寒甚 | 汗出更伤津液+刺激心脏→心脏力量不够→怕冷 |
| 温针 | 发热甚 | 津液已伤再加热 |
| 攻下 | 淋甚（小便浓稠） | 津液本已不够，再攻下或利尿→仅剩津液排出→浓稠 |

### 处方一：白虎加人参汤
- **适应症**：太阳中热者，暍是也，汗出恶寒，身热而渴
- **组成**：知母六两，生石膏一斤（碎），甘草二两（炙），粳米六合，人参三两
- **现代用量**：石膏一两起，知母五钱，甘草两三钱，粳米三钱（非六碗！），人参三钱
- **各药作用**：
  - 石膏：退高热（经方退热主力，不同于温病用犀角）
  - 知母：除烦（《本经》言除烦，手足躁扰），祛湿力强（科学中药暴露空气一夜变硬块）
  - 炙甘草：和中
  - 粳米：取土性，生津液
  - 人参：滋阴→去热（经方用人参目的：滋阴去热，用于高热时）

### 处方二：一物瓜蒂汤
- **适应症**：身热疼重，脉微弱，夏月伤冷水，水行皮中所致
- **病机**：夏天跳冷水→寒水瞬间束毛孔→汗出不来、水进不去→"真空状态"（皮中积水）
- **组成**：瓜蒂二十个，水一升煮取五合，顿服
- **原理**：吐法——吐→全身发微微一层汗→毛孔开→水出→津液回头
- **比喻**：吸管按头放水中提起→水悬在管内→手指放开→水出（吐法即"放开手指"）
- **关键**：中暑津液已伤，不能用开发表药（更伤津液），用吐法借反刍作用微微发汗

### ⚡ 阳极生阴 / 阴极生阳（倪师临床突破）
- **核心**：人体是圆形的（太极图），热极生寒，寒极生热
- **临床案例**：胖子一家三口→女儿一派热证→石膏从五两加到一斤仍热、胃口更大→改用生附子→一剂下去冷、开始瘦（小便不断）
- **寒极生热案例**：里寒盛手脚冰冷→生附子炮附子下去仍冷→改用黄芩黄连→热了（厥阴篇治疗尿毒/某些癌症晚期）
- **中暑应用**：白虎加人参汤三次越加越热→第三次即可确定→换热药
- **突破意义**：不要和病性对抗——越用寒药对抗热证越热，干脆加热药让它热极回头

## 三、百合狐惑阴阳毒证治第三——百合病

### 百合病定义
- 百合：症状很多（百脉一宗，悉致其病）
- **病因**：①大病初愈（体力未恢复）②长久忧郁（忧郁症/神经质）

### 五脏藏神体系
| 脏 | 所藏 | 失衡表现 |
|----|------|---------|
| 肝 | 魂 | 动（欲行不能行等） |
| 心 | 神 | — |
| 脾 | 意 | 意念想吃但不能吃、注意力不集中 |
| 肺 | 魄 | 白天见鬼（如有神灵）、幻觉 |
| 肾 | 志 | 没有志气、不想干事、晃晃荡荡 |

### 主症群
- 意欲食复不能食，常默默然
- 欲卧不能卧（肾水不足→不能生木→肝不静→动来动去）
- 欲行不能行
- 饮食或有美时，或有不欲闻食臭时
- 如寒无寒，如热无热
- 口苦，小便赤
- **诸药不能治，得药则剧吐利**（最核心特征——什么药都不受）
- 如有神灵者（看到奇怪东西=魄不宁）
- 身形如和，其脉微数

### 预后（小便反应法则）
| 小便反应 | 愈期 | 机理 |
|----------|------|------|
| 溺时头痛 | 60日 | 津液伤最重，小便再泄→头痛 |
| 溺时头不痛，淅淅然（耳中有风声） | 40日 | 肾较差 |
| 溺快然，但头眩 | 20日 | 较轻 |

### 百合病七方

#### 1. 百合地黄汤（纯百合病——不经吐下发汗）
- **组成**：百合七枚（擘）+ 生地黄汁一升
- **制法**：先煎百合（久煮至浓稠，不同于做菜要Q），内地黄汁（生地泡热水后布包拧汁），煎取一升五合
- **原理**：百合色白入肺→安魄→如见神灵状消失；生地黄色黑入肾→补水→水生木→肝静→能卧能睡
- **服后**：大便当如漆（黑色=排伏热余毒）
- **特点**：只用两味，轻推一下——其他药（桂附八味丸、四逆汤）百合病统统不受
- **适用**：大病初愈 + 忧郁症

#### 2. 百合知母汤（发汗后）
- **组成**：百合七枚 + 知母三两
- **原理**：知母除烦——发汗后津液更伤导致烦躁

#### 3. 百合滑石代赭汤（下之后）
- **组成**：百合七枚 + 滑石三两（绵裹） + 代赭石如弹丸大（绵裹）
- **原理**：代赭石涩剂止利；滑石利小便（让水从小便出=另路止利）
- **为什么用滑石不用茯苓/泽泻/猪苓**：百合病身体极虚→金石类药力量更强；滑石味淡无味

#### 4. 百合鸡子汤（吐之后）
- **组成**：百合七枚 + 鸡子黄一枚
- **制法**：百合煮好→放入鸡子黄搅匀（不能煮！煮了成蛋花汤）
- **原理**：吐后胃津液伤→甘草/生姜/人参统统不受→从食物找替代胃液的东西→鸡蛋黄滋阴
- **关键**：鸡蛋黄不煮=吃滋阴效果；煮成蛋花汤=如食不食（百合病不受）

#### 5. 百合洗方（一月不解，变成渴者）
- **组成**：百合一升，水一斗，渍一宿，以洗身
- **后续**：洗已食煮饼（面条），勿以盐豉（咸的会更渴）
- **原理**：渴但喝水不受→从表补水分（外洗法）

#### 6. 栝蒌牡蛎散（渴不解者）
- **组成**：栝蒌根 + 牡蛎（熬），等分，为细末，饮服方寸匕，日三服
- **原理**：栝蒌根止渴；牡蛎（味咸）去虚热——大病初愈的渴是虚热引起的，光止渴不够，必须去虚热
- **关键**：单用栝蒌根止不了百合病的渴——必须有牡蛎去虚热

#### 7. 百合滑石散（病变发热者）
- **组成**：百合一两（炙） + 滑石三两，为散，饮服方寸匕，日三服
- **注意**：当微利者止服（出现轻微下利即停药），热则除

### 百合病方剂总览
| 情况 | 方剂 | 核心加味 | 作用 |
|------|------|----------|------|
| 纯百合病 | 百合地黄汤 | 生地黄汁 | 安魄+补肾 |
| 误汗后 | 百合知母汤 | 知母 | 除烦 |
| 误下后 | 百合滑石代赭汤 | 滑石+代赭石 | 止利 |
| 误吐后 | 百合鸡子汤 | 鸡子黄 | 补胃液 |
| 一月仍渴 | 百合洗方 | 外洗 | 从表补水 |
| 渴不解 | 栝蒌牡蛎散 | 栝蒌根+牡蛎 | 止渴+去虚热 |
| 变发热 | 百合滑石散 | 滑石 | 清热利小便 |

## 四、新学到的关键概念

1. **中暑≠普通热病**：脉弦细芤迟（非浮数），误治后果严重，已脱水不能再发汗/攻下
2. **吐法的妙用**：中暑水行皮中→瓜蒂吐→微汗→毛孔开——不伤津液的解表法
3. **阳极生阴**：热证寒药越治越热→热药→热极回头→临床可验证（三次白虎加量仍热=信号）
4. **百合病的"不受药"特征**：诸药不能治，得药则剧吐利——必须用最轻最易消化的药食
5. **五脏藏神在临床的应用**：魄不宁=见鬼，意不宁=想吃不能吃，志不宁=没志气——不是精神病是脏腑失衡
6. **百合地黄汤的设计**：色白入肺+色黑入肾=金生水→水生木→五脏协调——只用两味食物级药物
7. **滑石 vs 茯苓的选择**：虚极用金石类药力量更强，淡味无负担
