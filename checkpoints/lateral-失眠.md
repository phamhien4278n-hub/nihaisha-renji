## 失眠总论（内经病机）

**核心病机链**：人卧血归于肝 → 肝藏魂 → 魂归肝则眠。血不归肝 → 魂不归 → 失眠。[C1 内经·五脏生成篇，segments/03行23]

**两类血不归肝**：
1. 肝实——肝脏长东西（实症），血回不去，反逆出来 → 人醒
2. 肝虚——回流力量不够，血回不去 → 也失眠 [C1 同上]

**心藏神**：心者君主之官，神明出焉。心脏有问题时神没有办法回到心脏 → 失眠。心肾相交（坎离既济）：肾水上济心火 → 心火不过亢 → 能眠。水火不交 → 失眠。[C1 内经·灵兰秘典论 + 倪师讲解]

**阴阳出入**：阳入于阴则寐，阳出于阴则寤。阳不入阴 = 失眠的总纲。[C2 内经推导]

**诊断三问（倪师铁律）**：①睡眠（心——好睡通宵/肝1-3醒/肺3-5醒）②手脚温否（心血压到末梢=手脚温）③胃口（有饿=有胃气=救得回）[C1 Hindsight 倪师多次强调]

## 倪师失眠诊断口诀

1. **定时醒 = 经络诊断**：1-3am = 肝（肝癌前兆可提前两年发现）；3-5am = 肺
2. **整夜不能睡 = 心脏问题**（心藏神）
3. **栀子豉 vs 黄连阿胶**：栀子豉=虚烦轻证(病后余热)；黄连阿胶=阴虚重症(心肾衰竭/贫血性失眠)
4. **酸枣仁 vs 黄连阿胶**：酸枣仁=虚劳(年老/大病后肝血不足·消导非强补)；黄连阿胶=心血不足(心藏神)
5. **烦=热=危险信号**：烦躁加重=病情恶化；安静=好转
6. **失眠+大便不规则+手脚冷+胸刺痛 = 动脉堵塞四大症**
