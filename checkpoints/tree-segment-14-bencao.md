## 倪师关键思路（本段提取）

### 1. 减肥方（防己段·倪师即兴开方）
**组成思路**：
- 石膏（大量，三两）→ 寒凉缩胃，食欲大减
- 知母 → 去热
- 防己 → 泻三焦油水
- 茯苓 → 甘淡渗湿，去肌肉水
- 泽泻 → 利全身水
- 加减：便秘加大黄，轻秘加麻子仁，失眠加酸枣仁

**核心理念**：不吃→身体释水→必须同时利尿，否则积水。

### 2. 紫参汤的辨证逻辑
下利+背痛 → 金匮紫参汤。破血通经，肺有淤血积血→咳逆下利背痛全可用。**但必须是热症**。

### 3. 金制木 → 补肝逻辑（萆薢段）
色白（金）→ 入肺 → 金克木 → 疏导肝脏（不是砍掉，是疏密）。肝脏累积毒素时，用金药（色白）把毒素排掉。

### 4. 月经=奶水（土瓜根段·侧面印证）
「妇人带下，下乳汁不通」→ 淤血散 → 奶水下 → 月经来。千年前中医认月经=奶水是天经地义，不需特别说明。

### 5. 内痔·外痔·大肠炎鉴别（地榆段）
- **内痔**：先血后便，血飙出来不痛（地榆）
- **外痔**：大便时肛门痛，纸擦见血
- **大肠炎**：血便混合，一天几十次（赤石脂+禹余粮）

### 6. 浮萍治风疹的「以意得」思维
浮萍 → 风一吹到处跑 → 象风疹（受风才发）。浮在水面 → 吸水强 → 利尿。味辛辣 → 解酒。面青背紫者佳。

### 7. 热极生寒·寒极生热（防己段呼应）
高热用石膏不退 → 改用咸味热药（如生附子）→ 热退。「阴阳是圆的，推到极限会回头。」

### 8. 久病必淤（丹皮段·唐容川理论）
癌症/硬块/肿瘤的治疗 = 热药开门 + 活血化瘀（丹皮是核心活血药）。

### 9. 竹叶 vs 黄芩黄连的寒性区分
都是寒药，但经方在妇科选竹叶不选黄芩黄连，在别处选黄芩黄连不选竹叶——性质不同但需经方具体讲才分得清。

### 10. 海藻的双重作用
- 咸软坚 → 甲状腺/淋巴/睾丸硬块
- 咸入水（海水潮汐=月经）→ 入子宫卵巢 → 稳定情志。未婚小姐多吃海藻。
