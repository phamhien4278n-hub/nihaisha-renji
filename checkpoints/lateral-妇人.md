## 一、黄帝内经 · 病机总纲

### 1.1 核心生理

| 生理要点 | 内经原文/倪师讲解 |
|---------|-----------------|
| 天癸节律 | 「七七四十九，天癸竭」——女子49岁月经当停 |
| 月经来源 | 奶水即月经——心火下行化赤为血，余者下行子宫为经水，上行为乳 |
| 周期 | 28天一个周期 |
| 妇科关键 | 心脏热力→小肠→子宫。心阳不足则子宫寒 |
| 经色诊断 | 色淡=小肠火不够(寒)；色黑=太热；正常=红色 |
| 经量诊断 | 滴漏不止=阳不足（固不住）；暴下=热或瘀 |

### 1.2 病机总纲

> 「妇人之病，因虚、积冷、结气，为诸经水断绝，至有历年，血寒积结，胞门寒伤，经络凝坚。」——金匮·妇人杂病脉证并治第二十二

三条根：
- **虚**：气血不足（尤其血虚）
- **积冷**：下焦寒凝
- **结气**：情志郁结，气机不畅

### 1.3 治则基石

- 「肝苦急，急食甘以缓之」→ 甘麦大枣汤
- 「瘀不去，新不生」→ 攻补兼施（攻三补七或攻补各半）
- 「急则治标，缓则治本」
- 「虚者补其母，实者泻其子」
