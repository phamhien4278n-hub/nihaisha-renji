## 一、核心病机（黄帝内经）

### 1.1 总纲：阳微阴弦

> 「师曰：夫脉当取太过不及，阳微阴弦，即胸痹而痛，所以然者，责其极虚也。今阳虚，知在上焦，所以胸痹心痛者，以其阴弦故也。」——《金匮》第九篇首条

**解读**：
- 寸脉（阳）微 = 上焦阳虚，胸阳不足
- 尺脉（阴）弦 = 下焦阴寒水气盛
- 心火衰微 → 小肠温度不够 → 大肠水无法气化 → 寒水从下上冲 → 胸痹而痛

### 1.2 水气凌心机理

心火移热小肠 → 正常时水气化上行如蒸汽。当心阳不足、小肠温度下降 → 水不成气，以液态寒水上冲 → 如冷水泼到热引擎上 → 胸痛。

### 1.3 胸阳定位

胸为诸阳之会，腹为阴。心为君主之官，不受邪。心脏病根源常在心包或心脏周围结构（痰/水/寒）。

### 1.4 脉诊判生死

- 正常：一息五至，五十四下一止
- 每五下一止 → 心脏可用五年（危）
- 每十下一止 → 可用十年
- 从十下变十五、二十 → 转好
- 从十五变十、变五 → 转坏

## 三、倪师心脏病治疗大法

### 3.1 治心必治肺
「火克金，金克木」，薤白即入肺治心。心脏三针中天突（肺经气所过）为第一针。

### 3.2 治心必强肝
肝为心之母，乌梅酸入肝。乌梅赤石脂丸用乌梅+蜀椒（辛入肺）=肺肝同治。

### 3.3 胸满用枳实，腹痛用白芍
张仲景铁律：胸满→枳实，腹满→厚朴，腹痛→白芍。

### 3.4 治心保冷
心脏病病人少穿衣，睡前冷水冲脚→刺激心脏自愈。夏天洗冷水澡。

### 3.5 四大症状判愈
①能睡 ②大便正常 ③手脚热（到脚趾尖）④胸口不痛不闷

### 3.6 动脉堵塞诊断
- T5（第五椎）有压痛→心脏结构问题（瓣膜/心包湿）
- 耳针心点压痛→动脉血管堵塞
- 两处都痛→气血双堵

## 四、针灸法则汇总

### 4.1 心脏三针（急救/主方）
**天突 → 巨阙 → 关元**：气从喉咙下到心脏→小肠→会聚第十椎下→再从第十椎下导气外出

### 4.2 胃心胸
**公孙 + 内关**：不分寒热虚实，所有胃/心/胸闷痛先下此二穴

### 4.3 急性心痛
加针**厉兑**（胃经井穴，井主心下满）：左痛针右，右痛针左，正中痛双针

### 4.4 背俞
心俞（T5旁）、肺俞（T3旁）、肝俞（T9旁）

### 4.5 第十椎下
无具体穴名，为心气导出孔道——针巨阙关后天突后，气自然汇聚此处，再针第十椎下导出

## 六、学习要点 & 临床提示

### 6.1 张仲景的胸痹递进治疗阶梯
```
最轻  →  栝蒌薤白白酒汤（胸背痛短气）
        ↓ 加重（痰多不得卧）
        →  栝蒌薤白半夏汤
        ↓ 气结胸中
        →  枳实薤白桂枝汤 / 人参汤（虚实分治）
        ↓ 偏水饮/偏气滞
        →  茯苓杏仁甘草汤 / 橘枳姜汤
        ↓ 时发时止
        →  薏苡附子散
        ↓ 心悬痛
        →  桂枝生姜枳实汤
最重  →  乌梅赤石脂丸（心痛彻背背痛彻心）
```

### 6.2 常规心脏病 vs 动脉血管堵塞
| | 结构/心包问题 | 动脉堵塞 |
|---|---|---|
| 诊断 | T5压痛点 | 耳针心点压痛 |
| 痛感 | 闷痛（虫蛀感） | 刺痛（刀刺感） |
| 治法 | 瓜蒌薤白系列 | 倪师活血温通方 |
| 用药原则 | 开胸涤痰通阳 | 热药化瘀+活血不破血 |

### 6.3 倪师独特贡献（C2标注）
1. 乌头赤石脂丸 → **乌梅赤石脂丸**（临床验证，乌梅+蜀椒+干姜+炮附子+赤石脂）
2. 动脉堵塞方：**黄连阿胶汤去白芍** + 桂枝+炙甘草+炮附子+川芎+丹皮+枳实
3. 治心保冷法（冷水冲脚/少穿衣）
4. 心脏三针 + 第十椎下导气法

### 6.4 待验证/缺漏
- 动脉堵塞方的原始条辨在《金匮》中已遗失（倪师自创补充）
- 紫参（疑为拳参或丹参）治肺癌肺痈——原文缺失
- 真心痛（内经死证）在《金匮》中未独立论述，乌梅丸或可通治
