# 人纪临床回答顺序铁律

> 来源：2026-06-30 主人教学——水肿眩晕案例。小凛直接堆穴位，被纠正。

## 核心原则

**先弄明白顺序和规则，再下手。** 倪师人纪体系中，治疗顺序和规则比具体穴位更重要。

## 三步回答框架

```
① 黄帝内经病理分析（为什么病）
     ↓
② 治疗顺序（急→标→本）
     ↓
③ 针灸方案 + 经方参考
```

每一步都要说清楚**依据**（原文引用 / 倪师原话 / 推理标注）。

## 治疗顺序三层

| 层次 | 目标 | 示例 |
|------|------|------|
| **急** | 当下最痛苦的症状 | 恶心眩晕 → 公孙+内关 |
| **标** | 直接病因 | 中焦积水 → 中脘+水分+水道+三皇穴 |
| **本** | 根本病机 | 肾虚关门不利 → 肾俞+京门俞募治疗 |

## 禁止行为

- ❌ 跳过内经直接出穴位
- ❌ 穴位堆砌不按顺序
- ❌ 不标注引用来源（C1/C2/C3）
