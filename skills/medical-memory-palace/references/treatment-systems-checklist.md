# 治则体系调用检查清单

## 使用方式

回答任何临床问题（"XX症状怎么治？"）时，逐条过此清单，确保不遗漏可用体系：

## 检查清单

- [ ] **经络体系**：症状属于哪条经？子母补泻取什么？原络治疗？
- [ ] **俞募体系**：该脏腑的募穴和俞穴在哪？俞募配对？
- [ ] **会郄体系**：该经的会穴和郄穴？原文支持此用法吗？（⚠️ 推断需标注）
- [ ] **前后对应/气汇导出**：前面问题是否在背后有对应点？心脏三针→第十椎下模式？
- [ ] **五门十变**：募穴/俞穴的五行属性？虚补母实泻子？化合取穴？
- [ ] **天应/阿是**：先辨经→在该经循行路线上按→找压痛点（不是简单哪痛扎哪）
- [ ] **统治法**：灸膏肓百壮？梅花灸？四花灸？
- [ ] **时间法**：灵龟八法、子午流注、地支时辰发病取穴？
- [ ] **脏腑相关**：表里经关联？五行生克关联（如肝病强脾）？

## 关键原则

- **原文有据** → 标注出处（段落号/行号）
- **推断** → 必须标注"⚠️ 小凛推断，原文未直接提及"
- 不确定时宁可少答，不可编造

## 常见易漏模式

| 症状 | 最容易被漏掉的方法 |
|------|-------------------|
| 胃病/反酸 | 梅花灸（中脘上下左右各一寸） |
| 膻中/胸痛 | 心脏三针→第十椎下前后对应、灸膏肓百壮 |
| 淋巴肿 | 天应：先辨三焦经→在三焦经上找压痛点 |
| 一切内科 | 灸膏肓百壮（"无所不疗"） |
| 一切痛症 | 会郄治疗（原文用于内脏痛+出血，经络痛不适用） |
