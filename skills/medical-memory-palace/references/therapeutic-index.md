# 治则索引表（Therapeutic Index）

> 所有课程共用的治则体系索引。每学完一段，提取涉及的治法填入此表。
> 表格按"症状×治则体系"组织，每格标注原文出处 + 原文/⚠️推断。
> ⚠️ 铁律：回答临床问题前必须先查此表，不能仅凭记忆。
