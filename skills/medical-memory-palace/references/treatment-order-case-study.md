# 临床治疗顺序实录：中焦积水致全身浮肿+眩晕

> 2026-06-30 主人手把手教学。核心教训：**顺序比穴位更重要。**

## 病例摘要

- 全身浮肿，手背脚背也肿
- 眩晕很厉害
- 伴有恶心

## 小凛初版（错误）

直接堆了一堆穴位：水分、阴陵泉、三阴交、复溜、太冲、百会——看起来每个穴都有依据，但**顺序是乱的**。先治急还是先治本？中焦积水先通哪？水气上逆的恶心怎么处理？——没有回答。

## 主人的正确逻辑

### 病理链

```
中焦积水（脾虚不运）
      ↓
水不平衡 → 水气上冲 → 眩晕 + 恶心（中焦水气扰胃）
         → 水湿泛溢 → 全身浮肿 + 手背脚背肿
```

**关键判断**：眩晕的根不是肝风，是中焦水气上冲。中焦的水不平衡才会引起眩晕——胃在心下，水停中焦，胃气不降反逆，所以兼恶心。

### 治疗顺序（三层）

| 层次 | 目标 | 穴位 | 原理 |
|------|------|------|------|
| **第一层：治急** | 止恶心眩晕 | 公孙 + 内关 | "公孙内关治胃心胸，不分寒热虚实"——倪师原话 |
| **第二层：治标** | 通中焦、利水 | 中脘 → 水分 → 水道 → 三皇穴 | 中焦积水不先通中焦，光利水道不够 |
| **第三层：治本** | 补肾 | 肾俞 + 京门 | 俞募治疗——肾主水，关门不利在肾 |

### 小凛漏掉的东西

| 漏了什么 | 为什么重要 |
|----------|-----------|
| **中脘** | 中焦积水不先通中焦，光利水道不够 |
| **公孙+内关** | 倪师课上反复讲——水气上逆犯胃导致恶心眩晕，正好在胃心胸范围内 |
| **水道** | 名字就在告诉你它是干嘛的 |
| **三皇穴** | 健脾利水的经典组合 |
| **太冲** | 归因错了——眩晕根在"水"不在"肝风"，自然不需要太冲 |

## 主人的总结

> "倪海厦在讲人纪的过程中，不仅会讲怎么治疗的方法，也会有很多治疗的顺序和规则，这些也是很重要的东西。先弄明白顺序和规则，再下手才是科学的逻辑。"

> "顺序和规则 > 具体穴位。先治急 → 再治标 → 再治本。先辨病位 → 再选治法 → 再配穴位。"

## 通用化

此三层顺序不仅适用于中焦积水，临床回答任何问题时都应遵循：

1. **先内经病理分析**（"为什么病"）
2. **再定治疗顺序**（"急→标→本"）
3. **再选具体治法**（针灸/经方）

不要跳过第2步直接给穴位——堆穴位看起来专业，实际没有灵魂。
