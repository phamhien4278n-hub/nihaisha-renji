## 关键发现：主流做法 = 三条腿

```
知识图谱 + 向量检索 + 间隔重复 → RAG 检索增强生成
```

## 关键洞察

- **AI不需要"记忆"，需要"防bit rot"** — 间隔重复对AI的意义不是对抗遗忘，而是验证知识库完整性
- **三条腿缺一不可** — 纯知识图谱无法语义检索，纯向量无法结构化推理，纯间隔重复无法发现跨科关联
- **治则提取比事后建矩阵更根本** — TCM-Sage等项目的实体提取是后处理，我们的治则提取嵌入学习流程，从源头防遗漏
