# Hindsight 实体提取模式

本文件记录人纪五科学习中向 Hindsight 写入的实体类型、关系类型和代码示例。

## 通用写入模板

```python
# 使用 hindsight_retain 工具
hindsight_retain(
    content="[科目·主题] 实体描述 + 核心关系",
    context="科目·段号·子主题",  # 用于按学习进度检索
    tags=["科目", "子标签"],     # 如 ["针灸","穴位"] 或 ["伤寒","方剂"]
)
```

### 针灸大成

| 实体类型 | 示例 | 关系类型 | 示例 |
|---------|------|---------|------|
| 穴位名 | 足三里、关元、合谷 | 能治 → 症状 | 足三里→所有胃病 |
| 穴位名 | 中极、关元 | 是X募穴 | 中极→膀胱募 |

| 治法 | 心脏三针 | 组成 → 穴位 | 心脏三针→天突+巨阙+关元 |

**写入节奏**: 每段学完写1~3条（小段1条，大段按子主题拆分2~3条）。每门课结束后触发 Consolidation。

**实战经验**: 针灸大成22段共写入25条 Hindsight 记忆，涵盖全部12经络+奇经八脉+手法+临床治症。

## 注意事项

1. **每段写入不宜过碎**：一段内同一主题的穴位/方剂合并为一条，避免一个穴位一条导致 Consolidation 负担过重
2. **Tags 规范**：一律使用科目简称——`针灸` `内经` `本草` `伤寒` `金匮`，子标签如 `穴位` `方剂` `药物` `病证` `概念`
3. **Context 规范**：`科目·段X·子主题`，便于按学习进度回溯
4. **同一实体跨科重复**：同名的实体（如"足三里"在针灸和伤寒中都出现）会自动在图谱中建立跨科边，不需手动声明
