# 临床推理模式（人纪病例分析实战经验）

## 一、反酸烧心鉴别诊断路径（2026-05-28 实战总结）

### 核心原则
- 反酸烧心是"症"，不是"病"。必须找到六经归属。
- 排除法 + 关键反证 + 阳性证据 = 最终诊断
- **"什么药有效"和"什么药加重"同等重要**

### 四步诊断法

**第一步：列出所有已试过的方剂及其结果**

| 方剂 | 结果 | 人纪解释 |
|------|------|---------|
| 旋覆花代赭石汤 | 无效 | 降逆治标，未治根本 |
| 半夏泻心汤+左金丸 | 无效 | 治痞+清肝火，不疏少阳枢机 |
| 吴茱萸汤 | 加重烧心 | 错判为寒，少阳有热→火上浇油 |
| 香砂养胃丸/养脾散 | 能控制但不断根 | 治脾（标），不解少阳（本） |

**第二步：找关键反证**
- 吴茱萸汤加重 → 排除胃寒、肝寒
- 左金丸无效 → 排除单纯的肝火犯胃
- 旋覆代赭石无效 → 排除单纯的胃气上逆

**第三步：找阳性证据**
- 口苦 → 少阳病标志（伤寒论277条）
- 肋胀痛 → 胸胁苦满（少阳核心证）
- 肝脉郁 → 少阳枢机不利

**第四步：验证诊断**
- 健脾有效但不能断根 → 标在脾，本在少阳
- "但见一证便是"（伤寒论113条）→ 口苦+肋胀已足够

### 最终诊断公式

```
少阳枢机不利（本病）
  ↓ 三焦气化失常
中焦脾胃升降失调
  ↓ 胃气不降反升
脾虚 + 反酸烧心（标病）
```

### 治疗：小柴胡汤（肝脾同治，标本兼顾）

- 柴胡+黄芩 → 解少阳（治本）
- 半夏+生姜 → 和胃降逆（治标）
- 参草枣 → 健脾（治标）

## 二、人纪关键原文索引

### 少阳病诊断标准
- **伤寒论277条**："少阳之为病，口苦、咽干、目眩也"
- **伤寒论113条**："伤寒与中风，有柴胡证，但见一证便是，不必悉具" — 黄金法则
- **伤寒论105条**：小柴胡汤四大主证 — 往来寒热、胸胁苦满、默默不欲食、心烦喜呕

### 少阳与脾胃的关系
- **伤寒论106条**："脏腑相连，其痛必下，邪高痛下，故使呕也，小柴胡汤主之" — 少阳病直接导致胃气上逆
- 倪师讲肝病："肝肿起来了顶到胃...胸胁苦满，恶心，默默不欲食" — 木克土的临床表现

### 反酸/烧心相关论述
- **伤寒论447行**：吴茱萸汤用于胃酸逆流的前提是"里虚"+"胃寒"
- **伤寒论448行**：吴茱萸汤+旋覆代赭石用于胃酸逆流，但要知道"胃酸怎么来的"
- **金匮893行**："胃的津液由肝的酸味、胆的苦味、脾的甜味混合而成"
- **金匮1426行**：吴茱萸止呕=胃寒（不渴+喜热饮）；半夏止呕=水饮（关脉弦+不渴）

### 三阳并病与和解
- **伤寒论**：三阳并病（太阳+少阳+阳明）→ 用小柴胡汤和解
- 小柴胡汤是"肝脾同治"的代表方

## 三、常见误诊陷阱

| 陷阱 | 表现 | 正确思路 |
|------|------|---------|
| 见酸治酸 | 直接降逆制酸 | 六经辨证，找到反酸的原因 |
| 见口苦就清肝火 | 左金丸、龙胆泻肝 | 分清肝火是原发还是因虚致郁 |
| 辛温药加重=更寒？ | 以为寒证用热药 | 可能是热证用热药火上浇油 |
| 健脾有效=病根在脾 | 香砂养胃丸控制病情 | 可能是标在脾，本在他处 |
