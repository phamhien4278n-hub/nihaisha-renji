# 痔疮反复发作——寒热鉴别与两型论治

> 2026-07-01 实战整理。主人纠正：内经"因而饱食筋脉横解"只解释了湿热型，不能套用于所有痔疮。

## 一、内经原文的范围

**《生气通天论》**：「因而饱食，筋脉横解，肠澼为痔。」

这句话只描述了**暴饮暴食→肠胃撑大→筋脉松弛**这一种病因。对于"吃得很少"但反复发作者，病机完全不同——不是撑松的，是提不住的。

## 二、两型鉴别表

| 维度 | 饱食湿热型 | 脾虚气陷型 |
|------|-----------|-----------|
| **病因** | 暴饮暴食→肠胃撑大→筋脉松弛 | 脾虚→清气不升→中气下陷→局部失于升提 |
| **胃口** | 正常或偏多 | **吃得很少** |
| **病性** | 湿热实证 | **虚寒证 / 气虚证** |
| **大便** | 黏滞不爽·臭秽 | 溏薄·或先干后溏·无力排出 |
| **痔核** | 红肿热痛·充血明显 | 不那么红肿·脱出感明显·易反复渗血 |
| **出血颜色** | 鲜红·量多·喷射状 | 淡红·量少·渗漏不止 |
| **疼痛** | 灼热刺痛 | 坠胀隐痛 |
| **小便** | 深黄 | 清白/淡黄 |
| **口渴** | 渴喜冷饮 | 不渴或喜热饮 |
| **手脚** | 热 | **凉** |
| **脉象** | 滑数 | 沉细/弱 |

## 三、核心病机链（脾虚型）

```
思虑过度 / 劳倦 / 久坐
        ↓
    脾虚不运
        ↓
  ┌──────────┬──────────┬──────────┐
  ↓          ↓          ↓
吃得很少    中气下陷    脾不统血

  │          │          │
  └──────────┴──────────┘
              ↓
    局部筋脉松弛·血渗不止
              ↓

```

## 四、分型治疗

### A. 湿热型（赤豆当归散方向）

| 层次 | 方法 |
|:--:|------|
| 治急 | 承山·天应放血 |
| 治标 | 赤豆当归散（赤小豆不发芽打散）·槐花散凉血止血 |
| 方药 | 赤小豆+当归+槐花+地榆 |
| 针灸 | 承山+孔最+长强+天应放血 |
| 出血 | 章门+蠡沟止血 |
| 倪师口诀 | **近血鲜红=痔疮** |

### B. 脾虚气陷型（补中益气方向）

| 层次 | 方法 |
|:--:|------|
| 治急 | 承山止痛（不分寒热） |
| 治标 | 补中益气汤升提 + 热药止血 |
| 治本 | 足三里常灸·隔盐灸神阙 |
| 方药 | ⚠️ 李东垣补中益气汤（黄芪+人参+白术+当归+升麻+柴胡） |
| 针灸 | 承山+百会(升提)+神阙隔盐灸(温中)+足三里(健脾)+孔最 |
| 出血 | **热药止血**：干姜(炮姜)·炮附子·艾叶 |
| **铁律** | **出血源于寒，用热药止血——与槐花散思路完全相反** |
| 外治 | 槐实熬膏塞肛门（不分寒热） |
| 辅助 | 醋炙五倍子（酸收敛·脱肛同治） |

## 五、关键知识点

1. **"出血源于寒"**（C1·倪师原文）——痔疮出血不等于热证。吃得少+手足凉+便溏=寒性出血，不可用凉血药（槐花·地榆·侧柏叶），应用热药止血（干姜·炮附·艾叶）。

2. **赤豆当归散的发芽用法**（C1·倪师原文）——赤小豆≠红豆。排脓用不发芽打散；发芽后生吃则治腹膜炎。

3. **脱肛同治**（⚠️ C2 推断）——中气下陷型痔疮本质是轻度脱肛，醋炙（酸收敛）和石榴皮坐浴可参考。

4. **槐实不分寒热**（C1·倪师原文）——熬膏塞肛门，内外痔均可，是痔疮专药。
