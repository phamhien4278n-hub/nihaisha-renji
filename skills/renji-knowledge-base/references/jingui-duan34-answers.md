# 金匮要略段34 应用题 参考答案

---

## 第1题参考答案

**正确答案：B. 大黄甘遂汤**

**选方理由：**
- 少腹膨隆如覆鼎、叩之空响 → "少腹如敦"（敦，古代盛食器，腹大中空），典型大黄甘遂汤证
- 小便微难而不渴 → 水结而非纯血瘀（抵当汤证小便自利，因血不在膀胱）
- 口不渴 → 水停而非津伤
- 面色尚可 → 非久病虚劳

**鉴别要点：**
| 方剂 | 核心病机 | 小便 | 疼痛 | 口渴 |
|------|----------|------|------|------|
| 大黄甘遂汤 | 水血俱结子宫 | 微难 | 胀为主 | 不渴 |
| 抵当汤 | 纯血瘀子宫 | 自利 | 固定点刺痛 | 不渴 |
| 桃核承气汤 | 血结膀胱 | 不利 | 少腹急结 | 渴 |

## 第2题参考答案

**(1) 应选用当归芍药散。**

理由：
- 月经期腹痛且经血色暗有块 → 实证血瘀
- 面色萎黄、身体困重 → 脾虚湿盛（当归芍药散含有茯苓、泽泻、白术化湿）
- 食欲尚可 → 排除小建中汤"无胃口"的核心指征
- 小建中汤用于虚劳腹痛：面色苍白、食欲不振、腹痛喜按，本病例不符

**(2) 应转用小建中汤。**

理由：
- 月经期过后转为面色苍白、食欲不振 → 虚劳征象出现
- 腹部隐痛喜按 → 虚证（"喜按为虚，拒按为实"）
- 舌淡脉弱 → 气血两虚
- 已非经期，湿瘀之标已除，虚劳之本显现

## 第3题参考答案

**(1) 选用大黄甘遂汤而非抵当汤的原因：**

- 关键鉴别点在小便：大黄甘遂汤证"小便微难"（水血同结），抵当汤证"小便自利"（纯血在子宫不在膀胱）
- 少腹如敦（膨隆、叩空） = 水血俱结 → 必须水血同治
- B超囊实性回声 = 现代印证"水裹血"的病理形态

**水血同病 vs 纯血瘀对比：**
- 水血俱结：少腹膨隆如敦，叩之空响，小便微难 → 大黄甘遂汤（大黄+甘遂逐水+阿胶补血）
- 纯血瘀：少腹固定点24h刺痛，小便利，大便黑，善忘 → 抵当汤（水蛭+虻虫+桃仁纯破血）

**(2) 呕吐黑色血水是治疗反应，非不良反应。**

机理：
- 甘遂逐水之力峻猛，水血结块被击破后，败血浊水循经上涌从胃而出（吐），下走从肠而出（泻）
- 倪海厦案例：8颗胶囊服后吐泻黑血水，是"邪有出路"的表现
- 黑色 = 瘀血排出；泻后少腹变软 = 药到病所、结块已散

**(3) 只开一剂原因：**

- 甘遂为峻下逐水药，药力极猛，中病即止
- 水血结块一旦击破排出，再服则伤正气
- 金匮原方就是"顿服"（一剂一次服完），不做疗程设计

## 第4题参考答案

| 患者 | 选方 | 鉴别关键点 |
|------|------|------------|
| 甲 | 枳实栀子豉汤 | 大病后过劳（="劳复"），胸中烦热+心下痞 = 热郁胸膈+气机不畅 |
| 乙 | 竹叶石膏汤 | 虚羸少气+欲呕+气逆欲吐 = 病后胃虚热，津液亏虚；口渴不欲多饮 = 虚热而非实热 |
| 丙 | 牡蛎泽泻散 | 腰以下水肿 = 水湿下注，牡蛎咸寒软坚+泽泻利水，专治下半身水肿 |

**不可选方排除：**
- 理中丸 → 胃有寒吐涎沫，三患者均无寒象
- 小柴胡汤 → 解表/攻下后余热往来，三患者无少阳证

## 第5题参考答案

**(1) 方剂选择：**

| 患者 | 方剂 | 用法 |
|------|------|------|
| A（厚白带） | 矾石丸 | 蜜丸枣核大，棉袋包裹，纳阴道 |
| B（阴痒） | 蛇床子散 | 蛇床子粉末，棉袋包裹，纳阴道 |
| C（溃烂） | 狼牙汤 | 狼牙草煮汤，浸棉条，纳入阴道 |

**(2) 矾石丸与蛇床子散的本质区别：**

| 对比维度 | 矾石丸 | 蛇床子散 |
|----------|--------|-----------|
| 核心作用 | 燥湿收敛（矾石） | 杀虫止痒（蛇床子） |
| 主症 | 厚白带（质稠如糊） | 阴痒（奇痒难忍） |
| 病机 | 湿浊下注 | 虫蚀/湿热生虫 |
| 停药指征 | 白带转正常即停 | 痒止即停（"一剂就停"） |
| 有无痒 | 无明显瘙痒 | 奇痒为主诉 |

本质区别：矾石丸治"带"，蛇床子散治"痒"。

**(3) 不可同时使用。**

原因：
- 狼牙汤治"疮"（溃烂糜烂），病位在表皮破损
- 蛇床子散治"痒"，病在皮下/黏膜完好
- 疮面溃烂时用粉剂刺激创面，加重损伤
- 金匮三方各有所主，不可混用
