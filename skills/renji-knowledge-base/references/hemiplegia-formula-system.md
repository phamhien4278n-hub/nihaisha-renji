# 半身不遂（中风偏枯）·方剂体系

> 2026-07-01。从倪师原文+Hindsight记忆中系统整理。

## 一、核心鉴别——寒则缩·热则纵

**看手脚形态即可定方，不复杂。**

| | 寒性中风（≈90%） | 热性中风（≈10%） |
|------|------|------|
| 手脚 | **拘挛蜷缩** | **瘫软摊开** |
| 脉 | 紧/弦 | 数/滑 |
| 主方 | 小续命汤 | 风引汤 |

---

## 二、三大主方鉴别

| 方剂 | 主症 | 占比 |
|------|------|:--:|
| 小续命汤 | 口不能言（最常用） | ~90% |
| 千金三黄汤 | 浅中风+能识人+恶寒心烦 | 较少 |
| 术附汤 | 极度眩晕+不能睁眼（中焦寒湿） | 较少 |

---

## 三、小续命汤（C1·倪师原文·最常用）

### 方义结构——攻补兼施

```
小续命汤 = 
  麻黄汤（麻黄+杏仁+桂枝+甘草）  ← 发汗解表·开太阳
  + 当归·川芎                    ← 补血活血
  + 石膏                          ← 清虚热（防麻桂过温）
  + 干姜                          ← 温中散寒
  + 人参                          ← 补气扶正
```

### 四重攻补平衡

| 药对 | 攻 | 补 |
|------|:--:|:--:|
| 麻黄+桂枝 | 开表驱风 | — |
| 当归+川芎 | — | 补血活血·血行风自灭 |
| 石膏 | 清郁热 | — |
| 人参+干姜 | — | 补气温中·扶正 |

### 指征

口不能言、手脚拘挛、脉弦紧。中风十有八九。

---

## 四、补阳还五汤（C1·王清任·倪师重点讲过）

### 方义

| 药物 | 特点 | 作用 |
|------|------|------|
| **黄芪** | 君·重用 | 大补元气·气行则血行 |
| 当归·川芎 | 臣 | 补血活血 |
| 桃仁·红花 | 佐 | 活血化瘀 |
| **地龙** | 佐·关键 | 钻入组织间隙·清红花清不到的瘀血 |
| 赤芍 | 使 | 凉血活血 |

### 倪师三点强调

1. **红花 vs 地龙**：红花走**血管内**清瘀（脑溢血后血块）；地龙走**组织间隙/脑膜间**清瘀。两线并进。
2. **攻血必配补血**：地龙伤血，必须配当归。血虚时减地龙增归芎。
3. **使用时机**：中风**恢复期**。急性期仍需小续命/风引汤，不可一上来就补阳还五。

---

## 五、风引汤（C1·热性中风专方）

### 核心思路——涩攻并用

```
赤石脂（涩）——留住好东西
大黄（攻）——排掉废物（热+痰+瘀）
```

中风病人大小便失禁，不能纯攻也不能纯补——赤石脂收涩固脱，大黄推陈致新。最精妙的设计。

### 其他药物

金石类（寒水石·滑石·紫石英·石膏等）——重镇清热。热极生风，金石压之。

## 六、其他方剂

### 千金三黄汤（C1）
```
麻黄+独活 → 发汗散风
黄芪+黄芩 → 一补一清
细辛 → 入少阴搜风散寒
```
比小续命汤证**轻**——能认人。

### 术附汤（C1）
```
白术+附子+甘草+生姜+大枣 → 温中散寒燥湿
```
治"晕到不能睁眼"——中焦寒湿上扰。

### 侯氏黑散（C1·金匮·先兆期）
```
菊花四十分（君·最多）+ 白术·防风各十分 + 细辛·茯苓·干姜·人参 + 矾石（化痰）
```
气血两虚者先吃侯氏黑散补虚，再考虑风引汤除热。菊花用量最大——"诸风掉眩皆属于肝"。

## 七、中风药对专论（C1·倪师原文）

| 药对 | 针对 | 原理 |
|------|------|------|
| **川芎+丹皮+桃仁** | 血管三胞胎 | 任何血管问题（剥离·堵塞·出血·跌打） |
| **红花** | 血管内瘀血 | — |
| **地龙** | 组织间隙瘀血 | 动物药·钻入红花到不了处 |
| **天南星** | 胸腔顽痰·中风痰壅 | 化痰力量最强 |
| **半夏**（去水） | 脑积水·肺积水·止呕 | — |
| **三生饮** | 南星+半夏+生附子 | 中风痰壅重症 |

---

## 八、治疗顺序总纲

```
先兆期 → 侯氏黑散补虚
急性期 → 小续命汤（寒）/ 风引汤（热）攻邪
恢复期 → 补阳还五汤补气活血通络
全程   → 有胃气则生·灸足三里
```

### 倪师核心心法

- **以通为补**：通了就是补
- **阴要自回才是痊愈**：不滋阴
- **攻补用藥艺术**：不能强补（恋邪）也不能强汗（伤正）

---

> ⚠️ 治疗顺序为C2推断——倪师分开讲了各方，顺序按病机逻辑排列。各方的组成和方义均为C1原文。
