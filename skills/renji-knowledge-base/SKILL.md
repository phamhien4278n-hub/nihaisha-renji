# 人纪知识库

## 一、针灸大成

### 1.1 经络体系
- 十二正经：手三阴(肺·心·心包)从胸走手，手三阳(大肠·小肠·三焦)从手走头，足三阳(胃·膀胱·胆)从头走足，足三阴(脾·肾·肝)从足走腹
- 奇经八脉：任(诸阴之海)·督(诸阳之会)·冲(血海)·带(横一圈)·阳蹻·阴蹻·阳维(肋骨外)·阴维(肋骨内)
- 十五络脉：十二经各一络+任脉络(鸠尾)+督脉络(长强)+脾大络(大包)

### 1.2 五腧穴体系
- 阴经：井(木)→荣(火)→俞(土)→经(金)→合(水)
- 阳经：井(金)→荣(水)→俞(木)→经(火)→合(土)
- 手不过肘，足不过膝
- 补井当补合，泻井当泻荣

### 1.3 五取穴法则
| 穴性 | 主治 | 示例 |
|------|------|------|
| 井主心下满 | 动脉堵塞·心脏病 | 厉兑→感觉心脏被扫洗 |
| 荣主身热 | 颜色病·热症 | 内庭(胃热)·鱼际(肺热) |
| 俞主体重结痛 | 时间病·按时辰发病 | 陷谷(7-9点头痛)·临泣(子时发病) |
| 经主喘咳寒热 | 声音病·感冒变声 | 间使·灵道·商丘 |
| 合主逆气而泻 | 饮食不节·胀满 | 足三里·曲泽·阴陵泉 |

### 1.4 原穴·络穴
- 阳经有原穴，阴经无（以俞代原）
- 络穴连接表里经
- 原络治疗：病经取原穴+表里经取络穴

### 1.5 八脉交会四组配对 ★★★
| 配对 | 主治 |
|------|------|
| 后溪(督脉)+申脉(阳蹻) | 项强·脖子僵硬 |
| 公孙(冲脉)+内关(阴维) | 胃·心·胸一切 |
| 外关(阳维)+临泣(带脉) | 腰痛·胸痛 |
| 列缺(任脉)+照海(阴蹻) | 喉咙·咽喉 |

### 1.6 关键穴位
**足三里**：胃经合穴本穴·胫骨高起外开一指·按之冲阳脉消失=找对·所有胃病(吐血除外)·常灸健康长寿·灸太过口干→灸足三里导气·血压高→足三里下五分斜刺·补泻：顺经快转慢回=补(单数)，逆=泻

**关元**：小肠募·脐下三寸·心脏病急救→针下心跳回来·练气所在·生精所在·灸到黑·孕妇禁针

**合谷**：大肠经原穴·气穴·四关之一·牙痛合谷(骨交叉点·上牙左病右治)·青春痘灸合谷·美白灸合谷·孕妇可泻不可补·合谷越大越高气越足·判断真假阵痛(脉跳=要生了)

**三阴交**：三条阴经交会·妇科大穴·血穴(合谷=气穴)·月经痛立竿见影·孕妇禁针(补合谷+泻三阴交=落胎)·腹寒诊断(压痛点)

**委中**：合穴·母穴(补井当补合)·腰背委中求·放血解毒(牛皮癣·带状疱疹)·不灸

**涌泉**：子穴(泻井当泻荣→然谷代)·头顶痛特效·急救大穴

**阳陵泉**：筋会阳陵·阳陵泉透阴陵泉治膝盖

**膈俞**：血会膈俞(第七椎)·贫血·瘀血·内伤出血·瘀血=口渴不喝水

**膻中**：气会膻中·天突与鸠尾中间是玉堂·玉堂下一寸半=膻中

**厉兑**：井穴·井主心下满→动脉血管堵塞→下针感觉有人扫地清洗

**承山**：痔疮(+孔最+长强)·脚抽筋重压解·尾椎新伤青筋放血

**至阴**：井穴·胎位不正24h回正

**肩井**：胸腔瘀血活血化瘀·必配足三里(否则气闷晕倒)·统治乳癌

**太冲**：肝经原穴·四关之一·太冲脉=肝气存亡

### 1.7 传经规律
太阳(开·表·寒热往来)→入阳明(合·纯热·不再传)→入少阳(枢·胸胁苦满·可转阴经)

## 二、黄帝内经

### 2.1 核心理论
- **道=阴阳**：形(看得见)=阴·神(力量)=阳·阴平阳秘精神乃治·阴阳离决精气乃绝
- **寒极生热·热极生寒**：热到极→寒药无效→热药下去产生寒症(400磅太太案例)
- **寒气生浊·热气生清**：所有肿瘤癌症都是寒(寒=水不动→累积；热=循环→不累积)
- **冬伤于寒春必温病**：温病不存在·必须冬天伤寒才有温病
- **阳化气·阴成形**：阴虚阳气浮→阳亢；阳虚阴独盛→畏寒

### 2.6 望诊
- **手掌**：正常粉红(肺白+心红)→肝病暗红→越红越暗=方向错·粉红出现=病退

### 5.2 配穴顺序：治急 → 治标 → 治本

> 2026-06-30 主人亲自教学扩展——从两层升级为三层。只治急治标不治本 = 暂时好转但会复发。「先弄明白顺序和规则，再下手才是科学的逻辑。」

多个症状并存时，**下针顺序**和**方案排序**必须遵循三层优先级：

| 层次 | 时机 | 目标 | 关键原则 |
|:--:|------|------|------|
| ① 治急 | 立即 | 解除最痛苦的急性症状 | 公孙+内关治胃心胸一切恶心·眩晕，不分寒热虚实 |
| ② 治标 | 急症缓解后 | 解决当前病变的核心病机 | 中焦积水→中脘（胃募·通中焦），然后水分+水道，再三皇穴 |
| ③ 治本 | 标症控制后 | 补益根本，防复发 | 肾主水→肾俞+京门（俞募治疗），或加复溜 |

| 实战案例 | 正确顺序（三层） | 错误做法 |
|---------|---------|---------|
| **恶心眩晕+全身浮肿**（中焦积水） | ①公孙+内关（急·止恶心眩晕）→ ②中脘→水分→水道→三皇穴（标·通中焦利水）→ ③肾俞+京门（本·俞募补肾） | ①直接堆穴位不排顺序 ②误判肝风配太冲 ③只到三皇穴为止不治本 |

> **核心诊断**：水肿+强烈眩晕 = 中焦积水，水不平衡引起眩晕（水气上冲清窍），**非肝风内动**。归经错误 → 配穴全错——选穴和归经是两层独立的事。

> 来源：倪师治症规则 + Hindsight 记忆 + 主人 2026-06-30 亲自教学。

```
> **附·XXX鉴别：** ……（补充内容）
```

### 6.3 常见遗漏模式

| 遗漏原因 | 典型例子 | 修复方式 |
|---------|---------|---------|
| 同篇小知识点未被主题目覆盖 | 五脏痿五色诊断中"脾热色黄肉蠕动"——Q1只用了肺痿，其余四色未触及 | 在Q1答案后加「五脏痿五色诊断速查」附注表 |
| 治法方药细节 | 酒风（泽泻+苍术+茜草=2:2:1）没有单独出题 | 在Q2（发狂治法）答案后加「酒风鉴别」附注 |
| 某经脉/类型的厥未被患者案例覆盖 | 厥阴厥逆、少阳厥未纳入四患者题 | 在Q4答案后加「厥阴厥逆」附注 |

### 6.4 参考答案格式标准

每题答案必须包含：
- **诊断**：病名（如"肺痿""太阳经厥"）
- **辨证依据**：引用经文原文 + 症状对应
- **病机分析**：因果链（如 外因→脏腑→病理产物→症状）
- **治则**：治法思路 + 具体方药方向（可选写方名，不可空谈）
- **附注**（如有）：鉴别补充、速查表、关联知识点

综合题额外要求：
- 明确标注"治急→治标→治本"三层顺序
- 包含病机链条图（ASCII或文字描述）
