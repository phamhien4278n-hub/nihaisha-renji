# 痔疮反复发作辨治 —— 脾虚气陷 vs 饱食湿热

> 2026-07-01 临床讨论。从"吃得很少"推翻内经"饱食筋脉横解"，重新建立脾虚气陷型痔疮的病机链路。

## 问题来源

患者反复发作痔疮，但**吃得很少**——不符合《内经》「因而饱食，筋脉横解，肠澼为痔」的描述。这揭示了痔疮的两种不同病因。

## 两型鉴别

| | 饱食型（内经原文） | 脾虚气陷型 |
|------|------|------|
| **病因** | 暴饮暴食→肠胃撑大→筋脉松弛 | 脾虚→清气不升→局部筋脉失升提 |
| **胃口** | 吃得多 | **吃得很少** |
| **病性** | 湿热实证 | **虚寒证/气虚证** |
| **出血** | 血色鲜红·量多·喷射状 | 血色偏淡·量少·渗漏不止 |
| **治疗方向** | 清热利湿·赤豆当归散 | **补气升提·热药止血** |

## 内经原文辨析

「因而饱食，筋脉横解，肠澼为痔」——《素问·生气通天论》只揭示了痔疮的一种成因。完整病机应是：

### 饱食型
```
暴饮暴食 → 肠胃撑大 → 筋脉松弛弛纵 → 湿热下注 → 痔疮
```
这是**实证**——外在因素撑松的。

### 脾虚气陷型
```
久坐/思虑/劳倦 → 脾虚不运 → 中气下陷 → 局部筋脉失于升提 → 痔疮
吃得很少 → 脾气不运 → 不知饥饿
```
这是**虚证**——内在气虚提不住的。

## 倪师核心口诀

> **"出血源于寒，用热药止血。"**
> **"血无止法，调阳回头血自停。"**

痔疮反复出血，如果是热性→凉血止血（槐花散）；如果是寒性→热药止血（干姜/炮附/艾叶）。

## 脾虚气陷型治疗方案

### 方剂

| 方 | 作用 | 时机 |
|------|------|------|
| 补中益气汤 | 黄芪重剂补气升提·参术健脾·升麻柴胡升阳 | 缓解期·治本 |
| 赤豆当归散 | 赤小豆发芽打散+当归·排脓清热 | 急性发作·化脓 |
| 热药止血 | 干姜/炮附子/艾叶 | 寒性出血·渗漏不止 |

### 针灸

| 穴位 | 作用 | 与热性痔疮的区别 |
|------|------|------|
| 承山 | 痔疮要穴·寒热通用 | 不变 |
| 孔最+长强 | 络穴+督脉·前后呼应 | 不变 |
| **百会** | 下病上取·**升提中气** | 热性不用 |
| **神阙·隔盐灸** | 温中散寒·升阳举陷 | 热性禁灸 |
| 章门+蠡沟 | 脏会+肝经郄穴·止血 | 不变 |

### 外治

- 槐实熬膏塞肛门——不分寒热，内外痔均可
- 醋炙五倍子研末敷——酸收敛脱

## 关键鉴别问诊

| 问 | 热性 | 寒性/气虚 |
|------|:--:|:--:|
| 胃口 | 正常或偏多 | **吃得很少** |
| 大便 | 臭秽黏滞 | **溏薄无力** |
| 小便 | 深黄 | **清白/淡黄** |
| 痔核 | 红肿热痛 | **不那么红·脱出感强** |
| 口渴 | 渴喜冷饮 | 不渴或喜热饮 |
| 手脚 | 热 | **凉** |
| 脉 | 滑数 | **沉细/弱** |

## 教训

- 内经"饱食筋脉横解"≠痔疮全部病因。脾虚气陷是另一大病机。
- 「吃得很少」是脾虚的直观体征——不能因为内经没写就忽略。
- 出血≠热。倪师明言「出血源于寒」——这是和温病派的分水岭。
- 百会升提治痔疮：下病上取，非倪师原文直接对应，属逻辑推导（C2）。
