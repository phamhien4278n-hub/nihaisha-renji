## 一、病症定位

半身不遂，中医称"中风""偏枯""偏风"。《金匮要略·中风历节病脉证并治》专篇论述，《黄帝内经》多处提及"偏枯""大厥""薄厥"。在倪师人纪体系中属于**3F内科·东（脑病）** 与**2F治疗·东（急救针灸）** 交叉领域。

| 维度 | 定位 |
|------|------|
| 病位 | 脑络（核心）+ 肢体经络（表现） |
| 相关脏腑 | 肝（主风藏血）、心（主血脉神明）、肾（主骨生髓通脑）、脾（主肌肉四肢） |
| 相关经络 | 督脉（入脑）+ 手足阳明经（治痿独取阳明）+ 肝胆经（主筋） |
| 宫殿房间 | 3F东·脑病 + 2F东·急救针灸 + 2F北·经方 |

## 二、内经病理链（C1·黄帝内经原文）

### 2.1 核心原文

> **《素问·调经论》**：「血之与气，并走于上，则为大厥，厥则暴死，气复反则生，不反则死。」
>
> **《素问·生气通天论》**：「大怒则形气绝，而血菀于上，使人薄厥。」
>
> **《灵枢·刺节真邪》**：「虚邪偏客于身半……发为偏枯。」
>
> **《素问·至真要大论》**：「诸风掉眩，皆属于肝。」
>
> **《素问·脉要精微论》**：「上盛则气高，下盛则气胀。」

### 2.2 六端病机（倪师人纪体系·C1）

中风非单一病因，倪师体系中归纳为**风·火·痰·瘀·虚·气**六端：

```
┌─────────────────────────────────────────────────────┐
│                   中风六端病机                         │
├──────────┬──────────────────┬───────────────────────┤
│ 风（肝）  │ 肝肾阴虚→水不涵木 │ 肝阳化风→上扰清窍       │
│          │ →阳亢化风        │ →眩晕·肢麻·卒然昏仆     │
├──────────┼──────────────────┼───────────────────────┤
│ 火（心肝）│ 五志过极→心火暴盛  │ 火性炎上→逼血上行       │
│          │ →肝火暴涨        │ →面赤气粗·阳闭证        │
├──────────┼──────────────────┼───────────────────────┤
│ 痰（脾）  │ 嗜食肥甘→脾虚生湿  │ 痰热互结→蒙蔽清窍       │
│          │ →湿聚成痰        │ →喉中痰鸣·舌强不语       │
├──────────┼──────────────────┼───────────────────────┤
│ 瘀（心肝）│ 气不运血→血行滞涩  │ 瘀阻脑络→肢体失用       │
│          │ →离经之血成瘀    │ →半身不遂·舌紫暗         │
├──────────┼──────────────────┼───────────────────────┤
│ 虚（肾脾）│ 年老体衰→肾精亏虚  │ 髓海不足→脑失所养       │
│          │ →气血两虚        │ →恢复期痿软不用          │
├──────────┼──────────────────┼───────────────────────┤
│ 气（肝）  │ 气机逆乱→升降失常  │ 气血并走于上→大厥       │
│          │ →怒则气上        │ →卒倒无知               │
└──────────┴──────────────────┴───────────────────────┘
```

### 4.1 药性结构精讲（倪师人纪·C1）

**补阳还五汤的君臣佐使内部分工**（C1·倪师重点讲过）：

```
黄芪 — 君 — 大补元气·气行则血行（用量占全方80%）
  │
  ├─ 当归·川芎 — 臣 — 补血活血（血为气母）
  ├─ 桃仁·红花 — 佐 — 活血化瘀（红走血管内）
  ├─ 地龙        — 佐 — 通络（红清不到的缝隙瘀血·膜间）
  └─ 赤芍        — 使 — 凉血活血引经
```

> **地龙 vs 红花**（C1·倪师原话）：
> 红花走**血管内**清瘀（脑溢血后血块）；地龙走**组织间隙/脑膜间**清瘀。两线并进，缺一不可。

**小续命汤的攻补平衡**（C1）：

```
麻黄汤（攻） ──→ 开表驱风
  + 石膏      ──→ 制麻桂之温（防过度发汗伤津）
  + 人参·干姜 ──→ 补气温中（扶正→正气足则邪自去）
  + 当归·川芎 ──→ 补血活血（血行风自灭）
```

### 5.3 小续命汤详解（C1·倪师原文·中风主力方）

```
麻黄10g  杏仁10g  桂枝10g  甘草6g    ← 麻黄汤：发汗解表开太阳
当归10g  川芎10g                            ← 补血活血·血行风自灭
石膏15g                                      ← 清郁热·制麻桂之温
干姜6g   人参10g                            ← 补气温中·扶正祛邪

九味药 = 攻（麻桂）+ 清（石膏）+ 补（归芎参姜）+ 和（甘草）
```

**使用时机**：中风后口不能言+手脚拘挛。**倪师名言："中风十有八九是小续命汤。"**

**关键加减**：
| 兼证 | 加减 |
|------|------|
| 烦躁 | + 石膏加倍 |
| 无热象 | 去石膏 |
| 体虚甚 | - 麻黄+杏仁，加黄芪30g |
| 痰涎壅盛 | + 竹沥、天竺黄 |

### 5.4 风引汤详解（C1·热性中风专方）

```
赤石脂（涩）—— 留住好东西（固脱）
大黄（攻）———— 排掉废物（热+痰+瘀）

核心精妙：硝攻并用——中风大便秘结、大小便失禁者
不能纯攻也不能纯补。赤石脂收涩固脱，大黄推陈致新。

金石类：寒水石·滑石·紫石英·石膏 —— 重镇清热
热极生风，金石压之，制其亢阳。
```

### 5.5 补阳还五汤详解（C1·王清任·倪师重点讲）

```
生黄芪 120g（起！可逐渐加至240g）    ← 君·大补元气
当归尾 6g                              ← 臣·养血活血
川芎 6g   赤芍 6g                     ← 佐·活血
桃仁 6g   红花 6g                     ← 佐·化瘀
地龙 6g                                ← 佐·通络走膜间

全方核心：黄芪占80% → "气行则血行，气足则瘀去"
黄芪与活血药比例 20:1
```

**使用时机**：中风**恢复期**。急性期不可用——必须先用小续命/风引汤开表/清热，待表邪已去再转补阳还五。

**倪师三点强调**（C1）：

1. **红花 vs 地龙**：红花走血管内清瘀，地龙走组织间隙清瘀——两线并进。脑溢血的瘀血在血管内和脑膜间都有。
2. **攻血必配补血**：地龙破血伤血，必须配当归。血虚时减地龙增归芎。
3. **先表后里**：小续命汤治表（祛风开表）→ 补阳还五汤治里（补气活血），顺序不能反。

### 6.4 倪师特殊刺法（C1）

**1. 巨刺法——左病右取，右病左取**

- 左上肢瘫 → 刺右侧曲池、合谷
- 左下肢瘫 → 刺右侧阳陵泉、足三里
- 目的：调动健侧经气以行患侧之滞

**2. 透刺法**

- 曲池→透→少海（上肢瘫·阴阳经贯通）

- 后溪→透→劳宫（手指拘挛）
