## 二、倪师核心心法——看手脚定方（C1·最重要的临床口诀）

```
寒则缩·热则纵

手脚拘挛蜷缩 = 寒 → 小续命汤（约90%）
手脚瘫软摊开 = 热 → 风引汤（约10%）

不复杂——看手脚就知道。
```

这是倪师人纪体系中**中风辨证的第一铁律**。不需要复杂的六端病机框架，抓这一个体征就能定方。

## 四、小续命汤——寒性中风主力方（C1·倪师原文）

### 4.1 方义

```
小续命汤 = 
  麻黄汤（麻黄+杏仁+桂枝+甘草）  ← 发汗开表·开太阳
  + 当归·川芎                    ← 补血活血·血行风自灭
  + 石膏                          ← 清郁热·制麻桂之温
  + 干姜                          ← 温中散寒
  + 人参                          ← 补气扶正
```

### 4.2 四重攻补平衡

| 药对 | 攻 | 补 |
|------|:--:|:--:|
| 麻黄+桂枝 | 开表驱风 | — |
| 当归+川芎 | — | 补血活血 |
| 石膏 | 清郁热 | — |
| 人参+干姜 | — | 补气温中扶正 |

### 4.3 指征

口不能言、手脚拘挛、脉弦紧。倪师名言：「中风十有八九是小续命汤。」

## 五、风引汤——热性中风专方（C1·金匮原文）

### 5.1 核心精妙——涩攻并用

```
赤石脂（涩）—— 留住好东西（固脱）
大黄（攻）———— 排掉废物（热+痰+瘀）
```

中风病人大小便失禁——不能纯攻也不能纯补。赤石脂收涩固脱，大黄推陈致新。最精妙的设计。

### 5.2 指征

手脚瘫软摊开、脉数滑、大小便失禁。~10%的中风是热性的。

## 六、补阳还五汤——恢复期主力（C1·王清任·倪师重点讲）

### 6.1 方义

```
生黄芪 120g（起！可加至240g）  ← 君·大补元气（占全方80%）
当归尾 6g                        ← 臣·养血活血
川芎 6g   赤芍 6g               ← 佐·活血
桃仁 6g   红花 6g               ← 佐·化瘀
地龙 6g                          ← 佐·通络走组织间隙

黄芪:活血药 = 20:1
```

### 6.2 倪师三点强调

1. **红花 vs 地龙**（C1）：红花走**血管内**清瘀（脑溢血后血块）；地龙走**组织间隙/脑膜间**清瘀。两线并进，缺一不可。
2. **攻血必配补血**：地龙伤血，必须配当归。血虚时减地龙增归芎。
3. **先表后里**：小续命汤治表（祛风开表）→ 补阳还五汤治里（补气活血），**顺序不能反**。急性期不可一上来就补阳还五。

## 七、先兆期——侯氏黑散（C1·金匮原文）

### 7.1 方义

```
菊花四十分（君·最多）  ← 诸风掉眩皆属于肝
白术·防风各十分
细辛·茯苓·干姜·人参
矾石（化痰）
```

气血两虚者先吃侯氏黑散补虚，再考虑除热。菊花用量最大——「诸风掉眩皆属于肝」。

### 7.2 倪师三个中风先兆信号（C1）

| 信号 | 经络 | 预警时限 |
|------|------|:--:|
| 拇指食指麻木 | 手太阴肺+手阳明大肠 | 三年内 |
| 足胫骨前麻木 | 足阳明胃经 | 三月内 |
| 无故头晕目眩 | 肝风内动 | 随时 |

> **倪师原话（C1）**：「中风急性期，十宣刺血是第一件事。血挤出来，人就醒一半。」

### 9.2 恢复期——治痿独取阳明（C1·内经+倪师）

```
曲池 + 合谷 + 足三里 → 全走阳明经
阳明多气多血，治瘫必用阳明
```

| 穴 | 归经 | 作用 |
|---|------|------|
| 曲池 | 大肠经（阳明） | 上肢不遂 |
| 合谷 | 大肠经（阳明） | 上肢+面瘫 |
| 足三里 | 胃经（阳明） | 下肢不遂+补气血 |
| 阳陵泉 | 胆经 | 筋会·舒筋活络 |

### 9.3 巨刺法（C1·《灵枢·官针》）

健侧取穴，患侧配合：「巨刺者，左取右，右取左。」——调动健侧经气以行患侧之滞。

### 9.4 透刺法（C1）

- 曲池→透→少海（上肢瘫·阴阳经贯通）
- 阳陵泉→透→阴陵泉（下肢瘫·筋脉并调）
- 地仓→透→颊车（面瘫）

## 十二、诚实声明

- 倪师人纪体系中风辨证的核心是**「寒则缩·热则纵」两型鉴别**，不是教科书式的六端病机（风火痰瘀虚气）。后者是后世TCM教材的归纳框架，虽不违背倪师精神但非倪师独有体系。
- **侯氏黑散**用于先兆期、**三生饮**用于痰壅重症，均为C1金匮原文+倪师讲解。
- 后世方如镇肝熄风汤、天麻钩藤饮、地黄饮子等，倪师虽有提及或引用，但**不属于人纪核心方剂体系**，此处不纳入主方。
- 针灸部分的对症配穴（流涎取地仓承浆、便秘取天枢支沟）部分为C2推断。
