## 一、病症定位

痔疮，《黄帝内经》称"肠澼为痔"，《金匮要略》归入"下血""肠痈"体系。病位在**魄门（肛门）**，属大肠末端。在倪师人纪体系中，痔疮横跨**内科（3F西·肠胃病）** 与**外科（4F南·疮疡）** 两个记忆宫殿房间。

| 维度 | 定位 |
|------|------|
| 病位 | 魄门（大肠末端） |
| 相关脏腑 | 大肠（主）、肺（表里）、脾（升清统血）、肝（疏泄藏血）、肾（司二便） |
| 相关经络 | 督脉（长强）、足太阳膀胱经（承山）、手太阴肺经（孔最） |
| 宫殿房间 | 3F西·肠胃病 + 4F南·疮疡 + 2F北·经方 + 1F东·望诊 |

## 二、内经病理链（C1·黄帝内经原文）

### 2.1 核心原文

> **《素问·生气通天论》**：「因而饱食，筋脉横解，肠澼为痔。」
>
> **《素问·阴阳应象大论》**：「清气在下，则生飧泄；浊气在上，则生䐜胀。」
>
> **《灵枢·百病始生》**：「卒然多食饮则肠满……起居不节，用力过度，则络脉伤。」

### 2.2 三因病理链

痔疮非单一病因，倪师人纪体系将其归纳为三条病理链：

```
链一（湿热实证·内经原文）：
  暴饮暴食 → 肠胃撑大 → 筋脉松弛弛纵 → 湿热下注魄门 → 痔核红肿出血
  └─ 吃得多、大便臭秽、出血鲜红、脉滑数

链二（脾虚气陷·内经推导 C2）：
  久坐劳倦/思虑过度 → 脾虚不运 → 中气下陷 → 魄门筋脉失于升提 → 痔核脱出
                       → 不知饥饿（吃得很少）
                       → 脾不统血（阳虚不摄）→ 渗漏出血
  └─ 吃得少、大便溏薄、血色淡、脉沉细

链三（肠风下血·金匮体系 C1）：
  风邪入肠 → 与热相搏 → 迫血妄行 → 便前下血如箭
  └─ 先血后便、血色鲜红、肛门瘙痒
```

### 2.3 关键病机辨析

| 误判 | 纠正（倪师观点） |
|------|-----------------|
| "出血一定是热" | **出血源于寒**——脾阳虚不摄血同样导致出血。热性出血鲜红量多；寒性出血淡红渗漏 |
| "痔疮就是湿热下注" | 湿热型只占一部分，脾虚气陷型在久坐人群中占比更高 |
| "痔疮就得凉血" | 寒性出血用热药止血（干姜/炮附/艾叶），凉血只会越清越漏 |
| "吃得多才会得痔疮" | 吃得很少+反复发作=脾虚提不住了，不是撑松的，是提不住的 |

## 五、经方治疗（伤寒论+金匮要略·C1）

### 5.1 主方：乙字汤（C1·倪师体系痔疮基础方）

```
当归 15g      —— 养血活血·润肠（血中之气药）
柴胡 10g      —— 升举清气·疏肝（配升麻成升提对）
黄芩 10g      —— 清热燥湿（中焦湿热→魄门）
升麻 6g       —— 升阳举陷·引药上行（下病上取之意）
甘草 6g       —— 调和诸药·缓急止痛
大黄 3-6g     —— 通腑泄热（釜底抽薪，便通热自去）
```

**方解（倪师人纪视角）**：
- 六味药升降有序：柴胡+升麻升清，大黄降浊，黄芩清中，当归活血，甘草和中
- "升麻引药上行"是乙字汤区别于其他痔疮方的关键——治下不忘升上
- 大便通畅时大黄减量或去之，脾虚者大黄慎用

| 证型 | 配穴 | 操作要点 |
|------|------|---------|

| 脱出难回 | + 百会灸、关元灸 | 重灸升阳固脱 |

### 6.3 倪师特殊手法

- **承山穴**：针感必须下传至足跟——"针到承山，感传到脚底才是对的"（C1）
- **长强穴**：不可直刺过深，避伤直肠。斜刺方向朝肛门（C1）
- **百会治痔**：下病上取——"提壶揭盖"思想在下焦病中的应用（C2·基于倪师升提理论推导）
- **对称取穴**：双侧承山、孔最同时取，增强疗效（C1）

## 九、诊断陷阱与临床珍珠

### 9.1 三大陷阱

| 陷阱 | 表现 | 纠正 |
|------|------|------|
| **出血=热** | 见血就凉血 | 看血色+胃口+手脚——寒性出血淡红渗漏，用干姜不用槐花 |
| **痔=湿热** | 一概清热利湿 | 脾虚气陷型占久坐人群大多数，用补中益气不用赤豆当归 |
| **通便=大黄** | 便秘就用大黄攻下 | 血虚便秘当用当归润肠，气虚便秘当用黄芪补气推动 |

### 9.2 倪师临床珍珠（C1）

> **"出血源于寒，用热药止血。"**
> ——痔疮反复渗血+手脚凉+脉沉细 = 干姜/炮附/艾叶，不是槐花地榆。

> **"血无止法，调阳回头血自停。"**
> ——不要见血就想"止住"，调阳气让身体自己收住。

> **"饱食指的是一餐吃很多撑到——不是每餐正常吃饱。"**
> ——内经"饱食筋脉横解"的"饱食"是暴食，不是正常饮食。
