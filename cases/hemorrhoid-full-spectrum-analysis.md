## 一、内经病理（C1）

### 1.1 核心原文

> 《素问·生气通天论》：「因而饱食，筋脉横解，肠澼为痔。」

**倪师解读**：暴饮暴食→肠胃撑大→筋脉松弛弛纵→湿热下注魄门→痔核形成。这是内经唯一明确讲痔疮的条文。

### 1.2 关键纠正——内经只讲了一种痔疮

> ⚠️ 倪师强调：「饱食」是指一餐吃很多撑到，不是每餐正常吃饱。

## 二、两型鉴别——倪师体系核心分型

这是整个痔疮辨证的**第一问**：

```
问胃口 → 吃得多 vs 吃得少 → 完全相反的治法
```

| 辨证要素 | 湿热下注型（饱食型） | 脾虚气陷型（吃得少型） |
|---------|:-----------:|:-----------:|
| **内经依据** | 因而饱食·筋脉横解 | ⚠️ 内经未直接描述·倪师临床补充 |
| **胃口** | 吃得多 | **吃得很少** |
| **病性** | 湿热实证 | 虚寒证/气虚证 |
| **病因** | 暴饮暴食撑松筋脉 | 久坐劳倦→脾虚→清气不升→中气下陷 |
| **本质** | 撑松的 | **提不住的** |
| **出血** | 鲜红·量多·喷射状 | 淡红·量少·渗漏不止 |
| **痔核** | 红肿热痛·充血明显 | 不那么红肿·脱出感明显 |
| **大便** | 黏滞不爽·臭秽 | 溏薄·先干后溏·无力排出 |
| **手脚** | 热 | **凉** |
| **脉** | 滑数 | 沉细/弱 |
| **治则** | 清热利湿·凉血 | **补中益气·升阳举陷·热药止血** |
| **证据** | C1 | C2（病机推导+倪师临床验证） |

## 三、倪师核心口诀（C1）

### 3.1 「出血源于寒，用热药止血」

这是倪师与温病派的分水岭。痔疮反复渗血+手脚凉+脉沉细 = **干姜/炮附/艾叶**，不是槐花地榆。

温病派见血就凉血→寒性出血越清越漏。

### 3.2 「血无止法，调阳回头血自停」

不去止血，去调阳气——阳气回头了血自己就停了。

### 3.3 「亡血家禁汗禁下」

反复渗血的人禁发汗禁攻下——攻下伤中气，发汗伤津液，越治越漏。

### 3.4 「醋炙取酸收敛治脱肛」

酸收法可辅助收痔核脱出。

## 四、针刺治疗（C1·针灸大成+倪师针灸篇）

### 4.1 核心三穴

| 穴 | 归经 | 作用 | 刺法 |
|---|------|------|------|
| **承山** | 膀胱经 | 痔疮第一要穴·消肿止痛 | 直刺1.5-2寸，针感至足跟 |
| **孔最** | 肺经 | 肺与大肠相表里·远道取穴 | 直刺1-1.5寸 |
| **长强** | 督脉 | 局部取穴·通督脉调肛肠 | 斜刺向肛门1-1.5寸 |

### 4.2 分型配穴

| 证型 | 加穴 | 要点 |
|------|------|------|
| 湿热下注 | + 阴陵泉、曲池 | 泻法清热利湿 |
| 脾虚气陷 | + 百会（下病上取·升提中气）+ 足三里灸 + 神阙隔盐灸 | 重灸升阳固脱 |
| 出血 | + 章门（脏会）+ 蠡沟（肝经郄穴） | C1·倪师止血配穴 |
| 实症瘀血 | 天应放血（痔核上）+ 火罐 | C1·倪师放血法 |

> ⚠️ 百会治痔为C2推断——基于倪师「下病上取」升提理论推导，倪师未直接说「百会治痔疮」。

## 五、方剂（C1为主）

### 5.1 湿热化脓型——赤豆当归散（C1·金匮原文）

```
赤小豆（发芽后）打散 + 当归 → 清热利湿排脓
```

**使用指征**：脉数·无表热·目四眦黑·能食。脓已成用之。赤小豆**发芽后**生吃打散，不煮。

### 5.2 脾虚气陷型——补中益气汤（C1·倪师引用李东垣）

```
黄芪（重剂30-60g）+ 升麻 + 柴胡 → 补气升提
干姜/炮附/艾叶 → 热药止血（倪师核心）
```

> ⚠️ 补中益气汤原方出自李东垣，倪师在伤寒课中引用并加了自己「热药止血」的改造。方剂本身C1（倪师课上讲过），热药止血是倪师独有贡献。

### 5.3 乙字汤（C1·倪师引用日本汉方）

```
当归15g + 柴胡10g + 黄芩10g + 升麻6g + 甘草6g + 大黄3-6g
```

升清降浊同用：柴升升清，大黄降浊，黄芩清中，当归活血，甘草和中。**升麻引药上行**是区别于其他痔疮方的关键。大便通畅时大黄减量或去之；脾虚者大黄慎用。

### 5.4 外用——槐实熬膏（C1·倪师原文·不分寒热）

槐实研末+猪脂熬膏→塞肛门。**痔疮专药，不分寒热，内外痔均可**。这是倪师最推崇的痔疮外治法。

### 5.5 五倍子醋炙外敷（C3·后世经验）

醋炙取酸收敛→痔核脱出外敷用。符合倪师「酸收」思路但非倪师原文。

## 七、诊断陷阱

| 陷阱 | 纠正 |
|------|------|
| 「出血=热·凉血止血」 | **出血源于寒**——看血色+胃口+手脚。淡红渗漏+吃得少+手脚凉=热药止血 |
| 「痔=湿热·一概清利」 | 脾虚气陷型在久坐人群中占比很高——用补中益气，不用赤豆当归 |
| 「通便=大黄」 | 血虚便秘用当归润肠，气虚便秘用黄芪补气推动，不都是大黄 |

## 八、倪师临床珍珠（C1）

> 「出血源于寒，用热药止血。」——痔疮反复渗血+手脚凉+脉沉细=干姜/炮附/艾叶，不是槐花地榆。

> 「血无止法，调阳回头血自停。」——不要见血就想「止住」，调阳气让身体自己收住。

> 「饱食指的是一餐吃很多撑到——不是每餐正常吃饱。」——内经只讲了暴食型痔疮，吃得少的脾虚型是另一回事。

> 「亡血家禁汗禁下。」——反复渗血的人禁发汗禁攻下，越治越漏。
