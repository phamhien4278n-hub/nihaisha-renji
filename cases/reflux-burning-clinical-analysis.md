# 反酸烧心全链路分析 —— 内经→经方→针灸

> 2026-07-01 临床考核。从黄帝内经病理推演到方剂方案，覆盖六经+金匮全路径。主人评分85分。

## 症状

长期反酸烧心，心下痞满，情志不畅后加重，伴嗳气。服用奥美拉唑可暂时抑制但停药复发。

## 内经病理链

《素问》：「诸逆冲上，皆属于火」——但反酸不全是火。核心病机在**胃气上逆**，寒热虚实皆可。

病理链：
```
情志不遂 → 肝气郁结 → 木横克土 → 胃气不降 → 心下痞塞 → 郁而化热 → 酸水上泛
```

**关键诊断点**：嗳气+脉弦 → 肝郁犯胃。心下痞但不痛 → 气滞为主，非实热。

## 六经+金匮方证全路径

| 证型 | 主方 | 指征 |
|------|------|------|
| 寒性·胃寒气逆 | 吴茱萸汤 | 吐清水、胃冷痛、脉弦迟 |
| 热性·胃热上冲 | 大黄甘草汤 / 栀子豉汤 | 口臭、便干、舌红苔黄 |
| 寒热错杂·心下痞 | **半夏泻心汤** | 痞满+肠鸣+寒热夹杂 |
| 肝胃不和·情绪诱发 | 四逆散合左金丸 | 情志波动加重、胁胀 |
| 肝郁重+便秘 | 大柴胡汤 | 心下急、大便不通 |
| 胃虚气逆·嗳气为主 | 旋覆代赭汤 | 嗳气频作、心下痞硬 |
| 痰饮内停 | 小半夏汤 / 小半夏加茯苓汤 | 呕恶、胃中振水声 |

## 主方：半夏泻心汤合四逆散

```
半夏泻心汤：半夏+黄芩+干姜+黄连+人参+甘草+大枣
  → 寒热并用 → 辛开苦降 → 消痞制酸

合四逆散：柴胡+枳实+白芍+甘草
  → 疏肝解郁 → 肝气畅则胃气自降
```

**加减法**：
- 肝火明显（口苦、舌边红）→ 加左金丸（黄连+吴茱萸6:1）
- 嗳气频作 → 合旋覆代赭汤
- 便秘心下急 → 改用大柴胡汤
- 湿重苔腻 → 加茯苓、苍术

## 针灸

| 归经 | 穴位 | 作用 |
|------|------|------|
| 肝经 | 太冲、期门 | 疏肝解郁 |
| 心包经 | 内关 | 降逆止呕 |
| 脾经 | 公孙、阴陵泉 | 健脾利湿 |
| 胃经 | 足三里 | 降胃气 |
| 任脉 | 中脘 | 胃募·消痞 |

核心组合：**公孙+内关** —— "公孙内关治胃心胸"，不分寒热虚实。

## 治急→治标→治本

| 层次 | 目标 | 方法 |
|:--:|------|------|
| 治急 | 止酸止痛 | 公孙+内关，左金丸 |
| 治标 | 消痞降逆 | 半夏泻心汤，中脘+足三里 |
| 治本 | 疏肝健脾 | 四逆散，太冲+期门，戒情绪 |

## 教训

- C1原文：半夏泻心汤、吴茱萸汤、大黄甘草汤、四逆散、旋覆代赭汤、公孙内关
- C2推断：半夏泻心汤合四逆散的合方用法为基于病机的推导
- 反酸≠全热——寒热虚实皆可，胃气上逆是共同病机
